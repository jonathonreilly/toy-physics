## External Bridge Imports

Used only to repair the B2 observable definition:

- `N_f=2+1` potential-scale literature: in full QCD the string breaks,
  so asymptotic string tension is not a clean full-QCD scale.
- `N_f=2+1+1` static-energy literature: static-energy fits can report a
  finite-window `sigma` and force scales, but those are bridge
  observables with uncertainties.
- `N_f=2` string-breaking literature: sea quarks can induce string
  breaking in four-dimensional full QCD.

No external number is promoted as a retained framework input in Cycle 1.
