# Assumptions And Imports

## Minimal premise set

- The note is a repository metadata object, not a physics theorem.
- The audit lane reads `Type: meta` as the intended claim class hint.
- Markdown links to docs seed citation graph dependencies; code-spanned
  filenames are navigation text only.
- A runner may certify syntactic repo invariants without importing physics
  content.

## Imports

No physics, textbook, observational, fitted, literature, normalization, or
boundary-condition imports are used.

The only repository inputs are:

- the source text of `docs/UNIVERSAL_QG_OPTIONAL_TEXTBOOK_COMPARISON_NOTE.md`;
- inbound mention contexts under `docs/`, used only to check that current
  mentions are guarded as optional packaging callouts.

## Forbidden imports

- Using the canonical textbook continuum closure theorem as a one-hop
  dependency for this meta row.
- Treating the optional note as authority for downstream universal-QG theorem
  claims.
- Treating any cross-reference in the note as load-bearing evidence.
