# Route Portfolio

## R1. Planck Double-Criticality Selector

Score: 14/18. Hard-residual pressure: 3.

Use `lambda(M_Pl)=0` plus `beta_lambda(M_Pl)=0` to select `y_t(M_Pl)`,
then run down through the SM RGE.  This is the selected route because it is
structurally independent of the top-sector mass pin and produces a falsifiable
number quickly.

Outcome: conditional support.  Three-loop readout is
`y_t(v)=0.9208739295`, `0.3568%` from the accepted comparator.  The new hard
residual is exactly the derivation of `beta_lambda(M_Pl)=0`.

## R2. Higgs-Mass Inversion

Score: 7/18. Hard-residual pressure: 1.

Invert the existing Higgs runner for the `y_t(v)` that gives the observed
Higgs mass.  Rejected as a primary route because it uses an observed Higgs
target unless an independent Higgs mass theorem is supplied.

## R3. IR Quasi-Fixed Point / Attractor

Score: 8/18. Hard-residual pressure: 2.

Scan broad UV `y_t(M_Pl)` data and test whether the SM flow naturally
attracts to the observed basin.  Useful as bounded support, but it does not
select a unique value without an extra UV boundary condition.

## R4. CKM / Quark-Mass Ratio Lift

Score: 5/18. Hard-residual pressure: 1.
