# Literature Bridges

2026-05-05 fresh artifact route review:

The literature bridge was used only to choose the next artifact contract, not
as proof authority.

- FMS / gauge-invariant Higgs composite literature: useful route shape for a
  same-source gauge-invariant `O_H` operator after an EW/Higgs action exists.
  It is not a PR230 source-to-Higgs identity and does not write
  `C_sH/C_HH` rows.
- Feynman-Hellmann lattice matrix-element literature: supports source-response
  measurement from action perturbations and two-point functions.  It does not
  fix the source-to-canonical-Higgs factor `kappa_s`.
- Lellouch-Luscher finite-volume matrix-element literature: relevant to future
  strict LSZ/FV normalization once spectra, row identities, and threshold data
  exist.  It is not closure from finite source-only FH/LSZ rows.
- RI/MOM nonperturbative renormalization: relevant as a possible composite
  operator normalization tool.  It is not a canonical-Higgs identity or
  source-pole-purity certificate.

Current conclusion: the cleanest target is the `O_H/C_sH/C_HH` contract via an
action-first FMS/canonical-operator route.  No literature source is imported as
`H_unit`, `yt_ward_identity`, observed-target, alpha/plaquette/u0, `g2`,
`kappa_s`, `c2`, or `Z_match` authority.
