1. **Action/canonical-operator theorem for `x=O_H`.**
   Highest-value lane.  Must derive or adopt same-surface EW/Higgs action,
   canonical normalization, and LSZ/metric authority.

2. **Strict source-Higgs pole rows.**
   Becomes executable only after `O_H` certification.  Requires
   `C_ss/C_sH/C_HH(t)`, pole isolation, covariance, FV/IR/model-class checks,
   and Gram flatness.

3. **Neutral rank-one theorem.**
   Would replace direct `C_sH/C_HH` rows by proving no orthogonal neutral
   source-pole admixture.  Requires physical transfer/off-diagonal generator,
   not finite equal-time covariance alone.

4. **W/Z response bypass.**
   Fallback if lane 1 stays blocked.  Requires accepted action, production W/Z
   rows, matched top/W covariance, strict non-observed `g2`, additive-top
   subtraction, and transverse/orthogonal correction authority.
