# Route Portfolio

1. **Neutral rank-one theorem.**
   Current best non-chunk bypass.  It would replace direct `C_sH/C_HH` rows by
   proving no orthogonal neutral source-pole admixture.  It must use a physical
   transfer/off-diagonal generator, not finite equal-time covariance alone.

2. **W/Z response bypass.**
   Fallback if lane 1 stays blocked.  Requires accepted response authority,
   production W/Z rows, matched top/W covariance, strict non-observed `g2`,
   additive-top subtraction, and transverse/orthogonal correction authority.

3. **Native scalar/action/LSZ theorem for `x=O_H`.**
   Block37 says the current surface does not already derive this.  Reopen only
   with a new primitive such as a derived scalar carrier, exact
   Hubbard-Stratonovich kernel, or same-surface action theorem.

4. **Strict source-Higgs pole rows.**
   Becomes executable only after `O_H` certification.  Requires
   `C_ss/C_sH/C_HH(t)`, pole isolation, covariance, FV/IR/model-class checks,
   and Gram flatness.
