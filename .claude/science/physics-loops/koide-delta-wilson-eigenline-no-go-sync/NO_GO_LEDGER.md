# No-Go Ledger

## Supported No-Go

The finite Wilson data select a same-character rank-two zero-mode sector, not
a canonical rank-one selected line.  A CP1 family of normalized rank-one lines
has the same retained Wilson zero-mode and `Z3` character data.
