# Assumptions And Imports

## Minimal Premise Set

- `rho > 0`, `eta > 0`, `mu > 0`.
- Radial scaling is defined by `(rho_bar, eta_bar) = (mu*rho, mu*eta)`.
- For the `(1,0)`-based finite tangent readout only, require
  `rho != 1` and `mu*rho != 1`.

## Imports

None load-bearing.

The framework instance `(rho, eta) = (1/6, sqrt(5)/6)` is retained only as a
non-load-bearing example/cross-check. No PDG values, fitted selectors,
literature comparators, admitted unit conventions, or CKM physical assignments
are consumed by the theorem.

## Forbidden Imports Check

- Observed target values: none.
- Fitted selectors: none.
- Literature comparators: none.
- Admitted conventions: none.
- Same-surface family assumptions: none.
