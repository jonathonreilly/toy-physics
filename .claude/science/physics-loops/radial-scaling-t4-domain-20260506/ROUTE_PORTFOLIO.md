# Route Portfolio

## Route A: Finite-Tangent Domain Restriction

State T4 only on `D_T4 = {rho > 0, eta > 0, mu > 0, rho != 1, mu*rho != 1}`.
This removes the auditor's overbroad universal-domain objection.

Verdict: executed.

## Route B: Numerator-Denominator Runner Check

Have the runner compute
`tan(beta_bar) - tan(beta) = eta*(mu - 1)/((1 - mu*rho)*(1 - rho))`,
then check the denominator and numerator separately.

Verdict: executed.
