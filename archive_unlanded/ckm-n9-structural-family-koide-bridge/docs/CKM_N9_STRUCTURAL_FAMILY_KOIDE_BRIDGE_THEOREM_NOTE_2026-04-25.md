### F5: NEW two paths to 5/9

**Path F5a: `(1 − A²)(1 + A²) = 1 − A⁴`**

```text

```


Wait: `N_color + N_pair = 3 + 2 = 5 = N_quark − 1` when N_pair = N_color − 1. So

```text
1 + A^2  =  (N_color + N_pair)/N_color  =  (N_quark - 1)/N_color    [in framework with N_pair = N_color - 1]
```

Then

```text
(1 - A^2)(1 + A^2)  =  (1/N_color) * ((N_quark - 1)/N_color)  =  (N_quark - 1)/N_color^2  =  5/9.
```

**Path F5b: `η² × N_pair² = (N_quark − 1)/N_color²`**

```text

                  =  (N_quark - 1) * N_pair^2 / N_quark^2
                  =  (N_quark - 1) * N_pair^2 / (N_pair * N_color)^2     [N_quark = N_pair × N_color]
                  =  (N_quark - 1) / N_color^2
```

So F5 has TWO independent paths from retained inputs to the same value `5/9`.
