## 2. Theorem: OP local-source protocol forces canonical descent before dimensionless readout

> **Theorem (OP local-descent protocol).** Let `K = sI + zZ` be any
> source in the retained `C_3`-commutant grammar `A = span{I, Z}`. By
> OP Theorem 2 (property 4: "local: it is generated by local projectors
> P_x"), every scalar bosonic observable on the lattice is **exactly** a
> coefficient in `W`'s expansion in **local onsite sources**
> `J = Σ_x j_x P_x`. Therefore evaluating any scalar bosonic observable
> at `K` requires expressing `K` in OP's local-source domain. By CD, the
> unique trace-preserving onsite descent gives
>
> ```text
> K = sI + zZ   ↦   E_loc(K) = (s − z/3) I.
> ```
>
> So any scalar bosonic observable evaluated at the commutant source `K`
> equals its evaluation at the descended source `E_loc(K)`. In
> particular, for the Q observable:
>
> ```text
> Q[K] = Q[E_loc(K)] = Q at z_eff = 0 = 2/3   (by CRIT).
> ```

**Proof.**

By OP Theorem 2: scalar bosonic observables on the lattice are exactly
the coefficients in `W = log|det(D+J)|`'s local-source expansion. The
local source domain is `J ∈ {Σ_x j_x P_x : j_x ∈ ℝ}`, a subspace of all
linear operators on the orbit Hilbert space. Equivalently, OP-source-
derivative observables operate on the subalgebra of LOCAL site projectors.

A commutant source `K = sI + zZ ∈ A` is generally **not** in OP's local
domain. Specifically (per CD §3 and `KOIDE_Q_ONSITE_SOURCE_DOMAIN_NO_GO_SYNTHESIS_NOTE_2026-04-25.md` §2):

```text
Z = -I/3 + (2/3)C + (2/3)C²
```

involves the cyclic shift `C` (a multi-site operator), so `Z` has
off-diagonal entries in the site basis and `Z ∉ span{P_x}`. The local
intersection `span{P_x} ∩ End_{C_3}(V) = span{I}` (per ONSITE §3).

For OP-derived scalar observables to be evaluated at `K`, `K` must first
be projected to OP's local domain. The natural projection (preserving
scalar normalization, trace on source observables, and strict onsite
target) is uniquely the canonical descent `E_loc` per CD Theorem 1:

```text
E_loc(X) = (Tr X / 3) I.
```

For `K = sI + zZ`:
- `Tr(K) = 3s + z·Tr(Z)`. Since `Tr(I) = 3` and
  `Tr(Z) = Tr(-I/3) + (2/3)·Tr(C) + (2/3)·Tr(C²) = -1 + 0 + 0 = -1`
  (cyclic permutations of order 3 on a 3-site basis have zero trace).
- Hence `Tr(K) = 3s − z`, so `E_loc(K) = (Tr(K)/3) I = (s − z/3) I`.

The descended source `E_loc(K) = (s − z/3) I` is a pure scalar (only the
identity component); its reduced traceless coordinate is `z_eff = 0`.

By CRIT §5 (retained):

```text
K = 0 in the reduced two-slot carrier ⇔ z = 0 ⇔ Q = 2/3.
```

For the OP-evaluated observable `Q[K] = Q[E_loc(K)]`, the descent maps
`K` to a source with `z_eff = 0` on the reduced carrier (since `(s − z/3) I`
is purely scalar). Therefore:

```text
Q[K] = Q at z_eff = 0 = 2/3
```

for any commutant source `K = sI + zZ`, including the physical undeformed
background.

This is **OP-protocol-forced**, not admitted: OP Theorem 2 specifies that
scalar bosonic observables are EXACTLY local-source-expansion coefficients.
A commutant source `K` evaluated as a scalar bosonic observable MUST go
through the local-source domain, which forces canonical descent. ∎

## 3. Why this is the OP-protocol alternative the canonical-descent note named

The canonical descent note (CD) §"Why this matters" explicitly identifies
two equivalent closure paths:

(a) "undeformed charged-lepton scalar backgrounds are strict onsite source
    functions"
(b) "projected commutant sources must be locally descended before
    dimensionless readout"

Path (a) requires deriving that the physical source domain is restricted
to onsite. Path (b) requires deriving that dimensionless readout forces
descent.

This note proves Path (b): **OP Theorem 2's local-scalar-observable
specification (= coefficients in local source expansion) forces any
commutant source to be canonically descended for dimensionless readout.**
The reasoning:
- OP's "local scalar observables" are EXACTLY (per Thm 2) the coefficients
  in `W`'s local-source expansion `J = Σ_x j_x P_x`.
- Q is dimensionless and scalar, hence is one such coefficient.
- A commutant source `K` containing non-local `Z` cannot be expressed as
  a local-source `J` directly (`Z ∉ span{P_x}` per ONSITE §3 + CD §3).
- The unique trace-preserving onsite projection of `K` to OP's local
  domain is `E_loc(K)` (CD Theorem 1).
- Therefore `Q[K] = Q[E_loc(K)]`.
- `E_loc(K)` has `z_eff = 0` (scalar), so `Q[K] = 2/3` by CRIT.

This is exactly Path (b) of CD's residual closure.

The reduced carrier is not admitted; CRIT is invoked only on the descended

The previous review's circular-dependency objection (Finding 1) is broken
by routing through CD (a retained theorem, not an admitted reduction)
