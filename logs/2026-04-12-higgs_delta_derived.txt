DERIVATION OF HIGGS Z_3 CHARGE delta = 1
============================================================

The staggered lattice taste states: (s1,s2,s3) in {0,1}^3
Z_3 acts as cyclic permutation: sigma(s1,s2,s3) = (s2,s3,s1)

Isospin from bipartite parity eps = (-1)^(s1+s2+s3):
  Up-type (eps=+1): weight 0 {(0,0,0)} + weight 2 {(1,1,0),(1,0,1),(0,1,1)}
  Down-type (eps=-1): weight 1 {(1,0,0),(0,1,0),(0,0,1)} + weight 3 {(1,1,1)}

3 generations of up = weight-2 Z_3 triplet
3 generations of down = weight-1 Z_3 triplet

Bit-flip (Yukawa/Higgs) coupling between orbits:
  Up[0] (1, 1, 0) -> Down[0] (1, 0, 0) (orbit index 0->0)
  Up[0] (1, 1, 0) -> Down[1] (0, 1, 0) (orbit index 0->1)
  Up[1] (1, 0, 1) -> Down[0] (1, 0, 0) (orbit index 1->0)
  Up[1] (1, 0, 1) -> Down[2] (0, 0, 1) (orbit index 1->2)
  Up[2] (0, 1, 1) -> Down[1] (0, 1, 0) (orbit index 2->1)
  Up[2] (0, 1, 1) -> Down[2] (0, 0, 1) (orbit index 2->2)

Orbit index mapping: 0->0, 1->2, 2->1
This is sigma^2 = sigma^{-1} (inverse cyclic permutation)

In Z_3 charge language:
  If up-generation has Z_3 charge z,
  the Higgs maps it to down-generation with charge sigma^{-1}(z) = (z-1) mod 3
  => q_down = q_up - 1 mod 3
  => delta = 1

VERIFICATION:
  Up z=0 -> Down z=2, shift = 1
  Up z=1 -> Down z=0, shift = 1
  Up z=2 -> Down z=1, shift = 1

ALL shifts equal 1 mod 3.

THEREFORE: delta = 1 is DERIVED from the bit-flip structure
of the Yukawa coupling between weight-2 (up) and weight-1 (down)
Z_3 orbits on the staggered lattice.

This is NOT an assumption — it is a CONSEQUENCE of:
  1. Bipartite parity assigns isospin (weight 0,2 = up; weight 1,3 = down)
  2. The Yukawa coupling is a single bit flip
  3. The bit flip permutes the Z_3 orbit by sigma^{-1}
  4. sigma^{-1} shifts Z_3 charge by -1, giving delta = 1
