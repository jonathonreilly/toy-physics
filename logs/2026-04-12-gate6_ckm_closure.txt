  APPROACH A: Z_3 GROUP THEORY
  ------------------------------------------------------------

  The FN mechanism: a heavy scalar (flavon) phi gets a VEV <phi>.
  The expansion parameter is eps = <phi>/M_FN.
  On the lattice, Z_3 symmetry breaks when the 3 spatial directions
  develop different hopping amplitudes (anisotropy).

  The Z_3 group has 3 elements: {1, omega, omega^2}.
  The representation theory gives the Clebsch-Gordan decomposition:
    3 x 3 = 1 + 1 + 1  (three singlets)
  The coupling between different Z_3 sectors is suppressed by
  the projection onto a single sector: P_z = (1/3) sum_k omega^(zk).

  The key: the probability of finding a Z_3-charged state in a
  specific Z_3 sector is 1/|Z_3| = 1/3.
  This 1/3 is the UNIVERSAL mixing suppression for any Z_3-breaking
  transition, because the flavon VEV selects one of 3 degenerate minima.

  VERIFICATION: Z_3 projector norms

    P_z = (1/3) sum_k omega^(z*k)
  The overlap between sectors z and z' is:

    <P_2|P_2> = +1.000000 -0.000000i  |overlap| = 1.000000

  |<z|z+1>| = 0 in the exact symmetry limit.  When the symmetry

  group structure: each minimum has equal weight 1/3 of the total,
  so the relative amplitude is |A| = 1/3.

  APPROACH D: INFORMATION-THEORETIC / PROBABILITY ARGUMENT
  ------------------------------------------------------------

  The FN parameter eps measures the SUPPRESSION of inter-generation
  transitions.  In a system with Z_3 symmetry:

  1. There are 3 degenerate minima (generations).
  2. A Z_3 transition operator maps sector z -> z+1.
  3. The PROBABILITY of transition z -> z+1 is P = 1/3
     (each of 3 sectors equally likely after one Z_3 rotation).
  4. The AMPLITUDE is sqrt(P) = 1/sqrt(3) ~ 0.577.

  But eps = 1/3, not 1/sqrt(3).  The resolution:
  eps is not the amplitude but the RATIO of VEVs:
    eps = <phi_broken>/M = v/M
  where v = <phi> is the VEV and M is the heavy scale.

  In a Z_3-symmetric potential V(phi) = lambda * (phi^3 - v^3)^2,
  the ratio v/M is determined by:
    v ~ (mu^2/lambda)^(1/3)  [cubic minimum]
    M ~ lambda * v^2 / mu  [heavy mode mass]
  giving v/M ~ mu / (lambda * v) ~ 1/(lambda * v^2 / mu^2)

  For the NATURAL choice where the potential's coefficients are
  related by the Z_3 structure:
    V = mu^2 * |phi|^2 * (1 - 2*Re(phi^3)/v^3 + |phi|^6/v^6)
  the ratio is controlled by the Z_3 Clebsch-Gordan coefficient
  for the fundamental x fundamental -> anti-fundamental coupling:
    C_{1,1}^{2} = 1/sqrt(3)  [Z_3 CG coefficient]
  and eps = |C|^2 = 1/3.

  NUMERICAL CHECK: Z_3 Clebsch-Gordan coefficients
  Z_3 representations: r=0 (trivial), r=1 (fundamental), r=2 (conjugate)
  Tensor product decomposition:
    r=0 x r=0 -> r=0
    r=0 x r=1 -> r=1
    r=0 x r=2 -> r=2
    r=1 x r=0 -> r=1
    r=1 x r=1 -> r=2
    r=1 x r=2 -> r=0
    r=2 x r=0 -> r=2
    r=2 x r=1 -> r=0
    r=2 x r=2 -> r=1

  For abelian groups, the CG coefficients are phases (magnitude 1).
  The 1/3 comes not from the CG but from the GROUP AVERAGING:
    <z+1|V_break|z> = (1/3) sum_k omega^k * V_k
  where V_k are the Fourier modes of the breaking potential.
  For a single-harmonic breaking V = V_1 * omega^x + h.c.,
  the 1/3 factor is the Fourier normalization.

  D) WHY delta = 0 FOR GENERATION 3
  ------------------------------------------------------------

  The observed pattern is delta = (1, 1, 0):
    Gen 1: q_up = 5, q_down = 4, delta = 1
    Gen 2: q_up = 3, q_down = 2, delta = 1
    Gen 3: q_up = 0, q_down = 0, delta = 0

  WHY does the Higgs not shift generation 3?

  PHYSICAL ARGUMENT:
  In the FN mechanism, the Yukawa coupling is:
    Y_ij ~ c_ij * (phi/M)^(q_i + q_j)
  For generation 3 with q = 0 in both sectors:
    Y_33 ~ c_33 * (phi/M)^0 = c_33 ~ O(1)
  The top quark mass is unsuppressed.  The Higgs VEV gives
  m_t = Y_33 * v ~ v, and m_b = Y_33 * v (same order).

  The Higgs Z_3 charge DOES enter the Yukawa, but for q_i = q_j = 0,
  the charge conservation is automatically satisfied:
    q_L(gen3) + q_H + q_R(gen3) = 0 + delta + 0 = delta mod 3
  This is nonzero only if delta != 0 mod 3.  But the Yukawa
  coupling is ALLOWED as long as the total charge is 0 mod 3.
  Since q_L = q_R = 0, we need delta = 0 mod 3 for gen 3.

  If the Higgs has delta = 1, then the gen-3 Yukawa would be
  FORBIDDEN unless there's a compensating flavon insertion.
  But the top Yukawa IS O(1), so there is NO suppression.

  RESOLUTION: The Higgs Z_3 charge enters the EFFECTIVE charge
  difference between up and down sectors for each generation.
  For gen 3 (q=0), the up-down splitting comes from the SU(2)
  Yukawa coupling directly, not from the FN mechanism.
  The 'delta = 0' for gen 3 means the FN mechanism does not
  DISTINGUISH up from down for the heaviest generation --
  both couple with strength O(1).

  This is CONSISTENT with the FN framework: the Higgs charge
  only matters for the FN-suppressed couplings (gen 1, 2),
  not for the unsuppressed ones (gen 3).

  REFINED MIXING ANALYSIS:
  The CKM matrix V = U_u^dag * U_d where U_u, U_d diagonalize
  the up and down mass matrices separately.

  For the mass matrix M_ij = c_ij * eps^(q_i + q_j):
    M_up = c * diag(eps^5, eps^3, 1)^T * diag(eps^5, eps^3, 1)
    M_down = c * diag(eps^4, eps^2, 1)^T * diag(eps^4, eps^2, 1)

  The LEFT rotation U_L diagonalizing M * M^dag has angles:
    (U_u)_12 ~ eps^(q1-q2) = eps^2  (up sector)
    (U_d)_12 ~ eps^(q1-q2) = eps^2  (down sector)
    V_us = (U_u)_12 - (U_d)_12 ~ O(eps^2)  [if equal, cancels!]

  The key: V_us involves a SUBTRACTION of two eps^2 terms.
  With c_ij != 1, the subtraction is incomplete and V_us can be
  larger (constructive) or smaller (destructive) than eps^2.
