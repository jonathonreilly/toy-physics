STAGE 1: Saddle-point closed form
------------------------------------------------------------------------------

  Per-step lateral characteristic function (C_arm support note):
      g(q; h) = a_0(h) + 2 a_pm(h) cos(q h)

  Saddle at q=0 (a_+ = a_-) gives complex Gaussian
      g(q; h)^N = g(0)^N * exp(-N r(h) (q h)^2 + O(q^4))
  with r(h) = a_pm(h) / g(0; h)  (complex).

  Fourier transform:
      A(y) = g(0)^N / sqrt(4 pi N r h^2) * exp(-y^2 / (4 N r h^2))

  Real part of -y^2/(4 N r h^2) gives the Gaussian magnitude
  (sigma_arm^2 = N h^2 |r|^2 / Re(r); C_arm support note).
  Imaginary part of -y^2/(4 N r h^2) gives the quadratic phase:

      arg A(y) ⊇ y^2 * Im(r) / (4 N h^2 |r|^2)

  So  c2(h) = (1/2) d^2/dy^2 arg A(y)
            = Im(r) / (4 N h^2 |r|^2)
            = Im(a_pm * conj(a_0)) / (4 L h |a_pm|^2).
