  Symmetry argument: at m = -4r, u_k = 2r(k-2). Pair (k, 4-k):
    binom(4, k) = binom(4, 4-k), and u_{4-k} = 2r((4-k)-2) = -2r(k-2) = -u_k.
    Each pair (k, 4-k): mult · u_k / (u_k^2 + 4u_0^2) + mult · u_{4-k} / (u_{4-k}^2 + 4u_0^2)
                       = mult · u_k / (u_k^2 + 4u_0^2) + mult · (-u_k) / (u_k^2 + 4u_0^2)
                       = 0.
    The k=2 self-paired term has u_2 = 0 directly. So all pair sums and the singleton
    contribute zero, giving dV^W/dm |_{m=-4r} = 0 identically.
