DECOHERENCE ARCHITECTURE ROUND VERDICT

Primary: entangling env (cos/sin split at mass nodes)
  Result: FAILED
  Purity: 0.94 → 0.87 → 0.90 (rises from N=12 to N=18)
  Insight: genuine superposition split gives HIGHER purity than
  deterministic labeling because amplitude spreads thinly across
  many env states → each state carries less weight → purity ∝ Σp² rises.

Fallback: substrate memory (split + phase kick at recorded nodes)
  Result: FAILED
  Purity: identical to entangling (0.94 → 0.87 → 0.90)
  Insight: phase kicks within existing branches don't create new
  branching. History-dependent propagation shifts phases but doesn't
  increase env balance.

Prior in this session: graph-memory scars (FAILED, 0.64→0.87),
  CA microstate coupling (FAILED, 0.80→0.86)

CUMULATIVE: 19 decoherence architectures tested across ~30 experiments.
  All fail the scaling criterion (purity not decreasing with N).

DEEP DIAGNOSIS:
  The decoherence scaling problem has two independent failure modes:
  1. Deterministic labeling: few balanced states, but path convergence
     makes labels shared → purity rises with N
  2. Entangling splitting: many unbalanced states, thin amplitude per
     state → purity starts high and stays high

  Neither mode produces decreasing purity with graph size because:
  - Mode 1: CLT makes both slits' labels converge
  - Mode 2: amplitude normalization makes thin states irrelevant to purity

  The model would need a coupling that:
  - creates balanced env states (like deterministic, not entangling)
  - while keeping env states slit-discriminating at large N (unlike deterministic)

  No tested architecture satisfies both simultaneously.

VERDICT: FAILED (both primary and fallback)
  The decoherence scaling problem resists all tested approaches.
  The next advance requires a new theoretical idea, not another
  architecture variant.
