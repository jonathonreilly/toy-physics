SECTION 2: DARK-TO-VISIBLE MASS RATIO
==============================================================================

  Observed: Omega_DM = 0.268, Omega_baryon = 0.049
  Ratio: Omega_DM / Omega_baryon = 5.47

  Framework prediction:
    n_dark = 2 singlet states (S_0, S_3)
    n_vis  = 6 triplet states (T_1 + T_2)
    Ratio = (2 * M_dark) / (6 * M_vis)

  Required: M_dark / M_vis = 5.47 * 3 = 16.4

  Wilson term predictions:
       r  m(S3)/m(T1)  m(S3)/m(T2)  Omega ratio (S3 only)   vs obs
  ------ ------------ ------------ ---------------------- --------
     0.5          3.0          1.5                   0.33     5.47
     1.0          3.0          1.5                   0.33     5.47
     1.5          3.0          1.5                   0.33     5.47

  Note: S_0 is massless under Wilson. If S_0 acquires dynamical mass
  m(S_0) from gauge interactions, the ratio becomes:
    Omega_dark/Omega_vis = (m(S0) + m(S3)) / (6 * m_vis_avg)

  r=0.5: Need m(S_0) = 46.22/a for correct DM ratio
    (m(S_3) = 3.0/a, m_vis_avg = 1.50/a)

  r=1.0: Need m(S_0) = 92.45/a for correct DM ratio
    (m(S_3) = 6.0/a, m_vis_avg = 3.00/a)

  SCENARIO A: Only S_3 is dark matter (S_0 is dark radiation)
    r=0.5: Omega_DM/Omega_vis = 0.33 (obs: 5.47)
    r=1.0: Omega_DM/Omega_vis = 0.33 (obs: 5.47)

  SCENARIO B: Both singlets are dark matter
    Requires dynamical mass for S_0, giving ratio 5.47

  KEY FINDING: Wilson masses alone give Omega_DM/Omega_vis ~ 0.3-1.0,
  which is O(1) but not 5.5. The right ballpark suggests the
  mechanism is qualitatively correct. Quantitative agreement requires
  either dynamical mass generation or non-thermal production.

SECTION 4: STABILITY ANALYSIS
==============================================================================

  4a. Z_3 charges of all orbits
  ----------------------------------------

  Z_3 representation matrix on T_1:
    D(sigma) = [[0, 1, 0], [0, 0, 1], [1, 0, 0]]
    Eigenvalues: ['-0.5000-0.8660j', '1.0000-0.0000j', '-0.5000+0.8660j']
    Z_3 charges: {0, 1, 2} (mod 3)

  Orbit Z_3 properties:
    S_0: Z_3 charge = 0 (invariant)
    T_1: decomposes into charges (0, 1, 2)
    T_2: decomposes into charges (0, 2, 1) [conjugate]
    S_3: Z_3 charge = 0 (invariant)

  4b. Decay channels for S_3
  ----------------------------------------

  S_3 (charge 0) -> possible final states with total charge 0 (mod 3):

  Channel                             Charges         Sum              Z3 OK?
  ----------------------------------- --------------- --------------- -------
  3 T_1 states                        1 + 1 + 1       0 mod 3             YES
  3 T_2 states                        2 + 2 + 2       6 = 0 mod 3         YES
  T_1(0) + T_1(1) + T_1(2)            0+1+2           3 = 0 mod 3         YES
  T_2(0) + T_2(2) + T_2(1)            0+2+1           3 = 0 mod 3         YES
  2 T_1 states                        various         not 0 mod 3 in general      NO
  T_1 + T_2                           various         depends on specific charges     YES
  S_0 + anything                      0 + X           requires X with charge 0     YES

  4c. Kinematic constraints
  ----------------------------------------

  Wilson masses (r=0.5):

  Channel                         m_initial    m_final   Allowed?
  ------------------------------ ---------- ---------- ----------
  S_3 -> 3 T_1                         3.00       3.00  THRESHOLD
  S_3 -> 3 T_2                         3.00       6.00         NO
  S_3 -> T_1 + T_2 + S_0               3.00       3.00  THRESHOLD
  S_3 -> T_1 + 2 T_1                   3.00       3.00  THRESHOLD
  S_3 -> S_0 + 2 T_1                   3.00       2.00        YES

  4d. Hamming weight parity as conservation law
  ----------------------------------------

  H-parity = (-1)^|s| (= staggered Gamma_5 chirality):
    S_0: H-parity = +1
    T_1: H-parity = -1
    T_2: H-parity = +1
    S_3: H-parity = -1

  Channel                         Initial P    Final P  H-parity OK?
  ------------------------------ ---------- ---------- -------------
  S_3 -> 3 T_1                           -1         -1           YES
  S_3 -> 3 T_2                           -1         +1     FORBIDDEN
  S_3 -> T_1 + 2 T_2                     -1         -1           YES
  S_3 -> S_0 + T_1 + T_2                 -1         -1           YES

  4e. Combined stability analysis
  ----------------------------------------

  For S_3 to decay, a channel must satisfy:
    1. Z_3 charge conservation (sum of charges = 0 mod 3)
    2. H-parity conservation (product of parities = -1)
    3. Energy conservation (m_initial >= sum m_final)

  Lowest-mass allowed channel: S_3 -> 3 T_1
    Z_3: 0 -> 0+1+2 = 0 mod 3  [OK]
    H-parity: -1 -> (-1)^3 = -1  [OK]
    Kinematics: m(S_3) = 3.0 = 3 * m(T_1) = 3.0  [AT THRESHOLD]

  RESULT: S_3 -> 3 T_1 is allowed but at exact mass threshold.
  At threshold, the phase space volume vanishes.
  The decay rate Gamma ~ (phase space) * |M|^2 -> 0.
  S_3 is KINEMATICALLY STABLE (or extremely long-lived).

  Is the threshold exact?
    Wilson mass: m(s) = 2r * |s| / a is EXACTLY linear in |s|.
    Therefore m(S_3) = 3 * m(T_1) is EXACT, not approximate.
    Any perturbation breaking this linearity determines stability:
    - If m(S_3) < 3*m(T_1): absolutely stable (kinematic)
    - If m(S_3) > 3*m(T_1): unstable, but lifetime depends on |M|^2
    - If m(S_3) = 3*m(T_1): marginal (zero phase space)

  Corrections beyond Wilson linearity:
    - Gauge interactions: shift masses by O(alpha * m)
    - Self-energy: m_phys = m_bare + Sigma(m)
    - The sign of the correction determines stability.
    - Lattice calculations typically show NEGATIVE self-energy
      corrections, meaning m_phys < m_bare.
    - If the correction is larger for S_3 (more mass -> larger
      self-energy), then m_phys(S_3) < 3 * m_phys(T_1).
    - This would make S_3 ABSOLUTELY STABLE.

SECTION 5: CHIRALITY AND MATTER-ANTIMATTER ASYMMETRY
==============================================================================

  Chirality assignment:
    T_1 (|s|=1): Gamma_5 = (-1)^1 = -1  -> LEFT-HANDED
    T_2 (|s|=2): Gamma_5 = (-1)^2 = +1  -> RIGHT-HANDED

  5a. SU(2) coupling strength
  ----------------------------------------

  Staggered phases eta_mu(x) = (-1)^(x_1 + ... + x_{mu-1}):
    eta_1(x) = 1              (always)
    eta_2(x) = (-1)^x_1
    eta_3(x) = (-1)^(x_1+x_2)

  Effective SU(2) coupling in taste space:
  State              C_1    C_2    C_3  Total |C|^2    Orbit
  --------------- ------ ------ ------ ------------ --------
  (0, 0, 0)           +1     +1     +1            3       S0
  (0, 0, 1)           +1     +1     +1            3       T1
  (0, 1, 0)           +1     +1     -1            3       T1
  (0, 1, 1)           +1     +1     -1            3       T2
  (1, 0, 0)           +1     -1     -1            3       T1
  (1, 0, 1)           +1     -1     -1            3       T2
  (1, 1, 0)           +1     -1     +1            3       T2
  (1, 1, 1)           +1     -1     +1            3       S3

  Coupling comparison by orbit:
    S0: couplings = [(1, 1, 1)], |C|^2 = [3]
    T1: couplings = [(1, 1, 1), (1, 1, -1), (1, -1, -1)], |C|^2 = [3, 3, 3]
    T2: couplings = [(1, 1, -1), (1, -1, -1), (1, -1, 1)], |C|^2 = [3, 3, 3]
    S3: couplings = [(1, -1, 1)], |C|^2 = [3]

  Average |C|^2:
    T_1 (left):  3.00
    T_2 (right): 3.00
    -> Equal: staggered phases alone do NOT break L-R symmetry
    -> Additional mechanism needed for parity violation

  5b. Parity violation mechanism
  ----------------------------------------

  Sign structure analysis:

  T1 (Left-handed):
    (0, 0, 1)       signs: (+, +, +)
    (0, 1, 0)       signs: (+, +, -)
    (1, 0, 0)       signs: (+, -, -)

  T2 (Right-handed):
    (0, 1, 1)       signs: (+, +, -)
    (1, 0, 1)       signs: (+, -, -)
    (1, 1, 0)       signs: (+, -, +)

  KEY RESULT: The staggered phases create different sign structures
  for T_1 (left) and T_2 (right), even though the total coupling
  magnitude is the same. This is the discrete lattice analog of
  chiral gauge coupling -- the weak force distinguishes L from R
  not by coupling strength but by coupling PHASE.

  Classification:

  Our singlets do NOT resemble any standard DM candidate closely:
    - Not WIMP: mass is at cutoff scale (Planck), not electroweak
    - Not axion: spin 3/2 (not 0), strong coupling to gravity
    - Not sterile neutrino: mass too large, different spin

  Closest analogy: SUPERHEAVY dark matter (SHDM) or 'WIMPzilla'
    - Mass at or near Planck/GUT scale
    - Produced gravitationally (not thermally)
    - Stable due to discrete symmetry or kinematics
    - Interacts primarily through gravity

  Novel features of taste singlet DM:
    1. Emerges from same structure as visible matter (no new sector)
    2. Multiplicity is PREDICTED: exactly 2 dark states per 6 visible

  Potential observational signatures:
    1. Gravitational: S_3 clusters gravitationally like CDM
    2. Annihilation: S_3 + S_3 -> T_1 + T_2 (if kinematically allowed)

       After mass weighting, Omega_DM/Omega_total depends on
       production mechanism
    4. No direct detection signal (too heavy for WIMP detectors)

  KEY ISSUES:
  1. ELECTRIC CHARGE: The biggest open question. If S_0 and S_3 carry
     electric charge, they are ruled out as dark matter. The charge
     assignment depends on how the lattice U(1) maps to QED.
  2. SU(2) NON-SINGLET: S_3 has j=3/2, meaning it participates in
     SU(2) interactions. This is problematic for 'dark' matter unless
     the SU(2) is broken at high scale (above S_3 mass).
  3. RELIC ABUNDANCE: Wilson mechanism gives O(1) ratio, not 5.4.
     Non-thermal production or dynamical mass could fix this.
  4. MASS SCALE: At cutoff (Planck) scale, S_3 is superheavy.
     This is consistent with gravitational DM but untestable.

  visible matter, no new physics). The main strengths are:
    - Automatic SU(3) singlet (no color)
    - Kinematic stability (threshold decay)
    - Predicted multiplicity (2 dark per 6 visible)
    - No free parameters
  The main weaknesses are:
    - Uncertain electric charge (could be fatal)
    - SU(2) non-singlet (j=3/2)
    - Mass ratio gives O(1), not 5.4
    - Superheavy mass makes direct tests impossible
