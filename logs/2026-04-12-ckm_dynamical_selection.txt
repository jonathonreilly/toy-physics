  PRINCIPLE: The 3 spatial directions are related by the cubic
  symmetry group.  The Z_3 charge assignment should be MAXIMALLY
  SYMMETRIC under permutation of spatial directions (S_3).

  This means: for each generation, choose the Z_3^3 vector that

  ----  ---------------  --------------------  ------------  --------

  For q_up = (q1, q2, 0), we need:
    q1 > q2 > q3 = 0 (mass ordering)
    Each generation's vector is maximally S_3-symmetric

  Step 1: Gen 3 (heaviest, q = 0)
    Vector: (0,0,0) -- fully symmetric, unique
    This is the ONLY choice.

  Step 2: Gen 2 (middle)
    Need q > 0 with maximally symmetric vector.
    Candidates by decreasing symmetry:
      q = 1: (0, 0, 1), partially_symmetric, orbit size 3
      q = 3: (1, 1, 1), fully_symmetric, orbit size 1
      q = 5: (1, 2, 2), partially_symmetric, orbit size 3
      q = 6: (2, 2, 2), fully_symmetric, orbit size 1

      q = 3: (1,1,1) is FULLY SYMMETRIC -- strongest candidate
      q = 6: (1,1,1) is FULLY SYMMETRIC -- strongest candidate

      q = 5: (1, 2, 2), partially_symmetric, orbit size 3, unique: yes
      q = 6: (2, 2, 2), fully_symmetric, orbit size 1, unique: yes

  THE SELECTION ARGUMENT:
  ========================

  FULLY SYMMETRIC vectors (all components equal) exist only for
  q = 0 (0,0,0), q = 3 (1,1,1), and q = 6 (2,2,2).
  These are special: they are S_3-INVARIANT (orbit size 1).

  For other q values, the best we can do is PARTIALLY SYMMETRIC
  (two components equal, orbit size 3).

  Hierarchy principle: assign symmetry levels to generations:
    Gen 3 (heaviest): FULLY symmetric -> q = 0 -> (0,0,0)
    Gen 2 (middle):   FULLY symmetric -> q = 3 -> (1,1,1)
    Gen 1 (lightest): PARTIALLY symmetric (best available for q > 3)

  For Gen 1 with q > 3, the partially symmetric options are:
    q = 4: (0, 2, 2) (orbit size 3)
    q = 5: (1, 2, 2) (orbit size 3)
    q = 6: (2, 2, 2) (FULLY symmetric -- but already used by Gen 2 logic)

  q = 6 gives (2,2,2) which is fully symmetric,
  but we already used the 'fully symmetric' level for Gen 2.
  Using ANOTHER fully symmetric vector would break the hierarchy:
  Gen 3 = (0,0,0) and Gen 1 = (2,2,2) would both be S_3-invariant,
  making them symmetry-equivalent.  The generation structure requires
  DISTINCT symmetry classes.

  Therefore Gen 1 should be PARTIALLY symmetric, which gives q = 4 or 5.

  ADDITIONAL CONSTRAINT: maximize q_1 for maximum mass hierarchy.
  Between q = 4 and q = 5, choose q = 5 for larger hierarchy.
