# Outputs Guide

`outputs/` contains generated visual artifacts and small retained output files.

These are secondary to the runner + note chain.

Use this directory for:

- figures referenced by notes
- quick visual sanity checks
- retained small output assets

For publication release prep, prefer:

- `outputs/figures/` for pinned figure inputs and regenerated figure assets
- [`docs/publication/ci3_z3/DERIVATION_VALIDATION_MAP.md`](/private/tmp/physics-publication-prep/docs/publication/ci3_z3/DERIVATION_VALIDATION_MAP.md) for claim-to-output mapping

Do not treat an output file as canonical without checking the corresponding
runner and note first.
