# Figure Output Staging

This directory is reserved for figure-prep artifacts used by the publication
package.

Rules:

- store only pinned figure inputs or regenerated publication figures here
- each artifact should map to a figure in
  `/private/tmp/physics-publication-prep/docs/publication/ci3_z3/FIGURE_PLAN.md`
- if a figure depends on a retained theorem, the corresponding runner and claim
  must already appear in
  `/private/tmp/physics-publication-prep/docs/publication/ci3_z3/DERIVATION_VALIDATION_MAP.md`
