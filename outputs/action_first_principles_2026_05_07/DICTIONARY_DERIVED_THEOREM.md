## Theorem statement (T-AT)

Let `H_KS` denote the framework's natural Hamiltonian on the spatial substrate `Z³` with continuous emergent time, derived in [`BLOCK_B_HAMILTONIAN_DERIVATION.md`](BLOCK_B_HAMILTONIAN_DERIVATION.md) under

- **A1**: physical local algebra `Cl(3)` at sites,
- **A2**: physical spatial substrate `Z³`,
- **canonical Cl(3) Tr-form**: `Tr(T_a T_b) = δ_ab/2`,
- **A11 (RP)**: reflection positivity in emergent time,
- **locality**: bounded-support Hamiltonian density,
- **single-clock**: Lieb-Robinson causal structure,
- **A2.5 (proposed)**: single-loop-traversal at minimal dimension for the magnetic operator (per [`BLOCK_B7_PRIMITIVE_OPS_FORCING.md`](BLOCK_B7_PRIMITIVE_OPS_FORCING.md))

so that

```
H_KS  =  (g²/(2 a_s))  Σ_e  Ĉ_2(e)  −  (1/(g² a_s))  Σ_p^{spatial}  (1/N_c) Re Tr_F(U_p)        (TKS)
```

where `g² = 1` is the canonical Cl(3) Tr-form value (per [`G_BARE_RIGIDITY_THEOREM_NOTE.md`](../../docs/G_BARE_RIGIDITY_THEOREM_NOTE.md)).

Let `T(a_τ) := exp(-a_τ H_KS)` denote the Euclidean transfer operator at temporal step `a_τ > 0`. Define the anisotropy ratio `ξ := a_s/a_τ`.

**Theorem (T-AT, Anisotropic Trotter Dictionary).** For any `a_τ > 0`, the Euclidean partition function `Z(a_τ, T_phys = a_τ N_τ) := Tr T(a_τ)^{N_τ}` admits the path-integral representation

```
Z  =  ∫ DU  exp(-S_AT[U; ξ])  ·  (1 + O(a_τ²) Trotter remainder)                              (T1)
```

with the **anisotropic Trotter action**

```
S_AT[U; ξ]  =  -  Σ_{spatial p}      β_σ  ·  (1/N_c) Re Tr_F U_p^{(σ)}
              +  Σ_{temporal p}    [-ln K_{s_t}(U_p^{(τ)})]                                    (TAT)
```

where `U_p^{(σ)}` denotes a spatial plaquette, `U_p^{(τ)}` denotes a temporal plaquette, and the **derived couplings** are

```
β_σ  =  1/(g² ξ)                                                                              (TC1)
s_t  =  g²/(2 ξ)                                                                              (TC2)
```

and `K_s(W) := Σ_λ d_λ e^{-s C_2(λ)} χ_λ(W)` is the SU(N_c) heat kernel on group elements `W ∈ SU(N_c)`.

In particular:

1. **The action is anisotropic.** Spatial plaquettes carry Wilson form, temporal plaquettes carry heat-kernel form. The two are functionally distinct as class functions on SU(N_c).

2. **The action is NOT 4D isotropic Wilson** for any value of `a_τ`, including `a_τ = a_s` (`ξ = 1`).

3. **In the Hamilton limit** `ξ → ∞` (`a_τ → 0` at fixed `a_s`), the heat-kernel temporal weight concentrates and `Z` recovers the canonical Hamiltonian path integral `Tr e^{-T_phys H_KS}` exactly (Trotter remainder vanishes).

## Proof

### Step 1 — Trotter expansion

Decompose `H_KS = H_E + H_M` where

```
H_E := (g²/(2 a_s)) Σ_e Ĉ_2(e),                                                              (P1a)
H_M := -(1/(g² a_s)) Σ_p^{spatial} (1/N_c) Re Tr_F(U_p).                                      (P1b)
```

The Suzuki-Trotter expansion (Suzuki 1976) gives

```
T(a_τ) = e^{-a_τ H_E/2} e^{-a_τ H_M} e^{-a_τ H_E/2} + a_τ³ R(H_E, H_M) + O(a_τ⁵)              (P2)
```

with the symmetric Trotter error `R(H_E, H_M)` a bounded operator on the finite-dimensional physical Hilbert space (bounded by the Lieb-Robinson [`AXIOM_FIRST_MICROCAUSALITY_LIEB_ROBINSON_THEOREM_NOTE_2026-05-01.md`](../../docs/AXIOM_FIRST_MICROCAUSALITY_LIEB_ROBINSON_THEOREM_NOTE_2026-05-01.md) bound on operator norms). For `N_τ` time steps, the cumulative error is `O(N_τ a_τ³) = O(T_phys a_τ²)` in the physical Hamiltonian limit.

### Step 2 — Insertion of complete sets

The eigenstates `|U⟩` of all spatial-link operators `U_e` form a complete basis of the gauge field Hilbert space (each link has an SU(N_c) Hilbert space generated by the regular representation; the spectrum of `U` covers SU(N_c) under Haar measure; cf. Kogut-Susskind 1975). Insert this complete set at each Trotter step:

```
Z = ∫ ∏_t (∏_e dU_e(t)) ⟨U(t+1)| T(a_τ) |U(t)⟩.                                              (P3)
```

### Step 3 — Magnetic matrix element

`H_M` is diagonal in the `|U⟩` basis (it is a function of `U_p`, which is a product of `U_e`'s). Thus

```
⟨U(t)| e^{-a_τ H_M} |U(t)⟩ = exp[(a_τ/(g² a_s)) Σ_p^{spatial} (1/N_c) Re Tr_F U_p(t)].         (P4)
```

The two halves of the symmetric Trotter step combine to evaluate at the average configuration; for our purposes the leading-order spatial-plaquette weight is (P4) with coefficient `a_τ/(g² a_s) = 1/(g²ξ) = β_σ`.

### Step 4 — Electric matrix element (heat kernel)

`H_E` is a sum over independent links of `(g²/(2 a_s)) Ĉ_2`. Operate on a single link. The Casimir operator `Ĉ_2 = Σ_a Ê_a²` (with `Ê_a` the canonical SU(N_c) electric momentum-conjugate to `U`) generates the heat semigroup on SU(N_c) (this is a standard result; see Helgason 1978, or Brian Hall "Lie Groups, Lie Algebras, and Representations" §11.3). The matrix element is

```
⟨U'| e^{-(a_τ g²/(2 a_s)) Ĉ_2} |U⟩ = K_{s_t}(U' U^†)                                          (P5)
```

with `s_t = a_τ g²/(2 a_s) = g²/(2ξ)`. Here `K_s` is the SU(N_c) heat kernel:

```
K_s(W) := Σ_{λ irrep} d_λ exp(-s C_2(λ)) χ_λ(W).                                              (P6)
```

This is the **standard heat-kernel identity** on a compact Lie group. Its derivation:
- The Casimir `Ĉ_2` is the Laplace-Beltrami operator on SU(N_c) with the bi-invariant metric (= negative of the Killing form, in our normalization the canonical Tr-form induces this exactly via `Tr(T_a T_b) = δ_ab/2`).
- `e^{-s Δ}` is the heat semigroup; its kernel is `Σ_λ d_λ e^{-s C_2(λ)} χ_λ` by spectral decomposition of `Δ` on SU(N_c).
- The matrix element `⟨U'| e^{-s Ĉ_2} |U⟩` is exactly this heat kernel evaluated at `U' U^†` (left-invariance).

### Step 5 — Assembling the partition function

Combine (P3), (P4), (P5):

```
Z = ∫ ∏_{x⃗,t,e} dU_e(x⃗,t)
    × ∏_{x⃗,t,e} K_{s_t}(U_e(x⃗,t+1) U_e(x⃗,t)^†)                                              (P7a)
    × ∏_{p,t} exp[(a_τ/(g² a_s)) (1/N_c) Re Tr_F U_p(x⃗,t)]                                  (P7b)
    × (1 + Trotter remainder).
```

### Step 6 — Temporal-plaquette identification

In temporal gauge `U_0(x⃗,t) ≡ I`, the heat-kernel argument `U_e(x⃗,t+1) U_e(x⃗,t)^†` is precisely the temporal plaquette `U_p^{(τ)}` built from one spatial link at `t`, one identity temporal link, one spatial link at `t+1` (reversed), and one identity temporal link (reversed). Restoring temporal gauge (which adds Faddeev-Popov determinants that cancel against the path-integral measure normalization; see Smit ch. 4.4):

```
K_{s_t}(U_e(x⃗,t+1) U_e(x⃗,t)^†)  ↦  K_{s_t}(U_p^{(τ)})                                       (P8)
```

i.e., the heat kernel on the temporal plaquette holonomy.

### Step 7 — Final form

Substituting (P8) into (P7) and identifying the action as `S_AT = -(ln of integrand)`:

```
S_AT[U; ξ] = -Σ_{spatial p,t} β_σ (1/N_c) Re Tr_F U_p^{(σ)}(t)
            + Σ_{temporal p,t} [-ln K_{s_t}(U_p^{(τ)}(x⃗,e,t))]                               (P9)
```

with `β_σ = 1/(g²ξ)` and `s_t = g²/(2ξ)`. This is exactly the claimed form (TAT).

The Trotter remainder is `O(a_τ²) = O(1/ξ²)` per timestep, accumulating to `O(T_phys a_τ²)` over `N_τ = T_phys/a_τ` timesteps. The remainder is bounded (uniformly in spatial volume) by Lieb-Robinson on the framework's finite-range Hamiltonian. ∎

## Corollaries

### Corollary T-AT.1 — Hamilton limit recovers `H_KS` exactly

In the limit `ξ → ∞` (`a_τ → 0` at fixed `a_s`), the heat-kernel parameter `s_t = g²/(2ξ) → 0`. The heat kernel `K_{s_t}(W)` concentrates at `W = I`:

```
K_{s_t}(W) → δ(W, I) as s_t → 0.                                                              (C1.1)
```

The temporal-plaquette term forces `U_e(x⃗,t+1) = U_e(x⃗,t)` at each step (no temporal evolution). The path integral reduces to the Hamiltonian trace `Tr e^{-T_phys H_KS}` exactly. Trotter remainder also vanishes. ∎

### Corollary T-AT.2 — At finite `ξ`, the action is structurally non-Wilson

The temporal-plaquette weight `-ln K_{s_t}(W)` and the Wilson form differ at `O(s_t)`. The leading-order asymptotic of the heat kernel near `W = I` is, by the standard Polyakov / Drouffe-Itzykson 1980 expansion:

```
-ln(K_s(W)/K_s(I)) = (1/(2s)) ‖log W‖²_Tr + O(s) + O(‖log W‖⁴/s)                              (C2.1a)
```

where `‖X‖²_Tr := -2 Tr_F(X²)` is the bi-invariant metric induced by the canonical Tr-form `Tr(T_a T_b) = δ_ab/2`. Near identity:

```
1 - (1/N_c) Re Tr_F W = (1/(2 N_c)) ‖log W‖²_Tr + O(‖log W‖⁴)                                 (C2.1b)
```

so that

```
-ln(K_s(W)/K_s(I)) = (N_c/s)·[1 - (1/N_c) Re Tr_F W] + O(s) + O((1-Tr/N_c)²/s)                (C2.1c)
```

The leading Wilson coefficient is **`N_c/s_t`**, not `2 N_c/s_t` as commonly misstated. The Wilson form omits the `O(s)` and higher-curvature terms. **At `s_t > 0`, the temporal-plaquette weight is genuinely heat-kernel, not Wilson**. ∎

### Corollary T-AT.3 — Naive replacement of heat kernel by Wilson at `ξ = 1` introduces `O(g²)` error

Setting `s_t = g²/2 = 0.5` at canonical `g² = 1` and `ξ = 1`, the difference between `(N_c/s_t)·[1 - (1/N_c) Re Tr_F W]` and `-ln(K_{s_t}(W)/K_{s_t}(I))` is `O(s_t) = O(g²) = O(0.5)`, with explicit class-function structure given by (C2.1c). Numerical verification (SU(2), Appendix A.1 of [DICTIONARY_DERIVATION_RESULTS.md](DICTIONARY_DERIVATION_RESULTS.md)):

| `s_t` | `ξ` | `θ` | rel.err |
|---|---|---|---|
| 0.01 | 50 | 0.5 | 0.35% |
| 0.05 | 10 | 0.5 | 0.32% |
| 0.50 | 1 | 0.5 | 8.55% |
| 0.50 | 1 | 1.0 | 6.92% |

Replacing the framework's derived temporal heat kernel by Wilson form is therefore **not equivalent** at `ξ = 1`: it introduces a numerically-verified ≈7-9% systematic error in the temporal-plaquette weight at canonical `g² = 1`. This is the **`O(g²)`** correction characterized in Approach 3 of [DICTIONARY_DERIVATION_RESULTS.md](DICTIONARY_DERIVATION_RESULTS.md).

### Corollary T-AT.4 — Isotropic Wilson at `β = 6` is NOT framework-derived

The 4D isotropic Wilson action

```
S_W[U] = -(β_W/N_c) Σ_{all p} Re Tr_F U_p                                                     (C4.1)
```

with `β_W = 6` requires both:
- (a) `β_σ = β_τ` (spatial-temporal coupling equality at `ξ = 1`).
- (b) Wilson-form weight on temporal plaquettes (replacing heat-kernel).

Neither is forced by A1+A2 + canonical Tr-form + RP + locality + single-clock + A2.5. Both are admissions:
- (a) is **Convention C-iso** (`a_τ = a_s` time-discretization at the spatial scale).
- (b) is the **Wilson-replacement convention** (`O(g²) ≈ 5-15%` finite-`s_t` correction).

Together, (a)+(b) give the standard mapping `g_KS²(=1) ↔ β_W(=6)` at leading order in `g²` and `1/ξ`, with explicit one-loop anisotropy correction `O(g²)` (Karsch 1982 coefficients applicable). ∎

## Strength assessment

### What this theorem establishes

- **Existence and uniqueness** of an exact path-integral representation of `Z = Tr e^{-T H_KS}` as a 4D Euclidean lattice integral.
- **Specific anisotropic structure**: Wilson on spatial plaquettes (forced by A2.5 + canonical Tr-form + Re-Tr_F primitive operation) and heat-kernel on temporal plaquettes (forced by Trotterization of the Casimir electric term).
- **Explicit coupling formulas**: `β_σ = 1/(g²ξ)`, `s_t = g²/(2ξ)`, derived from the framework's Hamiltonian without further conventions.
- **Hamilton-limit consistency**: the path integral exactly recovers the operator-theory trace as `ξ → ∞`.

### What this theorem does NOT establish

- **Isotropy at `ξ = 1`**. The framework primitives do not pin `a_τ = a_s`; this is Convention C-iso.
- **Wilson-form temporal plaquette**. The natural framework form is heat-kernel; Wilson is an approximation valid at small `s_t`, i.e., large `ξ`.
- **Continuum-limit equivalence to standard 4D Wilson**. The two actions agree at leading order in `g²` and `1/ξ`, but differ at one-loop with explicit Karsch-Klassen coefficients.

## Verification plan

A primary runner would verify:

1. **Heat-kernel identity** (numerical): for SU(N_c=2) and SU(3), evaluate `K_s(θ)` via partial sum; confirm normalization `∫ K_s dHaar = 1` and concentration `K_s → δ_I` as `s → 0`.

2. **Trotter convergence** (numerical): compute `⟨P⟩` from `T(a_τ) = e^{-a_τ H_E/2} e^{-a_τ H_M} e^{-a_τ H_E/2}` on a small lattice (e.g., 1-plaquette toy) for decreasing `a_τ`; verify `⟨P⟩` converges to ground-state value as `a_τ → 0`.

3. **Hamilton-limit recovery** (numerical): for `ξ = 1, 10, 100`, compare `⟨P⟩` from anisotropic Trotter action to direct ground-state evaluation of `H_KS`; verify convergence as `ξ → ∞`.

4. **Wilson approximation accuracy** (numerical): compute the maximum error of `Wilson_form(s_t, W)` vs `-ln K_{s_t}(W)` over `W ∈ SU(N_c)` as a function of `s_t`. Plot the relative error.

These checks are within scope of standard exact-diagonalization methods on small lattices.
