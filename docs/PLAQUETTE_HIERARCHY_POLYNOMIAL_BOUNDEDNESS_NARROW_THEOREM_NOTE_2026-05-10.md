# Plaquette-Hierarchy Polynomial-Boundedness Narrow Theorem

**Date:** 2026-05-10
**Type:** positive_theorem
**Claim scope:** the standalone polynomial-algebra fact that a real
polynomial `p(t) ∈ R[t]` with a finite limit as `t → +∞` is constant;
and the immediate corollary that no real polynomial can simultaneously
satisfy `p(0) = a` and `lim_{t → +∞} p(t) = b` with `a ≠ b`. This is
purely a fact of polynomial algebra over `R[t]`; no Wilson plaquette,
gauge group, Haar measure, source-deformed partition function, or
`Cl(3)` / `Z^3` framework input is consumed.

## Statement

Let `p(t) = a_0 + a_1 t + a_2 t^2 + ... + a_d t^d ∈ R[t]` be a real
polynomial of degree exactly `d ≥ 0` (so `a_d ≠ 0` when `d ≥ 1`).

**Conclusion (T1) (leading-term asymptotic).** If `d ≥ 1`, then
`|p(t)| → +∞` as `t → +∞`. Specifically,

```text
lim_{t → +∞}  p(t) / t^d  =  a_d  ≠  0.                                   (1)
```

**Conclusion (T2) (finite-limit forces constant).** If `lim_{t → +∞} p(t)`
exists and is finite in `R`, then `d = 0`, i.e. `p(t) = a_0` is constant.

**Conclusion (T3) (two-distinct-value obstruction).** Let `a, b ∈ R`
with `a ≠ b`. Then no real polynomial `p(t) ∈ R[t]` satisfies both

```text
p(0)                 =  a,                                                (2)
lim_{t → +∞}  p(t)   =  b   (finite limit).                               (3)
```

**Conclusion (T4) (boundedness corollary).** Let `M ∈ R` with `M > 0`.
If `p(t)` is bounded on `[0, +∞)` by `|p(t)| ≤ M` for all `t ≥ 0`,
then `p` is constant.

## Proof

`(T1)` Write `p(t) = a_d t^d (1 + (a_{d-1} / a_d) t^{-1} + ... + (a_0 / a_d) t^{-d})`.
Each negative-power term tends to `0` as `t → +∞`, so the parenthetical
factor tends to `1`, and `p(t) / t^d → a_d`. Since `a_d ≠ 0` (degree
exactly `d ≥ 1`) and `t^d → +∞`, the modulus `|p(t)| → +∞`.

`(T2)` Suppose `lim_{t → +∞} p(t) = L` is finite. If `d ≥ 1`, `(T1)`
gives `|p(t)| → +∞`, contradicting the finite limit. Hence `d = 0` and
`p(t) = a_0` is constant.

`(T3)` Suppose such a polynomial exists. By `(2)`, `p(0) = a`; by `(3)`,
`lim_{t → +∞} p(t) = b` is finite. By `(T2)`, `p` is constant, so
`p(t) = p(0) = a` for all `t`. Then `lim_{t → +∞} p(t) = a`. But by
hypothesis the limit equals `b ≠ a`, a contradiction.

`(T4)` If `|p(t)| ≤ M` on `[0, +∞)` and `d ≥ 1`, `(T1)` gives
`|p(t)| → +∞`, contradicting the bound. Hence `d = 0`. ∎

## What this claims

- `(T1)`: leading-term asymptotic `p(t) / t^d → a_d` for `d ≥ 1`.
- `(T2)`: any real polynomial with a finite limit at `+∞` is constant.
- `(T3)`: no real polynomial joins two distinct endpoint values
  `p(0) = a ≠ b = lim p(t)` with a finite limit.
- `(T4)`: any real polynomial bounded on `[0, +∞)` is constant.
