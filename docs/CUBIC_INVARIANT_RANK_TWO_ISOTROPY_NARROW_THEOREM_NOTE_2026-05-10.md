## Statement

Let `O_h` denote the full cubic group acting on `R^3` -- the symmetry
group of the cube, of order `48`, generated by

```text
S_1 = diag(-1,  1,  1),     R_xy = [[0, -1, 0], [1, 0, 0], [0, 0, 1]],
S_2 = diag( 1, -1,  1),     R_yz = [[1, 0, 0], [0, 0, -1], [0, 1, 0]],
S_3 = diag( 1,  1, -1),     R_zx = [[0, 0, 1], [0, 1, 0], [-1, 0, 0]],
```

(three coordinate sign-flips together with the three 90-degree axis
rotations).

**Conclusion (T1) (rank-1 vanishing).**
The only `O_h`-invariant rank-1 tensor `v_i` on `R^3` is `v = 0`.

**Conclusion (T2) (rank-2 isotropy / uniqueness).**
Every `O_h`-invariant rank-2 tensor `T_ij` on `R^3` has the form

```text
T_ij  =  c * delta_ij,        c in R,                                    (1)
```

so the space of `O_h`-invariant rank-2 tensors on `R^3` has real dimension
exactly `1`. In particular `T` is automatically isotropic (i.e.
`SO(3)`-invariant).

**Conclusion (T3) (rank-3 vanishing).**
The only `O_h`-invariant rank-3 tensor `T_ijk` on `R^3` is `T = 0`.

**Conclusion (T4) (corollary -- automatic second-moment isotropy of
cubic-symmetric measures).**
Let `S` be any finite `O_h`-orbit-stable subset of `Z^3` (i.e.
`g.S = S` for every `g in O_h`), embedded into `R^3` by the standard
inclusion. Let `mu : S -> R_{>=0}` be any probability measure on `S`
satisfying `mu(g.x) = mu(x)` for every `g in O_h` and every `x in S`
(cubic invariance of `mu`). Define the second-moment matrix

```text
M_ij  :=  sum over x in S of mu(x) * x_i * x_j.                          (2)
```

Then

```text
M_ij  =  (1/3) * <r^2> * delta_ij,
                <r^2>  :=  sum over x in S of mu(x) (x_1^2 + x_2^2 + x_3^2). (3)
```

Equivalently `<x_1^2> = <x_2^2> = <x_3^2>` and `<x_i x_j> = 0` for
`i != j`.

**Conclusion (T5) (sharpening boundary at rank 4).**
For comparison, the space of `O_h`-invariant rank-4 tensors on `R^3` has
real dimension `4`, strictly greater than the rank-2 dimension `1`. The
rank-2 uniqueness statement of `(T2)` is special to rank `2`.

## Proof

The rank-`r` `O_h`-invariant tensor space is, by Schur-Weyl-type
character counting on real tensor representations of the finite group
`O_h`, the multiplicity of the trivial irrep in the rank-`r` tensor
representation. Equivalently it is the kernel of the linear operator
`I - rho(g)` simultaneously over the six generators above, acting on
`R^{3^r}`. We compute each kernel directly.

`(T1)` Rank-1: any `v` invariant under `S_1 = diag(-1, 1, 1)` must have
`v_1 = -v_1`, hence `v_1 = 0`. Likewise `S_2, S_3` force `v_2 = v_3 = 0`.

`(T2)` Rank-2: under `S_1`, the invariance `S_1^T T S_1 = T` forces
`T_{1j} = -T_{1j}` for `j = 2, 3` (i.e. off-diagonal entries in the first
row-or-column with `j != 1` flip sign), so they vanish; symmetric
treatment with `S_2, S_3` forces all off-diagonal entries to zero. The
axis rotation `R_xy` then exchanges the diagonal entries
`T_{11} <-> T_{22}` (up to sign), and since `R_xy^T T R_xy = T` forces
equality, `T_{11} = T_{22}`. Similarly `R_yz` and `R_zx` give
`T_{22} = T_{33}` and `T_{11} = T_{33}`. Thus `T = c * I_3` for the
single free parameter `c = T_{33}`.

`(T3)` Rank-3: under any sign-flip `S_a`, an invariant component
`T_{ijk}` satisfies `T_{ijk} = epsilon_a(i) epsilon_a(j) epsilon_a(k) T_{ijk}`
where `epsilon_a(i) = -1` if `i = a` and `+1` otherwise. The product of
signs is `-1` precisely when the multiset `{i, j, k}` contains the index
`a` an odd number of times. Hence `T_{ijk} = 0` unless every index
appears an even number of times among `{i, j, k}`. Since the sum
`i + j + k` over multisets has size `3`, each index can appear at most
`3` times, and "every index even" with total size `3` is impossible
(parity contradiction): the only solution is `T_{ijk} = 0` for all `i, j, k`.

`(T4)` Cubic invariance of `mu` means `mu(g.x) = mu(x)` for every
`g in O_h, x in S`. Then for any rank-2 monomial `x_i x_j`,

```text
M_ij  =  sum over x of mu(x) x_i x_j,
sum over x of mu(g.x) (g.x)_i (g.x)_j  =  sum over x of mu(x) x_i x_j  ((change of variable y = g.x))
                                       =  M_ij;
on the other hand,
sum over x of mu(g.x) (g.x)_i (g.x)_j  =  g_ik g_jl M_kl  ((expanding (g.x)_i = g_ik x_k)).
```

The two displays must be equal for every `g in O_h`, so `M_ij` is itself
an `O_h`-invariant rank-2 tensor. By `(T2)`, `M = c * I_3` for some
`c in R`. Taking the trace, `tr M = 3 c = <r^2>`, so `c = <r^2> / 3`.
This gives `(3)`.

`(T5)` By direct dimension counting (see runner Part E): the joint
kernel of `I - rho(g)` over `g in {S_1, S_2, S_3, R_xy, R_yz, R_zx}` on
the rank-4 tensor space of `R^{81}` has dimension `4`, strictly greater
than the rank-2 dimension `1`. (The four cubic invariants are
`delta_ij delta_kl, delta_ik delta_jl, delta_il delta_jk` from the
isotropic part, plus the cubic-anisotropy invariant
`sum_{i in {1,2,3}} delta_ijkl` -- explicit forms are not needed for
this narrow note.) ∎
