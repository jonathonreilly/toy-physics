## Boundary

This note salvages the durable part of the staggered-Dirac closure attempt:
the C, H, and M_3(C) finite-algebra pieces can be represented on one
staggered C8 taste cube, and a minimal finite-Dirac candidate can be written
down. The same runner also shows why this is not a closure: the nontrivial
finite-Dirac examples tested here violate the order-one condition. The
remaining question is therefore D_F class selection, not a solved
left-handed-content theorem.

No new axiom, retained primitive, audit verdict, or retained-grade status is
introduced here.

## Construction

The runner uses the staggered Pauli-tensor realization

```text
Gamma_1 = sigma_1 x I x I
Gamma_2 = sigma_3 x sigma_1 x I
Gamma_3 = sigma_3 x sigma_3 x sigma_1
```

on C8. It verifies the Clifford relations exactly. With
`omega = Gamma_1 Gamma_2 Gamma_3`, it obtains:

- a C-summand generated by `I` and `omega`, with `omega^2 = -I`;
- an H-summand from `Cl+(3) = span{I, Gamma_1 Gamma_2,
  Gamma_1 Gamma_3, Gamma_2 Gamma_3}`;
- an M_3(C) action on the hamming-weight-one triplet, generated by the
  three projectors and a cyclic shift.

All of these are represented as 8 by 8 operators on the same C8. This is a
bounded assembly candidate, not a proof that the Standard Model finite
triple has been selected.

## Candidate D_F and KO Signs

The minimal candidate tested here is

```text
D_min = Gamma_1 + Gamma_2 + Gamma_3.
```

The runner verifies:

- `D_min` is self-adjoint;
- `D_min` is odd under `gamma_stag = sigma_3 x sigma_3 x sigma_3`;
- `J = omega K` has the KO-dim-6-style signs `J^2 = -I`,
  `J D_min = D_min J`, and `J gamma = -gamma J`.

These are useful bounded structural checks. They do not close the
finite-Dirac selection problem, because the order-one condition remains
decisive.

## Order-One Result

The runner builds a 14-element test set for the finite algebra pieces:
`{I, omega}`, the three `Cl+(3)` generators, and the rank-9 M_3(C) basis on
the hamming-weight-one triplet. The combined set has rank at least 13 because
the summand embeddings overlap on identity-like components. It then checks

```text
[[D_F, a], J b J^{-1}] = 0
```

for three representative finite-Dirac choices:

| D_F class | Runner result | Interpretation |
|---|---:|---|
| scalar `D_F = 0.7 I` | zero violation | vacuous order-one pass |
| `D_min = Gamma_1 + Gamma_2 + Gamma_3` | nonzero violation | not a closed SM finite Dirac operator |
| toy Yukawa-like perturbation | nonzero violation | selection problem remains |

This is the load-bearing salvage. The result makes the gate more precise:
C8 assembly is available, but order-one-compatible nontrivial `D_F`
selection is still open.
