## Statement

Let `C` be the standard `3 × 3` cyclic permutation matrix
```text
C = [[0, 1, 0],
     [0, 0, 1],
     [1, 0, 0]],                                                           (1)
```
so that `C³ = I`. Let `ω = e^{2πi/3}`, and let `(a, b) ∈ R × C` be
abstract symbols. Define the circulant Hermitian
```text
H  =  a I  +  b C  +  b̄ C².                                              (2)
```
Define the eigenvalue triple of `H`,
```text
λ_k  =  a + b ω^k + b̄ ω^{-k},   k = 0, 1, 2,                            (3)
```
and the `C_3` character coefficients of `λ = (λ_0, λ_1, λ_2)`,
```text
a_0  =  (λ_0 + λ_1 + λ_2) / √3,                                            (4)
z    =  (λ_0 + ω̄ λ_1 + ω λ_2) / √3.                                       (5)
```

**Conclusion (T1) (algebraic identity).** `a_0 = √3 · a`.

**Conclusion (T2) (algebraic identity).** `z = √3 · b`.

**Conclusion (T3) (algebraic identity, derived from T1+T2).**
```text
a_0² − 2 |z|²  =  3 a² − 6 |b|².                                           (6)
```

In particular, the equation `a_0² = 2 |z|²` on the character coefficients
is, as a fact of algebra, the same equation as `3 a² = 6 |b|²` on the
operator parameters.

## Proof

`(T1)` Sum (3) over `k ∈ {0, 1, 2}`:
```text
λ_0 + λ_1 + λ_2
   =  3 a  +  b (1 + ω + ω²)  +  b̄ (1 + ω^{-1} + ω^{-2}).
```
Using the standard root-of-unity identity `1 + ω + ω² = 0`, both `b` and
`b̄` terms cancel, leaving `λ_0 + λ_1 + λ_2 = 3 a`. Dividing by `√3`,
```text
a_0  =  3 a / √3  =  √3 · a.
```

`(T2)` Compute (5) using (3):
```text
λ_0 + ω̄ λ_1 + ω λ_2
   =  a (1 + ω̄ + ω)
    + b (1 + ω̄ ω + ω ω²)
    + b̄ (1 + ω̄ ω^{-1} + ω ω^{-2}).
```
The first bracket vanishes (`1 + ω + ω̄ = 1 + ω + ω² = 0`).
For the `b` bracket, `ω̄ ω = 1` and `ω ω² = ω³ = 1`, so the bracket
equals `1 + 1 + 1 = 3`. For the `b̄` bracket, `ω̄ ω^{-1} = ω^{-1} ω̄`,
and using `ω̄ = ω^{-1}` (since `|ω| = 1`), `ω̄ ω^{-1} = ω^{-2}`, while
`ω ω^{-2} = ω^{-1}`, so the `b̄` bracket equals
`1 + ω^{-2} + ω^{-1} = 1 + ω + ω² = 0`. Hence
```text
λ_0 + ω̄ λ_1 + ω λ_2  =  3 b,
```
and dividing by `√3`,
```text
z  =  3 b / √3  =  √3 · b.
```

`(T3)` From `(T1)`, `a_0² = 3 a²` (since `a` is real). From `(T2)`,
`|z|² = z · z̄ = (√3 b)(√3 b̄) = 3 |b|²`. Subtracting,
```text
a_0² − 2 |z|²  =  3 a² − 6 |b|².
```
∎
