# DM Broader Route Audit

**Date:** 2026-04-15  
**Branch:** `codex/dm-main-derived`  
**Purpose:** 30k-foot audit of whether the current `main` atlas and note/script
surface already contains a genuinely different all-axiom route to DM closure
that avoids the current Majorana absolute-scale / local-to-generation blocker

---

## Bottom Line

**No real alternative on the current axiom/atlas stack closes full zero-import
DM without confronting the denominator.**

The branch has one strong numerator program and one honest denominator program:

- numerator: direct-observable freeze-out / relic structure
- denominator: leptogenesis with a now-sharply isolated Majorana-scale blocker

Everything else on the current branch either:

1. strengthens the numerator only,
2. is bounded/open on a different lane,
3. depends on the same unresolved flavor / hierarchy inputs,
4. or simply moves the denominator problem somewhere else without removing it.

So the harsh recommendation is:

- **continue the current lane**
- **do not switch flagships**
- if bandwidth permits, run only one secondary lane in parallel:
  numerator-side direct-observable / thermodynamic cleanup
- do **not** pretend there is a hidden atlas-native denominator bypass already
  on `main`

---

## Ranked Routes

### 1. Current flagship: direct-observable numerator + leptogenesis denominator

**What it derives**

- the strongest current DM numerator on the branch
- `H -> T-matrix -> sigma_v` as the native annihilation route
- a strong structural relic result `R(eta)`
- a narrowed denominator program in which the local Dirac side is already
  highly constrained and the live problem is explicitly Majorana-side

**What it still needs**

- a genuinely new non-homogeneous local-to-generation Majorana bridge or a new
  absolute-scale datum
- then the three-generation `A/B/epsilon` lift without fitted leftovers

**Does it bypass the Majorana blocker?**

**No.** This is the lane that *contains* the blocker. It is still the best
lane because the alternatives are weaker, not because it is closed.

**Why it still ranks first**

- it is the only route in the atlas that already reaches an honest, sharp,
  theorem-grade denominator boundary
- the numerator side is materially stronger than every other DM route on the
  branch
- the branch already has the paper-safe fallback `R(eta)`

---

### 2. Direct-observable / `R(eta)` fallback

**What it derives**

- the DM numerator and structural relic ratio as a function of `eta`
- `R(eta)` with the observed `eta` giving the right number

**What it still needs**

- an independently derived `eta`, or an explicit decision to stop at the
  one-input theorem surface

**Does it bypass the Majorana blocker?**

**No.** It avoids the blocker only by refusing to derive the denominator.
That is a legitimate paper-safe fallback, not full DM closure.

**Harsh read**

This is the strongest *publication surface* but not an alternative closure
route. It is exactly the controlled fallback when the denominator remains open.

---

### 3. Wave direct-`dM` / matched-history lane

**What it derives**

- a bounded direct dynamical observable on a different lattice-response lane
- some controlled sign and amplitude-band structure in narrow families / seeds

**What it still needs**

- actual continuum convergence
- a canonical atlas role
- a clean bridge from the wave-response observable to the cosmological DM ratio
  or relic abundance

**Does it bypass the Majorana blocker?**

**No.** It does not currently derive the relic denominator, and it does not
replace the DM cosmology chain with a closed all-axiom alternative.

**Harsh read**

This lane is not a live DM-closure competitor. It is not even in the canonical
DM atlas row, and its own notes are mostly portability / control / boundary
audits with negative or bounded status. Treat it as exploratory companion work,
not as a rescue path.

---

### 4. Hierarchy / observable-principle / scalar-datum transplant

**What it derives**

- exact scalar source-response tooling on the hierarchy lane
- exact selector / normalization theorems for the hierarchy observable
- exact dimensionless scalar data that are reusable in some contexts

**What it still needs**

- a route from those scalar observables to a non-homogeneous Majorana
  staircase anchor

**Does it bypass the Majorana blocker?**

**No.** The branch already proved the obvious transplant fails: current scalar
atlas data only enter multiplicatively and leave the common Majorana scale
factored out.

**Harsh read**

This is the most tempting fake alternative because it is exact and atlas-native.
But the repo now already contains the obstruction theorem saying it does not
solve the DM denominator.

---

### 5. Flavor / CKM / three-generation structure

**What it derives**

- exact three-generation structural presence
- bounded CKM hierarchy patterns
- exact Schur-complement identities inside the open CKM lane

**What it still needs**

- full flavor closure
- a theorem-grade quantitative mass / texture law
- any demonstrated non-homogeneous Majorana scale lift

**Does it bypass the Majorana blocker?**

**No.** The current generation/flavor toolkit is structural or bounded. It does
not provide a closed absolute Majorana staircase law, and the current Majorana
notes already prove the retained `Z_3` lift is homogeneous. More sharply on
the current denominator surface, the new CKM-texture transfer no-go theorem
shows the current CKM/NNI tools do not even transfer to the exact universal
Dirac bridge: the singular spectrum stays triple-degenerate, the hierarchy
factors collapse to `1`, and phase-only companions still leave `Y^dag Y`
proportional to `I`.

**Harsh read**

This lane helps explain why a three-generation lift exists at all, but it does
not currently supply the missing absolute-scale theorem. Using CKM/flavor as a
DM rescue lane would be wishful relabeling.

---

### 6. EW baryogenesis / transport

**What it derives**

- bounded support on a separate `eta` route
- some numerically suggestive results on imported or calibrated transport
  surfaces

**What it still needs**

- resolution of the regime mismatch
- native transport closure
- removal of the detonation / EWPT contradictions

**Does it bypass the Majorana blocker?**

**Yes in a trivial sense, no in the sense that matters.**

It avoids Majorana because it uses a different denominator mechanism, but it
does not close on the current branch and is in worse shape than leptogenesis.

**Harsh read**

This is not a serious flagship candidate on the current branch. Switching back
to it would be a downgrade in rigor.

## Route Triage

### What is actually strong now

1. the numerator-side direct-observable stack
2. the `R(eta)` publication surface
3. the Majorana blocker itself as a sharply isolated theorem gap

### What looks different but is not a real bypass

1. hierarchy observable-principle reuse
2. CKM/flavor/generation structure
3. wave direct-`dM`

Each of these either remains homogeneous, bounded, or disconnected from the
actual denominator.

### What should not be promoted

1. EWBG as the main theorem route
2. any claim that the current atlas already hides a denominator-free DM route
3. any claim that `R(eta)` is equivalent to full DM closure

---

## Recommendation

### Main recommendation

**Continue the current lane. Do not switch flagships.**

This means:

1. keep the direct-observable numerator as the canonical DM front end
2. keep leptogenesis as the only serious theorem-grade denominator program
3. keep the paper-safe claim at `R(eta)` until the denominator is truly closed

### Secondary parallel lane worth running

If one secondary lane is run in parallel, it should be:

- numerator-side thermodynamic / relic-bridge cleanup

That lane can improve the paper and reduce residual caveats without pretending
to solve the denominator.

### What not to do

- do not pivot to wave direct-`dM`
- do not pivot back to EWBG transport
- do not sink time into scalar-atlas or CKM/flavor reuse as if they already
  contain the missing Majorana scale law
