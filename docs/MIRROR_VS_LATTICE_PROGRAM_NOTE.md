## Canonical Read

### Mirror remains the flagship

The exact mirror lane is still the main program story because it has the
strongest retained coexistence package:

- exact mirror / `Z2` remains the flagship same-graph coexistence lane
- exact 2D mirror provides the clean validation card for that story
- the retained mirror MI chain shows a clear bounded advantage over matched
  random baselines

The safe summary is not that mirror solves every axis. It is that mirror is the
best retained joint-coexistence architecture in the repo.

### Ordered lattice is the secondary law-side branch

The ordered-lattice line is now a real retained branch, but it is still split
across harnesses:

- standard-strength ordered lattice gives a same-family two-harness bridge
- the weak-field reopening is narrow but real
- the dense 3D spent-delay branch now adds a bounded same-family attractive
  window
- the no-barrier ordered-lattice branch gives a clean `|delta| ~ 1/b`-like
  distance law on that separate harness

That makes ordered lattice the strongest secondary branch for law-side work,
not a promoted replacement for mirror.

### Nearest-neighbor refinement is the frontier bridge

The nearest-neighbor lattice work strengthens the frontier side of the story,
but it does not close the continuum question:

- the raw NN lattice is Born-clean through `h = 0.25`
- the deterministic rescale path extends that refinement to `h = 0.0625`
- the raw kernel remains unresolved at `h = 0.125`
- the canonical notes explicitly keep the continuum claim open

So NN refinement is best described as a bounded refinement bridge, not a
finished continuum theorem.

## Safe Comparison

The retained comparison is:

- **mirror** is the flagship coexistence lane
- **ordered lattice** is the strongest secondary law-side branch

- **none of the NN results justify a finished continuum theorem**
