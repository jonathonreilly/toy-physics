## Derivation

### T1, T2: |V_us|² and |V_cb|² as Cabibbo powers

Direct from retained Wolfenstein:
- `|V_us|² = λ² = α_s/2 = α_s/N_pair`.
- `|V_cb|² = A² λ⁴ = (2/3)(α_s/2)² = (N_pair/N_color)(α_s/N_pair)² = (N_pair/N_color) |V_us|⁴`.

### T3: |V_ub|² as the 6th power

`|V_ub|² = A² λ⁶ |ρ - iη|² = A² λ⁶ × (ρ² + η²)`.

On retained surface: `ρ² + η² = (1/N_quark² + (N_quark − 1)/N_quark²) = N_quark/N_quark² = 1/N_quark`.
And `A² = N_pair/N_color`.

So `A²(ρ² + η²) = (N_pair/N_color)(1/N_quark) = N_pair/(N_color × N_quark) = N_pair/(N_color × N_pair × N_color) = 1/N_color²`.

Therefore `|V_ub|² = (1/N_color²) λ⁶ = (1/N_color²) |V_us|⁶`.

### T4: |V_td|² with circumradius factor

`|V_td|² = A² λ⁶ ((1 − ρ̄)² + η̄²)`. On retained surface,
`(1 − ρ̄)² + η̄² = (80 + α_s²)/96`. And `R̄² = (80 + α_s²)/320`, so
`(80 + α_s²) = 320 R̄²`. Thus `(1 − ρ̄)² + η̄² = 320 R̄²/96 = (10/3) R̄²`.

```text
|V_td|^2  =  A^2 lambda^6 * (10/3) R_bar^2
          =  (N_pair/N_color) (10/3) |V_us|^6 R_bar^2
          =  (10 N_pair / (3 N_color)) |V_us|^6 R_bar^2.
```

In structural integers: `10/3 × N_pair/N_color = 20/9 = N_pair²(N_quark − 1)/N_color²`.

So:

```text
|V_td|^2  =  (N_pair^2 (N_quark - 1) / N_color^2) |V_us|^6 R_bar^2
          =  (20/9) |V_us|^6 R_bar^2.
```

### T6, T7: |V_td/V_ub|² ratio

```text
|V_td/V_ub|^2  =  (N_pair^2 (N_quark - 1) / N_color^2) |V_us|^6 R_bar^2
                  / ((1/N_color^2) |V_us|^6)
              =  N_pair^2 (N_quark - 1) R_bar^2.
```

Substituting `R̄² = (80 + α_s²)/320`:

```text
N_pair^2 (N_quark - 1) R_bar^2
   =  4 * 5 * (80 + alpha_s^2)/320
   =  (80 + alpha_s^2)/16
   =  5 + alpha_s^2/16
   =  (N_quark - 1) + alpha_s^2/N_pair^4.
```

So `|V_td/V_ub|² ≈ N_quark − 1 = 5` with NLO correction `α_s²/N_pair⁴ = α_s²/16`.

At canonical `α_s ≈ 0.103`: correction is `0.0106/16 ≈ 0.000663`, i.e. ~0.013% deviation from 5. So `|V_td/V_ub|² = 5.000663` essentially exactly equal to `N_quark − 1 = 5`.

### T8, T9: α_s-independent ratios

```text
|V_cb|^2 / |V_us|^4  =  A^2 lambda^4 / lambda^4  =  A^2  =  N_pair/N_color  =  2/3.

|V_ub|^2 / |V_us|^6  =  A^2 (rho^2 + eta^2) lambda^6 / lambda^6  =  A^2 (rho^2 + eta^2)
                     =  (N_pair/N_color) (1/N_quark)  =  1/N_color^2  =  1/9.
```

Both are `α_s`-independent (universal on retained surface where `A²` and
`(ρ² + η²)` are fixed structural-integer ratios).
