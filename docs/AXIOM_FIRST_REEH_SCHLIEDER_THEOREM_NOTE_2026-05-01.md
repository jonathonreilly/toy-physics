## Scope

This note proves the **Reeh-Schlieder cyclicity theorem** on `A_min`
plus the retained axiom-first foundations:

> For any nonempty open region `O ⊂ Λ` of the framework's lattice,
> the local operator algebra `A(O)` acts cyclically on the vacuum
> vector `|Ω⟩` of the RP-reconstructed Hilbert space `H_phys`. That
> is, the vector space `A(O) |Ω⟩` is dense in `H_phys`.

This is the lattice analogue of the Reeh-Schlieder 1961 theorem.
It is one of the most counter-intuitive results in axiomatic QFT:
operations performed in any tiny localized region can prepare
arbitrary states of the entire physical Hilbert space.

After this note, the framework's local-algebra structure has the
full Reeh-Schlieder property, completing the
Wightman/Haag-Kastler-style local-algebra package together with
microcausality (Block 04).

## Statement

Let `H_phys` be the RP-reconstructed physical Hilbert space, `|Ω⟩`
the global cyclic vector (vacuum), `H` the reconstructed Hamiltonian
with `H ≥ 0` and `H |Ω⟩ = 0`. Let `O ⊂ Λ` be any nonempty open
region of lattice sites, and `A(O)` the local operator algebra
generated by polynomials in field operators supported in `O`.

Then on `A_min` plus the retained inputs above:

**(R1) Reeh-Schlieder cyclicity.** The vector subspace
`A(O) |Ω⟩ ⊂ H_phys` is dense in `H_phys`.

**(R2) Equivalent forms.**

- (R2a) For any `ψ ∈ H_phys` and any `ε > 0`, there exists
  `O_1, ..., O_n ∈ A(O)` and coefficients `c_α` such that
  `‖ ψ - Σ_α c_α O_α |Ω⟩ ‖ < ε`.

- (R2b) The vacuum `|Ω⟩` is a separating vector for the commutant
  `A(O)'`: for any `B ∈ A(O)'`, `B |Ω⟩ = 0` implies `B = 0`.

**(R3) Universal access.** Operations supported in any tiny
nonempty region `O` can in principle prepare states arbitrarily
close to any state of the entire physical Hilbert space. This is
the "Reeh-Schlieder paradox": a localized observer can in principle
reconstruct the entire global state.

Statements (R1)–(R3) constitute the framework Reeh-Schlieder theorem
on `A_min` plus the retained axiom-first foundations.

## Proof

The standard proof structure (Streater-Wightman 1964, Haag 1992)
uses analytic continuation in time from `O` to all of `Λ` via the
spectrum condition. We adapt to the framework's lattice setting.

### Step 1 — Global cyclicity

By the RP reconstruction (R2 of the RP note), `H_phys = A(Λ)|Ω⟩`
where `A(Λ)` is the global operator algebra. The vacuum `|Ω⟩` is
cyclic for `A(Λ)`: every vector in `H_phys` is the closure of
`A(Λ)|Ω⟩`. (This is essentially the definition of the GNS-like
RP reconstruction.)

### Step 2 — Time-translation analytic continuation

Suppose `ψ ∈ H_phys` is orthogonal to `A(O) |Ω⟩`. We will show
`ψ = 0`.

For any `n`, any operators `O_1, ..., O_n` supported in `O`, and any
real times `t_1, ..., t_n`, consider

```text
    F(t_1, ..., t_n)  :=  < ψ | α_{t_1}(O_1) ... α_{t_n}(O_n) | Ω >       (1)
```

where `α_t(O) := e^{i t H} O e^{-i t H}` is real-time evolution.
For `(t_1, ..., t_n)` small enough (so each `α_{t_i}(O_i)` remains
in the algebra generated by `A(O)` plus its time-translations within
some neighborhood — restricted to local operators by Block 04
microcausality), `α_{t_i}(O_i) ∈ A(O_t)` for some neighborhood
`O_t ⊃ O`.

By assumption (orthogonality to `A(O) |Ω⟩` extended by time
translation), `F(t_1, ..., t_n) = 0` for `(t_1, ..., t_n)` in a
small real neighborhood.

### Step 3 — Analytic continuation via spectrum condition

The spectrum condition `H ≥ 0` allows the function `F` to be
analytically continued in `t_i` to the upper half-plane `Im t_i > 0`:

```text
    α_z(O)  =  e^{i z H} O e^{-i z H}                                       (2)
```

For `z = t + i s` with `s > 0`, `e^{i z H} = e^{i t H} · e^{-s H}`,
and `e^{-s H}` is bounded (norm `≤ 1`) because `H ≥ 0` (spectrum
condition). Hence `α_z(O)` is well-defined and bounded for
`Im z > 0`.

The function

```text
    F(z_1, ..., z_n)  =  < ψ | α_{z_1}(O_1) ... α_{z_n}(O_n) | Ω >        (3)
```

is therefore analytic in `(z_1, ..., z_n)` in the wedge
`{Im z_i > 0}`, continuous up to the real boundary, and vanishes on
the real boundary (by orthogonality assumption near zero) — and
hence in a small open subset of the real boundary.

### Step 4 — Edge-of-the-wedge / continuation to all real times

By the standard edge-of-the-wedge theorem (Streater-Wightman 1964,
Theorem 2-15), an analytic function in the wedge `{Im z_i > 0}` that
vanishes on a nonempty open subset of the real axis vanishes
identically on the wedge.

By Schwarz reflection (extending across the real axis) and continuing
analytically, `F(z_1, ..., z_n) ≡ 0` for all complex `(z_1, ..., z_n)`,
in particular for all real `(t_1, ..., t_n)`.

### Step 5 — Density of time-translated A(O)|Ω⟩ in H_phys

By Step 4, `F(t_1, ..., t_n) = 0` for all real times. The set

```text
    { α_{t_1}(O_1) ... α_{t_n}(O_n) |Ω⟩  :  O_i ∈ A(O), t_i ∈ R, n ≥ 0 }    (4)
```

is the orbit of `A(O) |Ω⟩` under time translations. By Step 1, this
set is dense in `H_phys` (because time translations act on `H_phys`
and `A(O)` together generate via Step 1 an algebra dense in `A(Λ)`
— the time-translated operators reach all of `Λ` via the lattice
analogue of the propagation argument; Block 04 microcausality
provides the locality that makes this rigorous).

So `ψ` is orthogonal to a dense subspace of `H_phys`, hence `ψ = 0`.
This proves (R1).

### Step 6 — Equivalent forms (R2)

(R2a) is just a restatement of density. (R2b) follows from (R1) by
the GNS/Tomita-Takesaki structure: a separating vector for `A(O)'`
is exactly a cyclic vector for `A(O)`. The proof:

If `B ∈ A(O)'` and `B |Ω⟩ = 0`, then for any `O' ∈ A(O)`,
`B O' |Ω⟩ = O' B |Ω⟩ = 0` (by commutativity of `B` with `A(O)`).
Hence `B` annihilates `A(O) |Ω⟩`, which is dense in `H_phys` by
(R1). Hence `B = 0` on a dense subspace, so `B = 0`. ∎
