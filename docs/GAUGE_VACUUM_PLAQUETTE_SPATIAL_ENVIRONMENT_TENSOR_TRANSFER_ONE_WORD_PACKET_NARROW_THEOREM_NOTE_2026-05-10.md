## Statement

Let `N` be a nonnegative integer (default reference value `N = 4`). Let
`B_N = {(p,q) : 0 <= p, q <= N}` be the finite dominant-weight box, of
cardinality `|B_N| = (N+1)^2`.

Let `S` be the `|B_N| x |B_N|` permutation matrix defined by
`S_{(q,p), (p,q)} = 1` and zero otherwise. Then `S = S^T` and `S^2 = I`.

Let `D = diag(c_(p,q))_{(p,q) in B_N}` be a diagonal matrix indexed by
`B_N` with **strictly positive** entries satisfying
**conjugation symmetry**

```text
c_(p,q)  =  c_(q,p)   for all (p,q) in B_N.                              (1)
```

Equivalently, `S D = D S`.

Let `(N_f, N_fbar)` be a pair of `|B_N| x |B_N|` matrices with entries in
`{0, 1}` (or more generally in the nonnegative integers `Z_{>=0}`) related
by

```text
N_fbar  =  S N_f S.                                                       (2)
```

Define the assembled finite matrix

```text
M       :=  N_f + N_fbar,                                                 (3)
T       :=  D M D M^T D.                                                  (4)
```

Let `e_(0,0)` be the unit vector at the trivial-irrep index `(0,0) in B_N`
(i.e. the column vector with `1` at index `(0,0)` and `0` elsewhere).

**Conclusion (T1) (entry-wise nonnegativity).** Every entry of `T` is
nonnegative:

```text
T_{ab}  >=  0   for all  a, b in B_N.                                     (5)
```

**Conclusion (T2) (conjugation-swap symmetry).**

```text
S T  =  T S.                                                              (6)
```

**Conclusion (T3) (nonnegative trivial-readout, with strictly positive
trivial-channel amplitude).** Let `r := T e_(0,0)`. Then

```text
r_a  >=  0    for all a in B_N,                                           (7)
r_(0,0)  =  c_(0,0)^2 * sum_{a in B_N} M_{(0,0), a}^2 * c_a               (8)
         >  0   whenever there exists a in B_N with M_{(0,0), a} > 0.
```

Furthermore, `S r = r`, i.e. `r` is conjugation-swap-invariant.

## Proof

`(T1)` Each summand of `T_{ab} = sum_{x, y} D_{aa} M_{ax} D_{xx} M^T_{xy}
M^T_{yb} D_{bb}` rearranges (since `D` is diagonal) to

```text
T_{ab}  =  c_a * c_b * sum_{x in B_N} M_{ax} M_{bx} c_x.                  (9)
```

Each factor is nonnegative: `c_a, c_b, c_x > 0` by positivity of `D`, and
`M_{ax}, M_{bx} in Z_{>=0}` because `M = N_f + N_fbar` is a sum of two
nonnegative integer matrices. Hence `T_{ab} >= 0`. (In fact `T_{ab} = 0`
iff no `x` simultaneously satisfies `M_{ax} > 0` and `M_{bx} > 0`.)

`(T2)` Since `S = S^T` and `S^2 = I`, and `S D = D S` by (1), it suffices
to show `S M S = M`, i.e. `S M = M S`. By (2), `S N_f S = N_fbar` so
`S N_f = N_fbar S`. Also `S N_fbar S = S(S N_f S)S = S^2 N_f S^2 = N_f`,
so `S N_fbar = N_f S`. Hence

```text
S M  =  S (N_f + N_fbar)  =  N_fbar S + N_f S  =  (N_fbar + N_f) S  =  M S.
```

So `S` commutes with `D`, with `M`, and (since `(S X)^T = X^T S` for any
matrix `X` when `S = S^T`) with the assembly

```text
T  =  D M D M^T D
   →  S T  =  S D M D M^T D  =  D S M D M^T D  =  D M S D M^T D
            =  D M D S M^T D  =  D M D (M S)^T D                          (10)
            =  D M D M^T S^T D  =  D M D M^T S D  =  D M D M^T D S
            =  T S.
```

`(T3)` From (9),

```text
r_a  =  T_{a, (0,0)}  =  c_a * c_(0,0) * sum_{x in B_N} M_{ax} M_{(0,0), x} c_x.
```

This is a sum of nonnegative terms; hence `r_a >= 0`. The trivial-channel
entry simplifies because `M_{(0,0), x}^2 = M_{(0,0), x}` only when the
entries are in `{0,1}`; in general

```text
r_(0,0)  =  c_(0,0)^2 * sum_{x in B_N} M_{(0,0), x}^2 c_x,
```

which is strictly positive whenever some `M_{(0,0), x} > 0`. For the
default reference instance (`N = 4`, SU(3) Pieri primitives), there are
exactly two nonzero entries on row `(0,0)`: `M_{(0,0), (1,0)} = 1` and
`M_{(0,0), (0,1)} = 1`, giving

```text
r_(0,0)  =  c_(0,0)^2 * (c_(1,0) + c_(0,1))  >  0.                       (11)
```

Finally, `S r = S T e_(0,0) = T S e_(0,0) = T e_(0,0) = r`, where
`S e_(0,0) = e_(0,0)` because `(0,0) = (0,0)` is a self-conjugate weight
(`S` permutes only nontrivial conjugate pairs). ∎

## What this claims

- `(T1)`: entry-wise nonnegativity of the assembled finite matrix `T` on
  the finite box `B_N`, for any positive symmetric `D` and any nonnegative
  swap-related `(N_f, N_fbar)`.
- `(T2)`: conjugation-swap symmetry of `T`.
- `(T3)`: nonnegativity of the trivial-readout `r = T e_(0,0)`, with
  strictly positive trivial-channel amplitude `r_(0,0) > 0` whenever
  `M` has a nonzero `(0,0)`-row entry.

## What this does NOT claim

- Does **not** claim that the assembled matrix `T` equals any physical
  unmarked spatial-environment slice transfer operator. The narrow theorem
  treats `(D, N_f, N_fbar)` as abstract algebraic ingredients on a finite
  index set.
- Does **not** evaluate the boundary-character identity
  `z_(p,q)^env(beta) = <chi_(p,q), (T_beta^env,tensor)^(L_perp - 1)
  eta_beta^env>` for any `L_perp`, `(p,q)`, or `beta`. The parent
  bounded-theorem note states that identity structurally; the narrow
  theorem here does not consume or assert it.
- Does **not** construct or evaluate the full untruncated Wilson
  tensor-transfer operator at `beta = 6`, nor any Perron state, nor any
  `rho_(p,q)(6)` coefficients.
- Does **not** consume any PDG observed value, literature numerical
  comparator, fitted selector, or admitted unit convention.
- Does **not** claim that the explicit `SU(3)` Wilson character
  coefficients `c_lambda(6)` and Pieri multiplicities `(N_f, N_fbar)`
  satisfy the abstract hypotheses; the runner verifies this for the
  reference instance, but the narrow theorem itself only claims the
  finite-linear-algebra implication.

## Default reference instance (runner-verified)

The runner instantiates the abstract narrow theorem at the default
reference choice
```text
N            =  4,
c_(p,q)     :=  c_(p,q)(beta = 6) / (d_(p,q) * c_(0,0)(beta = 6)),
N_f         :=  exact SU(3) Pieri multiplicity matrix for fundamental
                fusion on B_4,
N_fbar      :=  exact SU(3) Pieri multiplicity matrix for antifundamental
                fusion on B_4,
```
where `c_(p,q)(beta)` is the Wilson character coefficient evaluated by
the Schur-Weyl Bessel-determinant sum at `MODE_MAX = 80` (matching the
parent runner). The runner verifies that this instance satisfies the
abstract hypotheses:
1. positivity of `c_(p,q)` on `B_4` (12-digit threshold);
2. conjugation symmetry `c_(p,q) = c_(q,p)` (12-digit threshold);
3. `N_f, N_fbar` entries in `{0, 1}`;
4. `S N_f S = N_fbar` and `S N_fbar S = N_f` exactly (integer equality);
5. `(T1)`, `(T2)`, `(T3)` hold for the assembled `T` (numerical
   thresholds for entry-wise nonnegativity, swap commutation, and
   trivial-channel positivity).

This instance is one finite tensor-word at the chosen `N = 4` truncation
and `MODE_MAX = 80` Bessel mode-sum truncation. It does not extend to
arbitrary `L_perp`, arbitrary `N`, or the full untruncated `beta = 6`
construction; those remain the open positive-theorem target of the parent
bounded-theorem note.

[`scripts/frontier_gauge_vacuum_plaquette_spatial_environment_tensor_transfer_one_word_packet_narrow.py`](./../scripts/frontier_gauge_vacuum_plaquette_spatial_environment_tensor_transfer_one_word_packet_narrow.py)
