## Step 1: scalar and projector observables are different grammars

The scalar observable principle starts from multiplicative partition data and
forces the additive scalar generator

`W = log |Z|`.

On the current boundary route this gives

`p_obs(L_Sigma) = p_vac(L_Sigma) = (1/(2n)) log det(L_Sigma)`.

That is exact, but it is a **scalar free-energy grammar**.

A projector-valued observable is different in kind: it asks a yes/no question
about a subspace of states. On a finite Hilbert carrier, such questions are
represented by orthogonal projectors.

So there are two distinct exact observable grammars on the branch:

- scalar additive observables from partition factorization;
- event/projector observables from the finite Hilbert event algebra.

The present lane can only close if the physical boundary readout belongs to the
second grammar rather than the first.

## Theorem 2: minimal projector-valued event observables on the cell are trace readouts

Fix the diagonal one-cell event algebra generated by the mutually orthogonal
atomic projectors `P_eta`.

Let `mu` be a physical yes/no event readout on that commuting family satisfying:

1. **normalization**
   `mu(0) = 0`, `mu(I) = 1`;
2. **exclusive-event additivity**
   `mu(P + Q) = mu(P) + mu(Q)` whenever `PQ = 0`;
3. **state covariance**
   the readout depends only on the chosen state `rho`.

Then on this commuting projector algebra the canonical event law is

`mu_rho(P) = Tr(rho P)`.

### Proof

On the atomic partition of the identity,

`I = sum_eta P_eta`,

normalization and orthogonal additivity determine `mu` completely from the
atomic weights `mu(P_eta)`.

For any diagonal projector

`P_S = sum_(eta in S) P_eta`,

one must have

`mu(P_S) = sum_(eta in S) mu(P_eta)`.

But this is exactly the trace rule on the same commuting algebra:

`Tr(rho P_S) = sum_(eta in S) Tr(rho P_eta)`.

So the minimal normalized additive projector-valued event readout is the trace
law on this event algebra.

This is the finite-packet version of the Born/event rule needed here. No new
coefficient enters.
