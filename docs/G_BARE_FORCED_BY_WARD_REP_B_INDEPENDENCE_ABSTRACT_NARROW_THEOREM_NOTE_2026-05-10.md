## Statement

Let `(F, g, N)` be abstract symbolic variables with `g > 0`, `N > 0`,
and `F` arbitrary in any field of characteristic zero (in particular `R`).
Let `c0 ∈ R_{>=0}` be a fixed constant.

**Hypothesis.** Suppose the following two algebraic constraints hold:

```text
(W1)  F^2  =  c0,                                                         (1)
(W2)  F^2  =  g^2 / (2 N).                                                (2)
```

**Conclusion (T1) (forcing identity).** Equating (1) and (2),

```text
g^2  =  2 N c0.                                                           (3)
```

**Conclusion (T2) (positive-branch unique solution).** On the positive
branch `g > 0`, the unique solution is

```text
g  =  sqrt(2 N c0),                                                       (4)
```

provided `c0 >= 0`. If `c0 = 0`, then `g = 0`, contradicting `g > 0`; if
`c0 > 0`, the positive branch yields a unique positive `g`.

**Conclusion (T3) (canonical numerical instance).** At
`(N, c0) = (3, 1/6)`,

```text
g^2  =  2 · 3 · (1/6)  =  1,    g  =  1   (positive branch).              (5)
```

The uniform pair `(N, c0) = (1, 1)` instead gives `g^2 = 2`, hence
`g = sqrt(2) ≠ 1`; the conclusion `g = 1` is therefore a non-trivial
consequence of the specific pair `(N, c0) = (3, 1/6)`.

## Proof

`(T1)` Direct substitution. From (1), `F^2 = c0`. From (2),
`F^2 = g^2 / (2 N)`. Equating both expressions for `F^2`,

```text
c0  =  g^2 / (2 N).
```

Multiplying both sides by `2 N` (valid since `N > 0`),

```text
g^2  =  2 N c0.
```

`(T2)` From (3), if `c0 = 0` then `g^2 = 0`, hence `g = 0`, which
contradicts the standing assumption `g > 0`. If `c0 > 0`, the equation
`g^2 = 2 N c0` has two real solutions `g = ± sqrt(2 N c0)`. Restricting
to `g > 0` selects the unique positive root `g = sqrt(2 N c0)`.

`(T3)` Direct substitution into (3): at `(N, c0) = (3, 1/6)`,

```text
g^2  =  2 · 3 · (1/6)  =  6 / 6  =  1,
```

so `g = 1` on the positive branch. At `(N, c0) = (1, 1)`,

```text
g^2  =  2 · 1 · 1  =  2,
```

so `g = sqrt(2) ≠ 1`. ∎

## What this claims
