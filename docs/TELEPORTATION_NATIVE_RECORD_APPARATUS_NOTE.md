# Native Teleportation Record Apparatus Note

**Date:** 2026-04-26
**Status:** planning / first native apparatus-carrier candidate
**Runner:** `scripts/frontier_teleportation_native_record_apparatus.py`

## Scope

This artifact attacks the largest remaining gap in the retained teleportation
lane: the Bell record should not merely be supplied to a classical channel. It
should be generated by a native measurement apparatus and then carried as a
3D+1 physical record.

The result is still a bounded model, not nature-grade closure. It does not
derive detector hardware, thermodynamic irreversibility, environmental
decoherence, a relativistic field equation, or a full conservation-ledger
theorem.

The scope remains ordinary quantum state teleportation only. No matter, mass,
charge, energy, object, or faster-than-light transport is claimed.

## Construction

The runner joins three pieces into one apparatus/carrier chain:

```text
Alice A,R Bell stabilizer transducer
  -> local durable pointer codeword
  -> 3D+1 local record-field pulses
  -> Bob decode
  -> retained-axis Pauli correction
```

### Bell-Stabilizer Transducer

The apparatus couples Alice's two registers to the commuting Bell stabilizers:

```text
Z_A Z_R  -> x bit
X_A X_R  -> z bit
```

The model uses the same Bell projectors and correction convention as the
protocol lane:

```text
P_zx = 1/4 (I + (-1)^x Z_A Z_R) (I + (-1)^z X_A X_R)
U_zx = Z^z X^x
```

Unlike the earlier causal-channel runner, the channel is not handed `(z,x)`
as free input. The record carrier is emitted from a `BellPointerRecord` whose
source is the local `bell_stabilizer_transducer`.

### Durable Pointer Code

The local pointer is encoded as an eight-component classical codeword:

```text
z z z | x x x | p p,   p = z xor x
```

The four Bell outcomes have minimum Hamming distance `5`. Nearest-codeword
decoding therefore corrects every one- and two-component bit flip in this
bounded model.

This is the first nontrivial record apparatus element: the Bell record is not
just a label. It is a redundant, locally copy-stable classical object.

### 3D+1 Record-Field Carrier

Each codeword component is represented as a local record-field pulse. The
default geometry matches the current 3D+1 causal lane:

```text
Alice site/tick = (1,1,1), t=4
Bob site = (5,3,2)
Manhattan distance = 7
earliest delivery tick = 11
```

The carrier moves by one 3D lattice step per tick. Bob cannot decode the
record before the pulse section reaches his site.

## What This Adds

This is the first artifact in the lane where the Bell record is generated,
encoded, carried, decoded, and used in one end-to-end model.

The carrier is still classical, but it is no longer an unexplained argument to
the channel. It is emitted from the Bell pointer and represented by local
record-field pulses with a finite propagation path.

The pointer code also gives a concrete durability target. Future native
detector models can now be judged against an explicit requirement:

```text
measurement must produce a redundant, locally stable Bell-record codeword
whose decoded payload is input-independent before delivery and branch-correct
after delivery.
```

## Nature-Grade Blockers

This artifact narrows the record gap but does not close it.

- The Bell-stabilizer transducer remains ideal/projective.
- The codeword is a classical model of durable memory, not a thermodynamic
  derivation of irreversibility.
- The record-field pulses are local on a 3D+1 lattice but are not derived from
  a retained field equation.
- The apparatus has no physical noise, leakage, threshold, bath, or detector
  calibration model.
- The conservation ledger is respected by construction, but not yet derived
  for a physical apparatus.
- The Bell resource and retained-axis readout/correction are still supplied by
  earlier ideal lane components.
