`D = D_1 ⊕ D_2`,

`Z[J_1 ⊕ J_2] = Z_1[J_1] Z_2[J_2]`.

`W[J_1 ⊕ J_2] = W[J_1] + W[J_2]`.

`W(r_1 r_2) = W(r_1) + W(r_2)`, with `r_i = |Z_i| > 0`.

Under the minimal regularity condition of continuity, the unique solution is
