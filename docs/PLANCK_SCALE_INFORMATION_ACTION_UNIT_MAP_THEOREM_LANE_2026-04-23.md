# Planck-Scale Information/Action Unit-Map Theorem Lane

**Date:** 2026-04-23  
**Status:** science-only lane note; sharpened obstruction with one surviving
future theorem target  
**Audit runner:** `scripts/frontier_planck_information_action_unit_map_theorem_lane.py`

## Question

Can the surviving Planck route come from an exact theorem that turns the
irreducible information/action object into the missing absolute unit map?

Equivalently:

- maybe one elementary information quantum is already the same thing as the
  elementary action phase `q_* = S_* / hbar`;
- maybe the exact finite partition family already carries the needed
  coefficient through `log Z`;
- maybe the current information/action surface can already break the gravity
  scale ray without importing a new physical constant.

## Bottom line

Not on the current admitted information surfaces.

The lane is now sharper than the earlier "open/speculative" status:

1. **direct information-as-action identifications are not admissible**
   because count/Shannon/von Neumann information is only canonical up to the
   positive logarithm-base convention;
2. **raw partition-log identifications are not admissible** because the
   finite partition theorem gives a chart density / measure class, not a
   chart-independent scalar `log Z`;
3. therefore the surviving information route is no longer vague. It must be a
   **new exact phase-per-information theorem**

   `q_* = kappa_info I_*`

   on the same elementary plaquette/hinge carrier, where

   - `I_*` is a canonical elementary information quantum,
   - `kappa_info` is a theorem-fixed base-compensating conversion constant,
   - and the pair closes the Planck coefficient relation

     `a^2 / l_P^2 = 8 pi kappa_info I_* / eps_*`.

The current package does **not** yet provide either that `kappa_info` theorem
or a canonical nonzero elementary `I_*` on the geometric carrier.

So the honest verdict is now:

- the information lane is **not** a current closure theorem;
- the obvious direct versions are ruled out cleanly;
- the only surviving information/action target is a new exact
  phase-per-information theorem.

## Three candidate subclasses

On the current information/action surface, the obvious unit-map candidates
split into three classes.

### Class I: direct count / entropy identification

Take the elementary phase directly from a finite information scalar:

- `q_* = log N_*`,
- `q_* = -sum p_i log p_i`,
- `q_* = S_vN(rho_*)`,
- or any equivalent count/Shannon/von Neumann scalar on the elementary
  carrier.

This is the most tempting route, because the information object is already
dimensionless and additive.

### Class II: raw partition-log identification

Take the elementary phase from the exact finite partition family:

- `q_* = log Z_*`,
- or an equivalent bare local partition/log-density object.

This is tempting because the direct-universal route already gives an exact
finite partition family and its stationary field reproduces the same discrete
gravity family.

### Class III: converted information theorem

Take the phase from a new exact conversion law

`q_* = kappa_info I_*`

with theorem-fixed `kappa_info` and canonical elementary `I_*`.

This is the only class that survives the obstructions below.

## Obstruction 1: count/Shannon/von Neumann information is log-base dependent

Finite information scalars are defined only up to a positive overall factor
coming from the logarithm base:

`I_base-b = I_nat / log b`.

So the same underlying finite alternative can be reported in nats, bits, or
any other logarithm-base convention.

That means a bare direct identification

`q_* = I_*`

cannot be a physical theorem. The physical phase `q_* = S_* / hbar` cannot
depend on an arbitrary reporting convention.

The only admissible form is a compensated conversion law

`q_* = kappa_info I_*`,

where `kappa_info` transforms inversely with the log-base convention.

So the current information object does **not** by itself fix the action phase.
It needs a new conversion constant theorem.

## Obstruction 2: the direct count/entropy subclass cannot land exact conventional Planck

Even if we momentarily ignore the base-choice objection and test the direct
subclass honestly, it still does not land exact conventional `a = l_P`.

For any nontrivial finite elementary information object, the smallest positive
count scalar is

`I_* >= log 2`.

The elementary action-phase reduction theorem gives

`a^2 / l_P^2 = 8 pi q_* / eps_*`.

If the direct subclass sets `q_* = I_*`, then for any geometric defect with

`0 < eps_* <= 2 pi`

we get the exact lower bound

`a^2 / l_P^2 >= 8 pi log 2 / (2 pi) = 4 log 2 ~= 2.77259 > 1`.

So even the most optimistic direct information-as-action subclass forces a
Planck-order or rationalized scale, not exact conventional unreduced
`a = l_P`.

On the natural minimal cubical defect `eps_* = pi/2`, the same direct
subclass gives

`a^2 / l_P^2 >= 16 log 2 ~= 11.09035`,

which is even farther from `1`.

This does **not** kill the whole information lane, because a converted law
`q_* = kappa_info I_*` could still work. But it does kill the most naive
direct information identification.

## Obstruction 3: raw `log Z` is a chart-density object, not a canonical phase quantum

The exact UV-finite partition theorem is strong, but it is explicit that the
raw partition value transforms under chart change as

`Z'_GR = |det T| Z_GR`.

Therefore

`log Z'_GR = log Z_GR + log |det T|`.

So the raw partition log is not a chart-independent scalar. The globally
meaningful object is the partition density / measure class, not the bare
number `log Z`.

That means the direct partition subclass

`q_* = log Z_*`

cannot be a canonical physical phase law on the current surface.

To feed the elementary action-phase reduction, the information lane would need
a chart-independent local information quantum, not the raw chart-dependent
partition value.

## The surviving theorem target

After the three obstructions above, the information lane reduces to one
sharper target:

> derive a canonical elementary information quantum `I_*` on the same
> plaquette/hinge carrier and derive an exact base-compensating conversion
> constant `kappa_info` such that
>
> `q_* = kappa_info I_*`
>
> is a chart-independent, convention-independent physical phase law.

Only then can the elementary action-phase reduction close:

`a^2 / l_P^2 = 8 pi kappa_info I_* / eps_*`.

This is a real reduction in the search space. It says the information lane no
longer needs "some way to connect information to Planck." It needs one very
specific theorem: a canonical phase-per-information law on the elementary
geometric carrier.

## The theorem-level statement

**Theorem (information/action unit-map obstruction on current carrier
classes).**
On the current admitted information/action surface, the obvious unit-map
subclasses are:

1. direct count/Shannon/von Neumann identifications `q_* = I_*`,
2. raw partition-log identifications `q_* = log Z_*`,
3. converted laws `q_* = kappa_info I_*`.

Subclass (1) is not an admissible physical theorem because finite information
is defined only up to positive logarithm-base rescaling; moreover, even if one
ignores that objection, it forces the exact lower bound

`a^2 / l_P^2 >= 4 log 2 > 1`,

so it cannot land exact conventional `a = l_P`.

Subclass (2) is not admissible because the exact finite partition family is a
chart density, with raw `log Z` shifting by `log |det T|` under chart change.

Therefore no currently realized direct information carrier furnishes the
missing unit map. The only surviving information/action route is subclass (3):
an exact theorem fixing both a canonical elementary information quantum `I_*`
and a base-compensating conversion constant `kappa_info` on the same
elementary geometric carrier.

## What this closes

This closes two loopholes cleanly:

1. maybe the Planck unit map is already hidden in a direct entropy/count
   scalar;
2. maybe the exact finite partition theorem already provides the physical
   action quantum through raw `log Z`.

Answer: no, not on the current admitted information surfaces.

## What this does not close

This lane note does **not** prove:

- that the information route is dead;
- that no exact `kappa_info` theorem exists;
- that the only surviving global route is boundary-density rather than
  information/action;
- that a future exact phase-per-information theorem could not still close
  Planck.

It proves only the sharper obstruction:

- the obvious direct information subclasses fail;
- the partition-log subclass is noncanonical on the current chart-density
  surface;
- the only serious information survivor is a new exact
  phase-per-information theorem.

## Safe wording
