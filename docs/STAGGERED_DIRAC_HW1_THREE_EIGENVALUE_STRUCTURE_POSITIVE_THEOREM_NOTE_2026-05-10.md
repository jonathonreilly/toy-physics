## Statement

Let `C` be the standard cyclic permutation matrix

```text
C = [[0, 1, 0],
     [0, 0, 1],
     [1, 0, 0]]
```

with `C^3 = I`. Let `omega = exp(2 pi i / 3)`. On the cited `H_hw=1`
carrier, assume the live operator being tested is `C_3`-equivariant, so it
commutes with `C`.

Then every Hermitian operator in this commuting class is a circulant

```text
H = a I + b C + bbar C^2,       a in R, b in C.
```

The Fourier vectors diagonalize `C`, so the eigenvalues of `H` are

```text
lambda_k = a + b omega^k + bbar omega^(-k),        k = 0, 1, 2.
```

Writing `b = |b| exp(i theta)` gives

```text
lambda_k = a + 2 |b| cos(theta + 2 pi k / 3).
```

For `b != 0` away from the finite coincidence set where two of these three
cosines are equal, the three eigenvalues are pairwise distinct real values.

## Proof

The matrix `C` is diagonalized by the Fourier basis
`v_k = (1, omega^k, omega^(2k))^T`, with eigenvalue `omega^k`. Since
`C^2` has eigenvalue `omega^(-k)` on the same vector, any matrix
`H = a I + b C + bbar C^2` has eigenvalue
`a + b omega^k + bbar omega^(-k)` on `v_k`.

Conversely, a Hermitian matrix commuting with the regular `Z/3Z`
permutation representation is in the commutant of that representation.
For the cyclic regular representation, the commutant is the polynomial
algebra in `C`; Hermiticity pairs the `C` and `C^2` coefficients as
complex conjugates and forces the identity coefficient to be real. Hence
the commuting Hermitian class is exactly `a I + b C + bbar C^2`.

The cosine form follows by substituting `b = |b| exp(i theta)` and using
`exp(i x) + exp(-i x) = 2 cos(x)`. Coincidences require two of
`cos(theta)`, `cos(theta + 2 pi/3)`, and `cos(theta + 4 pi/3)` to be equal;
outside that finite coincidence set and `b = 0`, the three eigenvalues are
pairwise distinct.
