## Statement

Let `C` be the standard `3 × 3` cyclic permutation matrix

```text
C = [[0, 1, 0],
     [0, 0, 1],
     [1, 0, 0]],
```

so that `C³ = I`, and let `ω = e^{2πi/3}`.

**Theorem (H_{hw=1} three-eigenvalue structure).** On premises
A1 + A2 + R1 + R2 (defined in §"Premises" below), the Hermitian
operator class on `H_{hw=1} ≅ ℂ³` reduces by Schur's lemma applied
to the `Z/3ℤ` regular representation generated by `C` to the
circulant Hermitian family

```text
H  =  a I  +  b C  +  b̄ C²,     (a, b) ∈ ℝ × ℂ.
```

The eigenvalue triple of `H` is

```text
λ_k  =  a + b ω^k + b̄ ω^{-k}
     =  a + 2 |b| cos(arg(b) + 2πk/3),     k = 0, 1, 2.
```

For every `b ≠ 0` with `arg(b) avoiding the discrete set where two of
the three cosines coincide` (mathematically, `arg(b) mod 2π/3 ∉ {0, π/3}`),
the three eigenvalues `(λ_0, λ_1, λ_2)` are **three distinct real
numbers**.

**Corollary (mass-ordering convention).** Once the three distinct
eigenvalues are ordered (e.g. `λ_min ≤ λ_mid ≤ λ_max`), assigning the
species labels `{e, μ, τ}` to the ordered triple is a standard
mass-ordering convention, identical in nature to the SM conventions
naming `(u, c, t)` quarks by charge and mass-ordering,
`(ν_1, ν_2, ν_3)` neutrino mass eigenstates by mass-ordering, and
`(K_S, K_L)` neutral kaons by lifetime ordering. The convention
consumes no retained-grade content and is not a derivation input.

## Proof

### Step 1: BZ-corner forcing of `M_3(ℂ)` on `H_{hw=1}` (premise R1)

Per
[`STAGGERED_DIRAC_BZ_CORNER_FORCING_THEOREM_NOTE_2026-05-07.md`](STAGGERED_DIRAC_BZ_CORNER_FORCING_THEOREM_NOTE_2026-05-07.md),
the Kawamoto-Smit staggered-Dirac kinetic operator on `Z³` APBC
diagonalizes at the BZ corners `k_μ ∈ {0, π}`, which distribute by
Hamming weight as `1 + 3 + 3 + 1 = 8`. The `hw=1` triplet
`{(1,0,0), (0,1,0), (0,0,1)}` carries the exact irreducible `M_3(ℂ)`
algebra generated by the three lattice translations together with
the `C_3[111]` cyclic generator that permutes the three corners
`(1,0,0) → (0,1,0) → (0,0,1) → (1,0,0)`. In the corner basis
`{|c_1⟩, |c_2⟩, |c_3⟩}`, the `C_3[111]` generator is the standard
`3 × 3` cyclic permutation matrix `C` of the Statement.

### Step 2: Schur reduction on the `Z/3ℤ` regular representation (premise R2)

The cyclic generator `C` acts on `H_{hw=1} ≅ ℂ³` as the regular
representation of `Z/3ℤ`: in the corner basis it is the standard
3-cycle permutation matrix with spectrum `{1, ω, ω²}`. The
`Z/3ℤ`-isotype decomposition has one one-dimensional component per
character, hence three one-dimensional irreducible components in
total. Any Hermitian operator `H` commuting with `C` is by Schur's
lemma a polynomial in `C`. The standard cyclic basis identity in the
corner basis (where `C` has zero diagonal) gives the explicit
circulant form

```text
H  =  a I  +  b C  +  b̄ C²,
```

with `a ∈ ℝ` (Hermiticity on the identity component) and `b ∈ ℂ`
(conjugate pairing on the two non-trivial characters). This step is
recorded explicitly in
[`KOIDE_CIRCULANT_CHARACTER_DERIVATION_NOTE_2026-04-18.md`](KOIDE_CIRCULANT_CHARACTER_DERIVATION_NOTE_2026-04-18.md)
§R1.

### Step 3: Eigenvalue formula

The Fourier basis `v_k = (1, ω^k, ω^{2k})^T / √3` for `k = 0, 1, 2`
diagonalizes both `C` (eigenvalue `ω^k`) and `C²` (eigenvalue
`ω^{-k} = ω^{2k}`), hence diagonalizes `H` with eigenvalue

```text
λ_k  =  a + b ω^k + b̄ ω^{-k}.
```

Writing `b = |b| e^{i arg(b)}` and using `e^{iθ} + e^{-iθ} = 2 cos θ`,

```text
b ω^k + b̄ ω^{-k}
   =  |b| (e^{i (arg(b) + 2πk/3)} + e^{-i (arg(b) + 2πk/3)})
   =  2 |b| cos(arg(b) + 2πk/3).
```

Hence

```text
λ_k  =  a + 2 |b| cos(arg(b) + 2πk/3).
```

This is recorded as R2 in
[`KOIDE_CIRCULANT_CHARACTER_DERIVATION_NOTE_2026-04-18.md`](KOIDE_CIRCULANT_CHARACTER_DERIVATION_NOTE_2026-04-18.md).

### Step 4: Generic three-distinctness

For two eigenvalues to coincide,

```text
λ_j  =  λ_k     ⟺     cos(arg(b) + 2πj/3)  =  cos(arg(b) + 2πk/3),
```

which for `j ≠ k` mod 3 requires the two cosine arguments to be
equal up to sign mod `2π`. This collapses to the discrete coincidence
set

```text
arg(b) mod (2π/3)  ∈  { 0,  π/3 }
```

(and `b = 0`, in which case the spectrum collapses entirely to `a`
with multiplicity 3). Outside this measure-zero set in the
`(|b|, arg(b))`-plane, the three cosine values
`cos(arg(b) + 2πk/3)`, `k = 0, 1, 2`, are pairwise distinct, and the
three eigenvalues `(λ_0, λ_1, λ_2)` are three distinct real numbers.

### Step 5: Mass-ordering convention assigns species labels

Once the three eigenvalues `(λ_0, λ_1, λ_2)` are three distinct real
numbers, ordering them by magnitude and assigning species labels
`{e, μ, τ}` to the smallest, middle, and largest eigenvalue is a
standard mass-ordering convention. This is precisely how the SM
fixes species names:

- `(u, c, t)` up-type quarks: named by electric-charge sign +
  mass-ordering, not by deriving "up-quark" from the SM Lagrangian
  alone.
- `(ν_1, ν_2, ν_3)` neutrino mass eigenstates: named by mass-ordering
  (or atmospheric-mixing convention in the inverted-ordering case).
- `(K_S, K_L)` neutral kaon mass eigenstates: named by lifetime
  ordering.

In each of these SM conventions, the underlying group/algebra
structure determines the multiplicity and irreducible character of
the eigenvalue tuple; the species names are conventional labels
attached after the derivation. The same applies here: the framework
derives the three-fold eigenvalue structure of `H_{hw=1}` (this
note's positive theorem); species names are conventional labels on
top.

QED. ∎
