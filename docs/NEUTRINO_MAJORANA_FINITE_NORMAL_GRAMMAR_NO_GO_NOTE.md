`c -> e^(i theta) c`, `c^dag -> e^(-i theta) c^dag`.

finite grammar.

## Exact selection rule

Let `O_q` be any operator with definite fermion-number charge `q`, meaning

`U(theta) O_q U(theta)^dag = e^(i q theta) O_q`

under the exact global `U(1)` generated by the total fermion number `N`.

If the finite state `rho` is `U(1)`-invariant, then

`<O_q> = Tr(rho O_q) = Tr(U rho U^dag O_q) = Tr(rho U^dag O_q U) = e^(i q theta) <O_q>`

for every `theta`.

For `q != 0`, the only solution is

`<O_q> = 0`.

So on the current finite retained grammar:

- charge-zero observables can be nontrivial
- charge-`+-2` Majorana bilinears vanish exactly
