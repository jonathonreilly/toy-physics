`rho_cell = I_16 / 16`.

It does not claim every prepared or reduced physical state is tracial. It claims
only that the canonical state of the bare no-extra-structure primitive cell is

## Theorem 1: full finite-cell automorphism naturality fixes the trace

Let `omega` be a state on `M_16(C)`. Since every finite-dimensional state is
represented by a density matrix `rho`,

`omega(X) = Tr(rho X)`.

Assume `omega` is natural under all automorphisms of the bare matrix cell:

`omega(U X U^*) = omega(X)`

for every unitary `U`.

Then

`rho = I_16 / 16`.

### Proof

The invariance condition implies

`Tr(rho U X U^*) = Tr(rho X)`

for every `X` and `U`. By cyclicity of trace,

`Tr(U^* rho U X) = Tr(rho X)`.

Since this holds for every `X`,

`U^* rho U = rho`

for every unitary `U`. Thus `rho` lies in the commutant of the full matrix
algebra. The commutant is the scalar center, so

`rho = lambda I_16`.

Normalization gives

`1 = Tr(rho) = 16 lambda`,

hence

`lambda = 1/16`.

Therefore

`rho = I_16 / 16`.

## Theorem 2: event-frame automorphism naturality fixes the same state

Let the retained primitive event-frame state be

`omega(P_eta) = p_eta`.

Assume the bare event frame has no extra labels beyond eventhood itself. Then
the automorphism group of the bare sixteen-atom event frame is transitive, so
for any `eta, xi` there is an event relabeling taking `P_eta` to `P_xi`.

Naturality gives

`p_eta = p_xi`

for all atoms. Let the common value be `p`. Normalization gives

`16 p = 1`,

so

`p = 1/16`.

Thus the event-frame state is again

`rho_cell = I_16 / 16`.
