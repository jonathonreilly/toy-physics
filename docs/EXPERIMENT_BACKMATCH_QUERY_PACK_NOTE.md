# Experiment Backmatch Query Pack Note

**Date:** 2026-04-06  
**Status:** repo-local prep note for later literature backmatching

## Purpose

This note is a compact shortlist of the retained observables most worth
searching against the literature later. It is not a literature review and it
does not claim prior experimental validation. The goal is to make a later
search disciplined: one observable, one match criterion, one or two strong
query strings.

## Top candidate observables

### 1. Causal propagating field ratio

**Observable**

- forward-only ratio and finite-cone ratio for the causal propagating-field
  observable
- especially the stable `c = 0.5` dynamic ratio versus the instantaneous /
  forward-only baselines

**What would count as a matching published outcome**

- a published result where a finite-propagation or retarded-field model
  produces a distinct observable relative to an instantaneous baseline
- ideally with a dynamic/static comparison, a causal-cone or retarded-potential
  interpretation, and a stable sign or ratio under parameter sweeps

**Search strings**

- `"retarded potential" discrete lattice observable dynamic static comparison`
- `"causal cone" field propagation observable finite speed ratio`
- `"wave propagation on graphs" finite speed response quadrature`

### 2. Impact-parameter deflection law

**Observable**

- deflection versus impact parameter on a portable grown row
- near-Newtonian tail exponent around `-1`

**What would count as a matching published outcome**

- a published weak-field or scattering result with a stable deflection law as a
  function of impact parameter
- ideally something that reports an exponent, a `1/b`-like tail, or a
  controlled finite-resolution/scattering falloff on a discrete substrate

**Search strings**

- `"impact parameter" deflection law 1/b discrete lattice`
- `"weak-field" scattering "impact parameter" exponent`
- `"gravitational lensing" computational model impact parameter scaling`

### 3. Moving-source directional bias

**Observable**

- odd-in-velocity centroid response for a moving-source proxy
- exact zero baseline, static control, and sign flip with `v`

**What would count as a matching published outcome**

- a published moving-source or retarded-source result with a signed directional
  observable that flips under source velocity reversal
- ideally with a static baseline, phase lag, or centroid shift readout

**Search strings**

- `"moving source" centroid shift sign flip retarded response`
- `"retarded source" directional bias phase lag`
- `"time-dependent source" lock-in centroid shift`

### 4. Quadrupole-like tidal / width response

**Observable**

- width change with near-zero centroid shift for a centered quadrupole-like
  source
- tidal/shape-sensitive response beyond the monopole or dipole card

**What would count as a matching published outcome**

- a published multipole, tidal, or shape-response result where the centroid
  stays pinned but a width/strain/second-moment observable grows
- ideally something quadrupole-like rather than pure dipole steering

**Search strings**

- `"quadrupole" tidal response width observable discrete model`
- `"multipole" shape-sensitive response centroid pinned`
- `"tidal" second-moment response lattice`

### 5. Diamond / NV phase-ramp discriminator

**Observable**

- lock-in quadrature `Y`, phase lag `phi`, and spatial phase ramp across an
  NV image
- the current in-repo claim is a phase-sensitive discriminator, not an
  absolute gravity amplitude

**What would count as a matching published outcome**

- a published diamond/NV or related precision-sensing result showing a stable
  quadrature channel, phase ramp, or spatially resolved lock-in phase
- ideally under a driven source, strain source, or other time-dependent probe

**Search strings**

- `"diamond NV" lock-in quadrature phase ramp spatial`
- `"NV center" phase-sensitive readout "quadrature"`
- `"widefield NV" phase gradient time-dependent source`

## Recommended search order

If we do a later literature pass, start here:

1. causal propagating field ratio
2. impact-parameter deflection law
3. moving-source directional bias
4. quadrupole-like tidal / width response
5. diamond / NV phase-ramp discriminator

That order matches the current retained hierarchy:

- causal propagation is the newest causal-field branch
- impact parameter is the strongest portable weak-field law
- moving-source and quadrupole/tidal are bounded but real extensions
- diamond/NV is the best external discriminator, not the core theory lane

## What this note is not

- It is not a claim that any of these observables already have a known
  literature match.
- It is not a substitute for a real search.
- It is not a request to broaden the physics story before the search.
