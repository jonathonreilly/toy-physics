## Scope

Records the lattice analogue of the Coleman–Mermin–Wagner theorem on
the Cl(3) ⊗ Z^d substrate. The result structurally confirms the
`d_s = 3` choice of `A_min`: `d = 3` is the *minimal* spatial
dimension that simultaneously admits a long-range `1/r^2` force law
(Newtonian gravity) AND spontaneous breaking of continuous global
symmetries.

In `docs/AXIOM_REDUCTION_NOTE.md` the dimensional choice `C1: d = 3`
is recorded as a *choice*, not a derived axiom. The Mermin–Wagner
analogue derived here narrows the choice further: any framework that
needs spontaneous symmetry breaking of a continuous global symmetry
*must* have `d ≥ 3`. Hence `d = 3` is the unique discrete choice
that is *both* large enough to admit a long-range force law AND
allow continuous SSB.

- **A3 — Hermitian Hamiltonian on Cl(3) ⊗ Z^d.** Used via the

## Statement

Let `H` be a translation-invariant Hermitian Hamiltonian on Cl(3) ⊗
Z^d with a continuous global symmetry generated by `Q = Σ_x q_x`,
acting locally with `[H, Q] = 0`. Then on `A_min`:

**(MW1) d ≤ 2 — no SSB at finite T.** For any finite temperature
`T > 0` (`β < ∞`), no continuous global symmetry can be spontaneously
broken: the order parameter `⟨q_x⟩_β = ⟨q_0⟩_β` (translation
invariance of the thermal state) vanishes.

**(MW2) d ≥ 3 — SSB is possible.** For `d ≥ 3` and certain
Hamiltonians, SSB is allowed: there exist symmetry-broken Gibbs
states with non-zero order parameter.

**(MW3) Critical IR integral.** The structural quantity controlling
both regimes is the lattice IR-integral

```text
    I_d   =   (1 / V)  Σ_{k ≠ 0}  1 / E_k                            (1)
```

where `V = L^d` and `E_k = 2 Σ_μ (1 - cos k_μ)` is the lattice
Goldstone-mode dispersion. In the thermodynamic limit `L → ∞`:

```text
    I_d   ~   ∫_{|k| > 1/L}  d^d k / k²                               (2)

    d = 1:   I_1 → ∞  (linearly with L)
    d = 2:   I_2 → ∞  (logarithmically with L)
    d = 3:   I_3 → finite
    d ≥ 4:   I_d → finite
```

**(MW4) d_s = 3 is the unique minimal dim with both long-range
force AND continuous SSB.** From `docs/AXIOM_REDUCTION_NOTE.md` step
D9, `d ≥ 2` is required for a non-divergent kernel; `d = 2` gives
only logarithmic potential (marginal). From (MW1)/(MW2), `d ≥ 3` is
required for continuous SSB. The intersection of the two
requirements is `d ≥ 3`. The minimal integer is `d = 3`, which is
exactly `A_min`'s `Z^3`.

## Proof

### Step 1 — Bogoliubov inequality

For any operators `A, B` and thermal state at inverse temperature
`β`, the Bogoliubov inequality reads

```text
    | ⟨ [A, B] ⟩_β |²   ≤   (β / 2)  · ⟨ {A, A†} ⟩_β  · ⟨ [B, [H, B]†] ⟩_β.   (3)
```

This is a finite-dim algebraic identity following from Hilbert–
Schmidt positivity; it does not import any continuum machinery.

### Step 2 — Choose `A` and `B` adapted to the Goldstone mode

Take `B = q_k := Σ_x e^{ikx} q_x` (the Fourier transform of the
charge density at lattice momentum `k ≠ 0`). For a translation-
invariant Hamiltonian with kinetic term `Σ_x (...)_x`,

```text
    ⟨ [B, [H, B]†] ⟩_β   ~   E_k · χ_q                               (4)
```

with `E_k` the Goldstone-mode dispersion (`~ k²` at small `k`) and
`χ_q = ⟨q² ⟩_β` the on-site susceptibility.

Take `A = m_k := Σ_x e^{ikx} m_x` for the order-parameter operator
`m_x` whose ground-state expectation `⟨m_0⟩` would signal SSB. By
the symmetry generated by `Q`, the commutator `[A, B] = -i M_k`
gives the order parameter at momentum `k`. So
`⟨[A, B]⟩_β = -i ⟨M_k⟩_β` which is the magnetisation.

### Step 3 — Apply Bogoliubov + sum over `k`

Substituting into (3):

```text
    |⟨M_k⟩_β|²   ≤   (β / 2)  · χ_m  · E_k · χ_q                     (5)
```

Hence

```text
    Σ_{k ≠ 0}  |⟨M_k⟩_β|² / E_k   ≤   (β / 2) · χ_m · χ_q · (V - 1).  (6)
```

The LHS, when divided by `V`, equals `(1/V) Σ |⟨M_k⟩|² / E_k`,
which by Parseval lower-bounds the magnetisation density:

```text
    |⟨M_0⟩_β|² / V · I_d   ≤   const · χ_m · χ_q                     (7)
```

with `I_d` from (1).

### Step 4 — Take `V → ∞`

In the thermodynamic limit:

- if `I_d → ∞` (`d ≤ 2`), then to keep the LHS of (7) bounded we
  need `|⟨M_0⟩_β|² / V → 0`, hence the magnetisation density
  vanishes. *No SSB.* (MW1).
- if `I_d → finite` (`d ≥ 3`), the bound is non-trivial but does
  not force the magnetisation density to zero. SSB is *not
  forbidden* and can occur for some Hamiltonians. (MW2).

### Step 5 — Combination with the long-range-force condition

From `docs/AXIOM_REDUCTION_NOTE.md` D9: the lattice transfer-norm
stability requires kernel power `p = d - 1` and `d ≥ 2` for a
non-divergent kernel. `d = 1` gives no long-range force; `d = 2`
gives only logarithmic potential (marginal).

Combined with (MW1)/(MW2):

| d | long-range force? | continuous SSB possible? |
|---|---|---|
| 1 | no | no |
| 2 | only log; marginal | no |
| 3 | yes (`1/r²`) | yes |
| 4+ | yes (`1/r^{d-1}`) | yes |

Hence `d = 3` is the *minimal* integer satisfying both. This is
A_min's substrate. ∎
