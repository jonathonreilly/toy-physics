## No-Go Theorem Statement

Let

```text
H_cell = C^2_t otimes C^2_x otimes C^2_y otimes C^2_z ~= C^16.
```

Let the substrate action on the one-axis coframe

```text
W = span(t,x,y,z)
```

be generated by:

1. the spatial `Cl(3)` bivector spin lift acting on `span(x,y,z)`;
2. the distinguished time-axis parity on `t`;
3. the CPT grading action;
4. the retained complex Hilbert structure;
5. tensor-local number operators on the four `C^2` factors.

Then the Hamming-weight-one projector

```text
P_1 = P_A
```

is a rank-four local equivariant projector, but it is not unique. The
Hamming-weight-three projector

```text
P_3
```

is also rank four, local, complex-linear, and equivariant under the same
spin/time/CPT substrate action. More generally, the local spin/time
decomposition contains 17 rank-four equivariant projector classes.

Therefore the requested implication fails:

```text
substrate spin lift + time axis + CPT + Born complex structure + locality
  -/-> unique P_A.
```

## Five-Step Derivation

### 1. Substrate Symmetry Action

Model the event cell as the exterior/Fock expansion of the four local axis
slots:

```text
H_cell ~= Lambda^* W,        W = span(t,x,y,z).
```

The spatial bivectors act as the standard `so(3)` spin lift on
`span(x,y,z)` and trivially on the distinguished time axis `t`. The runner
uses the three real antisymmetric one-particle generators and their induced
exterior-power action on `H_cell`.

Time parity acts by `(-1)^{n_t}`. The structural CPT grading is modeled by
`(-1)^N`, the grading action compatible with the `CPT_EXACT_NOTE.md`
statement that the combined CP operator maps each Clifford generator to minus
itself. Both actions preserve tensor locality and Hamming weight.

### 2. Irreducible Local Decomposition

The event cell decomposes into the following local spin/time blocks:

| Block | Description | Rank |
|---|---:|---:|
| `E0` | scalar, time-even, weight 0 | 1 |
| `Et` | scalar, time-odd, weight 1 | 1 |
| `EV` | spatial vector, time-even, weight 1 | 3 |
| `EtV` | spatial vector, time-odd, weight 2 | 3 |
| `EVV` | spatial bivector/vector-dual, time-even, weight 2 | 3 |
| `EtVV` | time times spatial bivector/vector-dual, time-odd, weight 3 | 3 |
| `EVVV` | spatial volume scalar, time-even, weight 3 | 1 |
| `EtVVV` | four-volume scalar, time-odd, weight 4 | 1 |

Each block commutes with the spatial spin generators, time parity, and CPT
grading. Each is also a tensor-local projector: it is diagonal in the four
axis occupation basis and is generated by the local number operators.

### 3. P_A Exists

The Hamming-weight-one projector is

```text
P_A = P_1 = E_t + E_V.
```

It has rank

```text
rank(P_A) = 1 + 3 = 4.
```

It is local, complex-linear, CPT-compatible, time-parity compatible, and
equivariant under the spatial `Cl(3)` bivector spin lift.

### 4. Non-P_A Candidates Also Exist

The Hamming-weight-three projector is

```text
P_3 = E_{tVV} + E_{VVV}.
```

It also has rank

```text
rank(P_3) = 3 + 1 = 4.
```

It is the local polynomial

```text
P_3 = sum_{|S|=3} prod_{i in S} n_i prod_{j notin S} (1-n_j),
```

so it respects the same tensor-factor locality as `P_A`. It commutes with the
same spatial spin-lift generators, time parity, and CPT grading. It is
complex-linear on the retained Hilbert surface.

Thus `P_3` violates none of the stated constraints (a)-(e). It is not
`P_A`, so uniqueness fails.

### 5. Exhaustion Of Local Rank-Four Equivariant Classes

Restricting to local projector sums of the eight spin/time blocks, every
rank-four equivariant projector is either:

- one rank-three vector block plus one rank-one scalar block; or
- the sum of all four scalar blocks.

There are four rank-three vector blocks and four rank-one scalar blocks,
giving `4 x 4 = 16` vector-plus-scalar classes, plus one all-scalar class:

```text
17 local rank-four equivariant projector classes.
```

`P_A` is only one of these 17 classes. `P_3` is another. Therefore the
requested uniqueness theorem cannot be derived from the stated substrate
constraints.
