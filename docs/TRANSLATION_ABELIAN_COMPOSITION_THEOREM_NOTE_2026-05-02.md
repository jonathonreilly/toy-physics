## Statement

Let H_phys be the framework's lattice Hilbert space, with basis {|x⟩ : x ∈ Λ}
indexed by lattice sites (Λ = Z^3 or (Z/N)^3 for a periodic lattice of size
N). For each a ∈ Z^3, define the translation operator

```text
    T_a |x⟩  :=  |x + a⟩    (mod N if periodic)                            (1)
```

Then:

**(G1) Closure (composition law).** For all a, b ∈ Z^3:

```text
    T_a T_b  =  T_{a+b}                                                    (2)
```

**(G2) Commutativity (abelian).** For all a, b ∈ Z^3:

```text
    [T_a, T_b]  =  T_a T_b - T_b T_a  =  0                                 (3)
```

**(G3) Unitarity & inverse.** Each T_a is unitary, with

```text
    T_{-a}  =  T_a^{-1}  =  T_a^†                                          (4)
```

**(G4) Identity.** T_0 = I.

**(G5) Faithfulness.** The homomorphism T : Z^3 → U(H_phys), a ↦ T_a, has
trivial kernel: T_a = I if and only if a = 0 (or a ∈ N·Z^3 in the
periodic case).

## Proof

### Step 1 — Translation symmetry of action (cited from N1)

By the cited lattice Noether theorem (N1), the framework's action is
invariant under the spatial translation group Z^3 acting by T_a : x ↦ x + a.
The translations are operators on H_phys defined by their action on basis
states (1).

### Step 2 — Closure (G1)

Direct computation on basis states:

```text
    T_a T_b |x⟩  =  T_a |x + b⟩  =  |x + b + a⟩  =  |x + (a + b)⟩  =  T_{a+b} |x⟩.    (5)
```

Since this holds on all basis states, T_a T_b = T_{a+b} as operators on H_phys.

### Step 3 — Commutativity (G2)

Z^3 addition is commutative: a + b = b + a for all a, b ∈ Z^3. Therefore
by (G1):

```text
    T_a T_b  =  T_{a+b}  =  T_{b+a}  =  T_b T_a.                          (6)
```

Hence [T_a, T_b] = 0.

### Step 4 — Unitarity & inverse (G3)

Each T_a is a permutation matrix on the orthonormal basis {|x⟩}, mapping
basis to basis. Permutation matrices are orthogonal; for the complex case
they are unitary, with inverse equal to the adjoint (transpose of a real
permutation):

```text
    T_a^†  =  T_a^{-1}.                                                    (7)
```

The inverse permutation x ↦ x - a is implemented by T_{-a}, so
T_a^{-1} = T_{-a} = T_a^†.

### Step 5 — Identity (G4)

T_0 |x⟩ = |x + 0⟩ = |x⟩, so T_0 = I.

### Step 6 — Faithfulness (G5)

If T_a = I, then T_a |x⟩ = |x⟩ for all basis states, i.e. |x + a⟩ = |x⟩
for all x. Since basis states are linearly independent and labeled by
distinct lattice sites, |x + a⟩ = |x⟩ requires x + a = x, i.e. a = 0
(or a ∈ N·Z^3 in the periodic case where a is identified with a + N e_i
for unit vectors e_i). Therefore the kernel is trivial. ∎
