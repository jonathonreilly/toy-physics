## 2. Proof

### 2.1 Failure (G1): Gouezel-Karlsson requires ergodic measure-preserving base system

The Gouezel-Karlsson theorem (arXiv:1509.07733, Theorem 1.1; JEMS 22
(2020)) is stated for the following setup:

```text
(Omega, P) probability space;                                       (G1.1)
T : Omega -> Omega ergodic measure-preserving transformation;       (G1.2)
phi : Omega -> Semi-Contractions(X, d) cocycle generator;           (G1.3)
phi_n(omega) = phi(omega) o phi(T omega) o ... o phi(T^{n-1} omega);(G1.4)
E_P[ d(phi(omega) x_0, x_0) ] < infinity (integrability).           (G1.5)
```

Conclusion: for `P`-a.e. `omega`, there exists `A = A(omega) >= 0`
(by ergodicity, `A` is `P`-a.s. constant) and a horofunction
`h_omega` such that

```text
d(phi_n(omega) x_0, x_0) / n  -->  A         (n -> infinity);       (G1.6)
h_omega(phi_n(omega) x_0) / n -->  -A.                              (G1.7)
```

The framework's 16-step staircase is a single fixed deterministic
finite sequence `(T_k)_{k=0..15}`. To apply (G1.1)-(G1.7), one must
embed `(T_k)` into an ergodic measure-preserving system. The
canonical option is the `Z_{16}`-periodic embedding:

```text
Omega = {0, 1, ..., 15};                                            (G1.8)
P = uniform measure (P({k}) = 1/16);                                (G1.9)
T(k) = (k + 1) mod 16  (cyclic shift);                              (G1.10)
phi(k) = T_k (the k-th blocking operator).                          (G1.11)
```

Under (G1.8)-(G1.11), `T` is ergodic (the cyclic group `Z_{16}`
acts transitively on `Omega`), so the Gouezel-Karlsson hypotheses
(G1.1)-(G1.5) are satisfied (provided integrability (G1.5) holds,
which is automatic for bounded operators on a finite-dim metric
space `X`). The asymptotic Lyapunov is then

```text
A  =  lim_{n -> infinity} (1/n) log ||phi_n(omega) x_0||
   =  (1/16) sum_{k=0}^{15} log ||T_k||_op,                         (G1.12)
```

i.e., the time-average of single-step log-norms.

The right-hand side of (G1.12) is precisely the elementary
submultiplicativity content of the Round 2 bounded-theorem
[`BOUGEROL_LACROIX_STAGGERED_BLOCKING_SUBMULT_TAUTOLOGICAL_BOUND_BOUNDED_THEOREM_NOTE_2026-05-10.md`](BOUGEROL_LACROIX_STAGGERED_BLOCKING_SUBMULT_TAUTOLOGICAL_BOUND_BOUNDED_THEOREM_NOTE_2026-05-10.md).
If each `||T_k||_op = alpha_LM` (the Round 2 1D scalar toy), (G1.12)
gives `A = log(alpha_LM)` tautologically. If the `||T_k||_op` are
not all equal to `alpha_LM`, no MET conclusion forces `A =
log(alpha_LM)`.

The Gouezel-Karlsson asymptotic conclusion is therefore **not a
purely deterministic non-iid result for fixed finite N=16**; it is an
ergodic-time-average over a periodic embedding, which reduces to
elementary submultiplicativity. No new content beyond Round 2 is
gained by invoking Gouezel-Karlsson. QED for (G1).

### 2.2 Failure (G2): Filip and Karlsson-Ledrappier inherit the same ergodic-base requirement

Filip's "Notes on the Multiplicative Ergodic Theorem"
(arXiv:1710.10694) treats integrable cocycles over ergodic
measure-preserving transformations with values in semigroups of
nonexpanding maps. The setup is

```text
(Omega, P, T) ergodic measure-preserving probability system;        (G2.1)
phi : Omega -> Semi(X) into a semigroup of nonexpanding maps;       (G2.2)
integrability E_P[log+ ||phi(omega)|| ] < infinity.                 (G2.3)
```

When `Semi(X) = GL_N(R)`, the theorem reduces to classical Oseledets
MET, which is the iid setting of Bougerol-Lacroix when the cocycle
is generated by iid random variables. Filip's generalisation extends
the codomain to nonexpanding semigroups but **does not remove the
`(Omega, P, T)` ergodic base requirement**.

Karlsson-Ledrappier "Noncommutative ergodic theorems" (Geom. Funct.
Anal. 21 (2011) 947-993) is the precursor: integrable cocycles into
group of isometries of a Hadamard manifold or a metric space, with
the same `(Omega, P, T)` ergodic base.

In all three published theorems (Gouezel-Karlsson, Filip,
Karlsson-Ledrappier), the asymptotic conclusion reduces to a time-
or ensemble-average over the ergodic measure `P`. For a fixed finite
deterministic sequence `(T_k)_{k=0..15}` on a Banach space, the only
canonical embedding is the `Z_{16}`-periodic one of (G1.8)-(G1.11),
under which the conclusion is (G1.12), i.e., elementary time-average
submultiplicativity.

The published deterministic-cocycle MET literature does not provide
a purely deterministic non-iid result for fixed finite N=16 that
adds content beyond Round 2 elementary submultiplicativity. QED for
(G2).

### 2.3 Failure (G3): no substrate-canonical operator gives top eigenvalue alpha_LM

By the K3 obstruction recorded in
[`HIERARCHY_BBS_STAGGERED_TASTE_BLOCKING_BRIDGE_KMS_SUBSTRATE_NARROW_NO_GO_NOTE_2026-05-15.md`](HIERARCHY_BBS_STAGGERED_TASTE_BLOCKING_BRIDGE_KMS_SUBSTRATE_NARROW_NO_GO_NOTE_2026-05-15.md),
linearising any Wilsonian RG at a Gaussian fixed point in 4D with
scale rescaling factor `b > 1` gives canonical eigenvalues on
operators `O_n` of canonical dimension `Delta_n`:

```text
lambda_{O_n}  =  b^{d - Delta_n}            (tree level, d=4),       (G3.1)
```

with one-loop corrections

```text
lambda_marginal  =  1 + c g^2 ln(b) + O(g^4)  (Delta_n = 4),         (G3.2)
```

where `c = O(1)` is a group-theoretic loop coefficient (Wilson-Kogut
1974; Salmhofer 1999 §3.4). At framework `g_s^2 ~= 1.139` and `b =
1/alpha_LM ~= 11.03`, the marginal eigenvalue is `O(1)`, not the
small ratio `alpha_LM ~= 0.0907`.

Wilson-Fisher fixed-point eigenvalues in `4 - epsilon` dimensions are

```text
lambda_relevant   =  2 - O(epsilon),                                 (G3.3)
lambda_irrelevant = -2 + O(epsilon),                                 (G3.4)
gamma_marginal    =  O(epsilon^2).                                   (G3.5)
```

None of (G3.1)-(G3.5) equals `alpha_LM` for any canonical scale
rescaling factor `b > 1`. For the framework substrate to
special-case the top eigenvalue equal to `alpha_LM`, the substrate
would have to impose a non-canonical norm choice — putting the
bridge identification in by hand, not deriving it. No such canonical
substrate-induced norm exists in the declared framework bridge-input
chain.

The Round 2 bounded-theorem
[`BOUGEROL_LACROIX_STAGGERED_BLOCKING_SUBMULT_TAUTOLOGICAL_BOUND_BOUNDED_THEOREM_NOTE_2026-05-10.md`](BOUGEROL_LACROIX_STAGGERED_BLOCKING_SUBMULT_TAUTOLOGICAL_BOUND_BOUNDED_THEOREM_NOTE_2026-05-10.md)
admitted that the 1D scalar model `A_k(x) = alpha_LM x` is the
"admitted mathematical carrier" with `||A_k||_op = alpha_LM` by
construction. Round 3's non-tautology gate requires a Banach
operator constructed from substrate primitives whose top eigenvalue
is `alpha_LM` without that input. By (G3.1)-(G3.5), no canonical
construction supplies this. QED for (G3).

### 2.4 Failure (G4): finite N = 16 is not asymptotic

The Gouezel-Karlsson asymptotic conclusion (G1.6) is

```text
A  =  lim_{n -> infinity} (1/n) log ||phi_n(omega) x_0||.            (G4.1)
```

At fixed finite `N = 16`, the finite-`n` version is

```text
(1/N) log ||Pi_N||_op  =  A  +  error(N),                            (G4.2)
```

where `error(N) -> 0` as `N -> infinity` but is not vanishing at
finite `N`. The cited Gouezel-Karlsson / Filip / Karlsson-Ledrappier
theorems do not provide an explicit finite-`N` error bound; they are
limit theorems, not finite-`N` rate theorems.

For the bridge identification `||Pi_16||_op = alpha_LM^16` to follow
non-tautologically from `A = log(alpha_LM)`, one needs

```text
||Pi_16||_op  =  exp(16 A)  =  alpha_LM^16  exactly,                 (G4.3)
```

which requires `error(16) = 0` exactly. This is a finite-`N`
strengthening beyond Gouezel-Karlsson / Filip; it is supplied by:

- the tautological setting where `||T_k||_op = alpha_LM` exactly for
  all `k` (Round 2, by-construction); OR
- a Pesin-type uniform hyperbolicity hypothesis with exponential
  separation of Oseledets subspaces and explicit finite-`N` rate,
  which is published research (Bochi-Viana, Avila-Viana,
  Bonatti-Viana on continuity / non-uniform hyperbolicity); none of
  these is supplied by Gouezel-Karlsson / Filip directly.

The non-stationary Furstenberg theorem of Gorodetski-Kleptsyn
(arXiv:2210.03805) and the parametric Furstenberg of
arXiv:2007.06326 do treat non-iid extensions but with their own
hypothesis sets (parametric or non-stationary) and asymptotic
conclusions, not finite-`N` exact-equality statements.

Finite `N = 16` exact equality is therefore not a consequence of
the Gouezel-Karlsson / Filip / Karlsson-Ledrappier route without
additional unsupplied hypotheses. QED for (G4).

### 2.5 Joint sufficiency

Each of (G1)-(G4) is independently sufficient to block the bridge
identification at the published-theorem level:

- (G1) blocks because Gouezel-Karlsson Theorem 1.1 still requires an
  ergodic measure-preserving base system `(Omega, P, T)`; the only
  canonical embedding of the fixed finite N=16 staircase reduces to
  elementary time-average submultiplicativity (Round 2 tautological).
- (G2) blocks because Filip and Karlsson-Ledrappier inherit the same
  ergodic-base requirement; the asymptotic conclusions add no
  content beyond (G1).
- (G3) blocks because no canonical Wilsonian RG construction at any
  Gaussian fixed point gives top eigenvalue `alpha_LM`; the marginal
  eigenvalue is `O(1)`, not the small ratio.
- (G4) blocks because all cited MET-type theorems are asymptotic-in-`N`
  limit theorems; fixed finite N=16 exact equality requires
  unsupplied uniform-hyperbolicity or finite-`N` rate hypotheses.

The four obstructions are independent in the sense that one could
hypothetically be removed without removing the others (e.g., a
purely deterministic non-iid MET theorem with no ergodic base would
address (G1) but not (G3) or (G4); a Pesin-type finite-`N` rate
theorem at the framework substrate would address (G4) but not (G1),
(G2), or (G3); a substrate-canonical norm picking top eigenvalue
`alpha_LM` would address (G3) only). Their conjunction is
overdetermined: the bridge is blocked at all four independent
substrate levels.

The negative theorem is therefore: at the level of published
deterministic-cocycle and semigroup multiplicative-ergodic theorems
plus framework bridge inputs, the bridge identification
"Gouezel-Karlsson / Filip / Karlsson-Ledrappier deterministic-cocycle
MET = non-tautological derivation of `lambda_1 = log(alpha_LM)` for
the framework's 16-step staggered taste-blocking" cannot be verified.
QED.

  1.1 (arXiv:1509.07733) is stated for an ergodic measure-preserving

- (G4) is the categorical distinction between asymptotic-in-`N` limit
  theorems and finite-`N` exact-equality statements; no Pesin-type
  uniform-hyperbolicity rate theorem is published for the framework
  substrate within the Gouezel-Karlsson / Filip scope.

This is load-bearing step class B: literature-domain inspection plus
a categorical distinction plus inheritance from a sibling structural
obstruction, evaluated against declared framework bridge inputs. The
declared framework dependencies supply the bridge's `alpha_LM`, the
