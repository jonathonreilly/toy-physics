   `rho_sf(C^2) = I_2/2`;

It also now has a cleaner package-native bridge in
[PLANCK_SCALE_SOURCE_FREE_STATE_SAME_OBJECT_SEMANTICS_BRIDGE_2026-04-23.md](./PLANCK_SCALE_SOURCE_FREE_STATE_SAME_OBJECT_SEMANTICS_BRIDGE_2026-04-23.md):

It does **not** show that the following state-law principle is already
contained in the accepted minimal stack:

That is now the actual promoted law doing the state selection.
