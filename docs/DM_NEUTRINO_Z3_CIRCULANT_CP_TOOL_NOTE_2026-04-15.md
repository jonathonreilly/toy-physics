## Exact theorem

### 1. The exact `1+2` split lifts to an even circulant texture

The weak-axis return structure gives a site-basis operator

`diag(a,b,b)`.

Conjugating by the canonical `Z_3` bridge gives

`Y_even = U_Z3^dag diag(a,b,b) U_Z3`.

Exactly,

`Y_even = mu I + nu (S + S^2)`

with

`mu = (a + 2b)/3`,
`nu = (a - b)/3`.

So the current exact split already picks a very specific texture class.

### 2. That current slice is still CP-degenerate

Its Hermitian kernel is

`H_even = Y_even^dag Y_even`

which depends only on `|a|^2` and `|b|^2` and stays in the same even
circulant slice. Its off-diagonal entries are therefore real, so the standard
CP tensor still vanishes.

### 3. The minimal `Z_3`-covariant extension is the odd circulant generator

The full complex circulant algebra is generated by

`I`, `S`, `S^2`.

Within the real-parameter Hermitian decomposition, the unique odd direction is

`i(S - S^2)`.

So the minimal `Z_3`-covariant CP-supporting family is

`Y_CP = mu I + nu (S + S^2) + i eta (S - S^2)`.

### 4. This family really can support nonzero CP

The `Z_3` bridge diagonalizes `Y_CP` with eigenvalues

`lambda_0 = mu + 2nu`,
`lambda_+ = mu - nu + sqrt(3) eta`,
`lambda_- = mu - nu - sqrt(3) eta`.

For generic `eta != 0`, the doublet pair splits and the Hermitian kernel gains
complex off-diagonal entries. In particular,

`H_01 = (2 mu nu + nu^2 - eta^2) + 2 i eta (mu - nu)`,

so generically

`Im[(H_01)^2] != 0`.

That is enough to support a nonzero standard leptogenesis CP tensor.
