# CKM CP-Phase rho-eta to delta Narrow Theorem

**Date:** 2026-05-10
**Type:** positive_theorem
**Claim scope:** the standalone polynomial-algebra implication that, given
abstract real positive symbols `w_A1`, `w_perp` satisfying
`w_A1 + w_perp = 1` and any abstract positive `r^2`, the radial
coordinates `rho^2 := r^2 * w_A1`, `eta^2 := r^2 * w_perp` and the angle
defined by `cos^2(delta) := rho^2 / (rho^2 + eta^2)`,
`sin^2(delta) := eta^2 / (rho^2 + eta^2)` satisfy the algebraic identities
`rho^2 + eta^2 = r^2`, `cos^2(delta) = w_A1`, `sin^2(delta) = w_perp`,
`tan^2(delta) = w_perp / w_A1`. Specialising to `w_A1 = 1/k`,
`w_perp = (k - 1)/k` for any integer `k >= 2` gives `cos^2(delta) = 1/k`,
`tan^2(delta) = k - 1`. The Jarlskog-shape area factorisation
`J_0 := lambda^6 A^2 sqrt(w_perp * r^2)` with abstract
`lambda^2 := alpha / n_pair`, `A^2 := n_pair / n_color` reduces to
`J_0 = alpha^3 * sqrt(w_perp * r^2) / (n_pair^3 * n_color^2)` by pure
substitution. This is purely a fact of polynomial algebra over abstract
positive symbols; no Wolfenstein parameterisation derivation, no CKM
atlas `1 + 5` projector-split authority, no `r^2 = 1/6` retained input,
no Wolfenstein `lambda^2 = alpha_s(v)/2` or `A^2 = 2/3` retained input,
no value of `alpha_s` or any `alpha` symbol, no CKM unitarity statement,
no PDG comparator, no quark-mass identification is consumed.

## Statement

Let `w_A1, w_perp, r^2, alpha, n_pair, n_color, k` be abstract positive
real symbols satisfying the partition identity

```text
w_A1 + w_perp  =  1,                                                      (P)
```

with `w_A1 > 0` and `w_perp > 0`, together with the radial-coordinate
definitions

```text
rho^2  :=  r^2 * w_A1,                                                    (D-rho)
eta^2  :=  r^2 * w_perp,                                                  (D-eta)
```

and the angle definitions

```text
cos^2(delta)  :=  rho^2 / (rho^2 + eta^2),                                (D-cos)
sin^2(delta)  :=  eta^2 / (rho^2 + eta^2),                                (D-sin)
tan^2(delta)  :=  sin^2(delta) / cos^2(delta).                            (D-tan)
```

**Conclusion (T1) (radius closes).** Under `(P)` and `(D-rho)`, `(D-eta)`,

```text
rho^2 + eta^2  =  r^2.                                                    (1)
```

**Conclusion (T2) (cos^2 reads w_A1).** Under the same hypotheses,

```text
cos^2(delta)  =  w_A1.                                                    (2)
```

**Conclusion (T3) (sin^2 reads w_perp).** Under the same hypotheses,

```text
sin^2(delta)  =  w_perp.                                                  (3)
```

**Conclusion (T4) (Pythagorean identity).** Under the same hypotheses,

```text
cos^2(delta) + sin^2(delta)  =  1.                                        (4)
```

**Conclusion (T5) (tan^2 ratio).** Under the same hypotheses with
`w_A1 > 0`,

```text
tan^2(delta)  =  w_perp / w_A1.                                           (5)
```

**Conclusion (T6) (integer-k specialisation).** Setting `w_A1 = 1/k`,
`w_perp = (k - 1)/k` for any abstract positive integer `k >= 2` (these
satisfy `(P)`),

```text
cos^2(delta)  =  1/k,                                                     (6a)
sin^2(delta)  =  (k - 1)/k,                                               (6b)
tan^2(delta)  =  k - 1,                                                   (6c)
delta         =  arccos(1/sqrt(k))  =  arctan(sqrt(k - 1)).               (6d)
```

**Conclusion (T7) (k = 6 numerical readout).** Specialising to the
integer `k = 6`,

```text
cos^2(delta)  =  1/6,                                                     (7a)
tan(delta)    =  sqrt(5),                                                 (7b)
delta         =  arccos(1/sqrt(6))  =  arctan(sqrt(5))                    (7c)
              =  65.905157447889 ...  degrees    (numerical).             (7d)
```

**Conclusion (T8) (Jarlskog-shape area factorisation).** Under the
additional abstract substitution hypotheses

```text
lambda^2  :=  alpha / n_pair,                                             (J-l)
A^2       :=  n_pair / n_color,                                           (J-A)
```

with `alpha > 0`, `n_pair > 0`, `n_color > 0`, and using `(D-eta)` to
write `eta = sqrt(r^2 * w_perp)`, the symbolic combination

```text
J_0  :=  lambda^6 * A^2 * eta                                             (J-def)
```

reduces by pure substitution to

```text
J_0  =  alpha^3 * sqrt(r^2 * w_perp) / (n_pair^2 * n_color).              (8)
```

**Conclusion (T9) (concrete numerical instance).** Specialising to the
abstract integer tuple `(n_pair, n_color, k) = (2, 3, 6)` and
`r^2 = 1/k = 1/6`, `w_perp = (k - 1)/k = 5/6`, identity `(8)` reduces to

```text
J_0  =  alpha^3 * sqrt((1/6) * (5/6)) / (2^2 * 3)                         (9a)
     =  alpha^3 * sqrt(5/36) / 12                                         (9b)
     =  alpha^3 * sqrt(5) / 72.                                           (9c)
```

**Conclusion (T10) (numerical witness on Wolfenstein-shape 3x3 unitary
matrices).** For any 3x3 unitary matrix V whose `(rho, eta)` coordinates
(in the standard Wolfenstein convention `bar(rho) + i bar(eta) =
- V_ud V_ub^* / (V_cd V_cb^*)` reduced to leading order) equal
`(1/6, sqrt(5)/6)`, the standard CP-phase angle satisfies
`arctan(eta / rho) = arctan(sqrt(5))` exactly under the (D-cos), (D-sin),
(D-tan) angle definitions. The narrow runner exhibits one such explicit
3x3 unitary V and verifies the angle readout numerically.

## Proof

`(T1)` By `(D-rho)`, `(D-eta)`, and `(P)`:

```text
rho^2 + eta^2  =  r^2 * w_A1 + r^2 * w_perp
              =  r^2 * (w_A1 + w_perp)
              =  r^2 * 1
              =  r^2.
```

`(T2)`, `(T3)` Substitute `(D-rho), (D-eta)` into `(D-cos), (D-sin)` and
use the algebraic identity `r^2 * w_A1 / (r^2 * w_A1 + r^2 * w_perp) =
w_A1 / (w_A1 + w_perp)` (`r^2 > 0` hypothesis); then apply `(P)`:

```text
cos^2(delta)  =  rho^2 / (rho^2 + eta^2)
              =  (r^2 * w_A1) / (r^2 * (w_A1 + w_perp))
              =  w_A1 / (w_A1 + w_perp)
              =  w_A1                          [by (P): w_A1 + w_perp = 1],
sin^2(delta)  =  eta^2 / (rho^2 + eta^2)
              =  w_perp / (w_A1 + w_perp)
              =  w_perp                        [by (P)].
```

The off-(P) intermediate identities
`cos^2 = w_A1/(w_A1 + w_perp)`, `sin^2 = w_perp/(w_A1 + w_perp)` hold
for any positive `(w_A1, w_perp, r^2)` (no partition needed) and are
recorded in the runner as separate diagnostic checks
`(T2-offP), (T3-offP)`.

`(T4)` Direct: `cos^2(delta) + sin^2(delta) = w_A1 + w_perp = 1` by `(P)`.

`(T5)` By `(T2)`, `(T3)`, `(D-tan)`, and `w_A1 > 0`:

```text
tan^2(delta)  =  sin^2(delta) / cos^2(delta)  =  w_perp / w_A1.
```

`(T6)` Direct substitution of `w_A1 = 1/k`, `w_perp = (k-1)/k` into
`(T2)`, `(T3)`, `(T5)`. The angle `delta = arccos(1/sqrt(k))` is the
positive-branch principal value; `arctan(sqrt(k-1)) = arccos(1/sqrt(k))`
on `(0, pi/2)` is a standard trigonometric identity.

`(T7)` Direct substitution of `k = 6` into `(T6)`; the numerical
`65.905157447889... deg` is `arccos(1/sqrt(6))` evaluated to
double-precision.

`(T8)` Direct substitution into `(J-def)` using `(J-l), (J-A), (D-eta)`:

```text
J_0  =  lambda^6 * A^2 * eta
     =  (lambda^2)^3 * A^2 * sqrt(r^2 * w_perp)
     =  (alpha / n_pair)^3 * (n_pair / n_color) * sqrt(r^2 * w_perp)
     =  (alpha^3 / n_pair^3) * (n_pair / n_color) * sqrt(r^2 * w_perp)
     =  alpha^3 * sqrt(r^2 * w_perp) / (n_pair^2 * n_color).
```

That is identity `(8)`.

`(T9)` Direct substitution of `(n_pair, n_color, r^2, w_perp) =
(2, 3, 1/6, 5/6)` into `(8)`:

```text
J_0  =  alpha^3 * sqrt((1/6) * (5/6)) / (2^2 * 3)
     =  alpha^3 * sqrt(5/36) / 12
     =  alpha^3 * (sqrt(5) / 6) / 12
     =  alpha^3 * sqrt(5) / 72.
```

(Using `sqrt(5/36) = sqrt(5)/6` since `6 > 0`.) This reproduces the
parent atlas' stated `J_0 = alpha^3 sqrt(5)/72` factorisation by pure
substitution under the abstract hypotheses, without invoking any value
of `alpha_s` or any retained CKM atlas projector-split / radius input.

`(T10)` By construction in the runner: define a 3x3 unitary V whose
Wolfenstein-shape `(rho, eta)` apex equals `(1/6, sqrt(5)/6)`, compute
the apex angle `arctan(eta/rho)`, and check equality with `arctan(sqrt(5))`
to floating-point precision.

`∎`
