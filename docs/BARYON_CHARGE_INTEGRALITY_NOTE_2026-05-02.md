## 0. Statement

**Theorem.** Given quark electric charges Q_u = +2/3, Q_d = −1/3 from
the cited SM charge table and SM convention that baryons consist of 3 quarks (in a color-singlet
combination), all baryons composed of u and d quarks have integer
electric charge:

```text
Q_baryon  =  N_u Q_u  +  N_d Q_d           (with N_u + N_d = 3)
            =  N_u · (2/3)  +  (3 − N_u) · (−1/3)
            =  (2 N_u − (3 − N_u)) / 3
            =  (3 N_u − 3) / 3
            =  N_u − 1.
```

For N_u ∈ {0, 1, 2, 3}: Q_baryon ∈ {−1, 0, +1, +2} — all integers.
