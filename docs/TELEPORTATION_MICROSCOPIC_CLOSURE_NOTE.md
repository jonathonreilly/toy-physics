## Native Stabilizer Hamiltonian

On the retained-axis taste factor, Alice's Bell stabilizers are:

```text
S_z = Z_A Z_R
S_x = X_A X_R
```

where `Z` and `X` are retained-axis site/taste operators

```text
Z_r = I_cells tensor Z_retained tensor I_spectator
X_r = I_cells tensor X_retained tensor I_spectator.
```

The runner checks this on the `3D side=2` surface and verifies that the native
site/taste operators factor exactly into the retained logical Pauli operators.

The Bell outcome bits are eigenvalue bits:

```text
z = 1 if S_x = -1, else 0
x = 1 if S_z = -1, else 0
p = z xor x = 1 if S_x S_z = -1, else 0.
```

The transducer Hamiltonian is:

```text
H_int = (pi/2T) [
    P_z^- sum_{i in z-domain} X_i
  + P_x^- sum_{i in x-domain} X_i
  + P_p^- sum_{i in parity-domain} X_i
]
```

where

```text
P_z^- = (I - S_x)/2
P_x^- = (I - S_z)/2
P_p^- = (I - S_x S_z)/2.
```

Because `S_x`, `S_z`, and `S_x S_z` commute, all Hamiltonian terms commute.
At time `T`, each pointer spin in the relevant domain flips iff its stabilizer
projector is active. The resulting codeword is exactly:

```text
z z z | x x x | p p.
```

This is the first artifact where the Bell-record write is generated by a
native retained-axis stabilizer Hamiltonian rather than an externally supplied
projector or abstract controlled unitary.

## Thermodynamic Detector Bound

Let `d_min` be the minimum Hamming distance between Bell-record codewords. For
the current code, `d_min = 5`.

If each pointer component has domain size `N`, pointer branch overlap base

```text
a = |cos(2 theta)|
```

and each domain spin couples to `b` bath spins with bath overlap base

```text
c = |cos(2 phi)|,
```

then any two distinct Bell-record branches have combined pointer+bath overlap
bounded by

```text
epsilon_N <= exp(-kappa N)
kappa = d_min [-log(a) - b log(c)].
```

For the default `theta=0.620`, `phi=0.350`, `b=4`, and `N=9`, the runner
finds:

```text
d_min = 5
kappa = 10.984501
epsilon_9 = 1.163e-43
```

At `N=21`, the entropy defect from a two-bit classical record is below
roundoff:

```text
2 - S(record) = 0.000e+00 bits
```

This gives a thermodynamic detector theorem for the candidate pointer+bath
family: record coherences vanish exponentially in domain size once the bath is
traced.

## Native Ledger Theorem Class

A native teleportation apparatus generator is any finite linear combination of
operators of the form:

```text
G_native = I_support tensor G_taste tensor A_apparatus
```

where `G_taste` is an arbitrary retained taste/stabilizer generator and
`A_apparatus` is an arbitrary pointer, bath, or record-field generator. The
conserved support, mass, and charge ledgers have the native form:

```text
L = L_support tensor I_taste tensor I_apparatus.
```

For all such `L` and all such `G_native`,

```text
[L, G_native] = 0.
```

The runner checks this both by random Hermitian samples and by the elementary
matrix basis identity:

```text
[E_ab_support tensor I_taste tensor I_app,
 I_support tensor E_mn_taste tensor E_uv_app] = 0.
```

It also checks that the stabilizer-controlled transducer terms are in this
class. They need not factor as `I_taste tensor A_apparatus`; they may be
controlled by `P_z^-`, `P_x^-`, and `P_p^-`, because those projectors act only
inside the retained taste sector. A device that couples a pointer directly to
support/species ledgers is outside the native taste-qubit teleportation
apparatus class and would be a different implementation claim.
