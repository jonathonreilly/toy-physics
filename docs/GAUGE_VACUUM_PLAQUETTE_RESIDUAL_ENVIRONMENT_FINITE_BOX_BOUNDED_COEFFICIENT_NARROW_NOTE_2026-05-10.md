## Statement

Fix the finite weight box `B = {(p, q) : 0 <= p, q <= NMAX}` with
`NMAX = 4`. Index it by `(p, q)`. For each `(p, q)`, define the standard
`SU(3)` irrep dimension

```text
d_(p,q) = (p + 1) (q + 1) (p + q + 2) / 2.                                (1)
```

Let `rho : B -> R_{>= 0}` be **any** real nonnegative coefficient sequence
that satisfies the conjugation-symmetry constraint

```text
rho_(p,q) = rho_(q,p)                          for all (p, q) in B.       (2)
```

Define on the finite-box class-function sector `H_B = span{chi_(p,q) :
(p,q) in B}` the diagonal operator

```text
R[rho] : H_B -> H_B,                                                      (3)
R[rho] chi_(p,q) = rho_(p,q) chi_(p,q)         for all (p, q) in B.
```

**Conclusion (N1) (finite-box diagonal-action identity).** For any `rho`
as above, the operator `R[rho]` is:

- diagonal in the orthonormal character basis `{chi_(p,q) : (p,q) in B}`,
- self-adjoint on `H_B`,
- positive semidefinite (positive definite if `rho_(p,q) > 0` on `B`),
- commuting with every diagonal coefficient multiplier in the character
  basis,
- conjugation-symmetric: `[S, R[rho]] = 0`, where `S` is the swap involution
  `S chi_(p,q) = chi_(q,p)` on `H_B`.

**Conclusion (N2) (finite-box Peter-Weyl convolution identification).** If
`Z(W)` is *any* central class function of marked plaquette holonomy
`W in SU(3)` whose character expansion truncated to `B` reads

```text
Z(W)|_B = c_0 sum_{(p,q) in B} d_(p,q) rho_(p,q) chi_(p,q)(W)             (4)
```

with `c_0 > 0`, then the normalized convolution operator `C_Z` on `H_B`
acts diagonally by

```text
C_Z chi_(p,q) = rho_(p,q) chi_(p,q)            for all (p, q) in B.       (5)
```

This is the standard Peter-Weyl convolution-on-characters identity
`chi_(p,q) * chi_(p',q') = (delta_((p,q),(p',q')) / d_(p,q)) chi_(p,q)`
restricted to the finite box `B`. The proof is the elementary character
orthogonality calculation, done line by line in the proof section below.

**Conclusion (N3) (finite-box bounded-coefficient instantiation).** Take
the bounded coefficient companion's runner-computed normalized single-link
`SU(3)` Wilson boundary values

```text
rho_(p,q)(6) = c_(p,q)(6) / (d_(p,q) c_(0,0)(6)),                         (6)
c_(p,q)(6) = integral_{SU(3)} chi_(p,q)(U) exp((6 / 3) Re tr U) dU,       (7)
```

for `(p, q) in B`. Substituting `rho := rho(6)` into `(N1)`/`(N2)`:

- the diagonal operator `R[rho(6)]` exists, is positive definite and
  conjugation-symmetric on `H_B`,
- it acts by `R[rho(6)] chi_(p,q) = rho_(p,q)(6) chi_(p,q)` for
  `(p, q) in B`,
- and it coincides on `H_B` with `R_6^env` as defined by the source-sector
  decomposition `K_6^src = exp(3 J) D_6^loc R_6^env exp(3 J)`, restricted
  to the same finite box.

This last coincidence is a definitional fact at finite-box scope: the
decomposition stripping defines the finite-box residual factor's action on
`chi_(p,q)` by a real diagonal scalar in this restricted class-function
sector, and the bounded coefficient companion supplies the numerical values

## Proof

`(N1)` is immediate from the definition `(3)`. The character basis is
orthonormal in `L^2(SU(3))^G` under Haar measure; restricting to `H_B`
preserves orthonormality. A real diagonal operator on an orthonormal
basis is self-adjoint. Nonnegative diagonal entries give positive
semidefiniteness. Two diagonal coefficient multipliers in the character
basis commute, so `R[rho]` commutes with every operator represented
diagonally in that basis. The swap involution `S` permutes characters by
`(p,q) <-> (q,p)`, so `[S, R[rho]]` vanishes iff `rho_(p,q) = rho_(q,p)`,
which is `(2)`.

`(N2)`. Normalized convolution on the class-function sector of a compact
group is the standard convolution: if `Z` has character expansion `Z = c_0
sum_b d_b rho_b chi_b` and `f = sum_a f_a chi_a`, then by the convolution-
on-characters identity

```text
chi_a * chi_b = (delta_(a,b) / d_a) chi_a,                                (8)
```

we have

```text
(Z * f)(W)
  = c_0 sum_{a, b in B} d_b rho_b f_a (chi_b * chi_a)(W)
  = c_0 sum_{a in B} d_a rho_a f_a (chi_a * chi_a)(W) / d_a · d_a
  = c_0 sum_{a in B} rho_a f_a chi_a(W),
```

(only `b = a` survives the Kronecker), so normalized convolution
`C_Z f := (1 / c_0) Z * f` gives

```text
C_Z f = sum_{a in B} rho_a f_a chi_a,                                     (9)
```

i.e. `C_Z chi_(p,q) = rho_(p,q) chi_(p,q)` on `B`. This is `(5)`.

`(N3)`. The narrow source-sector decomposition

```text
K_beta^src = exp[(beta / 2) J] D_beta^loc R_beta^env exp[(beta / 2) J]    (10)
```

is taken as the **definition** of `R_beta^env`. This note does not prove an
operator-theoretic inversion theorem for the full source-sector kernel. It
checks the finite-box residual factor directly:

- positivity: every imported `rho_(p,q)(6)` on `B` is positive;
- self-adjointness: `R[rho(6)]` is a real diagonal matrix on the character
  basis;
- class-sector diagonality: the restricted operator acts on the character
  basis by its coefficient eigenvalues;
- forward source-sector consistency: the runner builds
  `K_6^src = exp(3 J) D_6^loc R[rho(6)] exp(3 J)` and strips the two
  half-slice factors back to `D_6^loc R[rho(6)]` at machine precision.

For the narrow finite-box theorem, `R_beta^env|_{H_B}` is represented by a
positive self-adjoint diagonal operator in the character basis on `B`. Such
an operator is uniquely determined by its character eigenvalues `rho_(p,q)`.
The bounded companion's `rho_(p,q)(6)` values satisfy positivity and
conjugation symmetry on `B` (verified at runner time), and therefore
determine the defined `R_6^env|_{H_B}` on the finite box.
By `(N2)` applied to the unmarked spatial Wilson environment's boundary
class function `Z_6^env|_B` (which has those character coefficients by
construction in the bounded companion), this `R_6^env|_{H_B}` equals
`C_(Z_6^env)|_{H_B}`. ∎
