## Why this note exists

The natural science-fix attempt on the strong-CP audited_conditional
row is to use the already-retained RP norm-square identity to derive
the "no bare θ slot is admissible" premise. The argument would go:

> RP gives `⟨ Θ(F) · F ⟩ = ‖ψ²F‖² ≥ 0` for real Θ-invariant `S_+`.
> Adding `S_CP = i c h` with `c ≠ 0` and `h` real should make `S_+`
> complex-valued, exiting the RP-admissible class.

This no-go shows the argument **fails operationally**: the Θ-anti-
invariance of the CP-odd density `h` (which mirrors the continuum
property that `F̃F` is a total derivative / topological density)
causes the imaginary phase to cancel in every reflection-Hermitian
observable's reflected expectation. RP does not detect CP-odd
imaginary contamination of the topological-charge type. This is a
reusable negative result: future audit-fix iterations on the parent
strong-CP row should not attempt this direction.

## Claim scope

Let `(X, μ, Θ)` be the same finite measured space as in
`REFLECTION_POSITIVITY_GAUGE_HALF_CAUCHY_SCHWARZ_NARROW_THEOREM_NOTE_2026-05-10.md`:
finite or compact, measure-preserving involution. Let `S_+ : X → R`
be real-valued Θ-invariant and let `h : X → R` be real-valued
Θ-anti-invariant (`h(Θx) = -h(x)`). Let `c ∈ R \ {0}` and define the
modified half-action `S̃_+ := S_+ + i c h`.

**No-go.** For every reflection-Hermitian `F : X → C`
(`F(Θx) = overline{F(x)}`), the modified reflected expectation

```text
⟨ Θ(F) · F ⟩_{modified}
  := ∫_X dμ(x) exp(-S̃_+(x)) exp(-S̃_+(Θx)) F(Θx) F(x)
   = ∫_X dμ(x) exp(-2 S_+(x)) exp(-i c (h(x) + h(Θx))) F(Θx) F(x)
```

obeys, by Θ-anti-invariance of `h`,

```text
exp(-i c (h(x) + h(Θx))) = exp(-i c · 0) = 1
```

so the modified reflected expectation equals the original real
non-negative norm-square

```text
⟨ Θ(F) · F ⟩_{modified} = ‖ψ² F‖²_{L²(X, μ)}  ∈  R_{≥0}.
```

Even on the asymmetric single-side modification, where only the
integrand-`x` side carries the `exp(-i c h(x))` phase, summing over
the Θ-paired carrier `{k, Θk}` causes the imaginary phases to cancel
pointwise (because `|F(k)|² = |F(Θk)|²` for reflection-Hermitian `F`
and the Θ-anti-invariance of `h` flips the sign of the phase under
the Θ-image). The single-side modified reflected expectation
therefore also returns a real value for every reflection-Hermitian
observable on this carrier.

**Consequence.** Retained RP, as packaged in the cited norm-square
identity, **cannot** detect CP-odd imaginary additions of the
topological-charge type (`h` real Θ-anti-invariant). The "no bare
θ slot is admissible" premise of `STRONG_CP_THETA_ZERO_NOTE.md`
therefore cannot be derived from retained RP alone.

## Why the no-go is structural, not a research-effort gap

The cancellation `exp(-i c (h(x) + h(Θx))) = 1` is **forced** by:

1. Θ-anti-invariance of `h`. This mirrors the standard continuum
   property: the topological density `F̃F` integrates to an integer
   on closed manifolds and is parity/CP-odd, so on a Θ-symmetric
   carrier its Θ-image flips sign and the sum is identically zero.
2. The reflected expectation definition `∫ exp(-S̃(x)) exp(-S̃(Θx))
   F(Θx) F(x) dμ`, which integrates `S̃` against itself across the
   Θ-pair, exactly engaging the Θ-anti-invariance.
3. The reflection-Hermiticity of `F`, which makes `F(Θx) F(x) = |F(x)|²`
   real and `Θ`-invariant, so the only phase in the integrand is the
   `exp(-i c (h(x) + h(Θx)))` factor — and that factor is the
   identity by Θ-anti-invariance of `h`.

No tightening of this half-square RP identity, no extra observable
class inside this reflected-expectation form, and no extension of the
finite carrier removes this cancellation as long as `h` is required to
be Θ-anti-invariant and `F` is required to be reflection-Hermitian.
This finite-carrier algebra mirrors the CP-odd/topological parity of
the continuum `theta Ftilde F` density, but it should not be read as a
claim that the full continuum theta partition function is insensitive
to non-integer `theta`; the no-go is only that the retained RP
half-square identity does not itself exclude the slot.

## Proof

### Step 1. Reflected expectation under symmetric modification.

Under `S̃_+ = S_+ + i c h` applied symmetrically (both sides):

```text
⟨ Θ(F) · F ⟩_{sym-modified}
  = ∫_X dμ(x) exp(-S̃_+(x)) exp(-S̃_+(Θx)) F(Θx) F(x)
  = ∫_X dμ(x) exp(-S_+(x) - i c h(x))
              exp(-S_+(Θx) - i c h(Θx))
              F(Θx) F(x)
  = ∫_X dμ(x) exp(-S_+(x) - S_+(Θx))
              exp(-i c (h(x) + h(Θx)))
              F(Θx) F(x).
```

By Θ-invariance of `S_+`, `S_+(Θx) = S_+(x)`, so
`exp(-S_+(x) - S_+(Θx)) = exp(-2 S_+(x))`. By Θ-anti-invariance of
`h`, `h(Θx) + h(x) = 0`, so `exp(-i c (h(x) + h(Θx))) = 1`. By
reflection-Hermiticity of `F`, `F(Θx) F(x) = overline{F(x)} F(x) =
|F(x)|²`. Therefore

```text
⟨ Θ(F) · F ⟩_{sym-modified}
  = ∫_X dμ(x) exp(-2 S_+(x)) |F(x)|²
  = ‖ψ² F‖²_{L²(X, μ)}.                                          (A)
```

This is the original RP norm-square identity — the symmetric
CP-odd-imaginary modification is invisible.

### Step 2. Reflected expectation under asymmetric single-side modification.

Suppose the modification is applied only to one side, say the
integrand-`x` side:

```text
⟨ Θ(F) · F ⟩_{single-side}
  := ∫_X dμ(x) exp(-S_+(x) - i c h(x))
              exp(-S_+(Θx))
              F(Θx) F(x).
```

Re-index by Θ-pairing. Write the integral as a sum over
Θ-orbits `{x, Θx}`. Each orbit contributes

```text
exp(-S_+(x) - i c h(x)) exp(-S_+(Θx)) F(Θx) F(x)
 + exp(-S_+(Θx) - i c h(Θx)) exp(-S_+(Θ²x)) F(Θ²x) F(Θx).
```

Using `Θ² = id`, `S_+(Θx) = S_+(x)`, and
`h(Θx) = -h(x)`, this simplifies to

```text
exp(-2 S_+(x)) [exp(-i c h(x)) F(Θx) F(x)
              + exp(+i c h(x)) F(x) F(Θx)]
  = exp(-2 S_+(x)) F(Θx) F(x) · 2 cos(c h(x)).                (B)
```

The orbit contribution is **real**, because `F(Θx) F(x) = |F(x)|²`
is real for reflection-Hermitian `F` and `cos(c h(x))` is real.
Fixed points of `Θ` (orbits of size 1) trivially contribute reals.

Summing all orbits:

```text
⟨ Θ(F) · F ⟩_{single-side}
  = ∫_X dμ(x) exp(-2 S_+(x)) |F(x)|² cos(c h(x))             (C)
   ∈ R.
```

For small `c`, `cos(c h(x)) ≈ 1 - (c h(x))² / 2`, so the modification
appears as a **real second-order suppression**, not an imaginary
breakage. This single-side calculation is only a diagnostic that the
phase pairs to a real cosine envelope; it is not used to assert
positivity for arbitrary `c`.

### Step 3. RP detection failure.

By Steps 1-2, the symmetric reflected expectation is identical to the
original non-negative norm square, and the single-side diagnostic does
not acquire an imaginary part. The cited retained note's
RP-admissibility hypotheses (Θ-invariant real `S_+`,
reflection-Hermitian `F`) are exactly engaged so as to **cancel** the
CP-odd phase in the symmetric half-square pairing: Θ-invariance of
`S_+` and Θ-anti-invariance of `h` are mismatched in parity under the
very pairing the reflected expectation enforces.

Therefore retained RP is structurally insensitive to CP-odd
imaginary action additions of the topological-charge type. The "no
bare θ slot is admissible" premise cannot be derived as an RP
corollary. ∎
