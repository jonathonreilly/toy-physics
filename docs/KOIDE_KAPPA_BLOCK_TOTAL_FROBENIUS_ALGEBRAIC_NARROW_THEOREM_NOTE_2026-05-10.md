## Statement

Let `C` be the standard `3 x 3` cyclic permutation matrix
```text
C = [[0, 1, 0],
     [0, 0, 1],
     [1, 0, 0]],                                                           (1)
```
so that `C^3 = I`. Let `(a, b) in R x C` be abstract symbols, and define
the circulant Hermitian
```text
H  =  a I + b C + bbar C^2.                                                (2)
```
Define the canonical real-isotype projectors on `Herm_circ(3)`:
```text
pi_+(H)    :=  (tr H / 3) I  =  a I,                                       (3)
pi_perp(H) :=  H - pi_+(H)   =  b C + bbar C^2.                            (4)
```

**Conclusion (T1) (orthogonal decomposition and image dimensions).**

`pi_+` and `pi_perp` are mutually orthogonal Hermitian projectors on
`Herm_circ(3)` satisfying `pi_+ + pi_perp = id`. Their image dimensions
over R are
```text
dim_R image(pi_+)    =  1   (trivial isotype, spanned by I),               (5)
dim_R image(pi_perp) =  2   (doublet isotype, spanned by the Hermitian    (6)
                              basis {B_1, B_2} = {C + C^2, i (C - C^2)};
                              both B_1, B_2 are Hermitian -- B_1 is real
                              symmetric and B_2 has pure-imaginary
                              off-diagonals).
```

**Conclusion (T2) (block-total Frobenius identities).**

```text
E_+(H)    :=  || pi_+(H) ||_F^2     =  3 a^2,                              (7)
E_perp(H) :=  || pi_perp(H) ||_F^2  =  6 |b|^2.                            (8)
```

**Conclusion (T3) (equal-weight Lagrange extremum).**

Treat `(E_+, E_perp)` as positive real coordinates. The functional
```text
S(E_+, E_perp)  :=  log E_+  +  log E_perp                                 (9)
```
under the constraint `E_+ + E_perp = E_tot > 0` has unique interior
positive critical point
```text
E_+^*  =  E_perp^*  =  E_tot / 2,                                         (10)
```
equivalently, in `(a, b)` coordinates via `(7)`-`(8)`,
```text
a^2  =  2 |b|^2,    i.e.    kappa  :=  a^2 / |b|^2  =  2.                 (11)
```

**Conclusion (T4) (`d = 3` uniqueness of the `(1, 1)` multiplicity
pattern).**

For `d = 2, 3, 4, 5, 6`, the real-irrep multiplicity counts of the
`Z_d`-conjugation action on the real Hermitian circulants
`Herm_circ(d)` are
```text
d = 2:  (trivial, doublets, sign)  =  (1, (), 1),
d = 3:  (trivial, doublets, sign)  =  (1, (1,), 0),
d = 4:  (trivial, doublets, sign)  =  (1, (1,), 1),
d = 5:  (trivial, doublets, sign)  =  (1, (1, 1), 0),
d = 6:  (trivial, doublets, sign)  =  (1, (1, 1), 1).
```
The pattern `(trivial, doublets, sign) = (1, (1,), 0)`, i.e. exactly one
trivial irrep and one doublet and no sign irrep, is uniquely realized at
`d = 3` among `d in {2, 3, 4, 5, 6}`.

## Proof

`(T1)` `pi_+(H) = (tr H / 3) I` is the orthogonal projection onto the
1-dim trace-line in `Herm(3)` (well-defined since `tr` is a real linear
functional and `(I/sqrt(3))` is unit-norm under the Frobenius inner
product). On `Herm_circ(3)`, `tr H = 3 a`, so `pi_+(H) = a I`, hence
`pi_+(H) in span_R {I}` is 1-dim. Idempotency `pi_+(pi_+(H)) = pi_+(H)`
and Hermiticity `(pi_+(H))^H = pi_+(H)` are by direct evaluation.

`pi_perp(H) = H - pi_+(H) = b C + bbar C^2`. Write `b = (Re b) + i (Im b)`
and expand: `pi_perp(H) = (Re b)(C + C^2) + (Im b) i (C - C^2)`. Set
`B_1 := C + C^2` (Hermitian, real symmetric) and `B_2 := i (C - C^2)`
(Hermitian, with pure-imaginary off-diagonals: `B_2^H = -i (C^H - (C^2)^H)
= -i (C^2 - C) = i (C - C^2) = B_2`). Both lie in `Herm_circ(3)`, and the
real-parametrized expansion is `pi_perp(H) = (Re b) B_1 + (Im b) B_2`.
As `(Re b, Im b)` range over `R^2`, the image is the real-2-dim subspace
`span_R{B_1, B_2} subset Herm_circ(3)`, hence `dim_R image(pi_perp) = 2`.

Orthogonality of `pi_+(H)` and `pi_perp(H)` under the real Frobenius
inner product `<X, Y>_F := tr(X^H Y) = tr(X Y)` follows from
`tr(I (b C + bbar C^2)) = b tr(C) + bbar tr(C^2) = 0` (cyclic shifts
have zero trace at `d >= 2`). `pi_+ + pi_perp = id` by construction.

`(T2)` Using the Frobenius inner product:
```text
E_+(H)  =  tr((a I)^H (a I))  =  a^2 tr(I)  =  3 a^2.
E_perp(H)  =  tr((b C + bbar C^2)^H (b C + bbar C^2))
         =  |b|^2 tr(C^H C) + |b|^2 tr((C^2)^H (C^2))
            + b^2 tr(C^H C^2) + bbar^2 tr((C^2)^H C).
```
Now `C^H C = I` (since `C` is unitary), so `tr(C^H C) = tr(I) = 3`.
Similarly `tr((C^2)^H (C^2)) = 3`. Cross terms `tr(C^H C^2) = tr(C^{-1} C^2) = tr(C) = 0`
and `tr((C^2)^H C) = tr(C^{-2} C) = tr(C^{-1}) = 0`. Hence
`E_perp(H) = 3 |b|^2 + 3 |b|^2 = 6 |b|^2`.

`(T3)` Treat `E_+, E_perp > 0`. The constrained extremum of
`S = log E_+ + log E_perp` on `{E_+ + E_perp = E_tot}` has Lagrangian
`L = log E_+ + log E_perp - mu (E_+ + E_perp - E_tot)`, with stationary
conditions `1/E_+ = mu`, `1/E_perp = mu`, hence `E_+ = E_perp = E_tot / 2`.
Substituting via `(7)`-`(8)`: `3 a^2 = 6 |b|^2`, i.e. `a^2 = 2 |b|^2`,
i.e. `kappa = 2`. This is a positive interior critical point of `S`
since `S` is strictly concave on the positive orthant (Hessian
diagonal with negative entries `-1/E^2`).

`(T4)` A Hermitian circulant `H` on `C^d` is specified by `(a_0, b_1, b_2, ..., b_{d-1})`
with `a_0 in R` (coefficient of `I`) and `b_k = bbar_{d-k}` (so the off-diagonals
are pairwise conjugate-related). The `b_k` parametrization gives one
real scalar for `k = 0` (trivial), and for each conjugate pair
`{k, d - k}` with `k != d - k mod d` one complex `b_k` (a 2-real-dim
doublet), and if `d` is even one additional real scalar at `k = d/2`
(sign irrep). Counting:
```text
trivial irreps  :  1,
doublet irreps  :  floor((d - 1) / 2),
sign irreps     :  1  if d even,  else  0.
```
For `d = 2, 3, 4, 5, 6` this gives the table above. The pattern
`(1, (1,), 0)` requires `floor((d - 1) / 2) = 1` and `d` odd; the only
solution in `{2, 3, 4, 5, 6}` is `d = 3`. ∎

## Derivable corollary (Hessian sign of the extremum)

```text
At E_+^* = E_perp^* = E_tot / 2, S has Hessian
   d^2 S / dE_+^2     = -1 / (E_+^*)^2  = -4 / E_tot^2,
   d^2 S / dE_perp^2  = -1 / (E_perp^*)^2 = -4 / E_tot^2,
```
both strictly negative, so the critical point is a local (and global on
the positive simplex) maximum of `S` under the linear constraint.
