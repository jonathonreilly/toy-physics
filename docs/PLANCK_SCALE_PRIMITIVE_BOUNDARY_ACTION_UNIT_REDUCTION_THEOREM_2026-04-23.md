## Theorem 1: primitive unit-count semantics forces `mu = 1` inside the primitive object class

Let `S_1` be the four one-step worldtube atoms in the primitive cell. The
primitive boundary count is the additive functional generated by

`u(e) = 1`

on each primitive incidence atom `e in S_1`.

For a boundary surface tiled by primitive cells, the unit count is therefore

`N_boundary = A / a^2 + o(A/a^2)`.

The native source-free coefficient already derived on the cell is

`c_cell = 1/4`.

So the primitive unit boundary density is

`S_unit / k_B = c_cell A / a^2`.

If instead one writes

`S_mu / k_B = mu c_cell A / a^2`,

then:

- `mu = k`, with integer `k > 1`, counts `k` copies of each primitive
  incidence;
- `0 < mu < 1` assigns a fractional unit to an indivisible primitive incidence;
- non-integer `mu > 1` is a fractional copy/multiplicity weight;
- any `mu != 1` is therefore not the primitive unit-count object.

Thus, **inside the primitive unit-count object class**, `mu = 1`.

This is a real theorem, but it is not yet the full Planck theorem. A hostile
reviewer can still ask why the gravitational boundary action unit must be this
primitive unit-count object.

## Theorem 2: exact same-surface action kills multiplicative rescaling

On the exact boundary action lane, the retained microscopic action has the
quadratic form

`I_can(b ; j) = 1/2 b^T L_Sigma b - j^T b`.

If one tries to insert a multiplicative unit-map multiplier in the action,

`I_mu(b ; j) = mu/2 b^T L_Sigma b - j^T b`,

then the Hessian becomes

`mu L_Sigma`,

not

`L_Sigma`.

The stationary equation becomes

`mu L_Sigma b - j = 0`,

so the response becomes

`b = (1/mu) L_Sigma^(-1) j`.

Therefore `mu != 1` changes the microscopic operator and response. It is not a
same-surface normalization of the exact Schur boundary action.

So the exact action lane independently reaches the same conclusion:

> multiplicative `mu != 1` is not same-surface.

## Theorem 3: the surviving same-surface freedom is additive, not multiplicative

Fix the exact Hessian `L_Sigma` and source coupling `-j^T b`. Then a
same-surface action can differ from the canonical action only by a term
independent of `b`.

One-clock additivity forces this term to be extensive in worldtube duration:

`I_nu(tau ; b, j) = tau (1/2 b^T L_Sigma b - j^T b - nu)`.

`G_nu = nu I - L_Sigma`,

`p_*(nu) = nu - lambda_min(L_Sigma)`.

`nu = lambda_min(L_Sigma) + 1/4`.

`nu = 5/4`.
