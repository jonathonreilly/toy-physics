## Statement

For sites x, y ∈ Λ on the framework's N-site Fock space, define the
**symmetric hopping bilinear**

```text
    H_{xy}  :=  a_x^† a_y  +  a_y^† a_x.                                    (1)
```

Then:

**(B1) Hermiticity.** (H_{xy})^† = H_{xy} for all x, y.

**(B2) Translation covariance.** For every a ∈ Z^3:

```text
    T_a · H_{xy} · T_a^†  =  H_{x+a, y+a}                                  (2)
```

(periodic extension on a finite lattice).

**(B3) Charge conservation.** [H_{xy}, Q̂_total] = 0 (hopping conserves
total fermion number).

**(B4) Translation-invariant sum.** For any translation-invariant link
family L ⊂ Λ × Λ (i.e. (x, y) ∈ L iff (x + a, y + a) ∈ L for all a),

```text
    H_total  :=  Σ_{(x, y) ∈ L} H_{xy}                                      (3)
```

satisfies [T_a, H_total] = 0 for every a ∈ Z^3 (translation invariance)
and [H_total, Q̂_total] = 0 (charge conservation).

**(B5) Real spectrum.** H_total has real spectrum (from Hermiticity).

**(B6) Occupation-swap action.** H_{xy} maps the state |..., n_x = 1,
n_y = 0, ...⟩ to |..., n_x = 0, n_y = 1, ...⟩, and vice versa, preserving
all other site occupations and the total Q̂. Equivalently, H_{xy} acts as
the SWAP operator on the (n_x, n_y) = (1, 0) ↔ (0, 1) subspace.

## Proof

### Step 1 — Hermiticity (B1)

(a_x^† a_y)^† = a_y^† a_x. Adding: H_{xy}^† = (a_x^† a_y + a_y^† a_x)^† =
a_y^† a_x + a_x^† a_y = H_{xy} — establishing (B1).

### Step 2 — Translation covariance (B2)

By R7 Block 02 (translation covariance of local operators, on the same
cited N1 dependency), T_a a_x T_a^† = a_{x+a} (for the position label).
By R8 Block 01 / R7 Block 03 (per-site fermion construction), this
translates to T_a a_x^† T_a^† = a_{x+a}^† as well.

Hence T_a H_{xy} T_a^† = T_a (a_x^† a_y + a_y^† a_x) T_a^†
= a_{x+a}^† a_{y+a} + a_{y+a}^† a_{x+a}
= H_{x+a, y+a} — establishing (B2).

### Step 3 — Charge conservation (B3)

By R7 Block 03 (Q̂_total = Σ_x n̂_x with n̂_x = a_x^† a_x on Pauli C²),
[Q̂_total, a_x] = -a_x and [Q̂_total, a_x^†] = +a_x^† (lowering / raising
operators). Hence:

```text
    [Q̂, a_x^† a_y]  =  [Q̂, a_x^†] a_y  +  a_x^† [Q̂, a_y]
                    =  +a_x^† a_y  -  a_x^† a_y  =  0.                     (4)
```

Same for the conjugate term. So [H_{xy}, Q̂] = 0 — establishing (B3).

### Step 4 — Translation-invariant sum (B4)

By (B2), for any translation-invariant link family L:

```text
    T_a H_total T_a^†  =  Σ_{(x, y) ∈ L} T_a H_{xy} T_a^†
                       =  Σ_{(x, y) ∈ L} H_{x+a, y+a}
                       =  Σ_{(x', y') ∈ L} H_{x', y'}    (relabeling, L invariant)
                       =  H_total.                                          (5)
```

Hence [T_a, H_total] = 0 — establishing translation invariance.

Charge conservation [H_total, Q̂] = 0 follows by linearity of the sum from
(B3) — establishing (B4).

### Step 5 — Real spectrum (B5)

Hermitian operators have real spectrum (standard spectral theorem) —
establishing (B5).

### Step 6 — Occupation-swap action (B6)

Compute H_{xy} on the four-state subspace (n_x, n_y) ∈ {(0,0), (0,1),
(1,0), (1,1)}:
- a_x^† a_y · (1, 0) = 0 (a_y annihilates n_y = 0 ket).
- a_x^† a_y · (0, 1) = a_x^† · |n_x = 0, ..., n_y = 0, ...⟩ ·  (sign factor) — in our convention this is exactly |n_x = 1, n_y = 0⟩.

(Without Jordan-Wigner phases, on the framework's tensor-product fermion
construction, a_x and a_y commute as operators on disjoint factors. The
matrix element ⟨1, 0 | a_x^† a_y | 0, 1⟩ = 1.)

Symmetric: a_y^† a_x · (1, 0) = (0, 1).

The other 3 base states ((0,0), (1,1)) are annihilated by both terms.

Hence H_{xy} acts as the swap operator on the (1, 0) ↔ (0, 1) subspace,
preserving total occupation count — establishing (B6). ∎

## Corollaries

C1. **Tight-binding lattice Hamiltonian validity.** Any Hamiltonian of the
form H = Σ_{⟨xy⟩} t_{xy} H_{xy} with real coefficients t_{xy} and a
translation-invariant link family is automatically Hermitian, translation-
invariant, charge-conserving, and has real spectrum. This is the
foundational class of free-fermion Hamiltonians on the framework.

C2. **Lattice gauge generalization.** Adding a complex link variable U_{xy}
with U_{xy}^† = U_{yx} replaces H_{xy} → a_x^† U_{xy} a_y + h.c., giving
the gauge-invariant hopping bilinear used in lattice gauge theory. The
gauge-covariance machinery applies uniformly.

C3. **Q-sector preservation.** Since [H, Q̂] = 0, the Hamiltonian H_total
preserves each Q-eigenspace H_k (k ∈ {0, 1, ..., N}) of the Fock space.
Single-particle physics happens in H_1; many-body interactions happen in
H_k for k ≥ 2.

C4. **Spectrum is symmetric under particle-hole.** With nearest-neighbor
hopping on a bipartite lattice, the spectrum of H is symmetric around 0:
for every eigenvalue E, -E is also an eigenvalue (chiral symmetry from the
Cl(3) sublattice parity, see CPT_EXACT_NOTE for related structure).

C5. **Free-fermion eigenstates are plane waves.** On a translation-invariant
H_total, joint eigenstates of H and T_a are momentum eigenstates |k⟩ with
energies E(k) = -2 t cos(k·a) for nearest-neighbor hopping (standard
tight-binding dispersion).

C6. **Hopping induces nearest-neighbor entanglement.** Acting on a product
state, the hopping bilinear creates Bell-like pairings between adjacent
sites — the basic mechanism of entanglement spreading in lattice fermion
dynamics.
