## 2. Proof

The proof of inequalities (1)вҖ“(4) is classical Banach-space functional
analysis; we record it here for completeness so that the note is
self-contained on its load-bearing chain.

### 2.1 Inequality (1): N-step operator-norm bound

For any bounded linear operator `T` on a Banach space `(B, вҖ–В·вҖ–)`, the
operator-norm inequality `вҖ–T(x)вҖ– вүӨ вҖ–TвҖ–_op В· вҖ–xвҖ–` holds for every
`x вҲҲ B` by the definition of `вҖ–TвҖ–_op`. Iterating,

```text
вҖ–T^2(x_0)вҖ–  вүӨ  вҖ–TвҖ–_op В· вҖ–T(x_0)вҖ–  вүӨ  вҖ–TвҖ–_op^2 В· вҖ–x_0вҖ–,
```

and by induction on `N`,

```text
вҖ–T^N(x_0)вҖ–  вүӨ  вҖ–TвҖ–_op^N В· вҖ–x_0вҖ–  вүӨ  Оә^N В· вҖ–x_0вҖ–. в–Ў
```

### 2.2 Inequality (2): composition of distinct contractions

If `T_j` are bounded linear operators with `вҖ–T_jвҖ–_op вүӨ Оә_j`, sub-
multiplicativity of the operator norm gives

```text
вҖ–T_N вҲҳ вӢҜ вҲҳ T_1вҖ–_op  вүӨ  вҲҸ_{j=1}^{N} вҖ–T_jвҖ–_op  вүӨ  вҲҸ_{j=1}^{N} Оә_j  вүӨ  Оә^N.
```

Applied to `x_0` yields (2). The Lipschitz-self-map generalisation
replaces `вҖ–TвҖ–_op` by the Lipschitz constant; the same composition
argument applies. в–Ў

### 2.3 Inequality (3): Banach fixed-point error bound

When `T : B вҶ’ B` is a strict contraction with Lipschitz constant
`Оә < 1` on a complete metric space (in particular a Banach space),
the Banach fixed-point theorem (Banach, 1922) yields a unique fixed
point `x_* вҲҲ B` with `T(x_*) = x_*`. Subtracting `x_*` from both
sides of `T^N(x_0)` and applying (1) gives

```text
вҖ–T^N(x_0) вҲ’ x_*вҖ–  =  вҖ–T^N(x_0) вҲ’ T^N(x_*)вҖ–  вүӨ  Оә^N В· вҖ–x_0 вҲ’ x_*вҖ–. в–Ў
```

### 2.4 Inequality (4): geometric-series tail bound

For real `0 вүӨ Оә < 1` and integer `N вүҘ 0`, the geometric-series identity

```text
ОЈ_{k=0}^{вҲһ} Оә^k  =  1 / (1 вҲ’ Оә)
```

is the standard real-analysis statement; subtracting the truncated
head gives

```text
ОЈ_{k=N}^{вҲһ} Оә^k  =  Оә^N В· ОЈ_{j=0}^{вҲһ} Оә^j  =  Оә^N / (1 вҲ’ Оә). в–Ў
```
