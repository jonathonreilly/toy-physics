## Claim scope

Let `H` be a finite-dim complex Hilbert space with `dim_C H = D < ∞`.
Consider candidate canonical-quantisation rules for a single creation /
annihilation operator pair `(a, a†)` on `H` of one of the two standard
types:

- **Bosonic (CCR).** `[a, a†] = I`.
- **Fermionic (CAR).** `{a, a†} = I`, `{a, a} = 0`, `{a†, a†} = 0`.

Then:

- **(C1) Bosonic CCR forces `dim H = ∞`.** No finite-dim complex
  Hilbert space `H` (any `D < ∞`) admits operators `a, a†` satisfying
  `[a, a†] = I`. Concretely, taking the trace of both sides gives
  `Tr([a, a†]) = 0 = Tr(I) = D`, forcing `D = 0`, contradicting
  faithfulness of the algebra.
- **(C2) Fermionic CAR has a 2-dim irreducible single-mode
  realisation.** The Grassmann single-mode Hilbert space
  `F_x = span(|0⟩, |1⟩)` of dimension 2 admits operators
  `(c, c†)` satisfying CAR. Concretely, `c = [[0, 1], [0, 0]]`
  and `c† = [[0, 0], [1, 0]]` realise CAR on `C²`. General
  finite-dim representations of the one-pair CAR algebra may be
  direct sums of this 2-dim irreducible carrier; this note only uses
  the irreducible/minimal single-mode carrier.
- **(P1) Squared creation operator vanishes (Pauli exclusion).**
  From the CAR same-mode anticommutator `{a†, a†} = 2(a†)² = 0`,
  the squared creation operator on any single mode satisfies
  `(a†)² = 0` as an operator identity.
- **(P2) Single-mode occupation `n_φ ∈ {0, 1}`.** The number operator
  `n := a† a` on a single fermionic mode has spectrum `{0, 1}` —
  equivalently, `n² = n` (idempotent).

(C1)-(C2) are pure cardinality / linear-algebra statements about
finite-dim complex matrix algebras; (P1)-(P2) are one-line algebraic
corollaries of CAR.

## Proof

### (C1) Bosonic CCR forces `dim H = ∞`

Suppose `(a, a†)` are linear operators on a finite-dim complex
Hilbert space `H` with `dim_C H = D < ∞`, satisfying

```text
[a, a†]  =  a a† - a† a  =  I_H.
```

Take the trace of both sides:

```text
Tr(a a†) - Tr(a† a)  =  Tr(I_H).
```

By the cyclic property of trace, `Tr(a a†) = Tr(a† a)`, so the
left-hand side is `0`. The right-hand side is `Tr(I_H) = D`.
Therefore `0 = D`, forcing `D = 0`. Hence no `D ≥ 1` finite-dim
complex Hilbert space admits CCR operators `(a, a†)`. ∎

### (C2) Fermionic CAR realised on `dim H = 2`

Define on `C²` the matrices

```text
c     =  [[0, 1], [0, 0]],
c†    =  [[0, 0], [1, 0]].
```

Direct matrix computation gives

```text
c c†   =  [[1, 0], [0, 0]],
c† c   =  [[0, 0], [0, 1]],
{c, c†}  =  c c† + c† c  =  [[1, 0], [0, 1]]  =  I_2,
c²     =  [[0, 0], [0, 0]]  =  0,
(c†)²  =  [[0, 0], [0, 0]]  =  0.
```

So `{c, c†} = I`, `{c, c} = 0`, `{c†, c†} = 0`. Hence `dim H = 2`
is a realisable CAR Hilbert dimension. ∎

### (P1) Squared creation operator vanishes

Specialise the CAR same-mode anticommutator `{a†, a†} = 0` to
`(a†)²`:

```text
{a†, a†}  =  a† a† + a† a†  =  2 (a†)²  =  0,
⇒  (a†)²  =  0.
```

This holds as an operator identity on any CAR Hilbert space. ∎

### (P2) Single-mode occupation `n_φ ∈ {0, 1}`

Define the number operator `n := a† a`. Using `{a, a†} = I` and
`a² = (a†)² = 0`:

```text
n²  =  a† a a† a
    =  a† (I - a† a) a       (from a a† = I - a† a)
    =  a† a - a† a† a a
    =  a† a - 0              (from (a†)² = 0)
    =  n.
```

Hence `n² = n`, so `n` is idempotent. Its eigenvalues satisfy
`λ² = λ`, so `λ ∈ {0, 1}`. ∎

### (Corollary) Two-dim irreducible carrier per fermionic mode

Combining (C1)-(C2)-(P1)-(P2): the finite-dim bosonic CCR alternative
is forbidden by (C1), while the one-pair fermionic CAR algebra has a
2-dim irreducible Grassmann carrier. General finite-dimensional CAR
representations can be direct sums of that carrier, so this corollary
uses the irreducible/minimal single-mode reading. The two-dim
irreducible carrier matches the abstract Cl(3)-irrep dimension
