## Statement

Let `n_color` be an abstract positive integer. Fix the left-handed quark and
lepton hypercharges as
`Y_QL = a`, `Y_LL = b = -n_color a` (the trace-equation forcing from the
sister narrow theorem
`LH_TRACELESS_EIGENVALUE_RATIO_NARROW_THEOREM_NOTE_2026-05-10.md`),
with the labelling convention `b = -1` so `a = 1/n_color`.

Consider four right-handed SU(2)_L-singlet hypercharges
`(Y_uR, Y_dR, Y_eR, Y_nuR)`. Impose:

**(SHIFT) Standard singlet charge-readout / Y-shift relation** on the up-,
down-, charged-lepton, and neutral-lepton sectors, with
`Q(u) = a/2 + 1/2`, `Q(d) = a/2 - 1/2`, `Q(e) = b/2 - 1/2 = -1` under
`b = -1`, and `Q(nu) = b/2 + 1/2 = 0` under `b = -1`:

```text
(SHIFT.u)   Y_uR   =  2 Q(u)    =  a + 1,
(SHIFT.d)   Y_dR   =  2 Q(d)    =  a - 1,
(SHIFT.e)   Y_eR   =  2 Q(e)    =  b - 1,
(SHIFT.nu)  Y_nuR  =  2 Q(nu)   =  b + 1.
```

Equivalently, in the simpler shift-by-one form
(consistent with the standard `Q = T_3 + Y/2` hypercharge convention used in
the broad note's `Q = T_3 + Y/2` framing):

```text
(SHIFT.u)   Y_uR  =  a + 1,
(SHIFT.d)   Y_dR  =  a - 1,
(SHIFT.e)   Y_eR  =  b - 1,
(SHIFT.nu)  Y_nuR =  b + 1.
```

Under `b = -1` and `a = 1/n_color`, `(SHIFT)` forces `Y_eR = -2`,
`Y_nuR = 0`, `Y_uR = 1/n_color + 1`, and
`Y_dR = 1/n_color - 1`.

**(COLOR_ANOM) Gauge `[SU(N)]^2 U(1)_Y` anomaly cancellation**: the SU(2)
anomaly is satisfied because `Y_QL` and `Y_LL` are the only SU(2)-doublet
hypercharges and `2 Y_QL n_color + 2 Y_LL = 2 (n_color a + b) = 0` by trace
hypothesis. The SU(3)_c anomaly:

```text
(COLOR_ANOM)   2 n_color Y_QL  -  n_color Y_uR  -  n_color Y_dR  =  0,
             equivalently  2 a  -  (Y_uR + Y_dR)  =  0.
```

**(GRAV_ANOM) Mixed gauge-gravity `[grav]^2 U(1)_Y` (trace condition over all
fermions)**:

```text
(GRAV_ANOM)   2 n_color Y_QL  +  2 Y_LL  -  n_color Y_uR  -  n_color Y_dR  -  Y_eR  -  Y_nuR  =  0.
```

**(CUBIC_ANOM) Cubic `[U(1)_Y]^3` anomaly cancellation**:

```text
(CUBIC_ANOM)   2 n_color Y_QL^3  +  2 Y_LL^3
        -  n_color Y_uR^3  -  n_color Y_dR^3  -  Y_eR^3  -  Y_nuR^3  =  0.
```

**(NEUTRAL_BRANCH) Branch-selection convention** (named, not derived in this row):
the neutral-singlet branch keeps `Y_nuR = b + 1`, so `Y_nuR = 0` under the
`b = -1` convention. The `e_R <-> nu_R` relabelling swaps the two lepton
singlet values and is a discrete symmetry that `(COLOR_ANOM)`-`(CUBIC_ANOM)`
admit.

**Conclusion.** Under `(SHIFT)`-`(NEUTRAL_BRANCH)`, the four right-handed hypercharges are
forced to the closed forms

```text
(R1)  Y_uR    =  a + 1   =  (n_color + 1) / n_color,
(R2)  Y_dR    =  a - 1   =  (1 - n_color) / n_color,
(R3)  Y_eR    =  b - 1   =  -2     under  b = -1,
(R4)  Y_nuR   =  b + 1   =  0      under  b = -1, by branch convention
      (NEUTRAL_BRANCH).
```

The framework instance `n_color = 3` gives the rational closed-form readout

```text
Y_uR   =  4/3,
Y_dR   =  -2/3,
Y_eR   =  -2,
Y_nuR  =  0.
```

In the `Q = T_3 + Y/2` convention used by
`ONE_GENERATION_MATTER_CLOSURE_NOTE.md`,
`u_R : (1,3)_{+4/3}`, `d_R : (1,3)_{-2/3}`, `e_R : (1,1)_{-2}`, `nu_R : (1,1)_0`),
the values match the broad note's right-handed Y assignment at `n_color = 3`.

## Proof

`(R1)`, `(R2)`, `(R3)` follow directly from `(SHIFT)` by substitution.

For `(R4)`, the neutral-singlet branch in `(SHIFT.nu)` gives
`Y_nuR = b + 1`, so the convention `b = -1` gives `Y_nuR = 0`.

The anomaly checks are then direct:

- `Y_uR + Y_dR = (a + 1) + (a - 1) = 2a`, so `(COLOR_ANOM)` is
  `2a - 2a = 0`.
- `(GRAV_ANOM)` evaluates to
  `2 n_color a + 2 b - n_color(a + 1) - n_color(a - 1) - (b - 1) - (b + 1) = 0`.
- `(CUBIC_ANOM)` evaluates to
  `2 n_color a^3 + 2 b^3 - n_color[(a + 1)^3 + (a - 1)^3] - [(b - 1)^3 + (b + 1)^3]`
  `= -6(n_color a + b)`, which vanishes by the LH trace relation
  `b = -n_color a`.

The runner verifies these identities exactly and also checks that swapping
the two lepton singlet values is anomaly-consistent. ∎
