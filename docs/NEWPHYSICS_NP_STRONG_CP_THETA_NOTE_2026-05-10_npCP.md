## Operator-class basis and CP weights

Within the single-plaquette class, the admissible action term is
any real-valued class function `f(U_P)` on SU(3). The algebra of
SU(3) class functions over the complex field is generated by the
characters `chi_R(U_P) = tr_R(U_P)` for irreducible representations
`R`. For the fundamental representation, `chi_fund(U_P) = tr U_P`
is complex-valued; its real and imaginary parts span the
real-valued class-function space at the fundamental level:

```text
    O_+ := Re tr U_P,         O_- := Im tr U_P.                  (1)
```

CP transformation on a single plaquette acts by complex conjugation
of the SU(3) holonomy together with parity reversal of one spatial
axis (standard lattice CP):

```text
    CP : U_P  ->  U_P^*.                                          (2)
```

Under (2):

```text
    Re tr U_P  -> + Re tr U_P    (CP-even),
    Im tr U_P  -> - Im tr U_P    (CP-odd).                         (3)
```

Hence within the single-plaquette class there is exactly **one**
CP-odd dim-4 candidate at the fundamental-character level: the
operator `O_-` of (1). The analogous argument at higher characters
(e.g. `tr U_P^2`, adjoint character) gives further CP-odd
candidates `Im chi_R(U_P)`, but they are subleading at the
continuum F^2 order and are not the load-bearing slot for an `F̃F`
theta-term.

## Continuum-limit obstruction theorem

**Theorem (single-plaquette CP-odd operator does not encode `F̃F`).**
Let `S_- := sum_P g_- Im tr U_P` be any single-plaquette CP-odd
addition to the action, with real coupling `g_-`. Then in the
continuum expansion at small lattice spacing `a`,

```text
    S_-  =  O(a^6) (multi-derivative non-topological)              (4)
```

i.e. `S_-` contains *no* leading-order `F̃F` contribution. In
particular,

```text
    lim_{a -> 0} S_-  CANNOT  reproduce  (g^2/32 pi^2) integral d^4 x  F~F.
```

**Proof.** Let `T_a` (`a = 1, ..., 8`) be the canonical su(3)
generators with `Tr(T_a T_b) = delta_{ab}/2`, fixed by the Cl(3)
chain (Claims 1, 2 of
[`G_BARE_STRUCTURAL_NORMALIZATION_THEOREM_NOTE_2026-04-18.md`](G_BARE_STRUCTURAL_NORMALIZATION_THEOREM_NOTE_2026-04-18.md)),
all Hermitian. The continuum gauge field is `A_mu = A^a_mu T_a` and
the field strength

```text
    F_{mu nu} = partial_mu A_nu - partial_nu A_mu + i g [A_mu, A_nu]   (5)
```

is Hermitian as well.

For an elementary plaquette in the `(mu, nu)` plane (with `mu < nu`)
at site `x`, the standard parallel-transport expansion (Lüscher 1982;
Münster 1981) gives the SU(3) plaquette holonomy as a single-plane
matrix exponential

```text
    U_P^{(mu nu)}(x)  =  exp( i a^2 g F_{mu nu}(x)
                              + i a^3 g (D_rho F_{mu nu})(x) /2
                              + O(a^4) )                          (6)
```

with NO sum on `mu, nu`. Crucially, the entire `O(a^2)` and
`O(a^3)` content of `U_P^{(mu nu)}` involves field-strength
components from the *same* `(mu, nu)` plane only — the holonomy of
a single plaquette knows about only one spacetime plane.

**Step 1: each character is a class function on SU(3) with no
spacetime index structure.**

By gauge invariance, an action term that depends on a single
plaquette holonomy `U_P^{(mu nu)}(x)` must be a class function
`f : SU(3) -> R`. By the Peter-Weyl decomposition of class
functions on a compact Lie group, `f(U)` is a sum of characters:

```text
    f(U)  =  sum_R c_R chi_R(U) = sum_R c_R tr_R(U).               (7)
```

Each character `chi_R(U)` is a scalar function on SU(3); it carries
no spacetime index, only the matrix data of `U`. Hence each
`f(U_P^{(mu nu)}(x))` is a function whose only `(mu, nu)`-index
structure is inherited from the *single plane* `(mu, nu)` of the
plaquette — never from cross-plane Levi-Civita contractions
between distinct orientations.

**Step 2: continuum expansion of `Im tr U_P^{(mu nu)}` is cubic
in `F`, not bilinear, and lives entirely on one plane.**

Substituting (6) into `tr U_P^{(mu nu)}` and using `tr T_a = 0`,
`tr T_a T_b = delta_{ab}/2`:

```text
    tr U_P^{(mu nu)}
      =  N_c
         +  i a^2 g tr F_{mu nu}                                   (k=1, vanishes)
         +  (i a^2 g)^2 / 2  tr F_{mu nu}^2                         (k=2)
         +  (i a^2 g)^3 / 6  tr F_{mu nu}^3                         (k=3)
         +  O(a^8).                                                (8)
```

Term k=1: `tr F_{mu nu} = (F_{mu nu})^a tr T_a = 0`.
Term k=2: `(i a^2 g)^2 = -a^4 g^2` is real, so this contributes to
   `Re tr U_P`, not `Im tr U_P`. Concretely
   `-(a^4 g^2 / 2) tr F_{mu nu}^2` with `tr F^2` real and positive.
Term k=3: `(i a^2 g)^3 = -i a^6 g^3`, contributing to `Im tr U_P`
   as `-(a^6 g^3 / 6) tr F_{mu nu}^3`.

Therefore the leading non-vanishing imaginary part is

```text
    Im tr U_P^{(mu nu)}(x)  =  - (a^6 g^3 / 6) tr F_{mu nu}(x)^3
                                +  O(a^7).                          (9)
```

The result (9) is

(i) **single-plane**: contains only `F_{mu nu}` (no `F_{rho sigma}`
    from an orthogonal plane);
(ii) **cubic in `F`**: third power of the field strength, not
     bilinear; and
(iii) **`O(a^6)`**: subleading to the `O(a^4)` topological-density
      scaling.

**Step 3: F̃F is intrinsically bilinear and intrinsically
multi-plane.**

The continuum CP-odd topological density is

```text
    q(x)  =  (1 / 32 pi^2) epsilon_{mu nu rho sigma}
                tr [ F_{mu nu}(x) F_{rho sigma}(x) ].              (10)
```

It has three structural features inherited from continuum
Yang-Mills:

(I) **bilinear in `F`**, contracting two field-strength tensors;
(II) **two distinct planes**: `(mu, nu) != (rho, sigma)` in every
     nonvanishing term, since `epsilon_{mu nu rho sigma}` vanishes
     whenever any two of `mu, nu, rho, sigma` coincide;
(III) **Levi-Civita-weighted**: the four-index `epsilon` tensor
      enforces a specific orientation assignment between the two
      planes.

**Step 4: single-plane class-function terms cannot reproduce (10).**

By Step 1, any single-plaquette action term `f(U_P^{(mu nu)}(x))`
expands as a sum of single-plane characters; by Step 2, the
imaginary part of each character is at most cubic-in-`F` single-plane
content at `O(a^6)`.

Suppose, for contradiction, that the lattice action

```text
    S_-  =  sum_x sum_{(mu nu)}  c_{(mu nu)} Im tr U_P^{(mu nu)}(x)   (11)
```

reproduces `q(x)` at leading order. Then, summing (9) inside (11)
gives a leading content

```text
    S_-  ~  - (a^6 g^3 / 6) sum_x sum_{(mu nu)}
                c_{(mu nu)} tr F_{mu nu}(x)^3  +  O(a^7).            (12)
```

For (12) to match `int d^4 x q(x)` at leading order, we would need
`tr F_{mu nu}^3` summed with `c_{(mu nu)}` to equal
`epsilon_{mu nu rho sigma} tr F_{mu nu} F_{rho sigma}` summed with
`a^4` weight. But these two objects are structurally distinct:

- `tr F^3` is cubic; `tr F F` is quadratic. **Power counting fails.**
- `tr F_{mu nu}^3` is single-plane; the `epsilon`-contraction is
  two-plane. **Index structure fails.**
- `tr F^3` at `O(a^6)` is volume-suppressed relative to `tr F^2` at
  `O(a^4)`; the continuum limit `a -> 0` drives (12) to zero
  *faster* than the F̃F topological scale. **Scaling fails.**

No choice of `c_{(mu nu)}` can simultaneously upgrade `O(a^6)` to
`O(a^4)`, change cubic to bilinear, and change single-plane to
two-plane.

**Step 5: multi-plaquette structure is unavoidable for F̃F.**

The lattice realisation of (10) — the clover discretisation — is

```text
    q_L(x)  =  (1 / 32 pi^2) epsilon_{mu nu rho sigma}
               tr [ F_{mu nu}^{clov}(x) F_{rho sigma}^{clov}(x) ]   (13)
```

where `F_{mu nu}^{clov}(x)` is built from four plaquettes meeting
at `x` in the `(mu, nu)` plane (Smit-Vink 1987). The product of
two clover operators at the same site involves *eight* distinct
plaquettes — manifestly a multi-plaquette object.

Any single-plaquette action term reduces to a single field-strength
factor per site at the relevant order (Step 2) and cannot express
the bilinear two-plane structure of (13). QED.

**Corollary (single-plaquette action class cannot encode `F̃F` at
leading order).** Within the single-plaquette action class
`S(U) = sum_P f(U_P)` with `f` a real-valued class function on
SU(3), no `F̃F` coupling can appear at leading `O(a^4)` in the
continuum-limit action. The lattice class-function basis admits
CP-odd contributions only at order `a^6` (cubic-in-F, single-plane),
which are *not* the topological density. Equivalently, the `F̃F`
slot is structurally absent from this action class as an
**action-form admissibility statement**:

```text
    no F̃F coupling encodable by single-plaquette f(U_P) at
    leading O(a^4).                                                 (10)
```

**This is NOT a derivation of `theta_QCD = 0` in QCD.** Under the
admitted single-plaquette action-class scope (above), the bare-θ
slot has no place in the action; this is the *dual* statement to
"no CP-odd action-form admissible," not a dynamical-selection
result. A multi-plaquette generalization (clover-style two-plaquette
correlators) can, *a priori*, encode `F̃F` at leading order and is
the named open boundary of this note.

## What this closes and what it does not

**Closes (within the admitted single-plaquette action class scope):**

- The first Path A repair target from the parent's audit verdict
  ("retained operator-basis / action-surface theorem deriving from
  Cl(3)/Z^3 primitives ... that no gauge-invariant CP-odd theta term
  is an admissible action slot") — *bounded to* the admitted
  single-plaquette class. The closure is conditional on the
  action-class admission, not a derivation of the action-class itself
  from A1+A2.
- The structural origin of the parent runner's "no bare theta slot"
  support item: equation (10) gives an *action-form admissibility*
  derivation from A1+A2 + the admitted single-plaquette class,
  rather than taking it as part of the action-class definition.

**Does not close:**

- A multi-plaquette generalization. Two-plaquette operators of the
  form `sum_x epsilon_{mu nu rho sigma} f(U_P^{(mu nu)}(x))
  g(U_P^{(rho sigma)}(x))` can, *a priori*, encode `F̃F` at leading
  order. The single-plaquette action class restriction is the named
  open boundary of this note.
- The positive real quark-mass orientation
  (`arg det(M_u M_d) = 0`) derivation from framework primitives.
  This is the second Path A repair target and is explicitly *not*
  addressed by this note.
- Dynamical selection of `theta = 0` versus evaluation of a
  `theta`-free surface. Equation (10) shows that no `F̃F` coupling
  can be encoded by single-plaquette `f(U_P)` at leading `O(a^4)`;
  equivalently, the bare-θ slot is structurally absent from the
  admitted single-plaquette action class as an action-form
  admissibility statement. This is **NOT** a derivation of
  `theta_QCD = 0` and **NOT** a dynamical-selection statement.

The action-form-uniqueness no-go
([`BRIDGE_GAP_ACTION_FORM_UNIQUENESS_NO_GO_NOTE_2026-05-06.md`](BRIDGE_GAP_ACTION_FORM_UNIQUENESS_NO_GO_NOTE_2026-05-06.md))
already establishes that the framework's retained primitive stack
does *not* uniquely select an action functional — Wilson,
Heat-Kernel, and Manton are jointly compatible with all current
primitives, and each is a single-plaquette class function. The
*shared* feature of every candidate that survives that no-go's
continuum-limit screen is the single-plaquette structure.

Therefore: the natural action-class scope for any bridge derivation
that the parent strong-CP row needs is exactly the single-plaquette
class. Equation (10) closes the CP-odd action-slot question on
that class — the same class within which Wilson, Heat-Kernel, and
Manton already sit.

The companion runner [`scripts/frontier_np_strong_cp_theta_obstruction.py`](../scripts/frontier_np_strong_cp_theta_obstruction.py)
verifies each load-bearing step of the proof numerically:

1. The Cl(3) → su(3) canonical generators satisfy
   `Tr(T_a T_b) = delta_{ab}/2` and `tr T_a = 0`.
2. On random small-`a` SU(3) plaquette holonomies built from
   continuum field-strength inputs, the imaginary trace `Im tr U_P`
   scales as `a^6`, not `a^4`, and matches the cubic-`F` structure
   of (8) rather than the bilinear-`F` structure of `F̃F`.
3. The lattice topological charge from the two-plaquette clover
   construction (9) scales correctly as `a^4` and is *not*
   reproduced by any single-plaquette function `f(Im tr U_P)`.
4. Direct linear-fit verification: regressing `Im tr U_P` against
   the clover-`F̃F` density across an ensemble of small-`a`
   plaquettes gives a coefficient consistent with zero (within
   numerical noise), confirming that `Im tr U_P` does not encode
   the topological density.
5. Explicit CP transformation `U_P -> U_P^*` flips the sign of
   `Im tr U_P` (CP-odd label) and preserves `Re tr U_P` (CP-even
   label).
6. Class-function reality check: every Wilson / Heat-Kernel /
   Manton single-plaquette action evaluates to a real number on
   random SU(3) plaquette holonomies, confirming the single-plaquette
   class scope is closed under the retained action candidates.
