## Theorem statement

> **Three-Generation Observable Count Corollary.**
> Let `H_hw=1 = C^3` be the retained `hw=1` sector of the staggered
> Dirac / Wilson Hamiltonian on `Z^3`, with the retained operator
> algebra generated by `{T_x, T_y, T_z, C3[111]}`. Let
> `Q : H_hw=1 -> H_red` be any linear quotient preserving the full
> retained generation algebra (i.e. each generator descends to
> `H_red`). Then `H_red = H_hw=1`, and the observable count
> `card(hw=1) = 3` is exactly preserved.

## Proof skeleton

### 1. Orbit count gives the input cardinality

By the retained orbit/spectral structure (Hamming-weight degeneracy
`1 + 3 + 3 + 1` on `{0, pi}^3`), the lightest nonzero-mass corner
class has cardinality exactly three. This is the kinematic input.

### 2. Algebra irreducibility forbids proper quotient

By the no-proper-quotient theorem on `H_hw=1`, the retained
generation operator algebra equals `M_3(C)` and acts irreducibly.
Any quotient `Q` that preserves all retained generators must have
`ker(Q)` invariant under `M_3(C)`. The only `M_3(C)`-invariant
subspaces of `C^3` are `{0}` and all of `H_hw=1`. The case
`ker(Q) = H_hw=1` is excluded since `Q` is supposed to be a quotient
to a nonzero algebra-preserving target. Hence `ker(Q) = {0}` and
`Q` is an isomorphism.

### 3. Count preservation

Under `Q` an isomorphism, `dim(H_red) = dim(H_hw=1) = 3`. Combined
with the orbit-count step, the observable count `card(hw=1) = 3` is
exactly preserved.
