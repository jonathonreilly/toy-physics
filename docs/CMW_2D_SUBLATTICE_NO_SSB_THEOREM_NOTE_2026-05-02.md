## Admitted-context inputs

- **Sublattice definition.** A 1D sublattice of Z^3 is a coordinate-aligned line `{x ∈ Z^3 : x_2 = c_2, x_3 = c_3}` for fixed c_2, c_3. A 2D sublattice is a coordinate-aligned plane `{x ∈ Z^3 : x_3 = c_3}`. These are isomorphic to Z^1 and Z^2 as abstract lattices. Pure structural definition.
- **Restriction of global Hamiltonian to sublattice.** A Hamiltonian H_full on Z^3 with translation invariance restricts naturally to a 1D or 2D Hamiltonian on a sublattice if and only if the restriction is itself translation-invariant in the sublattice directions. Standard restriction map.
- **Translation invariance of restricted Hamiltonian.** Required hypothesis on the restriction. Pure statement of translation symmetry.

No physics conventions admitted beyond what the cited CMW theorem already provides.

## Statement

Let `Λ_2D ⊂ Z^3` be a 2-dimensional sublattice (plane) and `Λ_1D ⊂ Z^3` be a 1-dimensional sublattice (line). Let `H_2D` be a translation-invariant Hermitian Hamiltonian on `Λ_2D` carrying a continuous global symmetry generated by `Q_2D = Σ_{x ∈ Λ_2D} q_x` with `[H_2D, Q_2D] = 0`. Similarly for `Λ_1D` and `Q_1D`. Then, assuming the cited CMW authority:

**(S1) 2D no-SSB.** At any finite temperature T > 0, the Gibbs expectation value of the local symmetry density on `Λ_2D` vanishes:

```text
    ⟨q_x⟩_β  =  ⟨q_0⟩_β  =  0  for every x ∈ Λ_2D and every q in the symmetry algebra.    (1)
```

**(S2) 1D no-SSB.** Same conclusion for `Λ_1D`:

```text
    ⟨q_x⟩_β  =  0  for every x ∈ Λ_1D.                                                    (2)
```

**(S3) Continuous no-order conclusion.** No equilibrium phase on the sublattice can carry a nonzero order parameter for the continuous symmetry generated by Q at finite temperature.

**(S4) Universality across embeddings.** (S1)–(S3) hold independently of which specific 1D line or 2D plane is chosen, since CMW depends only on the dimension `d` of the substrate. The framework's cited Z^3 substrate admits 3 inequivalent classes of 2D sublattices (one per coordinate direction) and 3 classes of 1D sublattices; all 6 obey (S1)–(S2).

(S1)–(S4) constitute the no-SSB-on-low-dimensional-sublattice theorem on the framework's cited CMW surface.

## Proof

The proof is direct application of the cited CMW theorem MW1 to the sublattice setting.

### Step 1 — Identify sublattice as Z^d for d ≤ 2

The 2D coordinate-aligned plane `Λ_2D ⊂ Z^3` is isomorphic as a graph to the 2D lattice `Z^2`. Similarly `Λ_1D ≅ Z^1`. The lattice site algebra at each point of the sublattice inherits from Cl(3) ⊗ Z^3 (or any restriction of it).

### Step 2 — Apply CMW MW1

By the cited CMW theorem MW1 ([`AXIOM_FIRST_COLEMAN_MERMIN_WAGNER_THEOREM_NOTE_2026-04-29.md`](AXIOM_FIRST_COLEMAN_MERMIN_WAGNER_THEOREM_NOTE_2026-04-29.md)), for any translation-invariant Hermitian Hamiltonian on a substrate of dimension `d ≤ 2` carrying a continuous global symmetry, the order parameter vanishes at any finite T > 0.

Since `dim(Λ_2D) = 2 ≤ 2` and `dim(Λ_1D) = 1 ≤ 2`, MW1 applies directly. Therefore `⟨q_x⟩_β = 0` on each sublattice, establishing (S1) and (S2). ∎

### Step 3 — Continuous no-order conclusion (S3)

If a finite-temperature phase on the sublattice carried a nonzero order
parameter for the continuous symmetry generated by Q, it would be a
continuous-symmetry-broken phase. By (S1)/(S2), that order parameter
vanishes. ∎

### Step 4 — Universality across embeddings (S4)

The CMW theorem depends only on the substrate dimension d, not on the specific embedding. The framework's Z^3 admits 3 coordinate-aligned 2D planes (per the choice of fixed coordinate) and 3 coordinate-aligned 1D lines. Step 2's argument applies to each sublattice independently. ∎

## Corollaries

C1. **No 2D ferromagnetism on framework Z² planes at finite T.** The 2D Heisenberg ferromagnet has SU(2) global symmetry; (S1) forbids SSB of SU(2) at any T > 0. (Quantum analogue of Mermin-Wagner 1966 result, applied to framework substructures.)

C2. **No 2D antiferromagnetism either.** Same SU(2) global symmetry; same conclusion.

C3. **Layered 3D systems have anisotropic SSB.** A 3D system that is weakly coupled in one direction (effectively 2D) is on the boundary; weak inter-layer coupling can stabilize SSB in the full 3D system that would not exist in the strictly 2D restriction.

C4. **Goldstone modes only in d = 3 framework regimes.** If continuous symmetry is to be spontaneously broken on the framework, it must be broken at the full 3D Z^3 level (where MW2 of the cited CMW note allows SSB). 1D and 2D substructures cannot carry Goldstone bosons.

C5. **Validation of d = 3 substrate choice.** The cited CMW MW4 establishes that d = 3 is the minimal integer dimension simultaneously admitting long-range force and continuous SSB. Theorem (S1)–(S2) here is a constructive sub-result: no lower-dimensional substructure of Z^3 can host continuous SSB on its own.
