## Statement

Let `n_color` be an abstract positive integer and let `a, b` be real numbers
satisfying the single homogeneous trace equation on the left-handed `(2 n_color + 2)`
state count carried by one weak quark doublet (with `2 n_color` color-component
states) plus one weak lepton doublet (with `2` states):

```text
(L1)  2 n_color * a + 2 * b  =  0.
```

Then:

```text
(R1)  a : b  =  1 : (-n_color),
(R2)  setting the labelling convention b = -1,
        a  =  1 / n_color,
(R3)  the two left-handed quark electric charges under the standard
      `Q = T_3 + Y/2` hypercharge convention are
        Q(u_L)  =  (n_color + 1) / (2 n_color),
        Q(d_L)  =  (1 - n_color) / (2 n_color),
(R4)  the reduced (lowest-terms) denominator d_red(n_color) of
      Q(u_L), Q(d_L) is
        d_red(n_color)  =  n_color           if n_color is odd,
        d_red(n_color)  =  2 n_color         if n_color is even.
```

The framework-specific instance `n_color = 3` gives the rational closed-form readout

```text
a  =  1/3,
Q(u_L)  =   2/3,
Q(d_L)  =  -1/3,
d_red(3)  =  3.
```

This special case is shown for sanity, not as a load-bearing input. The algebra also
closes for any other positive integer `n_color` (e.g. the `n_color = 5` case verified
in the runner returns `1/5, 3/5, -2/5, d_red = 5`).

## Proof

`(R1)` Solve the single linear equation `(L1)`:

```text
2 n_color * a + 2 * b  =  0
        n_color * a + b  =  0
                      b  =  - n_color * a,
hence  a : b  =  1 : (-n_color).
```

`(R2)` Setting `b = -1` gives `-1 = -n_color * a`, hence `a = 1/n_color`.

`(R3)` Under `Q = T_3 + a/2`, the two SU(2) components of `Q_L` have `T_3 = +1/2`
(up-type) and `T_3 = -1/2` (down-type). Substituting `a = 1/n_color`:

```text
Q(u_L)  =  +1/2 + 1/(2 n_color)  =  (n_color + 1) / (2 n_color),
Q(d_L)  =  -1/2 + 1/(2 n_color)  =  (1 - n_color) / (2 n_color).
```

`(R4)` Consider `Q(u_L) = (n_color + 1)/(2 n_color)`. We need
`gcd(n_color + 1, 2 n_color)`. Write `gcd(n_color + 1, 2 n_color) = gcd(n_color + 1,
2 n_color - 2 (n_color + 1) + 2) = gcd(n_color + 1, 2)`. Hence:

- If `n_color` is odd: `n_color + 1` is even, so `gcd = 2`, reduced denominator is
  `2 n_color / 2 = n_color`.
- If `n_color` is even: `n_color + 1` is odd, so `gcd = 1`, reduced denominator
  remains `2 n_color`.

The analogous argument for `Q(d_L) = (1 - n_color)/(2 n_color)` uses
`gcd(1 - n_color, 2 n_color) = gcd(1 - n_color, 2)`; this is `2` iff `n_color`
is odd, giving the same reduced denominator. ∎

## Derivable corollaries

```text
(C1)  Q(u_L) - Q(d_L)  =  1,    independent of n_color.
(C2)  Q(u_L) + Q(d_L)  =  a   =  1 / n_color   under the b = -1 convention.
(C3)  At n_color = 3: third-integer denominator (lowest-terms 3, parity odd).
(C4)  At n_color = 1: integer charges (Q(u_L) = 1, Q(d_L) = 0; degenerate
       limit, supplies sanity-check parity-odd row only).
```
