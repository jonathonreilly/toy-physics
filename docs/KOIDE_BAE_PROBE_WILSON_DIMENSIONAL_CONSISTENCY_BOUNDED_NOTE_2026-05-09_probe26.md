**This probe interrogates the hypothesis** that the chain's

## Derivation

### Step 1: scalar / operator decomposition

Each structural factor in the `m_tau` chain is a real positive scalar:
- `(7/8)^(1/4) ∈ R_+` (APBC factor),
- `u_0 = <P>^(1/4) ∈ R_+` (tadpole),
- `alpha_LM ∈ R_+` (coupling),
- and any product / power of these.

A real positive scalar `r` acts on the matrix algebra `M_3(C)` by
`X -> r X`. This action:
- preserves the algebra structure (it's a *-homomorphism scaled by `r`),
- preserves the C_3 conjugation grading (since `r` commutes with the
  unitary `U_C3`),
- maps a Hermitian circulant `H = aI + bC + b̄C^2` to
  `rH = (ra) I + (rb) C + (rb̄) C^2`.

In particular, the BAE ratio is invariant under this action:
```
    |rb|^2 / |ra|^2 = r^2 |b|^2 / r^2 a^2 = |b|^2 / a^2.
```

This step is verified algebraically in the runner (`Sections 3.5,
4.2, 5.2, 6.2, 7.2*r`).

### Step 2: D1 — Frobenius-norm decomposition

The Frobenius norm of `H = aI + bC + b̄C^2` decomposes as
```
    ||H||^2_F = trace(H^* H) = 3 a^2 + 6 |b|^2,
```
with the `3 a^2` coming from the C_3-trivial isotype (the multiples
of `I`) and the `6 |b|^2` coming from the C_3-nontrivial isotype
(multiples of `C` and `C^2` paired by Hermiticity). This is
verified in Section 1.8 of the runner.

The candidate hypothesis "alpha_LM^16 = (alpha_LM)^N_taste enforces
equipartition" — in the Frobenius-norm sense — would require
`alpha_LM^16` to act on the C_3-trivial vs C_3-nontrivial isotypes
with *different* coefficients. But `alpha_LM^16` is C_3-trivial: it
commutes with `C_3` conjugation. Hence its action is `H -> alpha_LM^16 H`,
a scalar rescaling, with the SAME coefficient on every isotype. By
Step 1, this preserves `|b|^2 / a^2`.

To break the BAE ratio, an operator `T` would have to act with
*different* coefficients on the C_3-trivial vs C_3-nontrivial
isotypes, i.e.
```
    T(a I + b C + b̄ C^2) = (lambda_0 a) I + (lambda_1 b) C
                            + (lambda_1^* b̄) C^2
```
with `lambda_0 != lambda_1`. The retained content provides only
C_3-trivial scalar content (`alpha_LM`, `u_0`, `(7/8)^(1/4)`) in the
chain, hence `lambda_0 = lambda_1` is forced. This is verified in
Section 3.11–3.12 of the runner.

The 16 = N_taste counting is structural (it counts BZ corner
doublers, all of which contribute `alpha_LM` uniformly across all
sectors, NOT through a C_3-isotype-graded coupling). This is
consistent with the staggered-Dirac bounded theorem (`STAGGERED_DIRAC_BZ_CORNER_FORCING_THEOREM_NOTE_2026-05-07.md`).

### Step 3: D2 — Yukawa-vertex bilinear

The `alpha_LM^2 = (alpha_LM)^(N_Yukawa_legs)` factor enters as one
`alpha_LM` per chiral leg of the Yukawa bilinear. Bilinear coupling
on the matter sector takes `Y -> Y^* Y` for `Y` a complex circulant
(Probe 6 setting). Per Probe 6's explicit verification:
```
    Y = a_y I + b_y C + c_y C^2,   (a_y, b_y, c_y) ∈ C^3, 6 real DOFs
    Y^* Y = (|a_y|^2 + |b_y|^2 + |c_y|^2) I
            + (a_y^* b_y + b_y^* c_y + c_y^* a_y) C
            + h.c. C^2.
```
This is again a Hermitian circulant, with parameters
```
    alpha = |a_y|^2 + |b_y|^2 + |c_y|^2,
    beta  = a_y^* b_y + b_y^* c_y + c_y^* a_y.
```
The BAE condition on `Y^* Y` is `|beta|^2 / alpha^2 = 1/2`, which is
ONE algebraic constraint on the 6 real DOF `(a_y, b_y, c_y)`. Per
Probe 6, this is a 5-real-DOF constraint surface, NOT a forced
equality.

The bilinear `alpha_LM^2` factor multiplies `Y^* Y` by `alpha_LM^2`
(both legs contribute `alpha_LM`), again a scalar rescaling. By
Step 1, the BAE ratio on `Y^* Y` is preserved.

This is verified explicitly in Section 4.3–4.5 of the runner using
both BAE-violating and BAE-compliant complex-circulant `Y`'s, with
`alpha_LM^2` applied to both: the ratios remain distinct, so
`alpha_LM^2` cannot lift one to the other.

### Step 4: D3 — APBC factor (7/8)^(1/4)

The `(7/8)^(1/4)` factor enters from anti-periodic boundary
conditions on the temporal direction, summing over Matsubara
frequencies. The temporal direction is structurally orthogonal to
the spatial Z^3 substrate where the C_3[111] cyclic acts. After
temporal integration, the factor reduces to a scalar prefactor on
the spatial M_3(C) algebra on `hw=1`.

The runner verifies in Section 5.5 that any rational-fractional
power of `(7/8)` — `1/4`, `1/2`, `1`, `2`, `3/2` — preserves the
BAE ratio when applied as a scalar to `H`. The conclusion is
power-independent: `(7/8)^p * H` preserves `|b|^2 / a^2` for any
real `p`.

In particular, the alternative "lepton-sector form" `m_tau / v =
alpha_LM^2` (where `(7/8)^(1/4)` is reabsorbed into `v`) gives the
same operator-level scalar prefactor, with the `(7/8)^(1/4)`
dependence now appearing only at the lepton-vs-EW comparison level.

### Step 5: D4 — Plaquette factor u_0 = <P>^(1/4)

The `u_0` factor is computed from the gauge-link plaquette VEV `<P>`,
which is a trace over SU(3) color indices. The C_3[111] cyclic
action on `hw=1` operates on spatial / matter coordinates, NOT on
SU(3) color. Hence `u_0` is structurally orthogonal to the C_3[111]
spatial cyclic, and cannot carry C_3[111]-discriminating
operator-level content.

In particular, even though the diagonal C_3 ⊂ SU(3) center exists,
it acts on color indices, not on the spatial hw=1 triplet. Color
traces are blind to the spatial C_3[111]. This is verified in
Section 6.4 of the runner.

### Step 6: D5 — Five-form mutual consilience

The five algebraic forms for `m_tau` (per task brief) regroup the
same structural factors:
```
Form 1:   (7/8)^(1/4) * u_0 * alpha_LM^18
Form 2:   (7/8)^(1/4) * u_0 * alpha_LM^16 * alpha_LM^2     (16+2 split)
Form 3:   [(7/8)^(1/4) * alpha_LM^16] * [u_0 * alpha_LM^2]
Form 4:   [v / M_Pl] * u_0 * alpha_LM^2                     (v reabsorbs APBC)
Form 5:   u_0 * (7/8)^(1/4) * alpha_LM^16 * alpha_LM * alpha_LM
```
By the commutativity / associativity of real multiplication, all 5
forms produce the SAME numerical scalar prefactor. As scalars, all
5 induce the SAME operator-level transformation (`H -> r H` with
the same `r`). There is no operator-level disagreement between the
forms; the form-grouping freedom is purely associativity, not
operator-level content.

In particular, the "16+2 split" of the exponent 18 has two
interpretations (taste doublers + Yukawa legs) but they multiply to
the same scalar. This is verified in Sections 2.2–2.5 and 7.1–7.4
of the runner.

There is NO over-determination of `(a, b)` from the 5-form
consilience, because the 5 forms produce identical operator-level
content.

### Step 7: D6 — eigenvalue assignment under-determines (a, b)

Even if `m_tau` is identified with the LARGEST eigenvalue
`lambda_max(H)` (the Brannen / Rivero τ-assignment), distinct
`(a, b)` choices realize the same `lambda_max`. The runner
demonstrates this concretely (Sections 8.1–8.4):
- BAE-compliant `(a, b) = (1, e^{i 2/9} / sqrt(2))` gives
  `lambda_max ≈ 2.379`,
- BAE-violating `(a, b) = (lambda_max / 1.8, 0.4 a)` gives the same
  `lambda_max` with ratio `|b|^2/a^2 = 0.16 ≠ 0.5`.

A single scalar (one eigenvalue) cannot pin a 3-DOF parameter
family `(a, |b|, arg b)`. Even iterating to all three eigenvalues
(if the chain's sub-leading orders were extended to derive
`m_e`, `m_mu`, `m_tau` simultaneously), the resulting 3-equation
system in 3 unknowns admits multiple solutions on different
charged-lepton cones. BAE is the SPECIAL `Q = 2/3` solution; the
chain does not select `Q = 2/3` over other `Q` values without an
independent BAE selector.
