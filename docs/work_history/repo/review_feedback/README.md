# Review Feedback Archive

This directory holds **historical detailed review packets** and older
operational audit notes that should no longer live in the front-door `docs/`
surface.

Use this directory for:

- resolved code-review ledgers
- resolved rerun-required bug audits
- historical control-plane audits
- detailed review packets that are too long for the active queue

Do **not** use this directory as the live queue.

The canonical active surface is:

- `docs/repo/ACTIVE_REVIEW_QUEUE.md`

The process document is:

- `docs/repo/REVIEW_FEEDBACK_WORKFLOW.md`
