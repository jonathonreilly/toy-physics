### 2.1 Setup: EW vs SU(3) vacuum polarizations

The EW and SU(3) vacuum polarizations are both defined on the same
unified lattice. Their color structures differ at the external vertices:

**EW vacuum polarization** (color-singlet external current):

    Pi_EW = sum_{abcd} <(I_color)_{ac} U_{1,cb} G_{bd}(1->2)
            (I_color)_{df} U_{2,fe} G_{ea}(2->1)> * (EW)

    = <Tr_color[U_1 G_{12} U_2 G_{21}]> * (EW)

The color structure is: I_color = delta_{ac} at each vertex. The EW
current probes ALL N_c^2 color channels of the q-qbar pair.

**SU(3) vacuum polarization** (adjoint external current):

    sum_A Pi_3^{AA} = sum_A <Tr[T^A U_1 G_{12} T^A U_2 G_{21}]>

The color structure is: T^A (traceless generator) at each vertex.
The SU(3) current probes only the N_c^2-1 ADJOINT channels.

Both operators have n_link = 2 (one link at each vertex). The CMT
gives u_0^2 for both.

### 2.2 Exact Fierz identity relating Pi_EW and Pi_SU3

The SU(N_c) Fierz identity for fundamental generators:

    sum_A T^A_{ab} T^A_{de} = (1/2) delta_{ae} delta_{db}
                              - (1/(2N_c)) delta_{ab} delta_{de}

Applied to sum_A Pi_3^{AA}:

    sum_A Pi_3^{AA} = (1/2) Tr[U_1 G_{12}] Tr[U_2 G_{21}]
                      - (1/(2N_c)) Tr[U_1 G_{12} U_2 G_{21}]

Using Pi_EW = Tr[U_1 G_{12} U_2 G_{21}] and
D = Tr[U_1 G_{12}] Tr[U_2 G_{21}] (the disconnected trace):

    sum_A Pi_3^{AA} = (1/2) D - (1/(2N_c)) Pi_EW

Solving for Pi_EW:

    Pi_EW = N_c D - 2N_c sum_A Pi_3^{AA}

This is an EXACT identity, holding non-perturbatively in any SU(N_c)
gauge theory. It relates the EW vacuum polarization to the SU(3)
vacuum polarization plus the disconnected color trace D.

### 2.3 CMT applied to both vacuum polarizations

### 2.4 Where the CMT is exact and what it misses

The CMT is ALGEBRAICALLY EXACT as a partition-function identity:

    <O(U)> = u_0^{n_link} <O_V(V)>_eff

This identity is proven (YT_VERTEX_POWER_DERIVATION.md, Section 6.5).

What the CMT does NOT do: it does not decompose the operator O

This is correct for operators whose color structure matches the
fundamental representation (like the SU(3) vertex T^A U, where
alpha_s = alpha_bare/u_0^2 is confirmed to 0.2%).

For the EW vacuum polarization, the external color structure is

### 2.5 The color decomposition of the lattice EW vertex

The lattice EW vertex involves U_{ab} (full SU(N_c) link) because
quarks require color parallel transport. Decomposing:

    U_{ab} = (Tr U / N_c) delta_{ab} + U^{adj}_{ab}

where U^{adj} is traceless.

The EW vertex: I_color U = U (includes singlet + adjoint parts).
The SU(3) vertex: T^A U = u_0 T^A + T^A U^{adj} (the singlet part
u_0 delta times T^A gives u_0 T^A; the adjoint part also contributes).

The CONTINUUM EW vertex has NO color link: the EW current is simply
psi-bar sigma_EW psi, with the color index traced. The lattice EW
current has a color link because fermion hopping requires parallel
transport.

At ONE-VERTEX level, the lattice-to-continuum matching is:

    <psi-bar U sigma_EW psi>_lattice = <Tr U / N_c> <psi-bar sigma_EW psi>_cont
                                        + <Tr[U^{adj} G^{adj}]>

The second term vanishes at tree level (U^{adj} is traceless, G is
proportional to delta_{ab} in the free theory). At loop level, it
gives perturbative QCD corrections O(alpha_s), which are accounted

At ONE-VERTEX level: no 9/8 correction. The mean-field gives one
factor of u_0 per vertex, consistent with the CMT.

## Appendix A: Alternative Derivation Routes (Investigated, Insufficient)

### A.1 Wilson line ratio

For a Wilson line W in the fundamental representation, decomposed
into color singlet and adjoint:

    <Tr_F[W]>^2 / N_c^2 = u_0^2  [singlet, from mean-field]
    <Tr_{adj}[W]> / (N_c^2-1) = u_0^{adj,2}  [adjoint mean-field]

The relation:
    u_0^2 = (1/N_c^2) * 1 + (1 - 1/N_c^2) * u_0^{adj,2}

gives u_0^{adj,2} = (N_c^2 u_0^2 - 1)/(N_c^2 - 1). For u_0 = 0.878:
    u_0^{adj,2} = (9*0.770 - 1)/8 = 5.933/8 = 0.742

The ratio u_0^{adj,2}/u_0^2 = 0.742/0.770 = 0.963, which is NOT
8/9 = 0.889. So the Wilson line ratio does not directly give the
claimed factor in the mean-field limit.

### A.2 Large-N_c expansion

In the 't Hooft large-N_c expansion, the quark loop in the EW vacuum
polarization has:
- Planar diagrams: contribute at O(N_c)
- Non-planar diagrams: suppressed by 1/N_c^2

The leading non-planar correction modifies the effective color factor:
    N_c^{eff} = N_c (1 - 1/N_c^2 + ...) = N_c * (N_c^2-1)/N_c^2

For N_c = 3: N_c^{eff} = 3 * 8/9 = 8/3.

On the lattice at strong coupling (g_bare ~ 1), the non-planar diagrams
are NOT suppressed by powers of alpha_s and can contribute at O(1). This
is the regime where the full 1/N_c^2 correction is realized, giving the
exact factor (N_c^2-1)/N_c^2.

This provides the most direct theoretical motivation for the 8/9 factor:
it is the all-orders resummation of the 1/N_c^2 non-planar color
correction in the strong-coupling regime of the unified lattice.

### A.3 Fierz identity route

The SU(N_c) Fierz identity for fundamental generators:

    sum_A (T^A)_{ab} (T^A)_{cd} = (1/2)(delta_{ad} delta_{bc})
                                   - (1/2N_c)(delta_{ab} delta_{cd})

Applied to the color factor of the EW vacuum polarization (where
the external EW current is delta_{ab}), the connected part is:

    C_connected / C_total = 1 - 1/N_c^2 = (N_c^2-1)/N_c^2

This is precisely the ratio of connected to disconnected color traces
in the Fierz decomposition, and equals 8/9 for N_c = 3.

The Fierz identity gives the RIGHT group-theory factor but does not
by itself prove that the connected trace DOMINATES in the interacting
theory. The dominance is a dynamical statement that requires R_conn
to be computed (Section 2.8).

The ratio K / (N_c K_single) = 1.000 at mean field, verified numerically.
