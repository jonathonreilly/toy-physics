## Setup

Let `Λ ⊂ Z^3` be a finite block. Let `D` be the staggered Cl(3) hopping
operator on a fermion Hilbert space `V = C^N` over `Λ`. Let `J` be a
local diagonal source operator
`J = sum_{x in Λ} j_x P_x`, with `P_x` the on-site projector at `x`. The
finite-Grassmann partition is

```text
Z[J] = det(D + J),
```

forced by the finite Grassmann integral (no continuum QFT machinery; see
[`STAGGERED_DIRAC_GRASSMANN_FORCING_THEOREM_NOTE_2026-05-07.md`](STAGGERED_DIRAC_GRASSMANN_FORCING_THEOREM_NOTE_2026-05-07.md)
for the standalone forcing).

Let `(Λ_A, Λ_B)` be a partition `Λ = Λ_A ⊔ Λ_B` such that the staggered
hopping action contains no link bond from `Λ_A` to `Λ_B`. Then `D` is
block-diagonal in the `Λ_A`-`Λ_B` decomposition: there exist operators
`D_A` on `V_A = C^{N_A}` (the fermion subspace over `Λ_A`) and `D_B` on
`V_B = C^{N_B}` such that

```text
D = D_A (+) D_B,                                                       (B1)
```

with `(+)` the orthogonal direct sum of operators on `V = V_A (+) V_B`.
This is a structural fact about the staggered hopping operator on `Z^3`
with the cut chosen so that the support of `D` does not cross the cut.
For any source `J = J_A (+) J_B` with `J_A` supported on `V_A` and `J_B`
on `V_B`,

```text
D + J = (D_A + J_A) (+) (D_B + J_B).                                   (B2)
```

## Theorem 1: block-multiplicative partition (finite Grassmann)

For any partition `(Λ_A, Λ_B)` with the no-cut-bond property (B1):

```text
Z[J_A (+) J_B] = det((D_A + J_A) (+) (D_B + J_B))
              = det(D_A + J_A) · det(D_B + J_B)
              = Z_A[J_A] · Z_B[J_B].                                   (T1)
```

**Proof.** The orthogonal direct sum of operators has determinant equal
to the product of block determinants: this is a standard finite-
dimensional linear-algebra identity, since the matrix of `(D_A + J_A)
(+) (D_B + J_B)` in the basis `V_A ⊕ V_B` is block-diagonal, and a
block-diagonal matrix's determinant equals the product of its block
determinants. The first equality uses (B2); the second uses block
determinant multiplicativity; the third uses the definitions
`Z_A[J_A] = det(D_A + J_A)` and `Z_B[J_B] = det(D_B + J_B)`. ∎

**Provenance.** This is purely finite Grassmann calculus on a finite-
dim fermion Hilbert space. The "no cut bond" hypothesis is the structural
input that comes from `Z^3` lattice locality of the staggered Dirac
operator.

## Theorem 2: equivalence (A1) <=> (A2) (under continuity)

Suppose `W: R_+^{N_plaq+1} -> R` is a continuous functional of
`r = |Z[J]|` (depending on `J` only through `|Z[J]|`) satisfying (A1)
on every partition `(Λ_A, Λ_B)` with the no-cut-bond property. Then
there exist real constants `c` and `c_0` such that

```text
W[J] = c · log |Z[J]| + c_0,                                          (T2)
```

with `c_0` fixed by `W[0] = 0` (i.e. `c_0 = -c · log |det D|`) and the
canonical natural-log normalization sets `c = 1`.

**Proof.** By Theorem 1, `|Z[J_A ⊕ J_B]| = |Z_A[J_A]| · |Z_B[J_B]|`. So
(A1) gives `W(r_A · r_B) = W(r_A) + W(r_B)` for the function `W(r)`
viewed on the positive reals `r = |Z| > 0`. The continuous
multiplicative-to-additive functional equation `f(xy) = f(x) + f(y)`
on `R_+` with `f` continuous has unique solution `f(r) = c log r` for
some real constant `c` (Cauchy's functional equation, continuity branch;
no measurable-only-pathological branch needed since `r > 0` and
continuity is assumed in the bridge premise). Adding the zero-source
baseline `c_0 = -c log |det D|` gives (T2) with `W[0] = 0`. The natural-
log convention fixes `c = 1`. ∎

**Provenance.** (B1) + Theorem 1 + Cauchy functional equation on
`R_+` with continuity. The continuity input is supplied by the finite-
block polynomial structure of `j -> det(D + jI)` and `D` invertible on
the runner block, as already proved in
[`OBSERVABLE_PRINCIPLE_FROM_AXIOM_NOTE.md`](OBSERVABLE_PRINCIPLE_FROM_AXIOM_NOTE.md)
§"P3 (continuity)".

## Theorem 3: equivalence (A2) <=> (A3) (cumulant generating functional)

Suppose `W[J] = log |det(D + J)|` (up to additive constant) on a
neighborhood of `J = 0` where `det(D + J) != 0`. Then `W[J]` admits a
formal Taylor expansion around `J = 0`,

```text
W[J] = sum_{n>=1} (1/n!) sum_{x_1,...,x_n} K_n(x_1,...,x_n) J_{x_1} ... J_{x_n},  (T3)
```

with kernels `K_n` given by the source derivatives

```text
K_n(x_1,...,x_n) = ∂^n W / ∂j_{x_1} ... ∂j_{x_n} |_{J=0}.              (T3')
```

The kernels `K_n` are connected truncated `n`-point fermion-bilinear
moments of the finite-Grassmann measure: for any partition
`(Λ_A, Λ_B)` with the no-cut-bond property, if the support
`{x_1, ..., x_n}` splits across the cut (some `x_i ∈ Λ_A` and some
`x_j ∈ Λ_B`),

```text
K_n(x_1, ..., x_n) = 0.                                                 (T3'')
```

**Proof.** The polynomial `j -> det(D + jI)` is degree `N` with nonzero
constant term `det D` (since `D` is invertible on the runner block), so
on a neighborhood of `J = 0` the map `J -> log|det(D + J)|` is real-
analytic, and its Taylor expansion (T3) converges in that neighborhood.
The kernels `K_n` are by construction the source derivatives in (T3').
For the connected-truncated property: by Theorem 1, on any partition
with the no-cut-bond property,

```text
W[J_A (+) J_B] = log |Z_A[J_A] · Z_B[J_B]| - log |det D|
              = log |Z_A[J_A]| + log |Z_B[J_B]| - log |det D|
              = (W_A[J_A]) + (W_B[J_B])
                                                                       (T3a)
```

with `W_A[J_A] := log |Z_A[J_A]| - log |det D_A|`,
`W_B[J_B] := log |Z_B[J_B]| - log |det D_B|`, and `log |det D|
= log |det D_A| + log |det D_B|` from block multiplicativity (T1) at
`J = 0`. Therefore the mixed source derivative

```text
∂^n W / ∂j_{x_1} ... ∂j_{x_n}     ( with some x_i in Λ_A, others in Λ_B )
```

vanishes identically: from (T3a), `W[J_A (+) J_B]` is a sum of a function
of `J_A` only and a function of `J_B` only, so any mixed cross-derivative
between `J_A` indices and `J_B` indices is zero. Evaluating at `J = 0`
gives `K_n(x_1, ..., x_n) = 0` whenever the support splits across the cut.
This is exactly the connected-truncated property (T3''). ∎

**Provenance.** Theorem 1 + the analytic structure of `j -> log|det(D
+ jI)|` near `J = 0` on the runner block.

## Theorem 4: equivalence (A3) <=> (A1) (cumulant generator forces additivity)

Suppose `W[J]` satisfies (A3): it has a Taylor expansion (T3) around
`J = 0` with kernels `K_n` satisfying the connected-truncated property
(T3''). Then `W` is additive on independent blocks: for any partition
`(Λ_A, Λ_B)` with the no-cut-bond property,

```text
W[J_A (+) J_B] = W[J_A] + W[J_B].                                      (T4)
```

**Proof.** Expand `W[J_A (+) J_B]` using (T3) with the cumulant kernels
`K_n`. The expansion splits into three classes:

```text
W[J_A (+) J_B] = sum_n (1/n!) sum_{x_1,...,x_n in Λ_A} K_n J_{x_1}...J_{x_n}    (class I, pure A)
              + sum_n (1/n!) sum_{x_1,...,x_n in Λ_B} K_n J_{x_1}...J_{x_n}    (class II, pure B)
              + sum_n (1/n!) sum_{(x_1,...,x_n) split across cut} K_n J_{x_1}...J_{x_n}    (class III, mixed).
```

Class III vanishes term-by-term by (T3''). Class I is `W_A[J_A]`
(the same expansion of the same `W` restricted to sources on `Λ_A`).
Class II is `W_B[J_B]`. Hence (T4). ∎

**Provenance.** Pure algebraic substitution using only the connected-
truncated property (T3'') and the Taylor expansion (T3).

## Closure: (A1) <=> (A2) <=> (A3)

Combining Theorems 2, 3, 4 gives the three-way equivalence (A1) <=>
(A2) <=> (A3) on the finite-block surface where `Z[J]` is given by the
finite-Grassmann partition `det(D + J)`.

## What this closes (honest scope)

- The three formulations of "the scalar generator is additive on
  independent blocks" are mathematically the **same selection** in
  disguise on the finite-Grassmann surface:
  - (A1) "physical-scalar additivity" (the P1 admission of the parent
    observable-principle note);
  - (A2) `W = log|Z|` up to constant (the explicit functional form);
  - (A3) `W` is the cumulant generating functional (the canonical
    structural definition).
- No one of (A1), (A2), (A3) is independent of the other two given
  finite Grassmann calculus + `Z^3` locality of the staggered Dirac
  hopping operator + continuity at `J = 0`.
- The structural equivalence narrows the "P1 admission" by showing it
  is the same as the canonical cumulant-generator definition in finite

  cumulants play in finite-dim multivariate calculus and is checked

- The parent
