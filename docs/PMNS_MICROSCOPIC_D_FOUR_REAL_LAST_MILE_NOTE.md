# PMNS Microscopic D Four-Real Last Mile

## Question
Once the passive Hermitian block is derived from the microscopic passive
response law, how much of the PMNS-relevant microscopic operator `D` is still
not fixed?

## Exact Result
On the retained one-sided minimal PMNS classes:

- `q` remains fixed natively by passive support moments
- the passive Hermitian block fixes the three passive moduli exactly:
  `|a_i| = sqrt((D_pass D_pass^dag)_{ii})`
- `xbar` remains fixed natively by the active `C3`-even moment
- `sigma` remains fixed natively by the active `C3`-forward moment

The only unreduced active data are then four real orbit-breaking coordinates:

- `xi_1, xi_2` on the diagonal channel
- `rho_1, rho_2` on the cycle channel relative to the derived complex mean
  `sigma`

Therefore the PMNS-relevant microscopic `D` law is no longer a seven-real last
mile. It is exactly a four-real last mile:

`(xi_1, xi_2, rho_1, rho_2)`

together with the already derived native data `(tau, q, xbar, sigma)` and the
passive Hermitian moduli.

## Boundary
This is still a reduction theorem, not full sole-axiom closure. It does not yet
derive the four active orbit-breaking values from `Cl(3)` on `Z^3`.
