## 5. Bounded Support Claim (per-sub-step structural result)

**Claim (Higgs S7 Sub-step Accounting, Per-Sub-step Status).**

Under the premises of §2:

**(T1) Each of `δ_W, δ_CW, δ_lat, δ_R` is evaluated from cited inputs with one named admission.** The runner verifies this numerically (Sections 3-6 of the runner).

**(T2) No single sub-correction yields retained-grade closure of the +12% gap alone.** Numerical results:
- `δ_W` alone: `m_H_tree · sqrt(1 - δ_W) = 140.31 · sqrt(0.7849) = 124.27 GeV` (-0.78% vs PDG)
- `δ_CW` alone: `m_H_tree · sqrt(1 - δ_CW) = 140.31 · sqrt(0.7934) = 124.98 GeV` (-0.21% vs PDG)  [matches H1]
- `δ_lat` alone: `m_H_tree · sqrt(1 - δ_lat) = 140.31 · sqrt(0.7092) = 118.10 GeV` (-5.7% vs PDG)
- `δ_R` alone: `m_H_tree · sqrt(1 - δ_R) = 140.31 · sqrt(0.9976) = 140.14 GeV` (+11.9% vs PDG)

The closest single-correction match is `δ_CW` at -0.21% (this matches H1 exactly).

**(T3) The four sub-corrections are NOT mutually independent.** This is the **new structural content** of this note. Specifically:
- `δ_CW`, `δ_lat`, `δ_R` together encode the SAME SM RGE flow at successively higher orders (1-loop QCD vacuum-pol on top, 2-loop / lattice-perturbative running, full 3L+NNLO Buttazzo calibration).
- Composing all four multiplicatively over-corrects: `(1 - δ_W)(1 - δ_CW)(1 - δ_lat)(1 - δ_R) = 0.7849 · 0.7934 · 0.7092 · 0.9976 = 0.4404`, giving `m_H = 140.31 · sqrt(0.4404) = 93.10 GeV`, far below the PDG comparator. The over-correction is the numerical signature of non-independence.

**(T4) Two clean closure routes emerge.** Cleanly-separable sub-correction subsets:

- **Route A (Buttazzo only):** Set `δ_W = δ_CW = δ_lat = 0` and use `Δ²/m_H_tree² = 1 - (m_H_buttazzo/m_H_tree)² = 1 - (125.10/140.31)² = 0.2049`. This matches the comparator at `m_H = 125.10 GeV` (-0.12% vs PDG) with **one named Buttazzo-SM-RGE admission**, but Buttazzo uses imported SM RGE so the match is sister-authority support, not pure framework closure.
  
- **Route B (Wilson + CW only):** `(1 - δ_W)(1 - δ_CW) = 0.7849 · 0.7934 = 0.6228`, giving `m_H = 140.31 · sqrt(0.6228) = 110.78 GeV` -- **over-corrects below PDG**. This route evaluates with two named admissions (Wilson-r and CW-factor-two), but the over-correction shows that even within the "structurally separable" subset, sub-correction overlap remains (Wilson taste-breaking and 2-loop CW share content via the lattice perturbative expansion).

**(T5) Sharpened residue.** The S7 step's gap-closure functional `Δ²` is sharpened from a single 12% residue to a per-sub-step accounting with four named sub-corrections and four named admissions. The structural feature that the four sub-corrections are non-independent is **new** structural content beyond PR #865's per-step decomposition.

**(T6) The matching residual `M(curvature_lat) = m_H²_phys` per PR #865 is unchanged.** This note clarifies the S7 step but does not close the matching residual. The S4 step (tree-level vs post-EWSB readout) is unchanged.
