## Statement

Let `C` denote the 3x3 cyclic permutation matrix on `C^3` defined by
`C e_k = e_{k+1 mod 3}`, and let

```text
P_cyc(X)  =  (1/3) sum_{k=0}^{2} C^k X (C^{-k})       (1)
```

be the cyclic averaging operator on 3x3 matrices.

**Conclusion (T1).** `C^3 = I`, `C != I`, `C^2 != I` (genuine 3-cycle),
and `C` is real orthogonal (`C C^T = I`).

**Conclusion (T2).** Define

```text
B_0  =  I,
B_1  =  C + C^2,
B_2  =  i (C - C^2).
```

`B_0, B_1, B_2` are all Hermitian.

**Conclusion (T3).** `P_cyc` restricted to `Herm(3)` is the orthogonal
projector onto the 3-dimensional cyclic-invariant subspace, and

```text
image(P_cyc on Herm(3))  =  span_R { B_0, B_1, B_2 }.                 (2)
```

In particular, `P_cyc(B_i) = B_i` for `i = 0, 1, 2`, and `P_cyc` is
idempotent.

**Conclusion (T4).** On the standard Hermitian basis of `Herm(3)`
(`D_i = E_{ii}` for `i = 1, 2, 3`; `X_{ij} = E_{ij} + E_{ji}` for
`i < j`; `Y_{ij} = i(E_{ji} - E_{ij})` for `i < j`), `P_cyc` has the
explicit action:

```text
P_cyc(D_i)    =  B_0 / 3                       for i = 1, 2, 3,
P_cyc(X_{ij}) =  B_1 / 3                       for all i < j,
P_cyc(Y_{12}) =  P_cyc(Y_{23})  =  + B_2 / 3,
P_cyc(Y_{13}) =  - B_2 / 3.                                         (3)
```

So the 9-dimensional `Herm(3)` collapses under `P_cyc` to exactly the
3-dimensional `span_R { B_0, B_1, B_2 }`.

**Conclusion (T5).** For a generic Hermitian

```text
H_e  =  sum_i d_i D_i  +  sum_{i<j} x_{ij} X_{ij}  +  sum_{i<j} y_{ij} Y_{ij},
```

the cyclic compression is the explicit linear formula

```text
P_cyc(H_e)  =  ((d_1 + d_2 + d_3) / 3)  B_0
             +  ((x_{12} + x_{23} + x_{13}) / 3)  B_1
             +  ((y_{12} + y_{23} - y_{13}) / 3)  B_2.                (4)
```

## Proof

`(T1)` Direct computation: `C^3 = I`, `C^2` is the inverse cycle (not `I`),
and `C C^T = I` since `C` is a real permutation matrix.

`(T2)` `B_0 = I` is trivially Hermitian. `B_1 = C + C^2` is Hermitian
because `C^T = C^{-1} = C^2` (from `C^3 = I`), so `B_1^T = C^T + (C^2)^T
= C^2 + C = B_1`. `B_2 = i(C - C^2)` is Hermitian because
`B_2^dagger = -i (C^T - (C^2)^T) = -i(C^2 - C) = i(C - C^2) = B_2`.

`(T3)` `P_cyc` is idempotent on any matrix `X`: applying `P_cyc` twice
gives `(1/9) sum_{j, k} C^{j+k} X C^{-(j+k)}`. For each fixed `j+k mod 3
= m`, there are 3 pairs `(j, k)` summing to `m`, so the inner sum is
`3 * C^m X C^{-m}`. Hence `P_cyc(P_cyc(X)) = (1/3) sum_m C^m X C^{-m} =
P_cyc(X)`.

`P_cyc` is self-adjoint with respect to the Frobenius inner product
`<A, B>_F = Tr(A^dagger B)`: by Tr-cyclicity,
`Tr((C^k X C^{-k})^dagger Y) = Tr(X^dagger C^{-k} Y C^k) =
Tr(X^dagger (C^{-k} Y C^k))`. Averaging `k` over Z_3 gives
`<X, P_cyc(Y)>_F = <P_cyc(X), Y>_F`.

Idempotent + self-adjoint = orthogonal projector. The image is the
fixed-point set, which equals the cyclic-invariant subspace
`{X : C X C^{-1} = X}`. On Herm(3), this fixed-point set is
3-dimensional (one diagonal-trace direction + two off-diagonal cyclic
combinations) and is spanned by `B_0, B_1, B_2`. (Each `B_i` is fixed
by conjugation by `C`: `C B_0 C^{-1} = I = B_0`;
`C(C+C^2) C^{-1} = C + C^2`; `C i(C - C^2) C^{-1} = i(C - C^2)`.)

`(T4)` Direct computation. For `D_i = E_{ii}`:
`C^k E_{ii} C^{-k} = E_{(i+k) mod 3, (i+k) mod 3}`, so summing `k` over
`{0, 1, 2}` gives `E_{11} + E_{22} + E_{33} = I = B_0`, hence
`P_cyc(D_i) = I / 3 = B_0 / 3`.

For `X_{ij}` with `i < j`: similar enumeration of `C^k X_{ij} C^{-k}`
gives all three off-diagonal symmetric pairs `X_{12}, X_{23}, X_{13}`,
each appearing once in the sum, totaling `B_1 = C + C^2` (after the
`(1/3)` average gives `B_1 / 3`).

For `Y_{ij}`: the antisymmetric off-diagonal pairs `Y_{12}, Y_{23}` cycle
into each other and into `Y_{13}` with a sign flip (because the
antisymmetric Hermitian basis under cyclic shifts picks up a sign on
the "wraparound" pair). The sum gives `+B_2 / 3` for `Y_{12}, Y_{23}`
and `-B_2 / 3` for `Y_{13}`.

`(T5)` Linearity of `P_cyc` plus `(T4)`.

The runner verifies `(T1)`-`(T5)` at exact symbolic precision via sympy.
