## Positive closure (source-free-state mechanism)

### Statement

> **Theorem.** Let `rho_cell = I_16/16` be the maximally-mixed source-free
> state on `H_cell = (C^2)^{(x)4}` of the time-locked primitive event cell.
> Let `O_Schur` be any Hermitian operator on `H_cell` whose carrier
> reading agrees with the framework carrier:
>
> ```
> P_A · O_Schur · P_A = c_cell · P_A,    c_cell = 1/4.
> ```
>
> Then the parent-source Schur scalar
>
> ```
> p_Schur := Tr(rho_cell O_Schur) = c_cell + delta
> ```
>
> satisfies
>
> ```
> delta = (Tr(O_Schur) - Tr(P_A)) / 16
>       = Tr(O_Schur - P_A) / 16.
> ```
>
> Setting `Q := O_Schur - P_A`, the affine fiber direction `delta`
> decomposes as
>
> ```
> Q = (Tr(Q)/16) I_16  +  Q_traceless
> p_Schur reading from Q  =  Tr(rho_cell Q) = Tr(Q)/16
> ```
>
> with two cases:
>
> 1. **Traceless `Q` (`Tr(Q) = 0`):** `delta = 0` identically.
> 2. **Identity-multiple `Q = (q/16) I` for `q != 0`:** `delta = q/16` is
>    a normalization redefinition of `P_A` (i.e., `O_Schur = P_A + (q/16)
>    I` shifts `c_cell` to `c_cell + q/16` — a global trace shift, not an
>    independent affine direction in the parent-source fiber).
>
> In either case, `delta` is not an independent affine character: it is
> either zero, or absorbed into the carrier's trace normalization. The
> hypothesized "hidden direction `(0, 1)`" of the no-go (representing a
> nonzero `delta` with no carrier shift) is structurally absent on the
> source-free state.

### Proof

**Step 1 (trace structure of the maximally-mixed state).**
For any Hermitian operator `O` on `H_cell` (dim 16),

```
Tr(rho_cell O) = Tr((I_16/16) O) = Tr(O)/16.
```

This follows by linearity of trace and `rho_cell = I_16/16`.

**Step 2 (decomposition of `O_Schur - P_A`).**
Set `Q = O_Schur - P_A`. Then

```
delta = p_Schur - c_cell = Tr(rho_cell O_Schur) - Tr(rho_cell P_A)
      = Tr(rho_cell Q) = Tr(Q)/16.
```

**Step 3 (Hermitian decomposition).**
Any Hermitian `Q` decomposes uniquely as

```
Q = (Tr(Q)/16) I_16 + Q_traceless
```

where `Tr(Q_traceless) = 0` (verified explicitly in the runner Section 7,
trial loop: `Tr(rho_cell Q_traceless) = 0` to numerical zero).

**Step 4 (case analysis).**

- *Case (a): `Q` traceless.* Then `Tr(Q) = 0`, so `delta = 0`. The hidden
  direction contributes nothing to the Schur scalar on `rho_cell`.

- *Case (b): `Q` proportional to `I`.* Then `Q = (q/16) I` for some real
  `q`, and `delta = q/16`. But this corresponds to `O_Schur = P_A + (q/16) I`,
  which simply shifts the carrier reading uniformly: `Tr(rho_cell O_Schur) =
  Tr(P_A)/16 + q/16 = c_cell + q/16`. This is a renaming of `c_cell` (the
  primitive coefficient) to `c_cell + q/16`, not an independent affine
  hidden direction.

- *General case.* Decompose `Q = Q_I + Q_traceless`. The `Q_I` part is
  Case (b), the `Q_traceless` part is Case (a). The contribution to `delta`
  is `Tr(Q_I)/16`, which is either zero or absorbed into a carrier
  normalization redefinition.

**Step 5 (no genuine hidden direction).**
The hypothesized "hidden direction `(0, 1)`" of the no-go runner
parameterizes `(carrier shift, delta shift) = (0, 1)`: a `delta` shift
without a carrier shift. On the source-free state, this is structurally
impossible: `delta = q/16` requires `Tr(O_Schur) = Tr(P_A) + q`, which
shifts the carrier reading by exactly `q/16` (since the carrier itself
is the trace under `rho_cell`). So `(0, q/16)` for `q != 0` is not
achievable: any nonzero `delta` comes paired with a nonzero carrier
shift, contradicting the "hidden direction" definition.

The hidden direction collapses on the maximally-mixed state. ∎
