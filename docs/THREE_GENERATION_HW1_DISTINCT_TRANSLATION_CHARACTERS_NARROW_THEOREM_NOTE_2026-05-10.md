## Statement

Let `H = C^3` with ordered basis `(X_1, X_2, X_3)`. Let

```text
T_x = diag(-1, +1, +1),
T_y = diag(+1, -1, +1),
T_z = diag(+1, +1, -1)
```

be three diagonal matrices acting on `H`. Then:

1. (**Involutions, commute, distinct.**) Each `T_a` satisfies `T_a^2 = I_3`;
   the three matrices pairwise commute; and no two of them are equal.
2. (**Distinct joint characters.**) The joint character map
   `chi : { X_1, X_2, X_3 } -> { -1, +1 }^3` defined by
   `chi(X_i) = ( <X_i, T_x X_i>, <X_i, T_y X_i>, <X_i, T_z X_i> )`
   takes three distinct values:

   ```text
   chi(X_1) = (-1, +1, +1),
   chi(X_2) = (+1, -1, +1),
   chi(X_3) = (+1, +1, -1).
   ```

3. (**Joint sign-projector identity.**) For each sign triple
   `s = (s_x, s_y, s_z) in { -1, +1 }^3` the joint projector

   ```text
   P(s) = ( (I + s_x T_x) / 2 ) ( (I + s_y T_y) / 2 ) ( (I + s_z T_z) / 2 )
   ```

   evaluates as

   - `P(chi(X_i))` is the rank-1 projector onto `C X_i` for `i = 1, 2, 3`;
   - `P(s) = 0` for any other of the five remaining sign triples.

4. (**Rank-1 sector decomposition.**) The three rank-1 joint
   sign-projectors `P_i := P(chi(X_i))` satisfy

   ```text
   P_i^2 = P_i,  P_i P_j = 0 for i != j,  P_1 + P_2 + P_3 = I_3,
   ```

   and decompose `H = C^3` as a direct sum of three retained rank-1
   sectors `C^3 = C X_1 \oplus C X_2 \oplus C X_3`.

## Proof

Pure 3x3 matrix algebra.

**1.** `T_a^2 = diag(1, 1, 1) = I_3` for each `a`. The three diagonal
matrices commute (diagonal matrices always commute). Distinctness:
position of the `-1` entry differs (positions 1, 2, 3 respectively), so
no two are equal.

**2.** `T_x X_i = (-1)^{[i = 1]} X_i`, so `<X_i, T_x X_i> = -1` iff `i = 1`
and `+1` otherwise. By symmetry `<X_i, T_y X_i> = -1` iff `i = 2`,
`<X_i, T_z X_i> = -1` iff `i = 3`. Hence `chi(X_i)` takes the three
listed distinct sign triples.

**3.** Each factor `(I + s_a T_a) / 2` is the eigenprojector of `T_a` onto
the `s_a`-eigenspace. Diagonal matrices commute, so the three factor
projectors commute, and their product is the joint eigenprojector for
the joint character `s`. On `C^3`, the eight possible sign triples
`s in { -1, +1 }^3` partition into:
- three triples `chi(X_i)` each giving a rank-1 joint eigenspace `C X_i`,
  and `P(chi(X_i)) = e_i e_i^T` (the `e_i e_i^T` rank-1 projector);
- five triples giving the zero operator (since no `X_i` has those
  joint signs).

**4.** From (3), `P_i = e_i e_i^T`. The identity `e_i^T e_j = delta_{i,j}`
gives `P_i P_j = e_i (e_i^T e_j) e_j^T = delta_{i,j} P_i`, so
`P_i^2 = P_i` and `P_i P_j = 0` for `i != j`. The sum
`P_1 + P_2 + P_3 = diag(1, 1, 1) = I_3`. Hence the three projectors are
a complete orthogonal family and decompose `C^3` as the claimed direct
sum.

QED.

- The three sectors `C X_i` are eigenspaces of every joint translation
  function `f(T_x, T_y, T_z)` (any commutative-algebra generated
  observable on the `(T_x, T_y, T_z)` triple acts as a scalar on each
  `C X_i`).

  this is the minimal positive statement needed for the parent note's

- The eight joint sign-projectors `{ P(s) : s in {-1,+1}^3 }` form a
  partition of unity in `M_3(C)` with three nonzero rank-1 components
  and five zero components.

All three corollaries are pure linear-algebra consequences of items
(1)-(4) above.
