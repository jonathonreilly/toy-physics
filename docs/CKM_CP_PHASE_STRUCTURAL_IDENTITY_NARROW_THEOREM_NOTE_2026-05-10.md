## Statement

Let `alpha_s, n_pair, n_color, n_quark` be abstract positive symbols (with
`n_pair, n_color, n_quark` positive integers) and let `r > 0` be an abstract
positive real. Assume the four parametric input identities

```text
w_A1                =  1 / n_quark,                               (P1)
w_perp              =  (n_quark - 1) / n_quark,                   (P2)
r^2                 =  1 / n_quark,                               (R)
w_A1 + w_perp       =  1.                                         (PSUM)
```

The first two are the cited projector weights on the six-state quark block;
the third is the cited bright/tensor CP radius. Take the positive CKM branch
`rho = r sqrt(w_A1)`, `eta = r sqrt(w_perp)`.

Then the Wolfenstein CP-plane coordinates are forced

```text
rho                 =  1 / n_quark,                               (C1)
eta                 =  sqrt(n_quark - 1) / n_quark,               (C2)
rho^2 + eta^2       =  1 / n_quark,                               (C3)
```

the CKM phase identities are forced

```text
cos^2(delta_CKM)    =  1 / n_quark,                               (C4)
sin^2(delta_CKM)    =  (n_quark - 1) / n_quark,                   (C5)
tan(delta_CKM)      =  sqrt(n_quark - 1),                         (C6)
delta_CKM           =  arccos(1 / sqrt(n_quark))
                    =  arctan(sqrt(n_quark - 1)),                 (C7)
```

and the atlas-area Jarlskog factorisation (using `lambda^2 = alpha_s / n_pair`,
`A^2 = n_pair / n_color` as imported hypotheses) is forced

```text
J_0 = lambda^6 A^2 eta
    = (alpha_s / n_pair)^3 (n_pair / n_color) sqrt(n_quark - 1) / n_quark
    = alpha_s^3 sqrt(n_quark - 1) / (n_pair^2 n_color n_quark).   (J)
```

Under the additional count constraint `n_quark = n_pair * n_color`, this
reduces to

```text
J_0 = alpha_s^3 sqrt(n_quark - 1) / (n_pair^3 n_color^2).         (Jq)
```

At the framework counts `(n_pair, n_color, n_quark) = (2, 3, 6)`, equations
`(C1)`-`(Jq)` specialise to the closed forms

```text
rho               =  1/6,
eta               =  sqrt(5)/6,
rho^2 + eta^2     =  1/6,
cos^2(delta_CKM)  =  1/6,
sin^2(delta_CKM)  =  5/6,
tan(delta_CKM)    =  sqrt(5),
delta_CKM         =  arccos(1/sqrt(6)) = arctan(sqrt(5)),
J_0               =  alpha_s^3 sqrt(5) / 72.
```

These specialisations are shown for sanity, not as load-bearing inputs. The
parametric algebra closes for any other count tuple that satisfies the same
weight, radius, and count-constraint hypotheses.

## Proof

Pure substitution.

`(C1)`. `rho = r sqrt(w_A1)`. Substitute `r^2 = 1/n_quark` and
`w_A1 = 1/n_quark`:

```text
rho^2 = r^2 w_A1 = (1/n_quark)(1/n_quark) = 1/n_quark^2.
```

Take positive square root: `rho = 1/n_quark`.

`(C2)`. `eta = r sqrt(w_perp)`. Substitute `r^2 = 1/n_quark` and
`w_perp = (n_quark - 1)/n_quark`:

```text
eta^2 = r^2 w_perp = (1/n_quark)((n_quark - 1)/n_quark)
      = (n_quark - 1)/n_quark^2.
```

Take positive square root: `eta = sqrt(n_quark - 1)/n_quark`.

`(C3)`. By `(C1)` and `(C2)`,

```text
rho^2 + eta^2 = 1/n_quark^2 + (n_quark - 1)/n_quark^2
             = n_quark/n_quark^2 = 1/n_quark.
```

`(C4)`, `(C5)`. The standard CKM phase angle obeys

```text
cos(delta_CKM) = rho / sqrt(rho^2 + eta^2),
sin(delta_CKM) = eta / sqrt(rho^2 + eta^2).
```

Squaring and substituting `(C1)`-`(C3)`,

```text
cos^2(delta_CKM) = rho^2/(rho^2 + eta^2) = (1/n_quark^2)/(1/n_quark) = 1/n_quark,
sin^2(delta_CKM) = eta^2/(rho^2 + eta^2)
                 = ((n_quark - 1)/n_quark^2)/(1/n_quark)
                 = (n_quark - 1)/n_quark.
```

`(C6)`. `tan^2(delta_CKM) = sin^2/cos^2 = (n_quark - 1)`. Positive branch
gives `tan(delta_CKM) = sqrt(n_quark - 1)`.

`(C7)`. `cos(delta_CKM) = 1/sqrt(n_quark)` and `tan(delta_CKM) =
sqrt(n_quark - 1)` give `delta_CKM = arccos(1/sqrt(n_quark)) =
arctan(sqrt(n_quark - 1))`.

`(J)`. By definition of the atlas Jarlskog factor, `J_0 = lambda^6 A^2 eta`.
Substitute `lambda^2 = alpha_s / n_pair`, `A^2 = n_pair / n_color`, and `(C2)`:

```text
J_0 = (alpha_s / n_pair)^3 (n_pair / n_color) (sqrt(n_quark - 1)/n_quark)
    = alpha_s^3 sqrt(n_quark - 1) / (n_pair^2 n_color n_quark).
```

`(Jq)`. Apply `n_quark = n_pair * n_color`:

```text
J_0 = alpha_s^3 sqrt(n_quark - 1) / (n_pair^2 n_color (n_pair n_color))
    = alpha_s^3 sqrt(n_quark - 1) / (n_pair^3 n_color^2).
```

∎

## Derivable corollaries

```text
tan(delta_CKM) eta / rho   =  (n_quark - 1)            (purely combinatorial)
cos^2(delta_CKM) n_quark   =  1                        (radius normalisation)
```

All three follow algebraically from `(C1)`-`(Jq)`.
