## The conservative bridge

The minimal package surface fixes:

1. local algebra `Cl(3)`;
2. spatial substrate `Z^3`;
3. finite local Grassmann / staggered-Dirac dynamics;
4. canonical fixed evaluation surface.

The physical-lattice note then records that the lattice is not a disposable
regulator on the accepted package surface and that the physical-lattice reading
is derived on the accepted one-axiom Hilbert/locality/information surface.

The Planck lane does not use the one-axiom surface to change any of those
physical objects. The primitive cell remains

`H_cell = C^2_t otimes C^2_x otimes C^2_y otimes C^2_z ~= C^16`,

`N_cell = P_A`.

The one-axiom surface is used only to answer:

> what kind of local state datum belongs to the bare physical primitive cell
> when no source or preparation datum is supplied?

That is semantic state language, not a new physical carrier.

## Theorem: one-axiom use is conservative for the Planck lane

On the direct Planck lane, admitting the one-axiom information / Hilbert /
locality surface as local state semantics is conservative over the physical
`Cl(3)` / `Z^3` package in the following sense:

1. it introduces no new numerical constants;
2. it introduces no new tunable parameters;
3. it changes neither `H_cell` nor `P_A`;
4. it changes neither the physical-lattice reading nor the fixed evaluation
   surface;
5. it supplies only the semantics required to distinguish:
   - a prepared local state,
   - a dynamical reduced local state,
   - and a source-free default datum of the bare cell.

Therefore, for the Planck lane, the one-axiom surface is not an extra
scale-setting assumption. It is the conservative local-state semantics needed
to apply the already-fixed primitive-cell object.

## Proof

The only Planck-lane object whose status changes after invoking the one-axiom
surface is the interpretation of `rho_cell`.

Before the one-axiom surface, `rho_cell` can be treated as an arbitrary local
state, which leaves a family of possible packet masses.

After the one-axiom surface, the relevant state is not arbitrary. If the
coefficient is kinematic primitive-cell data and no source or preparation datum
is supplied, then a nontrivial local state would contain retrievable local
information not fixed by the bare cell. Such information would be hidden
preparation data unless it is part of the object itself.

So the one-axiom surface does not import the value `1/4`. It only licenses the
source-free default-datum question. The value then follows from the exact cell
and packet:

`dim(H_cell) = 16`,

`rank(P_A) = 4`.
