**Date:** 2026-05-10
**Claim type:** bounded_theorem
**Claim scope:** conditional on the lepton-block representation and
hypercharge packet cited below, and on the bare gauge-fermion action packet

single-particle exchanges between the `L_L : (2,1)` lepton doublet and the
`e_R : (1,1)` charged-lepton singlet reduce to a single nonzero

`Y_EW(L_L) Y_EW(e_R) = (-1/2)(-1) = 1/2`.

## Statement

Let the external fields be the lepton block
`L_L : (2,1)` and `e_R : (1,1)`, with the standard electroweak
hypercharges

```text
Y_EW(L_L) = -1/2,        Y_EW(e_R) = -1.
```

Equivalently, in the repo doubled convention `Q = T_3 + Y/2`,

```text
Y(L_L) = -1,             Y(e_R) = -2,
Y_EW = Y/2.
```

At order `g_1^2 + g_2^2 + g_3^2`, consider only single-particle exchanges
between the `L_L` and `e_R` bilinears in the bare gauge-fermion action packet.
Then:

1. **Single-gluon exchange is zero.** `SU(3)_c` generators act trivially on
   both `L_L` and `e_R`, so both gluon vertices vanish.
2. **Single-`W` exchange is zero.** `e_R` is an `SU(2)_L` singlet, so the
   `W e_R e_R` vertex vanishes.
3. **Single-`B` exchange is the only nonzero gauge-boson exchange.** Both
   fields have nonzero hypercharge, giving

```text
M_B  =  g_1^2 Y_EW(L_L) Y_EW(e_R) / q^2
     =  g_1^2 / (2 q^2)
```

up to the Lorentz-current factor and the usual sign convention for the
propagator/current ordering.
4. **A fundamental-Higgs exchange is outside the admitted action packet.**
   The bare action packet cited above has no independent fundamental scalar
   or bare Yukawa vertex, so this enumeration does not include a tree-level
   Higgs propagator.

The durable consequence is only the tree-exchange bookkeeping:

```text
lepton-block tree exchange factor = Y_EW(L_L) Y_EW(e_R) = 1/2.
```

This is an abelian charge product, not the non-abelian `SU(N_c)` Fierz
coefficient that appears in the top-Yukawa `Q_L`-block chain.

## Proof

For the gluon candidate, the coupling is proportional to
`g_3 T^a_color`. Both `L_L` and `e_R` are color singlets, so the trivial
representation has `T^a_color = 0` at both endpoints. The exchange amplitude
therefore contains a zero vertex and vanishes.

For the `W` candidate, the coupling is proportional to `g_2 T^a_weak`.
`L_L` is a weak doublet, but `e_R` is a weak singlet. The singlet
representation has `T^a_weak = 0`, so the `W e_R e_R` vertex vanishes and
the exchange amplitude is zero.

For the `B` candidate, the coupling is proportional to `g_1 Y_EW`. The two
hypercharges are both nonzero:

```text
Y_EW(L_L) Y_EW(e_R) = (-1/2)(-1) = 1/2.
```

Thus `B` exchange survives and produces the displayed charge product. If the
same calculation is written in the repo doubled convention, the product is
`Y(L_L)Y(e_R)=2`; that is the same bookkeeping after the corresponding
coupling-normalization convention is applied.

The scalar candidate is not a gauge-boson exchange and is absent under the
cited bare-action boundary. This completes the bounded enumeration.

## What This Claims

- The exact zero of the single-gluon and single-`W` tree-exchange amplitudes
  on the `L_L`/`e_R` block under the stated representation packet.
- The exact nonzero `B`-exchange charge product `1/2` in the standard
  electroweak hypercharge convention.
- The convention translation between the displayed `Y_EW` values and the
  repo doubled-hypercharge values.
- The structural distinction between an abelian charge product and the

  chain.

## Falsifiers

This bounded theorem fails if any one of the following is shown within the
same scoped action packet:

1. `L_L` or `e_R` carries a nontrivial `SU(3)_c` generator.
2. `e_R` carries a nontrivial `SU(2)_L` generator.
3. The cited hypercharge convention or assignments change the displayed
   `Y_EW(L_L)Y_EW(e_R)` product.
4. The admitted bare action contains an independent scalar or contact
   interaction that contributes at the same tree order being enumerated here.
