## Statement

Let `rho_grav` be a fixed normalized source density on a finite periodic
`Z^3` lattice, with the zero mode removed when solving the periodic Poisson
equation. Let `M >= 0` be a scalar source mass and define

```text
rho_mass(.; M) := M rho_grav(.).
```

Let `-Delta_lat` be the nearest-neighbor graph Laplacian

```text
(-Delta_lat f)(x) = 6 f(x) - sum_{|y-x|=1} f(y).
```

On the zero-mean subspace where the periodic inverse is defined, set

```text
V(.; M) := (-Delta_lat)^(-1) rho_mass(.; M).
```

Then `M -> V(.; M)` is a linear map:

```text
V(.; alpha M) = alpha V(.; M),
V(.; M1 + M2) = V(.; M1) + V(.; M2),
V(.; M) = M V_unit(.),    V_unit := (-Delta_lat)^(-1) rho_grav.
```

## Proof

The lattice operator is linear because it is a finite sum:

```text
(-Delta_lat)(alpha f + beta g)
 = alpha (-Delta_lat f) + beta (-Delta_lat g).
```

On the zero-mean subspace, its inverse is linear as well. If
`(-Delta_lat)V_f=f` and `(-Delta_lat)V_g=g`, then

```text
(-Delta_lat)(alpha V_f + beta V_g) = alpha f + beta g,
```

so uniqueness after fixing the zero mode gives

```text
(-Delta_lat)^(-1)(alpha f + beta g)
 = alpha (-Delta_lat)^(-1)f + beta (-Delta_lat)^(-1)g.
```

The mass-source map is linear in `M` by scalar multiplication:

```text
rho_mass(.; alpha M) = alpha rho_mass(.; M),
rho_mass(.; M1 + M2) = rho_mass(.; M1) + rho_mass(.; M2).
```

Composing the mass-source map with the linear Poisson inverse gives the
claimed `V`-linearity:

```text
V(.; alpha M)
 = (-Delta_lat)^(-1)(alpha rho_mass(.; M))
 = alpha V(.; M),

V(.; M1 + M2)
 = (-Delta_lat)^(-1)(rho_mass(.; M1) + rho_mass(.; M2))
 = V(.; M1) + V(.; M2).
```

Taking `M=1` defines `V_unit`, so `V(.; M)=M V_unit`.
