# PMNS Full Microscopic Operator To Pair

## Question
Can the retained PMNS-relevant microscopic pair be recovered starting from the
full charge-preserving microscopic operator `D`, without taking PMNS-side value
data as theorem inputs?

## Exact Result
Yes, at the full-microscopic operator level:

- `D` splits exactly into neutral and charge `-1` lepton sectors on the
  retained charge-preserving lane.
- The sector source-response packs are derived canonically from those extracted
  sector operators.
- Those derived packs recover the PMNS-relevant triplet pair
  `(D_0^trip, D_-^trip)` exactly.

So the current remaining gap is no longer “derive the PMNS pair” or even
“derive the sector response packs.” It is only the actual value law of the full
microscopic charge-preserving operator `D` from `Cl(3)` on `Z^3`.

## Scope
This is a reduction theorem, not yet sole-axiom full closure. The theorem input
is the full microscopic operator `D`; it does not derive the values of `D`
itself from `Cl(3)` on `Z^3`.
