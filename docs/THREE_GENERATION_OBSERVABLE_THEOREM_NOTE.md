## Proof skeleton

### 1. Exact retained generators

On the retained `hw=1` basis, the three elementary translations act as:

- `T_x = diag(-1, +1, +1)`
- `T_y = diag(+1, -1, +1)`
- `T_z = diag(+1, +1, -1)`.

The induced retained `C3[111]` operator is not inserted by hand. It is obtained
from the exact full taste-space `C3[111]` action by restricting to the three
retained `hw=1` eigenspaces, and it acts as

`X1 -> X2 -> X3 -> X1`.

### 2. Exact sector projectors

For a sign triple `(s_x, s_y, s_z)`, the joint projector is

`P(s) = ((I + s_x T_x)/2) ((I + s_y T_y)/2) ((I + s_z T_z)/2)`.

Only three sign triples give nonzero projectors, and they are exactly the
rank-1 projectors onto `X1`, `X2`, `X3`.

### 3. Observable-descent lemma

For a linear surjection `Q : H -> H_red`, an operator `O` descends to the
quotient exactly when `ker(Q)` is `O`-invariant.

This is the precise source of the intertwining law. It is not an extra
admissibility definition layered onto the theorem.

### 4. Full algebra generation

The projectors `P1, P2, P3` and the powers of `C3` generate every matrix unit:

`E_ij = P_i C3^k P_j`

for the unique `k in {0,1,2}` that maps `X_j` to `X_i`.

So the retained exact operator algebra is `M_3(C)`.

### 5. Irreducibility

If `W` is a nonzero invariant subspace and `w in W` has any nonzero component
`w_j`, then

`E_ij w = w_j e_i`

lies in `W` for every `i`. Therefore every nonzero invariant subspace contains
all basis vectors `e_i`, hence equals all of `H_hw=1`.

So the retained exact operator algebra has no nontrivial proper invariant
subspace, and no proper exact quotient preserving that algebra exists.
