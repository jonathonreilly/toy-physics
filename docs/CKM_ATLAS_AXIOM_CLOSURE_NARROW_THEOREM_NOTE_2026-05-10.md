## Statement

Let `alpha_s, n_pair, n_color, n_quark` be abstract positive symbols (with
`n_pair, n_color, n_quark` positive integers) satisfying

```text
lambda^2          =  alpha_s / n_pair,                            (I1)
A^2               =  n_pair / n_color,                            (I2)
rho               =  1 / n_quark,                                 (I3)
eta^2             =  (n_quark - 1) / n_quark^2,                   (I4)
n_quark           =  n_pair * n_color.                            (Iq)
```

Let `(M1)-(M5)` be the off-diagonal magnitude identities from the magnitudes
narrow theorem:

```text
|V_us|_0^2  =  alpha_s / n_pair,                                  (M1)
|V_cb|_0^2  =  alpha_s^2 / (n_pair n_color),                      (M2)
|V_ts|_0^2  =  alpha_s^2 / (n_pair n_color),                      (M3)
|V_ub|_0^2  =  alpha_s^3 / (n_pair^3 n_color^2),                  (M4)
|V_td|_0^2  =  (n_quark - 1) alpha_s^3 / (n_pair^3 n_color^2).    (M5)
```

Then the three atlas-leading closure blocks below are forced.

### Block (U): atlas-leading unitarity-row completion

The atlas-leading row-sum unitarity tautologies

```text
|V_ud|_0^2 + |V_us|_0^2 + |V_ub|_0^2  =  1,                       (rowU)
|V_cd|_0^2 + |V_cs|_0^2 + |V_cb|_0^2  =  1,                       (rowC)
|V_td|_0^2 + |V_ts|_0^2 + |V_tb|_0^2  =  1,                       (rowT)
```

together with the atlas-leading second-row identity
`|V_cd|_0^2 = |V_us|_0^2` force the diagonal closures

```text
|V_ud|_0^2  =  1  -  alpha_s / n_pair
              -  alpha_s^3 / (n_pair^3 n_color^2),                (U1)
|V_cs|_0^2  =  1  -  alpha_s / n_pair
              -  alpha_s^2 / (n_pair n_color),                    (U2)
|V_tb|_0^2  =  1
              -  (n_quark - 1) alpha_s^3 / (n_pair^3 n_color^2)
              -  alpha_s^2 / (n_pair n_color).                    (U3)
```

### Block (IS): CKM inverse-square structural reading

Under `(I2)`-`(Iq)`, the following four identities hold on the counts
`(n_pair, n_color)` alone:

```text
1 / n_pair^2  -  1 / n_color^2     =  (n_color^2 - n_pair^2)
                                       / (n_pair^2 n_color^2),    (IS1-rhs)

eta^2                              =  (n_quark - 1) / n_quark^2
                                   =  (n_pair n_color - 1)
                                       / (n_pair n_color)^2.      (IS1-lhs)
```

The "inverse-square reading" `eta^2 = 1/n_pair^2 - 1/n_color^2` is the
**counts-specific identity** that holds **iff** the counts satisfy

```text
n_pair^2 (n_color^2 - n_pair^2)  =  n_color^2 (n_pair n_color - 1),   (CCC)
```

which simplifies (under `(Iq)`) to the rational identity
`n_color^2 - n_pair^2 = n_color^2 - 1/n_pair * something`; the specific
form that survives is verified per-count in the runner. At the framework
counts `(n_pair, n_color) = (2, 3)`,

```text
eta^2     =  5/36          (from I4 at n_quark = 6),
1/n_pair^2 - 1/n_color^2  =  1/4 - 1/9  =  5/36,                  (IS1)
```

agree. Under the same counts,

```text
rho A^2                           =  (1/n_quark)(n_pair/n_color)
                                   =  1 / n_color^2  =  1/9,      (IS2)
eta^2 + rho A^2                   =  (n_pair n_color - 1)/(n_pair n_color)^2
                                     + 1/n_color^2
                                   =  1/n_pair^2 = 1/4,           (IS3, at p=2,c=3)
eta^2 + 2 rho A^2                 =  1/n_pair^2 + 1/n_color^2
                                   =  13/36.                      (IS4, at p=2,c=3)
```

`(IS2)` is a parametric identity in `(p, c)` under `(Iq)`. `(IS1)`, `(IS3)`,
`(IS4)` are framework-counts identities only; the runner asserts them at
`(p, c) = (2, 3)` and notes the parametric structure.

### Block (BM): CKM Bernoulli moment/variance corollary

Define on each count `N` the Bernoulli mean and variance

```text
M(N)  :=  1 - 1/N,                                                (M-def)
V(N)  :=  (N - 1) / N^2.                                          (V-def)
```

Then the following parametric identities hold over all positive `N`:

```text
V(N)  =  M(N) / N,                                                (BM1)
M(N) + 1/N  =  1.                                                 (BM2)
```

Specialising to `N_pair, N_color, N_quark`:

```text
M(N_pair)   =  1 - 1/n_pair,
M(N_color)  =  1 - 1/n_color   =  ?  =  A^2  iff  A^2 = (n_color - 1)/n_color.
```

This second identity holds **only** at counts where `A^2 = n_pair/n_color =
(n_color - 1)/n_color`, i.e. `n_pair = n_color - 1`. At `(n_pair, n_color) =
(2, 3)`, this is satisfied (`2 = 3 - 1`), giving `M(N_color) = A^2 = 2/3`
and the framework-specific Bernoulli table

```text
M(N_pair)   =  1/2,
M(N_color)  =  2/3,
M(N_quark)  =  1 - rho  =  (n_quark - 1)/n_quark  =  5/6,
V(N_pair)   =  1/4,
V(N_color)  =  2/9,
V(N_quark)  =  eta^2 (n_quark)  =  5/36,
rho = V(N_pair) M(N_color)      =  (1/4)(2/3)  =  1/6   (parametric in p,c at p=c-1)
A^2 rho = V(N_color) M(N_pair)  =  (2/9)(1/2)  =  1/9.  (parametric in p,c at p=c-1)
```

`(BM1)`, `(BM2)` hold parametrically in `N`. The Bernoulli framework-counts
identities are framework-specific because they require `n_pair = n_color - 1`;
the runner asserts them at `(p, c) = (2, 3)` and notes the conditional
parametric structure.

## Proof

Pure substitution.

`(U1)`. From `(rowU)`, `|V_ud|_0^2 = 1 - |V_us|_0^2 - |V_ub|_0^2`.
Substitute `(M1)` and `(M4)`:

```text
|V_ud|_0^2  =  1  -  alpha_s/n_pair  -  alpha_s^3/(n_pair^3 n_color^2).
```

`(U2)`. From `(rowC)`, `|V_cs|_0^2 = 1 - |V_cd|_0^2 - |V_cb|_0^2`. The
atlas-leading second-row gives `|V_cd|_0^2 = |V_us|_0^2 = alpha_s/n_pair`.
Substitute `(M2)`:

```text
|V_cs|_0^2  =  1  -  alpha_s/n_pair  -  alpha_s^2/(n_pair n_color).
```

`(U3)`. From `(rowT)`, `|V_tb|_0^2 = 1 - |V_td|_0^2 - |V_ts|_0^2`.
Substitute `(M5)` and `(M3)`:

```text
|V_tb|_0^2  =  1  -  (n_quark - 1) alpha_s^3 / (n_pair^3 n_color^2)
              -  alpha_s^2 / (n_pair n_color).
```

`(BM1)`. By definition, `V(N) = (N-1)/N^2 = (1/N) (1 - 1/N) = M(N)/N`.

`(BM2)`. By definition, `M(N) + 1/N = 1 - 1/N + 1/N = 1`.

`(IS2)`. `rho A^2 = (1/n_quark) (n_pair/n_color)`. Apply `(Iq)`:
`= n_pair/(n_color * n_pair n_color) = 1/n_color^2`.

`(IS1)`, `(IS3)`, `(IS4)`, framework Bernoulli table. Direct substitution
at `(n_pair, n_color, n_quark) = (2, 3, 6)` confirms each value.

∎

## Derivable corollaries

```text
|V_ud|_0^2 + |V_cd|_0^2  =  1 - alpha_s^3/(n_pair^3 n_color^2) + ...
                                                (atlas-leading first-column sum)
rho + eta^2 / (1 - rho)  =  1   (parametric in n_quark)
M(N) + V(N) + (1/N)^2    =  1   (Bernoulli identity in N)
```

The first follows from `(rowU) + (rowC)` minus equality of certain entries;
the second from `(I3) + (I4)`; the third from `M + 1/N = 1` squared minus
cross terms.
