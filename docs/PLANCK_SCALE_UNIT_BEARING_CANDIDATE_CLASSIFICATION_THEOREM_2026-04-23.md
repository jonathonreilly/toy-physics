**Date:** 2026-04-23

same-surface candidate menu  
**Audit runner:** `scripts/frontier_planck_unit_bearing_candidate_classification.py`

## Question

After the gravity/action scale-ray no-go, is there any **currently admitted
same-surface observable class** on the repo that could still break the Planck
scale ray and fix `a` without imports?

Equivalently:

- maybe the answer is still hidden somewhere inside the retained bulk/action
  family;
- maybe one of the existing spectral-gap/topology identities already carries a
  non-homogeneous absolute anchor;
- maybe one of the current entropy/information carriers already gives the
  missing unit-bearing observable.

## Bottom line

No.

On the current admitted same-surface menu, every candidate falls into one of
three classes:

1. **bulk/action and hierarchy identities**  
   These are homogeneous under the global unit-map ray and cannot fix `a`.

2. **spectral-gap / compactness identities**  
   These are exact function identities in a free radius `R`, but they do not
   select `R`, so they also cannot fix `a`.

3. **entropy / information carriers**  
   The current realized carriers either fail on their present surface
   (`1/6 != 1/4`, bounded boundary-law only) or remain purely structural
   (single-axiom information/Hilbert reductions).

So the current branch-level theorem target is no longer vague:

> any future no-import Planck derivation must introduce a **new** same-surface
> observable of one of the following two types:
>
> - an exact **boundary-density theorem** that lands the Einstein-Hilbert
>   horizon coefficient on a new admissible carrier, or
> - an exact **per-cell action/information quantum theorem** that feeds the
>   geometric action normalization.

## Candidate classes

### Class A: bulk/action and hierarchy identities

This class includes:

- weak-field gravity normalization in lattice units;
- Einstein-Hilbert-style geometric/action equivalence;
- hierarchy product identities like `a v`;
- cross-lane dimensionless combinations such as `G v^2`.

The scale-ray no-go theorem already proves that this entire class is
homogeneous under

`a -> lambda a`.

So Class A fixes exact relations on one positive ray but does not anchor the
ray itself.

### Class B: spectral-gap / compactness identities

This class includes:

- `Lambda_vac = lambda_1(S^3_R) = 3 / R^2`;
- `m_g^2 = 2 hbar^2 Lambda_vac / c^2 = 6 hbar^2 / (c^2 R^2)`.

These are exact retained identities, but they are explicit function identities
in the free compactness radius `R`.

They tell us:

- how `Lambda_vac` depends on `R`,
- how `m_g` depends on `R`,
- and how those quantities are related to each other,

but they do **not** select one `R`.

So Class B is exact and useful, but still homogeneous in the remaining scale
parameter.

### Class C: entropy / information carriers

This class splits into three realized subfamilies:

1. **current BH entropy / free-fermion horizon carrier**  
   Exact asymptote is `c_Widom = 1/6`, not `1/4`, so the current carrier is a
   no-go for an exact black-hole coefficient theorem.

2. **current boundary-law / holographic probes**  
   These are bounded companion probes showing boundary preference, but they do
   not establish Bekenstein-Hawking, AdS/CFT, or an exact horizon coefficient.

3. **single-axiom information/Hilbert reductions**  
   These are structural reductions of graph + locality + unitarity; they do
   not yet carry a physical unit map or per-cell action quantum.

So the currently realized entropy/information menu does not yet contain the
missing unit-bearing theorem either.

## The theorem

### 1. Class A cannot fix `a`

By the scale-ray no-go theorem, the current bulk/action and hierarchy class is
homogeneous under the global unit-map ray.

Therefore Class A fixes only scale-invariant relations and cannot select the
absolute lattice spacing.

### 2. Class B cannot fix `a`

The retained spectral-gap identities are exact functions of `R`, but both

- `Lambda_vac = 3 / R^2`, and
- `m_g^2 = 6 hbar^2 / (c^2 R^2)`

remain valid for every positive `R`.

So Class B supplies exact structure but no endpoint or stationary selector for
the absolute scale.

### 3. The current realized Class C carriers do not fix `a`

The BH entropy no-go theorem proves the current free-fermion carrier lands
`1/6`, not `1/4`.

The bounded boundary-law probes explicitly refuse stronger holographic or
Bekenstein-Hawking claims.

The single-axiom Hilbert/information notes explicitly remain structural rather
than unit-fixing.

So the currently realized entropy/information family does not furnish the
missing unit-bearing observable.

### 4. Therefore no current same-surface candidate breaks the scale ray

The current repo menu contains no further admitted same-surface candidate class
outside A/B/C above.

Since:

- Class A is homogeneous,
- Class B is homogeneous in `R`,
- and realized Class C carriers fail or remain structural,

no currently admitted same-surface observable can break the Planck scale ray.

## The theorem-level statement

**Theorem (current same-surface candidate classification).**
On the current admitted repo surface, every same-surface candidate for a
no-import Planck derivation lies in one of three classes:

1. bulk/action-hierarchy identities,
2. spectral-gap/compactness identities,
3. entropy/information carriers.

Classes (1) and (2) are homogeneous in the remaining scale parameter, and the
currently realized carriers in class (3) either fail on their present carrier
or remain structural only. Therefore no currently admitted same-surface
candidate breaks the Planck scale ray.

Equivalently: the next successful theorem must add a **new** unit-bearing
same-surface observable not already realized in the current candidate menu.

## Strongest surviving theorem classes

After this classification, the strongest surviving theorem targets are exactly
two:

1. **Boundary-density theorem**  
   Derive an exact horizon/boundary density that lands the
   Einstein-Hilbert/Bekenstein-Hawking coefficient on a new admissible
   carrier.

2. **Per-cell action/information quantum theorem**  
   Derive one exact action or entropy quantum per elementary cell and show that
   it feeds the geometric action normalization on the same surface.

Everything weaker than those looks like a repackaging of a class already ruled
out above.

## What this closes

This closes the loophole:

- maybe the missing unit-bearing observable is already somewhere in the
  existing same-surface theorem menu, just not yet organized correctly.

Answer: no.

The present candidate menu is now classified and exhausted.

## What this does not close

This theorem does **not** prove:

- that no future Planck derivation exists;
- that the gravity/action route is dead;
- that a new boundary-density or action-quantum theorem cannot be built;
- that a pinned Planck scale is the permanent final posture.

It proves only the sharper current statement:

- the needed theorem is not already present in the current same-surface menu.
