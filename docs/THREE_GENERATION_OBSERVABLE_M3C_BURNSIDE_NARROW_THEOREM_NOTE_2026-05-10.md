## Statement

Let `H = C^3` with the standard basis `(e_1, e_2, e_3)`. Define the
following operators:

```text
C  =  cyclic permutation matrix sending e_1 -> e_2 -> e_3 -> e_1.        (1)

T_1  =  diag(-1, +1, +1),
T_2  =  diag(+1, -1, +1),
T_3  =  diag(+1, +1, -1).                                                 (2)
```

Let `A = <C, T_1, T_2, T_3>_alg` be the associative subalgebra of `M_3(C)`
(complex linear span closed under products) generated by these four
operators.

**Conclusion (T1) (rank-1 projector lemma).** The three operators

```text
P_1  =  ((I - T_1)/2)((I + T_2)/2)((I + T_3)/2),
P_2  =  ((I + T_1)/2)((I - T_2)/2)((I + T_3)/2),
P_3  =  ((I + T_1)/2)((I + T_2)/2)((I - T_3)/2)                           (3)
```

lie in `A`, are pairwise orthogonal rank-1 self-adjoint projectors
satisfying `P_1 + P_2 + P_3 = I`, and their images are exactly the
joint eigenlines `span{e_1}`, `span{e_2}`, `span{e_3}` respectively.

**Conclusion (T2) (matrix-unit generation).** For all `i, j ∈ {1, 2, 3}`,
the matrix unit `E_ij` (the rank-1 operator `e_i e_j^*`) lies in `A`.
Concretely:

```text
E_ij  =  P_i C^k P_j     where   k = (i - j) mod 3.                       (4)
```

**Conclusion (T3) (algebra equality).** `A = M_3(C)`.

**Conclusion (T4) (irreducibility / no-proper-quotient).** No proper
nonzero subspace `W ⊊ C^3` is invariant under all of `{C, T_1, T_2, T_3}`.
Equivalently, no proper quotient `Q : C^3 -> C^3 / W` admits descent of
all four operators.

**Conclusion (T5) (Burnside corollary).** Any associative subalgebra
of `M_3(C)` that acts irreducibly on `C^3` and contains a rank-one
element equals `M_3(C)`. The algebra `A` realises both hypotheses
explicitly.

## Proof

`(T1)` Pure linear algebra. `T_i^2 = I` (squaring a diagonal `+/-1`
matrix), so the operators `(I + T_i)/2` and `(I - T_i)/2` are
orthogonal projectors, and they commute pairwise (each `T_i` is
diagonal).

The joint `(T_1, T_2, T_3)`-eigenvalues on the three basis vectors
read off from `(2)` are:

```text
e_1:  (-1, +1, +1),
e_2:  (+1, -1, +1),
e_3:  (+1, +1, -1).                                                       (3a)
```

Each basis vector has a distinct sign triple (a single `-1` on the
correspondingly-indexed `T_i`). Therefore the joint sign-triple
character separates the three basis vectors. Acting `P_1` on each:

```text
P_1 e_1  =  ((I - T_1)/2)((I + T_2)/2)((I + T_3)/2) e_1
         =  (1) (1) (1) e_1  =  e_1,
P_1 e_2  =  ((I - T_1)/2)((I + T_2)/2)((I + T_3)/2) e_2
         =  (1) (0) (1) e_2  =  0,
P_1 e_3  =  ((I - T_1)/2)((I + T_2)/2)((I + T_3)/2) e_3
         =  (1) (1) (0) e_3  =  0.
```

(In each line the `0` factor comes from the basis vector failing
the sign condition on that one Z_2 character.) So `P_1` is the rank-1
orthogonal projector onto `span{e_1}`. By the same calculation
`P_2 e_a = δ_{2a} e_2` and `P_3 e_a = δ_{3a} e_3`. Hence each `P_i` is
a rank-1 projector onto `span{e_i}`, the three projectors are pairwise
orthogonal (their images are orthogonal lines), and
`P_1 + P_2 + P_3 = I_3` (resolution of identity on `C^3`). ∎ (T1)

`(T2)` By `(T1)`, `P_i e_a = δ_{ia} e_i`. The cyclic permutation `C`
acts as `C e_a = e_{a+1 mod 3}`, equivalently `C^k e_a = e_{a+k mod 3}`.
For any target `(i, j)`, choose `k = (i - j) mod 3`; then
`C^k e_j = e_{j + k mod 3} = e_i`. Compute `P_i C^k P_j`:

```text
P_i C^k P_j  e_a  =  P_i C^k (δ_{ja} e_j)
                  =  δ_{ja} P_i (C^k e_j)
                  =  δ_{ja} P_i e_i
                  =  δ_{ja} e_i
                  =  (E_ij e)_a.
```

So `P_i C^k P_j = E_ij` for all `(i, j) ∈ {1,2,3}^2` and the chosen `k`.
∎ (T2)

`(T3)` The matrix units `{E_ij : i, j ∈ {1,2,3}}` form a `C`-linear basis
of `M_3(C)`. By `(T2)` all 9 of them lie in `A`, so `A ⊇ M_3(C)`. The
reverse inclusion is trivial because `A ⊆ M_3(C)` by construction.
∎ (T3)

`(T4)` Suppose `W ⊆ C^3` is a nonzero subspace invariant under all of
`{C, T_1, T_2, T_3}`. Then `W` is invariant under every element of
`A = M_3(C)` (by `(T3)`). But for any `0 ≠ w ∈ W`, there exists
`i ∈ {1, 2, 3}` such that the `i`-th coordinate `w_i ≠ 0`; then
`E_ki w = w_i e_k ∈ W` for every `k`. Therefore `e_k ∈ W` for every
`k`, so `W = C^3`. The only invariant subspaces are `{0}` and `C^3`.
∎ (T4)

`(T5)` This is classical Burnside (1905; see also Lomonosov-Rosenthal
2004 for the modern statement of the theorem on irreducible matrix
semigroups containing a rank-one element): an associative subalgebra
`A ⊆ M_n(C)` acting irreducibly on `C^n` and containing a rank-one
element equals `M_n(C)`. Our algebra `A` realises both hypotheses
because (a) `A` acts irreducibly by `(T4)` (irreducible action ↔ no
proper invariant subspace), and (b) `A` contains the rank-1
projectors `P_i` by `(T1)`. So Burnside gives `A = M_3(C)` directly.

`(T3)` is therefore a constructive realisation of the abstract
`(T5)` Burnside corollary: we exhibit the matrix units `E_ij`
explicitly via the formula `E_ij = P_i C^{(i-j) mod 3} P_j`, rather
than invoking Burnside as a black-box existential statement.
∎ (T5)
