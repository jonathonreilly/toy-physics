## Statement

Let `H := ⊗_{x=1}^{N} ℂ²` be the N-site tensor-product Fock space, with the
per-site standard Pauli basis `{|0>_x, |1>_x}` and ladder operators
`σ_+^{(x)}, σ_-^{(x)}` acting on the `x`-th tensor factor (Pauli convention:
`σ_+ |0> = 0, σ_+ |1> = |0>, σ_- |0> = |1>, σ_- |1> = 0`, `σ_3 |0> = +|0>,
σ_3 |1> = -|1>`). Define the per-site number operator and total fermion-number
operator

```text
    n̂_x  :=  σ_-^{(x)} σ_+^{(x)}  =  (I_x - σ_3^{(x)}) / 2  ,    Spec(n̂_x) = {0, 1},   (1)
    Q̂_total  :=  ∑_{x=1}^{N} n̂_x  ,    Spec(Q̂_total) = {0, 1, ..., N},               (2)
```

and the **fermion-parity operator**

```text
    F  :=  ⊗_{x=1}^{N} σ_3^{(x)}  =  exp(i π Q̂_total) .                       (3)
```

Then:

**(P1) Identity `F = exp(i π Q̂_total)`.** The product `⊗_x σ_3^{(x)}` equals
`(-1)^{Q̂_total} = exp(i π Q̂_total)` on each occupation-number basis state.

**(P2) Hermiticity and unitarity.** `F = F^†` and `F^† F = I`.

**(P3) Involution.** `F² = I`.

**(P4) Spectrum.** `Spec(F) = {+1, -1}` exactly; both eigenvalues are
realized.

**(P5) `Z_2` grading.** `H = H_even ⊕ H_odd`, with
`H_even := Ker(F - I)`, `H_odd := Ker(F + I)`, and `dim(H_even) = dim(H_odd)
= 2^{N-1}`.

**(P6) `Z_2`-odd action on ladder operators.** For each site `x`,

```text
    {F, σ_+^{(x)}}  =  F · σ_+^{(x)}  +  σ_+^{(x)} · F  =  0  exact,           (4)
    {F, σ_-^{(x)}}  =  0  exact.                                              (5)
```

Equivalently `F σ_±^{(x)} F^{-1} = -σ_±^{(x)}`.

**(P7) `Z_2`-even action on bilinears.** For all sites `x, y`,

```text
    [F, σ_+^{(x)} σ_-^{(y)}]  =  0  exact ,                                   (6)
    [F, n̂_x]  =  0  exact ,    [F, Q̂_total]  =  0  exact.                     (7)
```

## Proof

The proof is per-site Pauli algebra plus tensor-product factorization.

### Step 1 — Identity `F = ⊗ σ_3 = exp(i π Q̂_total)` (proves `(P1)`)

On the occupation-number basis `|ν> = |ν_1, ν_2, ..., ν_N>` with `ν_x ∈ {0, 1}`:

```text
    σ_3^{(x)} |ν>  =  (1 - 2 ν_x) |ν>  =  (-1)^{ν_x} |ν> .                    (8)
```

Hence

```text
    (⊗_x σ_3^{(x)}) |ν>  =  ∏_x (-1)^{ν_x} |ν>  =  (-1)^{∑_x ν_x} |ν>
                        =  (-1)^{Q̂_total |ν>}  =  exp(i π Q̂_total) |ν>      (9)
```

since `(-1)^k = exp(i π k)` for integer `k`. By linearity this extends to all
of `H`. ∎

### Step 2 — Hermiticity, involution, spectrum (proves `(P2), (P3), (P4)`)

Each `σ_3^{(x)} = (σ_3^{(x)})^†` is Hermitian, so the tensor product
`⊗_x σ_3^{(x)} = F` is Hermitian. Each `σ_3` has `σ_3² = I`, so
`F² = ⊗_x (σ_3^{(x)})² = ⊗_x I_x = I`. Hence `F = F^†` and `F^{-1} = F` (so
`F^† F = I`, unitary).

Since `F² = I` with `F ≠ 0`, the spectrum is `Spec(F) ⊆ {+1, -1}`. Both values
are realized: the all-empty state `|0, 0, ..., 0>` satisfies `F |0> = (+1)^N
|0> = |0>` (eigenvalue +1); the one-particle state `|1, 0, ..., 0>` satisfies
`F |1, 0> = (-1) · (+1)^{N-1} |1, 0> = -|1, 0>` (eigenvalue -1). ∎

### Step 3 — `Z_2`-grading and dimension balance (proves `(P5)`)

The eigenspaces are `H_+ = Ker(F - I)` and `H_- = Ker(F + I)`. On the
occupation basis, `|ν> ∈ H_+` iff `Q(ν) := ∑_x ν_x` is even, and
`|ν> ∈ H_-` iff `Q(ν)` is odd. The number of length-`N` binary strings with
even sum equals `2^{N-1}`; the number with odd sum also equals `2^{N-1}`
(classical combinatorial identity, equivalently `∑_{k even} C(N, k) =
∑_{k odd} C(N, k) = 2^{N-1}`). Hence `dim(H_+) = dim(H_-) = 2^{N-1}` and
`H = H_+ ⊕ H_-`. ∎

### Step 4 — `{F, σ_±^{(x)}} = 0` (proves `(P6)`)

At site `x`, the per-site Pauli relation

```text
    σ_3 σ_+  =  -σ_+ σ_3 ,    σ_3 σ_-  =  -σ_- σ_3                            (10)
```

holds (direct matrix product: `σ_3 σ_+ = diag(1, -1) · [[0, 1], [0, 0]] = [[0,
1], [0, 0]] = σ_+`, but `σ_+ σ_3 = [[0, 1], [0, 0]] · diag(1, -1) = [[0, -1],
[0, 0]] = -σ_+`, so `σ_3 σ_+ + σ_+ σ_3 = σ_+ - σ_+ = 0`, i.e.
`{σ_3, σ_+} = 0`).

At sites `y ≠ x`, the embedded `σ_+^{(x)}` acts as the identity on site `y`,
so `σ_3^{(y)} σ_+^{(x)} = σ_+^{(x)} σ_3^{(y)}` (commute).

Hence

```text
    F · σ_+^{(x)}  =  (⊗_y σ_3^{(y)}) · σ_+^{(x)}
                  =  (⊗_{y ≠ x} σ_3^{(y)}) · σ_3^{(x)} σ_+^{(x)}
                  =  (⊗_{y ≠ x} σ_3^{(y)}) · (-σ_+^{(x)} σ_3^{(x)})
                  =  -σ_+^{(x)} · (⊗_y σ_3^{(y)})
                  =  -σ_+^{(x)} · F                                           (11)
```

so `{F, σ_+^{(x)}} = 0`. Identical reasoning gives `{F, σ_-^{(x)}} = 0`. ∎

### Step 5 — `[F, σ_+^{(x)} σ_-^{(y)}] = 0` (proves `(P7)`)

By `(P6)`, `F σ_±^{(x)} F^{-1} = -σ_±^{(x)}`. Hence

```text
    F · σ_+^{(x)} σ_-^{(y)} · F^{-1}  =  (F σ_+^{(x)} F^{-1}) (F σ_-^{(y)} F^{-1})
                                      =  (-σ_+^{(x)}) · (-σ_-^{(y)})
                                      =  σ_+^{(x)} σ_-^{(y)} ,                (12)
```

so `[F, σ_+^{(x)} σ_-^{(y)}] = 0`. Specializing `y = x`: `[F, n̂_x] = 0` (since
`n̂_x = σ_-^{(x)} σ_+^{(x)}`, or equivalently `n̂_x` is a `Z_2`-even bilinear).
Summing over `x`: `[F, Q̂_total] = 0`. ∎
