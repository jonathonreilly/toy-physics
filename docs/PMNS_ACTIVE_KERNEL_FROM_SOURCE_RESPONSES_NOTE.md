# PMNS Active Kernel from Source Responses

Three source-response columns on the active `hw=1` triplet determine the
active projected kernel exactly. That kernel determines the active block, and
the active block determines `(xbar, ybar, xi_1, xi_2, eta_1, eta_2, delta)`.

This is a lower-level derivation theorem: the theorem input is the response
column set, not `ΔD_act` or explicit PMNS coordinates.
