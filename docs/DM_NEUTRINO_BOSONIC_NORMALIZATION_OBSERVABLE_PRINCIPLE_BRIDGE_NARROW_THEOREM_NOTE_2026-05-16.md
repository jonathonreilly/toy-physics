## Proof

Pure algebraic substitution closed by (X1)'s uniqueness.

**Step 1 ‚Äî `Y^2 = 0`.** From (P), `P_L P_R = 0` (and `P_R P_L = 0`).
Hence `Y^2 = (P_R Œì_1 P_L)(P_R Œì_1 P_L) = P_R Œì_1 (P_L P_R) Œì_1 P_L
= 0`. (Runner verifies on the canonical `C^16` representation.)

**Step 2 ‚Äî `Y + Y^‚ÄÝ = Œì_1`.** Compute `Y^‚ÄÝ` using `(P_R Œì_1 P_L)^‚ÄÝ =
P_L^‚ÄÝ Œì_1^‚ÄÝ P_R^‚ÄÝ = P_L Œì_1 P_R` (using `P_L^‚ÄÝ = P_L`, `P_R^‚ÄÝ = P_R`,
`Œì_1^‚ÄÝ = Œì_1`). So

```text
Y + Y^‚ÄÝ  =  P_R Œì_1 P_L  +  P_L Œì_1 P_R.
```

Using `{Œ≥_5, Œì_1} = 0`, one has `Œì_1 P_L = P_R Œì_1` and
`Œì_1 P_R = P_L Œì_1`. Substituting,

```text
P_R Œì_1 P_L  =  P_R P_R Œì_1  =  P_R Œì_1,
P_L Œì_1 P_R  =  P_L P_L Œì_1  =  P_L Œì_1,
Y + Y^‚ÄÝ      =  (P_R + P_L) Œì_1  =  Œì_1.
```

**Step 3 ‚Äî `det(m I + j Y) = m^{16}` identically in `j`.** Because
`Y^2 = 0`, the matrix `j Y` is nilpotent for any real `j`. The
determinant of `m I + j Y` is unchanged by adding a nilpotent that
commutes with itself (more directly: the characteristic polynomial of a
strictly upper-triangular-in-some-basis nilpotent matrix is `x^{16}`,
so all eigenvalues of `j Y` are zero and `det(m I + j Y) = m^{16}`).
Therefore `W[j Y] = log|m^{16}| - log|m^{16}| = 0` identically in `j`,
as claimed in (WZ). Runner verifies at exact sympy precision by
expanding `det(m I_{16} + j Y_{C^{16}})` as a polynomial in `j` and
checking all `j`-power coefficients above the leading `m^{16}` vanish.

**Step 4 ‚Äî `det(m I + j Œì_1) = (m^2 - j^2)^8`.** Because `Œì_1^2 = I`,
the eigenvalues of `Œì_1` on `V = C^16` are each `¬±1`. The number of
`+1` eigenvalues is `dim ker(Œì_1 - I)` and the number of `-1`
eigenvalues is `dim ker(Œì_1 + I)`. Because `Œì_1` is a Hermitian
involution with `Tr Œì_1 = 0` on the canonical `C^16` representation
(verified in the runner), the two eigenspaces have equal dimension 8.
Therefore

```text
det(m I + j Œì_1)  =  prod_k (m + j Œª_k)
                 =  (m + j)^8 (m - j)^8
                 =  (m^2 - j^2)^8.
```

Hence `W[j Œì_1] = 8 log|1 - j^2/m^2|`, as claimed in (WG). Runner
verifies this at exact sympy precision via the characteristic polynomial
of `Œì_1`.

**Step 5 ‚Äî application of (X1) uniqueness.** By the retained
block-local uniqueness (X1), the unique admissible `W`-source-response
coefficient in the source direction `M` is `c ¬∑ ‚àÇ_j W[j M]|_{j=0}`
(modulo `c`, fixed to `1`). For `M = Y`, this is identically zero
(Step 3, derived from `Y^2 = 0`). For `M = Œì_1`, this is the
nontrivial response from Step 4. Hence the raw chiral bridge direction
`Y` is not an admissible observable source direction on the real-D
block of (X2); only the Hermitian-completed direction `Œì_1` is.

**Step 6 ‚Äî trace ratios `8` and `16`.** Direct computation:
`Tr(Œì_1^‚ÄÝ Œì_1) = Tr(I_{16}) = 16` (since `Œì_1^2 = I`).
`Tr(Y^‚ÄÝ Y) = Tr(P_L Œì_1 P_R Œì_1 P_L)`. Using `Œì_1 P_R = P_L Œì_1`
(Step 2 derivation) and `Œì_1 P_L = P_R Œì_1`,

```text
P_L Œì_1 P_R Œì_1 P_L  =  P_L (P_L Œì_1) Œì_1 P_L
                    =  P_L P_L Œì_1^2 P_L
                    =  P_L ¬∑ I ¬∑ P_L
                    =  P_L.
```

Hence `Tr(Y^‚ÄÝ Y) = Tr(P_L) = 8` on the canonical `C^16` representation
(where `P_L` projects onto the left-chiral 8-dimensional subspace).
Runner verifies `Tr(P_L) = 8` and `Tr(P_R) = 8` symbolically.

**Step 7 ‚Äî collecting.** By Step 6,
`sqrt(Tr(Y^‚ÄÝ Y) / Tr(Œì_1^‚ÄÝ Œì_1)) = sqrt(8/16) = 1/sqrt(2)`. By Step 5,
