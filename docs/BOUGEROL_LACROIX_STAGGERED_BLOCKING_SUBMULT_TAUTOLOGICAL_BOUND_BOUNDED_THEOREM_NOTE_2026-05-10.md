## 3. Proof

### 3.1 Construction of the 1D scale operator family `(A_k)`

Let `V = R` with the Euclidean inner product. Let `α_LM = α_bare / u_0`
be the canonical-surface readout (♢). Define

```text
   A_k : V → V,   A_k(x) = α_LM · x,   for k = 0, 1, …, 15.            (3.1)
```

Each `A_k` is the same 1×1 linear map: multiplication by the scalar
`α_LM`. Since `0 < α_LM < 1`, each `A_k` is bounded and invertible.

The operator norm in 1-dim is the absolute value of the scalar:

```text
   ‖A_k‖_op  =  sup_{‖x‖ ≤ 1} ‖A_k(x)‖  =  |α_LM|  =  α_LM.            (3.2)
```

### 3.2 Sub-multiplicativity identity in 1-dim

For two bounded linear operators `A, B : V → V` on any inner-product
space, the operator norm is sub-multiplicative:

```text
   ‖A B‖_op  ≤  ‖A‖_op · ‖B‖_op.                                       (3.3)
```

In the 1-dim case `V = R`, both sides are absolute values of scalar
products, so (3.3) reduces to the **equality**

```text
   ‖A_k A_j‖_op  =  ‖A_k‖_op · ‖A_j‖_op  =  α_LM²,                     (3.4)
```

since `(A_k A_j)(x) = α_LM² · x`. Iterating across 16 steps gives the
sub-multiplicative chain

```text
   ‖Π_16‖_op  =  ‖A_15 · A_14 · ⋯ · A_0‖_op
              =  ∏_{k=0}^{15} ‖A_k‖_op
              =  α_LM^16.                                              (★)
```

Numerically with `α_LM = 0.09066783601728631…`,

```text
   α_LM^16  =  2.0855(…) × 10^{−17}.                                   (3.5)
```

The identity (★) holds at exact-rational precision for `α_LM` written
as the algebraic combination `α_bare / u_0 = (1/(4π)) / ⟨P⟩^{1/4}`,
and the numerical value (3.5) is the exact mathematical consequence.

per [`YT_P2_TASTE_STAIRCASE_BETA_FUNCTIONS_NOTE_2026-04-17.md`](YT_P2_TASTE_STAIRCASE_BETA_FUNCTIONS_NOTE_2026-04-17.md)
equation (1.1). Each per-step scale ratio is

```text
   μ_{k+1} / μ_k  =  α_LM.                                             (3.7)
```

The 1D scale operator `A_k(x) = α_LM · x` is exactly the operator
that implements (3.7) on a scalar carrier. The 16-step product (★)

Therefore (★) is **not an independent derivation** of `α_LM^16`. It

The honest accounting is therefore:

## 4. Why the 1D operator norm is not the canonical blocking operator's spectrum

The 1D scale operator of §3.1 is a deliberately minimal construction
to give precise mathematical content to "operator-norm bounded by
`α_LM`". It is **not** identified with any canonical Wilson-Kadanoff
blocking operator on the framework's lattice gauge theory. Three
distinct categorical objects are kept rigorously separate:

### 4.1 Staggered Dirac taste spectrum (the wrong object to use)

The staggered Dirac operator `D_stag(p)` on `Z^4` has 16 zero modes
at the corners of the Brillouin zone (the naive species count parent
narrow theorem
[`NAIVE_LATTICE_FERMION_TWO_POWER_D_SPECIES_COUNT_NARROW_THEOREM_NOTE_2026-05-10.md`](NAIVE_LATTICE_FERMION_TWO_POWER_D_SPECIES_COUNT_NARROW_THEOREM_NOTE_2026-05-10.md)).
Interacting taste-breaking splits these zero modes by amounts of
`O(α_s · (aΛ)²)` per Lee-Sharpe staggered ChPT (Phys. Rev. D60 (1999)
114503). These taste eigenvalue splittings are kinematic features of
the Dirac operator at fixed scale; they do **not** equal `α_LM` in any
QFT-canonical reading. Conflating "Dirac taste eigenvalue spectrum" with
"per-step blocking operator spectrum" is a category error.

### 4.2 Wilson-Kadanoff blocking eigenvalues at a marginal fixed point

The canonical Wilson-Kadanoff RG step `R_b` (cell-doubling with
blocking factor `b = 2`) linearised at a fixed point `K^*` of the
gauge-theory action space has eigenvalues `b^{y_i}`, where `y_i` are
the scaling dimensions of operators in the theory (Wilson, "The
renormalization group and critical phenomena", Rev. Mod. Phys. 55
(1983) 583, §III). For a **marginal coupling** (such as the
asymptotically-free gauge coupling in `4D` SU(3)), `y = 0` and the
eigenvalue is exactly `1`, with corrections from the 1-loop beta
function that produce slow logarithmic running. The canonical blocking
operator's marginal eigenvalue is therefore `O(1)`, not `α_LM`.

The 1D scale operator of §3.1 with eigenvalue `α_LM ≈ 0.0907` is
therefore not the linearised Wilson-Kadanoff blocking operator on the
4D SU(3) gauge theory. The two objects share no QFT-canonical
identification.

### 4.3 Lüscher transfer matrix spectral gap

The transfer matrix `T` of a finite Euclidean lattice gauge theory is
selfadjoint and strictly positive (Lüscher, Commun. Math. Phys. 54
(1977) 283), with spectrum `0 < λ_top > λ_2 > …` corresponding to the
mass spectrum of the theory via `E_n = -log(λ_n / λ_top) / a_t`. The
transfer-matrix spectral gap is governed by the lightest particle's
mass in lattice units. There is no QFT-canonical identification of the
gap with `α_LM` either; on a coarse lattice the gap is `O(1)` and on a
fine lattice (continuum limit) the gap approaches zero as `a → 0`.

### 4.4 Summary of the categorical distinction

| Object | Domain | Spectral content | Relation to α_LM |
|---|---|---|---|
| `D_stag` taste spectrum | Z^4 momentum | 16 zero modes, O(α_s a² Λ²) splittings | none |
| `R_b` linearised at fixed pt | RG flow space | b^{y_i}, marginal y=0 → eigenvalue 1 | none |
| Lüscher transfer matrix | Hilbert space at fixed time | mass eigenvalues, O(1) gap on coarse lattice | none |
| **§3.1 1D scale operator** | `V = R` | single singular value α_LM | tautological |

The 1D scale operator family of §3.1 is the only one of the four that
matches the framework's per-step scale ratio. The identity (★) is
mathematically genuine but **categorically restricted** to this 1D
identification.

## 5. Relation to landed external MET narrow theorem

The landed external MET narrow theorem
[`BOUGEROL_LACROIX_OSELEDETS_MET_EXTERNAL_NARROW_THEOREM_NOTE_2026-05-10.md`](BOUGEROL_LACROIX_OSELEDETS_MET_EXTERNAL_NARROW_THEOREM_NOTE_2026-05-10.md)
records the classical Bougerol-Lacroix/Oseledets statement: for an
i.i.d. sequence `(A_n)` of invertible matrices with
`E log^+ ‖A_0‖ < ∞`, `E log^+ ‖A_0^{-1}‖ < ∞`, the limit `(1/N) log
‖A_{N-1} ⋯ A_0 v‖ → λ_1` exists for a.e. realization. The note
explicitly disclaims that any framework blocking map satisfies the MET
hypotheses.

For the 1D scale-operator family of §3.1, the MET trivially holds:
each `A_k = α_LM · I` is i.i.d. (a single point measure on `GL(R)`),
`log ‖A_k‖_op = log α_LM`, `log ‖A_k^{-1}‖_op = -log α_LM`, both
integrable, and

```text
   (1/N) log ‖Π_N‖_op  =  log α_LM   for every N.                       (5.1)
```

So `λ_1 = log α_LM` by direct computation. The MET output is again the
identity (★) up to logarithm.

Generalised non-i.i.d. MET statements (Gouëzel-Karlsson 2015
arXiv:1509.07733; Karlsson-Ledrappier random walks on metric spaces;
Filip 2017 arXiv:1710.10694) extend the classical theorem to
deterministic cocycles `T : X → X` on a measure-preserving system
`(X, T, μ)` with subadditive cocycle structure. The framework's
finite 16-step staircase is not naturally a cocycle: there is no
`(X, T, μ)` and no infinite-N limit; the framework specifies a
**single** finite product. Periodic extension of the 16-step pattern
to an infinite cocycle gives `λ_1 = (1/16) log ‖Π_16‖_op = log α_LM`,
the same answer as (5.1). This is internally consistent but again
tautological.

The honest conclusion is that **none of the MET literature provides
new derivational content beyond the 1D model already considered in §3**,
because the 1D model already saturates the per-step scaling identity
without needing the asymptotic apparatus.
