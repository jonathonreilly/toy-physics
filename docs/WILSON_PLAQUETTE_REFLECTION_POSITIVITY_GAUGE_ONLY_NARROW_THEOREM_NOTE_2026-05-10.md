## Statement

Let `G` be a compact Lie group equipped with normalised bi-invariant Haar
measure `dU` (so `‚à´_G dU = 1` and `dU = d(VU) = d(UV)` for every `V ‚àà G`).
Let `Œõ` be a finite lattice block of the form

```text
Œõ  =  (Z / L_œÑ Z)  √ó  (Z / L_s Z)^d_s                                     (1)
```

with `L_œÑ` even, fix the temporal reflection plane between the slices
`t = -1` and `t = 0`, and define the link reflection map `Œò` on
oriented gauge links by

```text
Œò U_t(x‚Éó, t)  =  U_t(x‚Éó, -1 - (t + 1))^‚ÄÝ,                                  (2a)
Œò U_i(x‚Éó, t)  =  U_i(x‚Éó, -1 - t),    i = 1, ‚Ä¶, d_s.                        (2b)
```

`Œò` is an antilinear involution on the gauge-field algebra: `Œò¬≤ = id`,
`Œò(Œ± U + Œ≤ V) = ·æ± Œò(U) + Œ≤ÃÑ Œò(V)`. Decompose `Œõ = Œõ_+ ‚à™ Œõ_- ‚à™ ‚àÇ` with
`Œõ_+ = { x : t ‚â• 0 }`, `Œõ_- = Œ∏(Œõ_+) = { x : t ‚â§ -1 }`, and `‚àÇ` the set of
links crossing between `t = -1` and `t = 0`. The Wilson plaquette action

```text
S_G[U]  =  Œ≤  ‚àë_{P}  Re [ 1  ‚àí  (1/N_c)  tr U_P ],     Œ≤ > 0              (3)
```

(sum over unoriented plaquettes `P` of `Œõ`, with `U_P` the canonical
ordered product of the four oriented links bounding `P`) splits as
`S_G = S_G^+ + S_G^- + S_G^‚àÇ` where `S_G^+` collects plaquettes with all
four corners in `Œõ_+`, `S_G^-` is the `Œò`-image, and `S_G^‚àÇ` is the
contribution of plaquettes that straddle the reflection plane.

Let `A_+` be the algebra of polynomial functions in the gauge-link
variables localised on `Œõ_+ ‚à™ ‚àÇ` (i.e. polynomials in matrix entries of
links with at least one endpoint in `Œõ_+`).

**Conclusion (T1) (image action equals reflected action).** For every
gauge-link configuration `U`,

```text
S_G^-[U]  =  S_G^+[Œò U]                                                   (4)
```

and `S_G^‚àÇ` is invariant under `Œò` term-by-term up to Hermitian
conjugation of the crossing link variables.

**Conclusion (T2) (factorisation identity).** For every `F ‚àà A_+`,

```text
< Œò(F) ¬∑ F >_{Wilson}
  =  ‚à´_{links in ‚àÇ}  dU_‚àÇ  | I_+(U_‚àÇ; F) |¬≤                              (5)
```

where the inner integral

```text
I_+(U_‚àÇ; F)
  :=  ‚à´_{links in Œõ_+ \ ‚àÇ}  dU_+  e^{-S_G^+[U] ‚àí (1/2) S_G^‚àÇ[U]}  F(U)   (6)
```

is a complex-valued function of the crossing-link variables `U_‚àÇ`. The
square `|I_+|¬≤` is a non-negative real number for every `U_‚àÇ`, the Haar
integral over `U_‚àÇ` is therefore non-negative, and (5) is the
Osterwalder-Seiler / Menotti-Pelissetto factorisation in its native form.

**Conclusion (T3) (gauge-only reflection positivity).**

```text
< Œò(F) ¬∑ F >_{Wilson}  ‚â•  0    for every F ‚àà A_+.                         (7)
```

Equality holds iff `I_+(U_‚àÇ; F) = 0` for Haar-almost-every `U_‚àÇ`.

**Conclusion (T4) (sesquilinear-form positivity).** The map

```text
G_W(F, F')  :=  < Œò(F) ¬∑ F' >_{Wilson}                                   (8)
```

is a positive semi-definite Hermitian sesquilinear form on `A_+`. The
quotient `A_+ / Null(G_W)` carries a positive-definite inner product;
its completion is the gauge-only physical Hilbert space `H_W`.

## Proof

`(T1)` Each plaquette `P` of `Œõ` falls into one of three classes
according to where its four corners lie: (a) all four corners in `Œõ_+`,
(b) all four corners in `Œõ_-`, (c) corners on both sides of the
reflection plane (a temporal plaquette straddling `t = -1` and `t = 0`).
The reflection map (2a)-(2b) carries class (a) plaquettes bijectively
onto class (b) plaquettes and conjugates the trace inside the Wilson
weight, so

```text
Re tr U_{P_+}  =  Re tr U_{Œò(P_+)}^‚ÄÝ  =  Re tr U_{P_-},                  (9)
```

using `Re tr A^‚ÄÝ = Re tr A`. Thus the contribution to `S_G^+` from any
class-(a) plaquette equals the contribution to `S_G^-` from its image
under `Œò`, giving (4).

For class (c) (crossing) plaquettes, the four corners decompose as two
in `Œõ_+` and two in `Œõ_-`. Each crossing plaquette contains one or two
crossing-temporal links, which acquire a Hermitian conjugate under (2a)
and stay invariant in modulus under `Œò`. The collected
crossing-plaquette contribution `S_G^‚àÇ` is real and invariant under
`Œò` term-by-term.

`(T2)` Insert the action splitting `S_G = S_G^+ + S_G^- + S_G^‚àÇ` into
the path integral expression for `< Œò(F) ¬∑ F >_{Wilson}`:

```text
< Œò(F) ¬∑ F >_{Wilson}
  =  Z^{-1}  ‚à´  dU_total  e^{-S_G^+}  e^{-S_G^-}  e^{-S_G^‚àÇ}  Œò(F)  F.  (10)
```

Group the integration variables as `(U_+, U_-, U_‚àÇ)` for links in
`Œõ_+ \ ‚àÇ`, `Œõ_- \ ‚àÇ`, and the crossing set respectively. The Haar
measure factorises link-by-link, and each integration `dU_+` involves
the link variables of `Œõ_+ \ ‚àÇ`, each `dU_-` involves the link
variables of `Œõ_- \ ‚àÇ`. By construction `S_G^+` depends only on `U_+`
and `U_‚àÇ`, and `S_G^-` only on `U_-` and `U_‚àÇ`. The crossing-plaquette
weight `S_G^‚àÇ` couples the crossing links to both halves; we **split**
this weight evenly using `e^{-S_G^‚àÇ} = e^{-(1/2) S_G^‚àÇ} ¬∑ e^{-(1/2) S_G^‚àÇ}`,
attach one half to the `Œõ_+`-integration and one half to the
`Œõ_-`-integration, and define

```text
I_+(U_‚àÇ; F)  :=  ‚à´  dU_+  e^{-S_G^+ ‚àí (1/2) S_G^‚àÇ}  F.                  (11)
```

Then by (T1) and the Haar reflection invariance applied link-by-link
to `Œõ_-` (the change of variables `U_- ‚Ü¶ Œò(U_-)` has unit Jacobian
because `dU_-` is bi-invariant Haar), the `Œõ_-` integration with
weight `e^{-S_G^- ‚àí (1/2) S_G^‚àÇ}` and observable `Œò(F)` reduces to the
complex conjugate `I_+(U_‚àÇ; F)*` of the same integral:

```text
‚à´  dU_-  e^{-S_G^- ‚àí (1/2) S_G^‚àÇ}  Œò(F)  =  I_+(U_‚àÇ; F)*.               (12)
```

(Equation (12) is the heart of the OS argument: `Œò` is an antilinear
involution that commutes with Haar integration on `Œõ_-`, so its action
on the `Œõ_-` half-action and observable just complex-conjugates the
result of integrating against `e^{-S_G^+ ‚àí (1/2) S_G^‚àÇ}` on `Œõ_+`.)
Substituting (11) and (12) into (10),

```text
< Œò(F) ¬∑ F >_{Wilson}
  =  Z^{-1}  ‚à´  dU_‚àÇ  I_+(U_‚àÇ; F)*  ¬∑  I_+(U_‚àÇ; F)
  =  Z^{-1}  ‚à´  dU_‚àÇ  | I_+(U_‚àÇ; F) |¬≤                                  (13)
```

which is exactly (5)-(6) up to the partition-function normalisation
constant `Z = ‚à´ dU_total e^{-S_G}`, which is itself a strictly positive
real number (the integrand is everywhere positive on a finite
configuration set). Absorbing `Z` into the normalisation gives (5).

`(T3)` `|I_+(U_‚àÇ; F)|¬≤` is the squared modulus of a complex number, hence
a non-negative real number for every `U_‚àÇ`. Haar measure is a positive
measure on a compact group, so the integral of a non-negative function
is non-negative: `‚à´ dU_‚àÇ |I_+|¬≤ ‚â• 0`. Therefore `< Œò(F) ¬∑ F >_{Wilson} ‚â• 0`.
Equality holds iff `|I_+(U_‚àÇ; F)|¬≤ = 0` Haar-almost-everywhere, i.e. iff
`I_+(U_‚àÇ; F) = 0` for Haar-a.e. `U_‚àÇ`.

`(T4)` Polarisation of (T3): for `F, F' ‚àà A_+`, write
`< Œò(F) ¬∑ F' >_{Wilson} = ‚à´ dU_‚àÇ I_+(U_‚àÇ; F)* I_+(U_‚àÇ; F')`. This is
the `L¬≤(Haar)` inner product of `I_+(¬∑; F)` and `I_+(¬∑; F')`, hence
Hermitian sesquilinear and positive semi-definite. The null space
`Null(G_W) = { F : G_W(F, F) = 0 }` is exactly the kernel of the linear
map `F ‚Ü¶ I_+(¬∑; F)` into `L¬≤(Haar)`. Quotient by the null space and
complete to obtain `H_W`. ‚àé
