## 1. Claim Scope

> **Theorem (dimensional fourth-root compression at `d = 4`).** Let
> `d >= 1` be an integer and let `f` be an abstract positive quantity
> assigned mass dimension `d`. For a simple-power extraction
> `M = C_M f^alpha` with dimensionless prefactor `C_M`, the output has
> mass dimension one if and only if
>
> ```text
> alpha = 1/d.
> ```
>
> Therefore the `d = 4` instance is
>
> ```text
> M = C_M f^(1/4),
> ```
>
> and among integer `d >= 1`, the rational value `1/d = 1/4` occurs
> uniquely at `d = 4`.

This theorem is only a dimensional-algebra statement. It does not derive

## 2. Proof

By hypothesis, `f` has mass dimension `d`. A simple-power extraction
`M = C_M f^alpha` has mass dimension

```text
[M] = [C_M] + alpha [f] = 0 + alpha d.
```

The condition `[M] = 1` is therefore the linear equation

```text
d alpha = 1.
```

Since `d >= 1`, the unique rational solution is `alpha = 1/d`. Thus the
unique simple-power extraction of mass dimension one is

```text
M = C_M f^(1/d).
```

Substituting `d = 4` gives `alpha = 1/4`. The map `d -> 1/d` is
injective on positive integers, so `1/d = 1/4` has the unique integer
solution `d = 4`.

## 3. Load-Bearing Step

The single load-bearing step is the dimension-counting identity

```text
[f] = d and [C_M] = 0  =>  [C_M f^alpha] = d alpha.
```

Solving `d alpha = 1` gives `alpha = 1/d`. This is exact algebra on an
explicit dimensional premise and has zero ledger dependencies.
