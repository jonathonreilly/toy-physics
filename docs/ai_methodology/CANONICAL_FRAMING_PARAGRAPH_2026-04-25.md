# Canonical Methodology Framing Paragraph

**Date adopted:** 2026-04-25  
**Status:** active per-paper disclosure paragraph for this framework

This is the short disclosure/accountability paragraph intended for paper-level
reuse. It is narrower than the package-level methodology note and should be the
default manuscript/talk insertion point.

## Verbatim Paragraph

> This work was developed using AI-assisted theoretical-physics tools
> (Claude Sonnet 4.5, OpenAI Codex). The tools were used to generate candidate
> derivations, write and run checks, search for obstructions, and review claim
> boundaries. A retained claim is paired with a note, a runner or proof surface,
> and an explicit status label at [repo URL]. The author is responsible for the
> physics: the framework choice, physical interpretation, acceptance or
> rejection of results, and final scope of the claims.

## Usage Guidance

- Replace `[repo URL]` with the specific public repository or per-paper code
  artifact URL.
- If a paper used only one tool, narrow the tool list rather than overstating
  it.
- Keep the final accountability sentence intact unless a venue policy forces a
  different format.
