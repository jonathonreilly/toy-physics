## 1. Claim scope

> **Theorem (sin^2 uniformity at L_t = 4).** For an integer
> `L_t >= 2` with `L_t` even, define the APBC fermion Matsubara mode
> set
>
> ```text
> M(L_t)  :=  { omega_n  =  (2 n + 1) pi / L_t  :  n = 0, 1, ..., L_t - 1 }
> ```
>
> and the per-mode weight `w(omega) := sin^2(omega)`. Then:
>
> **(a)** At `L_t = 2`, `w(omega) = 1` uniformly for both modes
> `omega in {pi/2, 3 pi/2}`.
>
> **(b)** At `L_t = 4`, `w(omega) = 1/2` uniformly for all four modes
> `omega in {pi/4, 3 pi/4, 5 pi/4, 7 pi/4}`.
>
> **(c)** At every even `L_t in {6, 8, 10, 12, 14, 16}`, the multiset
> `{ w(omega) : omega in M(L_t) }` contains at least two distinct
> values (i.e. `w` is not uniform on `M(L_t)`).
>
> **(d)** **Uniqueness in scanned range.** Among even
> `L_t in {2, 4, 6, 8, 10, 12, 14, 16}`, uniformity of `w` on `M(L_t)`
> holds exactly at `L_t in {2, 4}` and at no other scanned value.
>
> **(e)** **Klein-four orbit closure (single-orbit at L_t = 4).** The
> Klein-four group `V_4 = Z_2 x Z_2` generated by the involutions
> `sigma_1 : omega -> pi - omega` and
> `sigma_2 : omega -> omega + pi (mod 2 pi)` acts on `R / 2 pi Z`,
> leaves `w(omega) = sin^2(omega)` invariant
> (`sin^2(pi - omega) = sin^2(omega + pi) = sin^2(omega)`), and
> partitions `M(L_t)` into orbits. At `L_t = 4`, all four modes form
> a single `V_4` orbit, so the orbit-constant `w` value is shared by
> every mode (uniformity (b)). At `L_t = 6`, `M(6)` splits into two
> distinct `V_4` orbits with different `w` values (`{1/4}` on one

## 2. Proof

The proof reduces to elementary trigonometric arithmetic and the
finite-orbit decomposition of an explicit finite group action.

### 2.1 Two `V_4` invariances of `sin^2`

For real `omega`,

```text
sin(pi - omega)  =  sin(omega),                                   (A1)
sin(omega + pi)  =  -sin(omega).                                  (A2)
```

Squaring,

```text
sin^2(pi - omega)  =  sin^2(omega),                               (A3)
sin^2(omega + pi)  =  sin^2(omega).                               (A4)
```

Hence the involutions

```text
sigma_1 : omega -> pi - omega,
sigma_2 : omega -> omega + pi   (mod 2 pi),
```

both preserve `w(omega) = sin^2(omega)`. Their composition is
`sigma_3 = sigma_1 sigma_2 : omega -> -omega (mod 2 pi)` and also
fixes `w` (`sin^2(-omega) = sin^2(omega)`). The group
`{ id, sigma_1, sigma_2, sigma_3 }` is the Klein-four group
`V_4 = Z_2 x Z_2` (each non-identity element has order 2;
`sigma_i sigma_j = sigma_k` for distinct `i, j, k`).

### 2.2 Direct verification at `L_t = 2` — case (a)

The two modes are `omega_0 = pi/2` and `omega_1 = 3 pi/2`. By the
identity `sin^2(pi/2) = 1` and `sin^2(3 pi/2) = sin^2(pi/2 + pi) = 1`
(applying (A4)), both modes have `w = 1`. Uniformity holds. ∎

### 2.3 Direct verification at `L_t = 4` — case (b)

The four modes are `omega_n = (2 n + 1) pi / 4` for `n in {0, 1, 2, 3}`,
i.e.

```text
{ pi/4, 3 pi/4, 5 pi/4, 7 pi/4 }.
```

Compute each `w` value directly:

```text
sin(pi/4)    =  sqrt(2)/2,           sin^2(pi/4)    =  1/2,
sin(3 pi/4)  =  sin(pi - pi/4)
             =  sin(pi/4)
             =  sqrt(2)/2,            sin^2(3 pi/4)  =  1/2,         (B1)
sin(5 pi/4)  =  sin(pi/4 + pi)
             =  -sin(pi/4)
             =  -sqrt(2)/2,           sin^2(5 pi/4)  =  1/2,
sin(7 pi/4)  =  sin(2 pi - pi/4)
             =  -sin(pi/4)
             =  -sqrt(2)/2,           sin^2(7 pi/4)  =  1/2.
```

All four `w` values equal `1/2`. Uniformity holds. ∎

Equivalently, applying the `V_4` action of §2.1 to the seed mode
`pi/4`,

```text
sigma_1(pi/4)  =  pi - pi/4    =  3 pi/4,
sigma_2(pi/4)  =  pi/4 + pi    =  5 pi/4,
sigma_3(pi/4)  =  -pi/4        =  7 pi/4   (mod 2 pi),
```

so `{pi/4, 3 pi/4, 5 pi/4, 7 pi/4}` is a single `V_4` orbit of size 4,
which is the entire mode set `M(4)`. Since `w` is `V_4`-invariant
(§2.1), `w` is constant on the orbit, hence on `M(4)`.

### 2.4 Non-uniformity at `L_t in {6, 8, 10, 12, 14, 16}` — case (c)

For each scanned `L_t`, exhibit two modes with distinct `w` values
(exact algebraic computation; see runner §6 for the full mode-by-mode
list):

```text
L_t = 6:  sin^2(pi/6)   = 1/4,       sin^2(pi/2)   = 1.            (C6)
L_t = 8:  sin^2(pi/8)   = 1/2 - sqrt(2)/4,
          sin^2(3 pi/8) = 1/2 + sqrt(2)/4.                          (C8)
L_t = 10: sin^2(pi/10)  = 3/8 - sqrt(5)/8,
          sin^2(pi/2)   = 1.                                        (C10)
L_t = 12: sin^2(pi/12)  = 1/2 - sqrt(3)/4,
          sin^2(pi/4)   = 1/2.                                       (C12)
L_t = 14: sin^2(pi/14) != sin^2(pi/2),
          values verified non-equal in runner.                       (C14)
L_t = 16: sin^2(pi/16) != sin^2(3 pi/16),
          values verified non-equal in runner.                       (C16)
```

In each case at least two distinct `w` values appear, so `w` is not
uniform on `M(L_t)`. ∎

For (C6), the explicit values

```text
sin^2(pi/6)  = (1/2)^2  =  1/4,
sin^2(pi/2)  = 1^2      =  1,
```

are elementary; for (C8)–(C16) the surd values are standard
half-angle identities verified at exact algebraic precision in §6.

### 2.5 Uniqueness in scanned range — case (d)

Combining (a)–(c): at `L_t in {2, 4}` uniformity holds; at
`L_t in {6, 8, 10, 12, 14, 16}` it fails. Therefore among even
`L_t in {2, 4, 6, 8, 10, 12, 14, 16}`, uniformity of `w` on `M(L_t)`
holds exactly at `L_t in {2, 4}`. ∎

### 2.6 Klein-four orbit closure at `L_t = 4` — case (e)

By §2.3 the action of `V_4` on the seed mode `pi/4` produces all four
elements of `M(4)`. Equivalently `M(4)` is a single `V_4` orbit. Since
`w` is constant on every `V_4` orbit (§2.1), `w` takes a single value
on `M(4)`. The orbit-constant value `1/2` is read off from
`sin^2(pi/4) = 1/2`.

For contrast, at `L_t = 6` the mode set
`M(6) = { pi/6, pi/2, 5 pi/6, 7 pi/6, 3 pi/2, 11 pi/6 }` decomposes
into two `V_4` orbits:

```text
orbit_A  =  { pi/6, 5 pi/6, 7 pi/6, 11 pi/6 }     (size 4),  w = 1/4,
orbit_B  =  { pi/2, 3 pi/2 }                       (size 2),  w = 1.
```

Both orbits are `V_4`-closed (verified by direct application of
`sigma_1, sigma_2, sigma_3`), but they carry distinct `w` values, so
`w` is non-uniform on `M(6)`. ∎

The general structural mechanism: at `L_t = 4` the entire APBC mode
set is a single `V_4` orbit of maximal size `|V_4| = 4`; at every
other scanned `L_t > 4`, the mode set splits into multiple `V_4`
orbits, generically with different `w` values. (`L_t = 2` is the
trivial case where the two modes both have `w = 1` since
`sigma_1(pi/2) = pi/2` is an internal fixed point.)
