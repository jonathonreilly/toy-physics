## Statement

Let `Lambda` be a finite set of `N := |Lambda|` sites. Let
`(chi_x, chi-bar_x)_{x in Lambda}` be `2N` generators of the complex
Grassmann algebra `Lambda_{2N}` satisfying

```text
{chi_x, chi_y}     = 0,                                                       (G1)
{chi-bar_x, chi-bar_y} = 0,                                                   (G2)
{chi-bar_x, chi_y}     = 0,                                                   (G3)
```

for all `x, y in Lambda` (including `x = y`). The Berezin integration
rules are

```text
int dchi_x = 0,    int chi_x dchi_x = 1,                                      (B1)
int dchi-bar_x = 0,    int chi-bar_x dchi-bar_x = 1.                          (B2)
```

Let `M = (M_xy)_{x, y in Lambda}` be a complex `N x N` matrix. Then:

**(D) Berezin determinant identity.**

```text
Z_F  :=  int prod_{x in Lambda} dchi-bar_x dchi_x
          exp(-sum_{x, y in Lambda} chi-bar_x M_xy chi_y)
    =  det(M).                                                                 (1)
```

**(A) Two-point antisymmetry.** For any pair of odd-degree
Grassmann-polynomial operators `O_x` and `O_y` (each an odd-graded
element of the algebra), under the Berezin integral

```text
<O_x O_y>_F  =  -<O_y O_x>_F                                                   (2)
```

(equivalently `<O_x O_y>_F + <O_y O_x>_F = 0`, where the expectation
is taken with respect to the same Z_F).

The framework-specific instance — `Lambda` a finite block of a cubic
lattice `Z^3` with `M` the staggered Dirac-Wilson operator — inherits
(D) and (A) once (G1)-(G3) and (B1)-(B2) are accepted on the
generators `(chi_x, chi-bar_x)`. This special case is shown for
sanity in the runner at small `N`; the algebra also closes for any
other choice of finite `Lambda` and complex matrix `M`.

## Proof

Pure Berezin calculus.

**(D).** Expand the exponential using the nilpotency
`{chi-bar_x, chi_y} = 0` implies the product
`chi-bar_x M_xy chi_y chi-bar_z M_zw chi_w` has each `chi-bar` and
`chi` appearing at most once per site in the surviving terms. The
Berezin integral over each pair `(chi-bar_x, dchi-bar_x)` and
`(chi_x, dchi_x)` selects exactly the term with one factor of
`chi-bar_x chi_x` per site (by (B1), (B2)). Reordering via (G1)-(G3),
that selection is exactly the determinant of `M`:

```text
Z_F  =  sum over permutations pi of Lambda of sign(pi) prod_x M_{x, pi(x)}
    =  det(M).
```

The runner exhibits this exactly for small `N` (`N = 1, 2, 3`) using
sympy Grassmann-symbol multiplication with explicit anticommutation
rules; the result matches `sympy.Matrix(M).det()` exactly.

**(A).** For odd-graded `O_x` and `O_y`, the operator product
`O_x O_y` swaps to `-O_y O_x` by a single Grassmann anticommutation
(each odd-graded element contains a single Grassmann generator factor
that anticommutes with the corresponding factor in the other operator).
For Gaussian-weighted Berezin integration with respect to (1),
linearity gives `<O_x O_y>_F = -<O_y O_x>_F`. The runner exhibits this
for representative odd-graded operators `O_x = chi_x` and
`O_x = chi-bar_x chi_x chi_y` at small `N`.

∎

## Derivable corollaries

**(C1) Pauli exclusion at coincident sites.** From (G1) at `x = y`:
`chi_x chi_x = 0`. This is the algebraic content of
"no two identical fermions occupying the same single-particle mode."
This follows from (G1) alone, before (D) or (A) are invoked.

**(C2) Determinant positivity surface.** If `M` is real and antisymmetric
(so `det(M) = Pf(M)^2`), then `Z_F = det(M) >= 0`. If `M` is complex
but satisfies `gamma_5 M gamma_5^dagger = M^dagger` (gamma_5-Hermiticity),
then `det(M)` is real. These are standard corollaries of the
determinant identity (D); the runner exhibits both.

Both follow algebraically from (D), (A), and standard linear algebra.
