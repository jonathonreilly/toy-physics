## Definitions

Let `V` be a finite set of vertices and let `R subset V x V` be the one-step
dependency relation. The ordered pair `(u, v) in R` means that the value at
vertex `v` after one update tick is allowed to use the value at vertex `u` from
the previous tick.

For an undirected nearest-neighbor lattice update, `R` contains the self-edge
`(v, v)` and the directed versions of the nearest-neighbor edges. For a DAG
update, `R` contains the directed parent-to-child dependency edges, again with
self-edges only if the recurrence carries the previous value forward.

For a source set `S subset V`, define the forward causal neighborhoods
recursively:

```text
C_0(S) = S
C_{t+1}(S) = C_t(S) union {v in V : there exists u in C_t(S) with (u, v) in R}.
```

An update rule is `R`-local when, for every tick `t` and vertex `v`,

```text
x_v(t + 1) = F_{v,t}({x_u(t) : (u, v) in R})
```

for some function `F_{v,t}` and for no arguments outside the listed dependency
set. If the update uses external randomness, the comparison below fixes the
same external random choices in both histories; no stochastic independence or
physical noise model is part of the claim.

## Theorem

For any finite `R`-local update rule and any two initial configurations that
differ only on `S`, the two evolved configurations can differ at tick `t` only
on `C_t(S)`.

Equivalently, no vertex outside the forward reachability set of `S` in the
one-step dependency graph can be influenced by the initial perturbation within
`t` update ticks.

## Proof

The proof is by induction on `t`.

At `t = 0`, the two configurations differ only on `S = C_0(S)` by assumption.

Assume the claim holds at tick `t`: every vertex outside `C_t(S)` has the same
value in the two histories. Let `v` be a vertex outside `C_{t+1}(S)`. By the
definition of `C_{t+1}(S)`, no predecessor `u` with `(u, v) in R` lies in
`C_t(S)`. Every input value used by `F_{v,t}` is therefore equal in the two
histories at tick `t`. Because the update rule for `v` has no arguments outside
those predecessors, both histories produce the same value at `v` at tick
`t + 1`.

Thus all differences at tick `t + 1` are confined to `C_{t+1}(S)`. Induction
gives the result for every finite tick.
