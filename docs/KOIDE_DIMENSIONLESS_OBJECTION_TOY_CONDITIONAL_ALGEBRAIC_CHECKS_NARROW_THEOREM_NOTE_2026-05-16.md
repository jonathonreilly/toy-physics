## Conditioning admissions (explicit antecedents)

All statements below are conditional on the following named admissions,
which are **stated, not derived**:

- **(A1)** Admitted two-channel source-response carrier. Work in the
  reduced complex vector space `V = C^2` with central commutant source
  grammar generated by `I` and `Z = diag(1, -1)`. A general background
  source has the form `J_0 = (s + z, s - z)` with `s, z ∈ Q` (the
  rationals suffice for the algebra). The local probe coefficient is
  `Y = (1, 1)`. The dimensionless ratio is
  ```text
  Q(s, z)  :=  (1/3) * ( 1  +  y_perp / y_plus ),
  y_plus   :=  1 / (1 + s + z),
  y_perp   :=  1 / (1 + s - z).
  ```
- **(A2)** Admitted central `Z` label. `Z = P_plus - P_perp` with
  `Z^2 = I`. Expectation `<Z>` on a source state with channel weight
  `w ∈ [0, 1]` is `<Z>(w) = 2w - 1`.
- **(A3)** Admitted selected-line endpoint algebra. Work in a basis
  adapted to a selected line `L_chi ⊂ V`, with selected-line projector
  `P_chi = diag(1, 0)`. The selected-line endpoint algebra
  `End(L_chi)` is one-dimensional; the ambient endpoint algebra
  `End(V)` is four-dimensional.
- **(A4)** Admitted based-endpoint section. The endpoint torsor shift
  is `c = 0` (based).
- **(A5)** Admitted APS scalar. The APS-type scalar transferred to the
  open endpoint is `eta_APS = 2/9`. The open-endpoint formula is
  ```text
  delta(s_chi, c)  :=  eta_APS * (1 - s_chi)  +  c,
  ```
  where `s_chi ∈ Q` is a spectator/ambient endpoint weight and `c ∈ Q`
  is the endpoint torsor shift.

Each `(Ak)` is an admission. None of `(A1)-(A5)` is derived from a
retained framework axiom by this note. The narrow theorem statement is

## Conditional algebraic identities

Under `(A1)-(A5)` the following identities hold exactly in `Q`.

**(T1) Q zero-background identity.** Under `(A1)`,
```text
Q(0, 0)  =  2 / 3.
```

**(T2) Q common-source invariance.** Under `(A1)`, for any `s ∈ Q`
with denominators nonzero,
```text
Q(s, 0)  =  2 / 3.
```
That is, the common (trace-part) source background drops out of `Q`.

**(T3) Q traceless-background departure.** Under `(A1)`, for any
`z ∈ Q \ {0}` with denominators nonzero,
```text
Q(0, z)  ≠  2 / 3.
```
Two specific exact counterdomains:
```text
Q(0,  1/4)   =  8 / 9,
Q(0, -1/4)   =  8 / 15.
```

**(T4) Z survives under `(A2)`.** Under `(A2)`,
```text
Z^2          =  I,
<Z>(w)       =  2 w - 1,
<Z>(w) = 0   <=>   w = 1 / 2.
```
In particular, the central `Z` label is nonzero on every non-midpoint
source state and therefore distinguishes such states; observable
completeness inside `(A2)` does not erase `Z`.

**(T5) Selected-line endpoint dimensionality and projector identities.**
Under `(A3)`,
```text
dim End(L_chi)  =  1,
dim End(V)      =  4,
P_chi^2         =  P_chi,
P_chi * P_chi   =  P_chi  (selected channel weight 1, spectator weight 0).
```

**(T6) delta selected-line conditional value.** Under `(A3) + (A4) + (A5)`,
the selected-line local positive endpoint source is the projector
`P_chi`, giving `(s_chi, c) = (0, 0)` in `(A5)`, hence
```text
delta(0, 0)  =  eta_APS  =  2 / 9.
```

**(T7) delta ambient countermodels.** Under `(A5)`, with the other
arguments varied to model the ambient countermodels described in the
parent packet,
```text
delta(0, 0)     =  2 / 9      (closing endpoint),
delta(1, 0)     =  0          (spectator endpoint),
delta(1/2, 0)   =  1 / 9      (mixed endpoint),
delta(0, 1/9)   =  1 / 3      (shifted endpoint, c = 1/9).
```
In particular, the closing value `delta = 2/9` is not the only value of
`delta` reachable from `(A5)` alone — at minimum the additional
admissions that select `(s_chi, c) = (0, 0)` are necessary to pick out
the closing countermodel.

**(T8) Q retained-only blocker.** Under `(A1) + (A2)`, observable
completeness (i.e. tracking `<Z>` on source states) does not force
`Q = 2/3` because `Q(0, z) ≠ 2/3` for `z ≠ 0` while still admitting a
nonzero `<Z>` on `(A2)`-allowed source states. Therefore, inside
`(A1) + (A2)`, the conditional `Q = 2/3` requires an additional
admission selecting the `z = 0` background (equivalently the
`Z`-erasure subdomain).

**(T9) delta retained-only blocker.** Under `(A3) + (A4) + (A5)`,
`delta = 2/9` requires the joint selection `(s_chi, c) = (0, 0)`. The
ambient countermodels in `(T7)` show this is not the only value reachable
inside `(A5)`; therefore `delta = 2/9` requires additional admissions
beyond `(A1)-(A5)` to fix the selected-line + based subdomain.

## Proof

Pure rational arithmetic in `Q`.

**Proof of (T1).** Substituting `s = z = 0`:
`y_plus = y_perp = 1 / 1 = 1`, so `Q = (1/3) * (1 + 1) = 2/3`. ∎

**Proof of (T2).** For `z = 0`: `y_plus = y_perp = 1 / (1 + s)`, so
`y_perp / y_plus = 1`, hence `Q = 2/3`. ∎

**Proof of (T3).** For `s = 0`,
`y_plus = 1 / (1 + z), y_perp = 1 / (1 - z)`, so
`y_perp / y_plus = (1 + z) / (1 - z)` and
`Q = (1/3) * (1 + (1 + z) / (1 - z)) = (1/3) * (2 / (1 - z))
   = 2 / (3 (1 - z))`. For `z ≠ 0` this is not `2/3` (which would
require `1 - z = 1`, i.e. `z = 0`). The two concrete substitutions are
direct: `Q(0, 1/4) = 2 / (3 * 3/4) = 8/9`, `Q(0, -1/4) = 2 / (3 * 5/4)
= 8/15`. ∎

**Proof of (T4).** Direct computation:
`Z * Z = diag(1, -1) * diag(1, -1) = diag(1, 1) = I`;
`<Z>(w) = w * 1 + (1 - w) * (-1) = 2 w - 1`;
`2 w - 1 = 0 <=> w = 1/2`. ∎

**Proof of (T5).** `End(L_chi)` is one-dimensional because `L_chi` is a
line in `V`; `End(V) = End(C^2)` is four-dimensional; idempotency of
`P_chi = diag(1, 0)` is direct: `diag(1, 0) * diag(1, 0) = diag(1, 0)`.
Selected channel weight `e_1^T P_chi e_1 = 1` and spectator weight
`e_2^T P_chi e_2 = 0`. ∎

**Proof of (T6).** `(s_chi, c) = (0, 0)` substituted into `(A5)`:
`delta(0, 0) = eta_APS * (1 - 0) + 0 = eta_APS = 2/9`. ∎

**Proof of (T7).** Direct substitution into `(A5)`:
`delta(0, 0) = 2/9 * 1 + 0 = 2/9`;
`delta(1, 0) = 2/9 * 0 + 0 = 0`;
`delta(1/2, 0) = 2/9 * 1/2 + 0 = 1/9`;
`delta(0, 1/9) = 2/9 * 1 + 1/9 = 2/9 + 1/9 = 3/9 = 1/3`. ∎

**Proof of (T8).** From `(T3)`, `Q(0, z) ≠ 2/3` for any `z ∈ Q \ {0}`
with denominators nonzero. From `(T4)`, every non-midpoint source state
in `(A2)` has `<Z> ≠ 0`, so observable completeness on `(A1) + (A2)`
does not entail `z = 0`. Hence the inference `[observable completeness on
(A1)+(A2)] => Q = 2/3` fails; an extra admission selecting `z = 0` (or
equivalently `Z`-erasure) is required. ∎

**Proof of (T9).** From `(T7)`, the four listed countermodels each
satisfy `(A5)` exactly and give four distinct values
`{2/9, 0, 1/9, 1/3}`. Hence `(A5)` alone does not entail
`delta = 2/9`; the joint selection `(s_chi, c) = (0, 0)` is required. ∎
