## 0. Question (Gap #2 probe restatement)

The framework's mean-field staggered Dirac log-determinant per site is

```text
V_taste(m) = -8 log(m^2 + 4 u_0^2)        (HIGGS_MASS_FROM_AXIOM eq.[2])
```

At the Yukawa identification `m = y_t * phi`, Taylor-expanding around `phi = 0` gives

```text
V_taste(y_t phi) = const - (2 y_t^2 / u_0^2) phi^2
                + (y_t^4 / (4 u_0^4)) phi^4 + O(phi^6)
```

so the **bare** φ⁴ coefficient (per channel) is

```text
lambda_bare = y_t^4 / (4 u_0^4).
```

Standard scalar Wilsonian / Coleman-Weinberg λ has a `1/(16 π²)` loop factor (the d=4 heat-kernel measure `(4π)^{-d/2}`); V_taste's φ⁴ coefficient does **not** carry this factor. The Gap #2 probe identified a candidate pathway: perform the missing 1-loop scalar fluctuation integral around the V_taste saddle and generate the heat-kernel measure from a Gaussian integral.

This note performs that 1-loop scalar Coleman-Weinberg correction. The deliverables are:

1. The auxiliary scalar identification (Hubbard-Stratonovich field σ ~ ψ̄ψ identified with `y_t · φ` after Yukawa rescaling).
2. The Gaussian fluctuation integral `Z_σ = ∫Dσ exp[-V_taste(σ_saddle + δσ)]` at one loop, generating the standard CW form `V_CW = (1/(64π²)) M⁴(σ) [log(|M²|/μ²) − 3/2]` with `M²(σ) = V_taste''(σ)`.
3. The corrected effective potential `V_corrected = V_taste + V_CW^scalar` and its φ⁴ coefficient under `σ = y_t · φ`.
4. Verification that the `1/(16 π²)` loop factor emerges from the Gaussian measure rather than being inserted by convention alone, with citations to Vassilevich 2003 (heat-kernel manual) and Dunne 2008 (functional determinants).
5. Numerical comparison of corrected vs uncorrected φ⁴ at the lattice scale and naive m_H readout, with honest scope.

## 2. Elon first-principles strip-down (cap 300 words)

Strip the framework. Standard QFT:

1. A four-fermion interaction `(σ_a ψ̄ψ)² / (constant)` can be re-expressed via Hubbard-Stratonovich as `Z = ∫DψDσ exp[-ψ̄(D̸ + σ)ψ - σ²/(2g)]`.
2. Integrating out the fermion gives `Z_ψ = det(D̸ + σ)`. For staggered Dirac with mean-field links `U → u_0 I`, this determinant is exactly `V_taste(σ) = -8 log(σ² + 4u_0²)` per site (HIGGS_MASS_FROM_AXIOM Step 1 eq.[1]).
3. The remaining σ path integral is `Z_σ = ∫Dσ exp[-V_taste(σ) - σ²/(2g)]`.
4. Saddle-point evaluation: `Z_σ ~ exp(-V_taste(σ_saddle)) · (det V''(σ_saddle))^{-1/2}`.
5. The Gaussian measure `(det V'')^{-1/2} = exp[-(1/2) Tr log V'']`.
6. In d-dim Euclidean field theory: `Tr log K = (1/(2π)^d) ∫d^d k log(k² + M²) = -(M^d/(4π)^{d/2}) Γ(-d/2)` (zeta regularization).
7. At d=4: `-(M⁴/(16π²)) · (3/2 - log(M²/μ²))` (MS-bar). Equivalently, `V_CW = (1/(64π²)) M⁴ [log(M²/μ²) - 3/2]`. The `1/(16π²)` is the heat-kernel measure `(4π)^{-d/2}` at d=4.

**Cited references** (each is the standard textbook authority for the Gaussian-measure derivation, no framework-specific content):

- Vassilevich 2003, Phys. Rep. 388, 279 (heat kernel user's manual; eq. 2.36 gives the d=4 measure explicitly).
- Dunne 2008, J. Phys. A 41, 304006 (functional determinants in QFT; zeta regularization).
- Capitani 2003, Phys. Rep. 382, 113 (lattice perturbation theory; matches MS-bar up to constant).
- Coleman-Weinberg 1973, Phys. Rev. D 7, 1888 (original 1-loop CW formula).

The factor `1/(16π²)` is the standard d=4 Gaussian-measure factor; the additional `1/4` (giving `1/(64π²)`) comes from the M⁴ Taylor structure and the `1/2` from `det^{-1/2}`. This is a standard QFT import used inside a bounded calculation, not a new framework axiom.

## 5. Calculation (symbolic, cross-checked by SymPy)

### Step 5.1 — Hessian M²(σ)

Per V_taste = -8 log(σ² + 4 u_0²),

```text
M^2(sigma) := V_taste''(sigma)
            = 16 (sigma^2 - 4 u_0^2) / (sigma^2 + 4 u_0^2)^2
```

at the symmetric saddle σ=0:

```text
M^2(0) = -4 / u_0^2     (tachyonic, drives EWSB; cited eq.[3]).
```

### Step 5.2 — Series expansion

The Taylor series of M²(σ) and (M²(σ))² around σ=0:

```text
M^2  = (-4/u_0^2) + (3/u_0^4) sigma^2 - (5/(4 u_0^6)) sigma^4 + O(sigma^6)
M^4 = (16/u_0^4) - (24/u_0^6) sigma^2 + (19/u_0^8) sigma^4 + O(sigma^6)
```

The log of |M²| (treating the tachyonic region |M²| = -M² > 0 around σ=0):

```text
log(|M^2|/mu^2) = log(4/(u_0^2 mu^2)) - (3/(4 u_0^2)) sigma^2
              + (1/(32 u_0^4)) sigma^4 + O(sigma^6).
```

### Step 5.3 — V_CW^scalar series

Apply `V_CW = (1/(64 π²)) M^4 [log(|M^2|/μ^2) − 3/2]` and Taylor in σ. The σ⁴ coefficient (the only one we need for the φ⁴ readout under σ = y_t φ) decomposes into a log term and a constant term:

```text
V_CW(sigma^4 coef.) = A * log(4/(u_0^2 mu^2)) + B
                   = (19/(64 pi^2 u_0^8)) log(4/(u_0^2 mu^2))
                   - 5/(32 pi^2 u_0^8).
```

This is verified symbolically by the runner (SymPy `simplify` returns 0 on the residual; PASS in §5 of runner output).

### Step 5.4 — V_CW phi⁴ coefficient (Yukawa identification σ = y_t φ)

Multiply the σ⁴ coefficient by y_t⁴:

```text
V_CW(phi^4 coef.) = y_t^4 * [(19/(64 pi^2 u_0^8)) log(4/(u_0^2 mu^2))
                          - 5/(32 pi^2 u_0^8)].
```

### Step 5.5 — Loop-factor verification

The prefactor 1/(64 π²) decomposes as:

```text
1/(64 pi^2) = (1/2) * (1/(16 pi^2)) * (1/2)
```

where:
- `1/(16 π²) = (4π)^{-d/2}` at d=4 is the standard heat-kernel / Gaussian-measure factor (Vassilevich 2003 eq. 2.36; Dunne 2008).
- The first `1/2` is from `det^{-1/2}` (Gaussian integral around the saddle).
- The second `1/2` is from the M⁴ Taylor structure of the standard CW formula.

This is exactly the Wilsonian / heat-kernel measure factor that V_taste's φ⁴ coefficient `y_t⁴/(4 u_0⁴)` was missing, as identified by the Gap #2 probe.
