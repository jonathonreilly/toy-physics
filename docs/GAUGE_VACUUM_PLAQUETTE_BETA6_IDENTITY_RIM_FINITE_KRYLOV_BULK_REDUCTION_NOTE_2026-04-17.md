## Question

After the identity-rim reduction and the cyclic-bulk reduction, what is the
strongest honest fixed-depth bulk object for explicit class-sector `beta = 6`
closure?

Is the live bulk target still the whole compressed operator?

Or even the whole `eta_6(e)`-cyclic bulk object?

And if the exact propagated retained triple were fixed, would that at least
determine the first finite bulk matrix-element block?

## Answer

For fixed propagation depth `d = L_perp - 1`, not even the whole cyclic bulk
object is the sharpest target.

The exact propagated vector is

`v_6 = (S_6^env)^d eta_6(e)`.

So only the finite Krylov space

`K_d(eta_6(e), S_6^env) = span{eta_6(e), S_6^env eta_6(e), ..., (S_6^env)^d eta_6(e)}`

matters for fixed-depth class-sector closure. Equivalently, the only bulk
matrix elements needed at that depth are the matrix elements of the finite
compression `Q_d S_6^env Q_d`, where `Q_d` projects onto that Krylov space.

That is strictly sharper than saying "the whole compressed bulk operator" and
also sharper than saying "the whole cyclic bulk object".

But the current finite propagated target still does **not** determine even the
first nontrivial finite block on that front.

The runner gives two positive self-adjoint bulk witnesses with:

- one common fixed identity rim state `eta`,
- the same exact propagated retained triple
  `(Z_6^env(W_A), Z_6^env(W_B), Z_6^env(W_C))`,
- but different first `2 x 2` Krylov/Lanczos bulk blocks.

So the current propagated triple still does not close even the first finite
bulk matrix-element block after the identity-rim reduction.

## Theorem 1: exact finite-Krylov reduction of fixed-depth class-sector closure

Let

`eta = eta_6(e)`,

`d = L_perp - 1`,

and let `Q_d` denote the orthogonal projector onto

`K_d(eta, S_6^env) = span{eta, S_6^env eta, ..., (S_6^env)^d eta}`.

Then for every `0 <= n <= d`,

`(Q_d S_6^env Q_d)^n eta = (S_6^env)^n eta`.

Therefore:

- the propagated vector `v_6 = (S_6^env)^d eta` depends only on the finite
  compression `Q_d S_6^env Q_d`,
- every fixed-depth class-sector coefficient
  `z_(p,q)^env(6) = <chi_(p,q), (S_6^env)^d eta>`
  depends only on that same finite compression,
- and every fixed-depth boundary amplitude
  `Z_6^env(W) = <K(W), (S_6^env)^d eta>`
  depends only on that same finite compression.

So the sharp fixed-depth bulk target is not the whole operator and not even the
whole cyclic bulk object. It is the finite Krylov bulk block generated by
`eta_6(e)` up to depth `d`.

## Corollary 1: exact finite bulk matrix-element closure target

Choose any orthonormal basis of `K_d(eta, S_6^env)` beginning with the
normalized identity rim state.

Then fixed-depth class-sector closure is determined by:

- the matrix elements of `Q_d S_6^env Q_d` in that finite basis,
- the identity rim state `eta`,
- and the already-fixed downstream evaluation functionals `chi_(p,q)` or `K(W)`.

So the honest explicit bulk matrix-element front is finite at fixed depth.

## Corollary 2: exact irrelevance of complement data once the finite Krylov block is fixed

If `S_A` and `S_B` are positive self-adjoint bulk operators with the same
finite compression on `K_d(eta, S_A) = K_d(eta, S_B)` and the same identity rim
state `eta`, then

`(S_A)^n eta = (S_B)^n eta`

for every `0 <= n <= d`.

So differences outside that finite Krylov block are invisible to fixed-depth
class-sector closure.

## Theorem 2: fixed identity rim plus fixed propagated retained triple still does not determine even the first finite bulk block

Work on the explicit higher-orbit slice

`{(0,2), (0,3), (0,4), (0,5)}`.

Let `E_3` denote the exact propagated three-sample evaluator to

`(Z_6^env(W_A), Z_6^env(W_B), Z_6^env(W_C))`.

`k = (-0.4885403546..., 0.7722564727..., -0.3682858155..., 0.1712127927...)`

with

`E_3(k) = 0`.

Fix the common identity rim state

`eta = (1,1,1,1) / 2`

and the two positive propagated depth-`d` targets

`v_P = (1.2, 0.9, 1.1, 0.8)`,

`v_Q = v_P + 0.4 k`

`= (1.0045838581..., 1.2089025891..., 0.9526856738..., 0.8684851171...)`.

Then

`E_3(v_P) = E_3(v_Q)`.

Define the positive rank-one bulk witnesses

`S_v = lambda_v u_v u_v^*`,

`u_v = v / ||v||`,

`lambda_v = (||v||^2 / <v, eta>)^(1/d)`.

Then

`(S_v)^d eta = v`.

So `S_P` and `S_Q` have:

- the same identity rim state `eta`,
- the same exact propagated retained triple,
- but different first `2 x 2` Krylov/Lanczos bulk blocks

`         [0.1554841239..., 0.0193294873...]]`.

Therefore the current exact propagated retained triple still does **not**
determine even the first finite bulk matrix-element block after the
identity-rim reduction.
