# Gauge Temporal-Gauge Mixed-Kernel Spatial-Link Factorization Narrow Theorem

**Date:** 2026-05-10
**Claim type:** positive_theorem
**Type:** positive_theorem
**Claim scope:** the standalone algebraic fact that, on a finite hypercubic
Wilson lattice in temporal gauge (`U_t = 1`) with any compact group `G`
and any one-link bi-invariant Wilson weight `w_beta` that is a class
function on `G`, the one-step mixed kernel between two adjacent time
slices factorizes exactly as a tensor product of independent one-link
Wilson convolution factors, one factor per spatial link of the slice;
and the per-link factor acts on every Peter-Weyl character `chi_lambda`
by the exact convolution eigenvalue `c_lambda(beta) / d_lambda`, with
the normalized (trivial-channel) eigenvalue `a_lambda(beta) =
c_lambda(beta) / (d_lambda c_0(beta))` satisfying `a_0(beta) = 1`. This

of the Wilson action in temporal gauge; no `SU(3)`, `beta = 6`, marked

## Statement

Let `G` be a compact group with normalized Haar measure `dg`. Let `w_beta`
be a real positive class function on `G` (the one-link Wilson weight),
and let `{chi_lambda : lambda in Irr(G)}` be the irreducible characters
with respective dimensions `{d_lambda}`. Define the character expansion

```text
w_beta(g)  =  sum_lambda c_lambda(beta) chi_lambda(g),                    (1)
```

with coefficients

```text
c_lambda(beta)  =  integral_G overline{chi_lambda(g)} w_beta(g) dg.       (2)
```

Consider the finite hypercubic Wilson lattice with `d`-dimensional
spatial substrate `Lambda_s` (any finite cardinality `|Lambda_s|`) and
temporal extent `L_t`, in temporal gauge `U_t = 1` so that every
temporal link variable is fixed to the identity. The spatial link
variables on time slice `tau` are denoted `U_tau = (U_tau(x, mu))` with
`x in Lambda_s`, `mu in {1, ..., d}`. The one-step mixed kernel between
adjacent slices is the standard Wilson kernel

```text
K(U_(tau+1), U_tau)
  =  prod_{(x,mu) in Lambda_s x {1,...,d}}
       w_beta( U_(tau+1)(x, mu) U_tau(x, mu)^{-1} ),                       (3)
```

where each factor depends only on the **same** spatial link at the two
adjacent slices, because in temporal gauge the mixed-plaquette
holonomy on plaquette `p = ((x, tau), mu, t)` reduces to
`U_(tau+1)(x, mu) U_tau(x, mu)^{-1}` (no temporal link variable
remains).

**Conclusion (T1) (temporal-gauge tensor factorization).** The one-step
mixed kernel `K` factorizes exactly as an independent product over
spatial links:

```text
K(U_(tau+1), U_tau)
  =  prod_{(x,mu)} K_{(x,mu)}( U_(tau+1)(x, mu), U_tau(x, mu) ),          (4)
```

where each one-link factor

```text
K_{(x,mu)}(U', U)  =  w_beta( U' U^{-1} )                                  (5)
```

depends only on the two link variables at the same spatial link
`(x, mu)` on the two adjacent slices.

**Conclusion (T2) (per-link convolution diagonalization).** For every
irrep `lambda` and every matrix-coefficient slot `(i, j)` of `lambda`,

```text
integral_G K_{(x,mu)}(U', U) lambda_{i,j}(U) dU
  =  ( c_lambda(beta) / d_lambda ) lambda_{i,j}(U'),                       (6)
```

so each per-link kernel acts on the matrix coefficients of `lambda` by
the exact convolution eigenvalue `c_lambda(beta) / d_lambda`. On the
character subspace, equivalently,

```text
integral_G K_{(x,mu)}(U', U) chi_lambda(U) dU
  =  ( c_lambda(beta) / d_lambda ) chi_lambda(U').                          (7)
```

**Conclusion (T3) (trivial-channel normalization).** Define the
normalized per-link convolution eigenvalue

```text
a_lambda(beta)  :=  c_lambda(beta) / ( d_lambda c_0(beta) ).               (8)
```

Then `a_0(beta) = 1` for every `beta > 0`. Equivalently, the trivial
irrep `lambda = 0` is fixed by the normalized per-link kernel.

**Conclusion (T4) (marked vs non-marked compression map).** Let
`f : G^|Lambda_s x {1,...,d}| -> C` be a function that depends only on a
finite "marked" subset `M subset Lambda_s x {1,...,d}` of spatial links
through their irrep matrix coefficients
`{lambda^{(x,mu)}_{i_{x,mu}, j_{x,mu}}}_{(x,mu) in M}`. Then on every
such `f`, the one-step kernel acts as

```text
integral K(U_(tau+1), U_tau) f(U_tau) prod_{(x,mu)} dU_tau(x, mu)
  =  prod_{(x,mu) in M} ( c_{lambda^{(x,mu)}}(beta) / d_{lambda^{(x,mu)}} )
   * prod_{(x,mu) NOT in M} c_0(beta)
   * f(U_(tau+1)).                                                          (9)
```

After dividing out the overall trivial-channel scalar
`prod_{(x,mu)} c_0(beta) = c_0(beta)^{|Lambda_s x {1,...,d}|}`, the
normalized one-step compression on `f` is

```text
prod_{(x,mu) in M} a_{lambda^{(x,mu)}}(beta)
  *  prod_{(x,mu) NOT in M} 1
  *  f(U_(tau+1)).                                                          (10)
```

In particular, non-marked spatial links contribute exactly the
trivial-channel scalar `c_0(beta)` before normalization and `1` after
normalization, while each marked spatial link `(x, mu) in M` contributes
the exact local irrep eigenvalue `a_{lambda^{(x,mu)}}(beta)`.

## Proof

**(T1).** By definition the Wilson action `S_W` is a sum over plaquettes
of `-(beta/N_c) Re Tr U_p`, equivalently `S_W = -sum_p f_beta(U_p)`
where `f_beta` is a fixed real central function of the plaquette
holonomy. In temporal gauge `U_t = 1`, the plaquettes split into two
disjoint classes:

```text
(a) purely spatial plaquettes inside one fixed time slice tau;
(b) mixed plaquettes between adjacent slices tau and tau + 1.
```

Class (a) does not couple distinct slices and is therefore absent from
the one-step mixed kernel `K(U_(tau+1), U_tau)`. Class (b) consists of
plaquettes whose holonomy in temporal gauge is exactly the product of
**one** spatial link on slice `tau`, **one** temporal link `U_t = 1`,
**one** spatial link on slice `tau + 1` (inverted), and **one**
temporal link `U_t = 1`. Writing it out, the holonomy reduces to

```text
U_p  =  U_(tau+1)(x, mu) * 1 * U_tau(x, mu)^{-1} * 1
     =  U_(tau+1)(x, mu) U_tau(x, mu)^{-1}.                              (11)
```

So each mixed plaquette in temporal gauge depends only on the **same**
spatial link `(x, mu)` at the two adjacent slices. Two distinct mixed
plaquettes touch two distinct spatial links of the slice (the spatial
link `(x, mu)` enters at most once at slice `tau` and once at slice
`tau + 1`, and the Wilson action sums over plaquettes, not over
slice-pairs of links). Therefore the contribution of class (b) to the
single-step kernel `K` factorizes as

```text
K(U_(tau+1), U_tau)
  =  prod_{(x,mu)} exp[ -S_W^{(x,mu)} ]
  =  prod_{(x,mu)} w_beta( U_(tau+1)(x, mu) U_tau(x, mu)^{-1} ),         (12)
```

where `w_beta(g) = exp[ +(beta/N_c) Re Tr g ]` is the standard one-link
Wilson weight, recognized here as a class function on `G`. This is
exactly equations `(4)` and `(5)`.

**(T2).** For any compact group `G` with normalized Haar measure, every
class function `w(g) = sum_lambda c_lambda chi_lambda(g)` acts by
convolution on the matrix coefficients of any irrep `lambda` by the
scalar `c_lambda / d_lambda`. Explicitly, for `lambda_{i,j}(g)` the
`(i, j)`-matrix coefficient,

```text
integral_G w(U' U^{-1}) lambda_{i,j}(U) dU
  =  sum_mu c_mu integral_G chi_mu(U' U^{-1}) lambda_{i,j}(U) dU.        (13)
```

By the standard Schur orthogonality for class functions versus matrix
coefficients on a compact group,

```text
integral_G chi_mu(U' U^{-1}) lambda_{i,j}(U) dU
  =  delta_{mu, lambda} (1 / d_lambda) lambda_{i,j}(U').                  (14)
```

(See e.g. Folland, *A Course in Abstract Harmonic Analysis*, Theorem
5.11; or Bump, *Lie Groups*, Theorem 2.3.) Substituting,

```text
integral_G w(U' U^{-1}) lambda_{i,j}(U) dU
  =  ( c_lambda / d_lambda ) lambda_{i,j}(U').                            (15)
```

Specializing to `w = w_beta`, this is equation `(6)`. Equation `(7)`
follows by tracing in `(i, j)`.

**(T3).** The trivial irrep has dimension `d_0 = 1` and character
`chi_0(g) = 1`, so `c_0(beta) = integral w_beta(g) dg > 0` for any real
positive Wilson weight. Substituting in `(8)`,

```text
a_0(beta)  =  c_0(beta) / ( 1 * c_0(beta) )  =  1.                       (16)
```

**(T4).** Since the kernel `K` factorizes as a product of one-link
factors `(4)` and the integration measure
`prod_{(x,mu)} dU_tau(x, mu)` factorizes as a product of independent
Haar measures, the integral factorizes:

```text
integral K(U_(tau+1), U_tau) f(U_tau) prod dU
  =  prod_{(x,mu)} integral_G K_{(x,mu)}(U_(tau+1), U_tau(x, mu))
       * f_{(x,mu)}(U_tau(x, mu)) dU_tau(x, mu),                          (17)
```

where `f_{(x,mu)}` denotes the per-link factor of `f` at link
`(x, mu)`. For `(x, mu) in M`, `f_{(x,mu)} = lambda^{(x,mu)}_{i, j}`
and `(T2)` gives the local irrep eigenvalue
`c_{lambda^{(x,mu)}}(beta) / d_{lambda^{(x,mu)}}` times the same
matrix-coefficient evaluated at `U_(tau+1)(x, mu)`. For `(x, mu) NOT in
M`, `f_{(x,mu)} = 1` is the trivial character, so by `(7)` with
`lambda = 0`, `chi_0 = 1`,

```text
integral_G K_{(x,mu)}(U', U) * 1 dU  =  c_0(beta).                       (18)
```

Multiplying the factors yields `(9)`. Dividing by the overall
trivial-channel scalar `prod_{(x,mu)} c_0(beta)` yields `(10)`. ∎

   `c_k * c_k * c_0 * c_0 = c_k^2 c_0^2` on the marked subspace, and

   coupling between distinct spatial links of the same slice, breaking
   `(4)`. This rules out a vacuous-true reading.
