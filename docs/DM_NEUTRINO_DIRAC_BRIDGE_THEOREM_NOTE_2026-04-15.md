`gamma_5 = G_0 G_1 G_2 G_3`, with left/right projectors
`P_L = (1 + gamma_5)/2`, `P_R = (1 - gamma_5)/2`.

Let the local spatial Higgs family be

`M(phi) = phi_1 Gamma_1 + phi_2 Gamma_2 + phi_3 Gamma_3`.

Then:

1. `M(phi)` is Hermitian.
2. `M(phi)^2 = |phi|^2 I`.
3. `{M(phi), gamma_5} = 0`.
4. Therefore `P_L M(phi) P_L = P_R M(phi) P_R = 0`, so `M(phi)` is purely
   chiral off-diagonal.
5. The exact selector `V_sel` has minima only at the axis vectors `e_1,e_2,e_3`.
6. At those minima, `M(e_i) = Gamma_i`.

Therefore after EWSB axis selection the local post-EWSB Dirac operator is
