## Statement

Let `c, G, A ∈ Q_+` be three abstract positive-rational symbols. Define
the two abstract evaluation forms

```text
S_Wald(c, A)  :=  A · c,                                                  (1)
S_BH(G, A)    :=  A / (4 G).                                              (2)
```

**Conclusion (T1) (algebraic equivalence).** For all `A ∈ Q_+`,

```text
S_Wald(c, A)  =  S_BH(G, A)   for all A > 0
              ⇔
                  c  =  1 / (4 G)                                          (3)
              ⇔
                  4 G c  =  1.                                             (4)
```

**Conclusion (T2) (specializations).** Under the constraint `4 G c = 1`,

```text
c = 1/4   ⇒   G = 1,                                                       (5)
G = 1     ⇒   c = 1/4.                                                     (6)
```

**Conclusion (T3) (constraint set is a non-trivial hyperbola).** The
constraint set `{(c, G) ∈ Q_+^2 : 4 G c = 1}` is a smooth rational
hyperbola in the positive quadrant. It contains the distinct points
`(c, G) = (1/4, 1)`, `(c, G) = (1, 1/4)`, and `(c, G) = (1/2, 1/2)`. By
contrast, `(c, G) = (1, 1)` does **not** lie on the constraint, since
`4 · 1 · 1 = 4 ≠ 1`. The constraint is therefore a non-trivial proper
subset of `Q_+^2`.

## Proof

`(T1)` Suppose first `c = 1/(4G)`. Then for any `A`,

```text
S_Wald(c, A)  =  A · c  =  A · 1/(4G)  =  A/(4G)  =  S_BH(G, A).
```

Conversely suppose `S_Wald(c, A) = S_BH(G, A)` for all `A > 0`. Then
`A · c = A/(4G)` for all `A > 0`, so dividing by `A > 0` gives
`c = 1/(4G)`. Cross-multiplying by `4G` gives `4 G c = 1`. The two
conditions are therefore equivalent.

`(T2)` Substituting `c = 1/4` into `4 G c = 1` gives `G = 1`. Substituting
`G = 1` gives `c = 1/4`. Both specializations are direct rational
substitutions.

`(T3)` By direct verification: `4 · (1/4) · 1 = 1`, `4 · 1 · (1/4) = 1`,
and `4 · (1/2) · (1/2) = 1`, so `(1/4, 1)`, `(1, 1/4)`, and `(1/2, 1/2)`
all lie on the constraint. By contrast, `4 · 1 · 1 = 4 ≠ 1`, so `(1, 1)`
does **not** lie on the constraint. The constraint is therefore a
non-trivial smooth rational hyperbola in the positive quadrant.
∎
