## Setup

### 1. Exact one-clock slice factorization

On a periodic `L_s^3 x L_t` Wilson lattice, split the plaquettes at fixed time
into:

- `3 L_s^3` spatial plaquettes inside one time slice,
- `3 L_s^3` mixed plaquettes linking adjacent slices.

Writing the spatial links on consecutive slices as `U` and `U'`, and the
intervening temporal links as `V`, define the exact one-step kernel

`K_(L_s,beta)(U',U)
 = integral dV exp[(beta/3)
     ( 1/2 sum_(p subset U) Re Tr U_p
     +     sum_(mixed p between U,U',V) Re Tr U_p
     + 1/2 sum_(p subset U') Re Tr U'_p ) ]`.

Because the Wilson weight is real and positive, this kernel is pointwise
positive. By time-reversal / Haar invariance it is symmetric in `U,U'`.
Hence it defines a positive self-adjoint transfer operator `T_(L_s,beta)`.

### 2. Exact local plaquette character

For a plaquette holonomy `W in SU(3)`,

`X(W) = (1/3) Re Tr W = (chi_(1,0)(W) + chi_(0,1)(W)) / 6`.

So the plaquette source is exactly multiplication by the real class function
`X`.

### 3. Exact `SU(3)` dominant-weight recurrence

For the `SU(3)` irreducible characters `chi_(p,q)`,

`chi_(1,0) chi_(p,q)
 = chi_(p+1,q) + chi_(p-1,q+1) + chi_(p,q-1)`,

`chi_(0,1) chi_(p,q)
 = chi_(p,q+1) + chi_(p+1,q-1) + chi_(p-1,q)`,

again omitting all negative-label terms.

Therefore multiplication by `X` is the exact weighted dominant-weight adjacency
operator `J`.

## Theorem 1: exact local character-recurrence law

Multiplication by the plaquette source `X` closes exactly on the dominant-weight
character basis:

`J chi_(p,q)
 = (1/6) [ chi_(p+1,q) + chi_(p-1,q+1) + chi_(p,q-1)
         + chi_(p,q+1) + chi_(p+1,q-1) + chi_(p-1,q) ]`.

This is an explicit nearest-neighbor recurrence on the `SU(3)` dominant-weight
graph.

## Corollary 1: exact self-adjoint recurrence operator

Since `X` is real, multiplication by `X` is self-adjoint on class functions.
In the orthonormal character basis, `J` is therefore a symmetric weighted
adjacency operator.

Because `X(W) in [-1/2, 1]` for all `W in SU(3)`, the spectrum of `J` is
contained in `[-1/2,1]`.

## Theorem 2: exact transfer-operator realization of the plaquette generating object

Let `A_p` be the multiplication operator by the marked plaquette source `X`.
Then on the finite Wilson source surface,

`<f(X_p)>_(beta,L_s,L_t)
 = Tr[T_(L_s,beta)^(L_t) f(A_p)] / Tr[T_(L_s,beta)^(L_t)]`

for every bounded Borel function `f`.

Restricting to the source-cyclic class-function subspace generated by repeated
applications of `A_p`, this is exactly

`<f(X_p)>_(beta,L_s,L_t)
 = integral f(x) dmu_(beta,L_s,L_t)(x)`,

where `mu_(beta,L_s,L_t)` is the spectral measure of the explicit self-adjoint
character-recurrence operator `J` in the positive transfer state induced by
`T_(L_s,beta)`.

So the plaquette generating object is now explicit at the operator level:

- one positive self-adjoint transfer operator `T_(L_s,beta)`,
- one explicit self-adjoint character-recurrence operator `J`,
- one induced transfer state on the source-cyclic class algebra.
