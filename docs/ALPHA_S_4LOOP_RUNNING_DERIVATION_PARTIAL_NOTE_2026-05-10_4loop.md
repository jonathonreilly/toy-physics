## 1. Setup — the 4-loop QCD β-function

The standard SM 4-loop QCD β-function (Vermaseren-Larin-van Ritbergen 1997, Czakon 2005) is

```text
β(α_s) = -β_0 α_s² - β_1 α_s³ - β_2 α_s⁴ - β_3 α_s⁵ + O(α_s⁶)        (BETA)
```

with coefficients

```text
β_0 = (33 - 2 N_f) / (12π)                                            (LO)
β_1 = (153 - 19 N_f) / (24π²)                                         (NLO)
β_2 = (2857/2 - (5033/18) N_f + (325/54) N_f²) / (64π³)              (NNLO; MS-bar)
β_3 = ... (3-loop, MS-bar specific)                                    (NNNLO; MS-bar)
```

where `N_f` is the number of dynamical quark flavors at the running scale.

Equivalent group-theoretic expressions (universal, scheme-independent for β_0, β_1; scheme-dependent for β_2, β_3):

```text
β_0 = ((11/3) C_A - (4/3) T_F N_f) / (4π)                                  (LO group)
β_1 = ((34/3) C_A² - 4 C_F T_F N_f - (20/3) C_A T_F N_f) / (4π)²            (NLO group)
β_2 = (...) / (4π)³                                                          (NNLO; scheme)
```

with SU(3) Casimirs C_F = 4/3, C_A = 3, T_F = 1/2.

At `N_f = 6` (UV regime, above all SM thresholds): `β_0 = (33 - 12)/(12π) = 21/(12π) = 7/(4π)`, equivalent to the comparator value `b_3 = 7` in the inverse-coupling convention.
