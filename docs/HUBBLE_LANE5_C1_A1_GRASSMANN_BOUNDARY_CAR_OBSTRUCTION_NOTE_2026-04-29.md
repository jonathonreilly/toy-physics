## Result

Let `H_bulk` carry a finite CAR representation generated by annihilation
operators `c_i`. Let `P` be a rank-four projection. The compressed operators

```text
a_i = P c_i P
```

obey CAR on `P H_bulk`,

```text
{a_i, a_j^dagger} = delta_ij P,  {a_i, a_j}=0,
```

only under an additional compatibility condition such as:

```text
P c_i = c_i P
```

for the selected modes, or an equivalent statement that `P H_bulk` is a
reducing module for the selected Clifford/CAR algebra. Without that condition,
there are rank-four projections with the same trace/rank data for which the
compressed operators fail CAR.

Thus A1 does not retire `(G1)` from `A_min` as currently written. It exposes a
newly sharp residual import:

```text
P_A is a Clifford/CAR module morphism, or equivalently P_A H_cell is a
reducing active boundary module for the selected edge modes.
```
