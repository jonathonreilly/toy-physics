The question is whether the order-one condition

```text
[[D_F, a], J b J^-1] = 0
```

can be removed as an independent P-LH bridge by deriving the finite
Connes-Chamseddine-style data directly from the repo baseline: the physical
`Cl(3)` local algebra on the `Z^3` spatial substrate.

The review-loop salvage is an open gate, not a primitive admission and not a
theorem-grade status claim. The runner supports three bounded statements:

1. Several algebra types resembling the summands of
   `A_F = C + H + M_3(C)` can be located in existing local structures.
2. Those located summands live on different carrier sectors, so they do not
   yet assemble into one finite Hilbert space `H_F`.
3. Even after imposing a toy direct-sum `A_F`, order-one is not automatic:
   scalar/block-diagonal `D_F` choices satisfy it vacuously, while a generic
   Yukawa-like off-diagonal `D_F` violates it.

## 2. Located Algebra Types

The runner checks the following finite algebra facts.

### 2.1 Complex summand candidate

For Pauli generators of `Cl(3)`, the pseudoscalar

```text
omega = gamma_1 gamma_2 gamma_3
```

is central in the local representation and satisfies `omega^2 = -I`. The real
algebra `R[omega]` is therefore two-dimensional with multiplication

```text
(a + b omega)(c + d omega) = (ac - bd) + (ad + bc) omega,
```

so it is isomorphic to `C` as a real algebra.

### 2.2 Quaternion summand candidate

The even subalgebra of `Cl(3)` is generated by bivectors and satisfies the
quaternion relations. This is the existing `Cl^+(3) ~= H` structure described
in [`CL3_SM_EMBEDDING_THEOREM.md`](CL3_SM_EMBEDDING_THEOREM.md).

### 2.3 Matrix summand candidate

On the hw=1 triplet from the generation-candidate surface, the diagonal
projectors `P_X_i` plus the cyclic permutation `C_3` generate the nine matrix
units of `M_3(C)`. This is the abstract matrix-algebra fact used by
[`THREE_GENERATION_OBSERVABLE_NO_PROPER_QUOTIENT_NARROW_THEOREM_NOTE_2026-05-02.md`](THREE_GENERATION_OBSERVABLE_NO_PROPER_QUOTIENT_NARROW_THEOREM_NOTE_2026-05-02.md).

The caveat is central to the boundary: this `M_3(C)` is located on a generation triplet,
while the Connes-Chamseddine finite algebra uses `M_3(C)` as a color algebra.
Same abstract algebra type is not the same physical bridge.

## 3. Assembly Obstruction

The located summands do not yet form one native finite spectral-triple
algebra on a single `H_F`.

| Candidate summand | Located carrier | Status here |
|---|---|---|
| `C` from `omega` | per-site `Cl(3)` carrier | located |
| `H` from `Cl^+(3)` | per-site `Cl(3)` carrier | located |
| `M_3(C)` from `P_X_i, C_3` | hw=1 generation triplet | located as abstract algebra, not as color |

The first two are per-site local-algebra structures. The third lives on the
generation/taste surface that depends on the staggered-Dirac open gate. The
runner therefore treats the assembly into a single finite Hilbert space as
open. It does not derive the Connes-Chamseddine `H_F`, its bimodule/Krajewski
data, or a physical color/generation identification.

## 4. Order-One Is Not Automatic

The runner builds a toy direct-sum model

```text
H_toy = C^1 + C^2 + C^3
```

with block-diagonal action of `C + H + M_3(C)`. This toy model is not the
physical finite Hilbert space; it is a bounded obstruction test.

The checks show:

- a block-scalar `D_F` satisfies order-one vacuously because `[D_F, a] = 0`;
- `D_F = 0` also satisfies order-one vacuously;
- a Yukawa-like off-diagonal `D_F` violates order-one for generic algebra
  elements.

Thus the direct-sum algebra shape does not force the order-one condition.
Order-one remains an active selection on admissible `D_F` choices unless a
future staggered-Dirac closure derives the relevant `H_F`, algebra embedding,
real structure, and Dirac operator class.

## 5. Boundary

This open gate does not establish:

- an additional repo-wide premise;
- a retained positive theorem;
- a derivation of the physical finite Dirac operator `D_F`;
- a derivation of the Connes-Chamseddine finite Hilbert space `H_F`;
- a derivation of the color `M_3(C)` carrier from the generation
  `M_3(C)` carrier;
- automatic order-one satisfaction for the physical theory.

The useful result is narrower: the remaining P-LH order-one question should be
tested through the existing staggered-Dirac/H_F assembly gate, because the
toy and finite-algebra checks show the order-one condition is downstream of
the still-open carrier, embedding, and `D_F` choices.
