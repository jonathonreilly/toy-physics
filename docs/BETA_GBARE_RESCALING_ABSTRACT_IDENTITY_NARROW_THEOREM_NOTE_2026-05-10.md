## Statement

Let `(g, c, N)` be abstract symbolic variables with `g > 0`, `c > 0`, and
`N` arbitrary in any field of characteristic zero (in particular `Q`).
Define the rational function

```text
β(g, N)  :=  2 N / g²                                                     (1)
```

and consider the multiplicative rescaling map

```text
σ_c : g  ↦  g / c.                                                        (2)
```

**Conclusion (T1) (algebraic rescaling identity).** For all symbolic
`(g, c, N)` with `g > 0`, `c > 0`,

```text
β(g / c, N)  =  c² · β(g, N).                                             (3)
```

**Conclusion (T2) (joint-rescaling invariance of the product).** The
product `β(g, N) · g²` is invariant under the joint rescaling
`(g, β) ↦ (g / c, c² · β)`:

```text
( c² · β(g, N) ) · ( g / c )²  =  β(g, N) · g²  =  2 N.                   (4)
```

**Conclusion (T3) (orbit is non-degenerate).** For any concrete numerical
choice `(g₀, N₀, c) ∈ Q_{>0} × Q_{>0} × Q_{>0}` with `c ≠ 1`, the rescaled
β satisfies `β(g₀/c, N₀) ≠ β(g₀, N₀)` whenever `N₀ ≠ 0`. The orbit of `g₀`
under `{σ_c : c ∈ Q_{>0}}` is the entire positive ray `Q_{>0}`, and the
fiber `β = β₀` over each value `β₀ = 2 N₀ / g₀² > 0` is a single point
in `g`.

## Proof

`(T1)` Direct substitution into (1):

```text
β(g / c, N)  =  2 N / (g / c)²
             =  2 N · c² / g²
             =  c² · ( 2 N / g² )
             =  c² · β(g, N).
```

This is class (A) — pure polynomial-algebra over the rationals.

`(T2)` Direct substitution of the joint rescaling into the product:

```text
( c² · β(g, N) ) · ( g / c )²
   =  c² · β(g, N) · g² / c²
   =  β(g, N) · g²
   =  ( 2 N / g² ) · g²
   =  2 N.
```

So the product is invariant under the joint rescaling and equals the
constant `2 N`.

`(T3)` From (1), `g₀² = 2 N₀ / β(g₀, N₀)`. If `N₀ ≠ 0`, then `g₀ > 0`
determines `β(g₀, N₀) > 0` uniquely, and conversely each `β₀ > 0`
determines a unique positive `g₀ = sqrt(2 N₀ / β₀)`. Under `σ_c` with
`c ≠ 1`, the rescaled `g₀ / c ≠ g₀`, hence by (3),
`β(g₀ / c, N₀) = c² · β(g₀, N₀) ≠ β(g₀, N₀)`. The orbit
`{ g₀ / c : c ∈ Q_{>0} }` is `Q_{>0}` because for any positive rational
`g'`, the choice `c = g₀ / g'` (in `Q_{>0}`) sends `g₀ ↦ g'`. ∎
