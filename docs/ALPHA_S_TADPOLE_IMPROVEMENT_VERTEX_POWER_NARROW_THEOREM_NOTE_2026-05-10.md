## Statement

Let `(alpha_bare, u_0)` be any abstract pair of positive reals, and define
the two tadpole-improved couplings

```text
alpha_LM    :=  alpha_bare / u_0,                                          (1)
alpha_s(v)  :=  alpha_bare / u_0^2.                                        (2)
```

**Conclusion (T1) (vertex-power identity).** For all
`(alpha_bare, u_0) âˆˆ R^+ Ã— R^+`,

```text
alpha_s(v) * u_0^2  =  alpha_bare,                                         (3)
alpha_LM   * u_0    =  alpha_bare.                                         (4)
```

Identities `(3)` and `(4)` are immediate restatements of the definitions
`(1)`, `(2)` and require no further input.

**Conclusion (T2) (geometric-mean identity).** For all
`(alpha_bare, u_0) âˆˆ R^+ Ã— R^+`,

```text
alpha_LM^2  =  alpha_bare * alpha_s(v),                                    (5)
```

equivalently

```text
log alpha_LM  =  ( log alpha_bare + log alpha_s(v) ) / 2.                  (6)
```

**Conclusion (T3) (constant-ratio chain).** For all
`(alpha_bare, u_0) âˆˆ R^+ Ã— R^+`,

```text
alpha_LM    / alpha_bare  =  1 / u_0,                                      (7)
alpha_s(v) / alpha_LM     =  1 / u_0,                                      (8)
```

so the three couplings sit on a geometric progression with common ratio
`1/u_0` (i.e. their logarithms are arithmetic in `log u_0`).

**Conclusion (T4) (boundary at no-improvement).** When `u_0 = 1`,

```text
alpha_LM   =  alpha_bare,                                                  (9)
alpha_s(v) =  alpha_bare,                                                  (10)
```

so the entire tadpole improvement collapses to the bare coupling. This is
the algebraic boundary that recovers the unimproved Wilson surface.

**Conclusion (T5) (inverse map).** For any positive `(alpha_bare, alpha_s(v))`,
the unique positive `u_0` satisfying `(2)` is

```text
u_0  =  sqrt( alpha_bare / alpha_s(v) ).                                  (11)
```

In particular, the pair `(alpha_bare, alpha_s(v))` and the pair
`(alpha_bare, u_0)` carry the same algebraic content.

**Conclusion (T6) (plaquette-fourth-root substitution).** Substituting the
explicit fourth-root convention `u_0 = P^(1/4)` for any positive `P`,
identity `(2)` becomes

```text
alpha_s(v)  =  alpha_bare / sqrt(P).                                      (12)
```

This is an algebraic substitution; whether `P` equals any particular
numerical plaquette expectation is **not** part of this theorem.

## Proof

`(T1)` Substituting `(1)`, `(2)`:

```text
alpha_s(v) * u_0^2  =  (alpha_bare / u_0^2) * u_0^2  =  alpha_bare,
alpha_LM   * u_0    =  (alpha_bare / u_0)   * u_0    =  alpha_bare.
```

`(T2)` From `(1)`, `alpha_LM^2 = alpha_bare^2 / u_0^2`. From `(2)`,
`alpha_bare * alpha_s(v) = alpha_bare^2 / u_0^2`. Hence
`alpha_LM^2 = alpha_bare * alpha_s(v)`. Taking logarithms (both sides
positive since `(alpha_bare, u_0) âˆˆ R^+ Ã— R^+`) gives `(6)`.

`(T3)` From `(1)`, `alpha_LM / alpha_bare = 1/u_0`. From `(2)` divided by
`(1)`, `alpha_s(v) / alpha_LM = (alpha_bare / u_0^2) / (alpha_bare / u_0)
 = 1/u_0`.

`(T4)` Substituting `u_0 = 1` in `(1)` and `(2)`.

`(T5)` Solving `(2)` for `u_0` with `alpha_s(v) > 0` and `alpha_bare > 0`:
`u_0^2 = alpha_bare / alpha_s(v)`, so the unique positive root is
`u_0 = sqrt(alpha_bare / alpha_s(v))`.

`(T6)` Substituting `u_0 = P^(1/4)` (so `u_0^2 = P^(1/2) = sqrt(P)`)
