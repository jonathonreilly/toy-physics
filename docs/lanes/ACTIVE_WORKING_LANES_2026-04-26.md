## Maintenance Rule

- Add a lane here when it is an active work target, not merely a theorem note.
- Keep the status honest: `retained`, `support`, `bounded`, `scaffold`, `open`,
  or `planning`.
- If a lane closes, replace the active-lane row with the retained theorem
  authority and open any remaining sub-lanes explicitly.
- Do not use this file to promote support-grade material into retained claims.
