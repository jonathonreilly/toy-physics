## Statement

Let `v = (v_1, v_2, v_3) ∈ ℝ^3_+` be an abstract positive 3-vector. Let
`omega = e^{2 pi i / 3}` and let

```text
e_+        :=  (1, 1, 1) / sqrt(3),                                  (trivial character chi = 1)
e_omega    :=  (1, omega, omega^2) / sqrt(3),                        (nontrivial chi = omega)
e_{omega^2} :=  (1, omega^2, omega) / sqrt(3),                       (nontrivial chi = omega^2)
```

be the unitary `C_3`-character basis on ℂ^3. Decompose `v` over this basis:

```text
v  =  a_0 e_+  +  z e_omega  +  zbar e_{omega^2},                      (D1)
```

with components

```text
a_0  =  (v_1 + v_2 + v_3) / sqrt(3),                                  (D2)
z    =  (v_1 + omegabar v_2 + omega v_3) / sqrt(3),                   (D3)
```

where `omegabar = omega^{-1} = omega^2`. Reality of `v` forces the
`e_omega` and `e_{omega^2}` coefficients to be complex conjugates,
giving `a_0 ∈ ℝ` and `z ∈ ℂ`.

Define the Koide invariant

```text
Q(v)  :=  (v_1^2 + v_2^2 + v_3^2) / (v_1 + v_2 + v_3)^2,              (D4)
```

equivalent to the standard Koide form on `v_i = sqrt(m_i)`:

```text
Q(v) | _{v_i = sqrt(m_i)}  =  (m_1 + m_2 + m_3) / (sqrt(m_1) + sqrt(m_2) + sqrt(m_3))^2.
```

Then the algebraic equivalence

```text
Q(v)  =  2 / 3       <=>       a_0^2  =  2 |z|^2                       (P1)
```

holds for any positive 3-vector `v`.

## Proof

Pure substitution via Plancherel/Parseval on the `C_3`-character basis.

**Step 1.** Plancherel/Parseval (unitarity of the discrete Fourier
transform on `C_3`) gives

```text
|v|^2  =  v_1^2 + v_2^2 + v_3^2  =  a_0^2 + 2 |z|^2.                  (E1)
```

The factor `2 |z|^2` arises from the two complex-conjugate nontrivial
characters: `|z|^2 + |zbar|^2 = 2 |z|^2`.

**Step 2.** From `(D2)`,

```text
v_1 + v_2 + v_3  =  sqrt(3) * a_0,                                    (E2)
```

so

```text
(v_1 + v_2 + v_3)^2  =  3 a_0^2.                                      (E3)
```

**Step 3.** Substitute `(E1)` and `(E3)` into `(D4)`:

```text
Q(v)  =  (a_0^2 + 2 |z|^2) / (3 a_0^2).                               (E4)
```

**Step 4 (forward direction `Q = 2/3 => a_0^2 = 2 |z|^2`).** Set `Q = 2/3`
in `(E4)`:

```text
(a_0^2 + 2 |z|^2) / (3 a_0^2)  =  2 / 3.
```

Multiply both sides by `3 a_0^2`:

```text
a_0^2 + 2 |z|^2  =  2 a_0^2,
```

which gives `a_0^2 = 2 |z|^2`.

**Step 5 (reverse direction `a_0^2 = 2 |z|^2 => Q = 2/3`).** Substitute
`a_0^2 = 2 |z|^2` (equivalently `|z|^2 = a_0^2 / 2`) into `(E4)`:

```text
Q(v)  =  (a_0^2 + a_0^2) / (3 a_0^2)  =  (2 a_0^2) / (3 a_0^2)  =  2 / 3.
```

Both directions hold for any positive 3-vector `v` with `v_1 + v_2 + v_3
> 0`, which is automatic on `ℝ^3_+`. ∎
