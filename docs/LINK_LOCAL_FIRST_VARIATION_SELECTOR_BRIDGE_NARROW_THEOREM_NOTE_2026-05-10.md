## Statement

Let `E = {t, x, y, z}`. Let

```text
H_cell = Lambda^* span(t,x,y,z)
```

have the orthonormal basis `|S>` indexed by subsets `S subset E`. Let
`P_S = |S><S|`, and define the Hamming-weight projectors

```text
P_k = sum_{|S| = k} P_S,      k = 0, 1, 2, 3, 4.
```

In particular,

```text
P_1 = sum_{a in E} P_{ {a} }
```

is the projector onto the one-axis subspace.

Assume the following two hypotheses.

1. **One-link source-map hypothesis.** There is a four-dimensional source
   space `U = span(u_t, u_x, u_y, u_z)` and an affine-linear map

   ```text
   S_link(u) = sum_{a in E} u_a J_a
   ```

   such that each basis image `J_a` is supported on the corresponding
   one-axis basis state `{a}`.

2. **Active-response/support bridge hypothesis.** The active primitive
   response projector `P_A` is defined as the orthogonal projector onto
   `span(image(dS_link))`.

Then

```text
P_A = P_1.
```

Also,

```text
P_3 * P_1 = 0,
```

so `P_3` is not the support projector of `dS_link` under the one-link
source-map hypothesis.

## Proof

Because `S_link` is affine-linear, its differential sends the basis
variation in direction `u_a` to `J_a`. By the one-link source-map
hypothesis, the four vectors `J_t, J_x, J_y, J_z` span exactly the
one-axis subspace of `H_cell`. Therefore the orthogonal projector onto
`span(image(dS_link))` is

```text
sum_{a in E} P_{ {a} } = P_1.
```

The active-response/support bridge hypothesis identifies `P_A` with that
same support projector, so `P_A = P_1`.

Every basis state in the range of `P_3` has Hamming weight three, while
every `J_a` has Hamming weight one. These subspaces are orthogonal in the
subset basis. Hence `P_3 P_1 = 0`, and `P_3` is not the support projector
of `dS_link`.
