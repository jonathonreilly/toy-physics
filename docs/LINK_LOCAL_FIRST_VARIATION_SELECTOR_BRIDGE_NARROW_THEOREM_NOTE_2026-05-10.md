## Statement

Let `E = {t, x, y, z}` and let

```text
H_cell ~= Lambda^* span(t,x,y,z) ~= (C^2)^4
```

denote the Boolean event-cell algebra with orthonormal basis `|S>` indexed
by subsets `S subset E`. Let `P_S = |S><S|` and let

```text
P_k = sum_{|S|=k} P_S,    k = 0,1,2,3,4
```

denote the Hamming-weight-`k` rank-binomial projector classes on `H_cell`.
In particular `P_1` is the unique rank-four projector class supported on
the four one-axis basis states `{ {t}, {x}, {y}, {z} }`.

**Hypothesis (i): one-link source domain.** There is a four-dimensional
abstract source space

```text
U = span(u_t, u_x, u_y, u_z)
```

and four basis source images `J_t, J_x, J_y, J_z in H_cell` with

```text
support(J_a) = {a},    a in E,
```

defining an affine-linear link-local source map

```text
S_link : U -> H_cell,
S_link(u) = sum_{a in E} u_a J_a.
```

Hypothesis (i) records that each fundamental source variable `u_a` couples
to exactly one one-axis basis state.

**Hypothesis (ii): selector-bridge premise.** The active primitive response
projector `P_A` is identified with the support projector of the algebraic
differential `dS_link`,

```text
P_A  =  proj_{span(image(dS_link))}                                       (B)
     =  proj_{span(J_t, J_x, J_y, J_z)}.
```

Hypothesis (ii) is the "active primitive response = first-variation support"
bridge premise named as the load-bearing not-yet-derived selector in the two
prior audit verdicts. This narrow theorem treats (B) as an input hypothesis,
not as a derived fact. The premise's audit-grade derivability from `A_min`
plus retained authorities is the responsibility of upstream rows, not this
narrow theorem.

**Conclusion.** Under (i) and (ii), the active primitive response projector
is forced to be the rank-four Hamming-weight-one projector:

```text
P_A  =  P_1  =  sum_{a in E} P_{ {a} }.                                   (C1)
```

Furthermore, among the rank-four projector classes on `H_cell`, the
Hamming-weight-three projector `P_3` is excluded as the support of `dS_link`
because `J_a` has Hamming weight one, not three:

```text
P_3 != proj_{span(image(dS_link))}                                        (C2)
```

under hypothesis (i). The Hodge-dual ambiguity `*: P_1 <-> P_3` recorded in
the retained no-go
[`FIRST_ORDER_COFRAME_UNCONDITIONALITY_NO_GO_THEOREM_NOTE_2026-04-30.md`](FIRST_ORDER_COFRAME_UNCONDITIONALITY_NO_GO_THEOREM_NOTE_2026-04-30.md)
is broken on the source-map surface (i) because Hodge complementation sends
a one-link source `J_a` to a three-link composite, which is not in the image
of `dS_link`. The no-go itself remains valid: the substrate symmetry package
alone does not distinguish `P_1` from `P_3`. (i)+(ii) supplies the
distinguishing structure as input hypotheses.

## Proof

Pure substitution and Boolean-projector arithmetic on the stated hypotheses.

### Step 1: differential of the link-local source map

By hypothesis (i), `S_link(u) = sum_a u_a J_a` is affine-linear in the
source variables. Its algebraic differential is

```text
dS_link(du) = sum_{a in E} du_a J_a.
```

Evaluated on the basis variation `du_a`, the four basis images are
`dS_link(du_a) = J_a`. By hypothesis (i),

```text
support(J_a) = {a},
```

so

```text
proj_{span(image(dS_link))} = proj_{span(J_t, J_x, J_y, J_z)}.
```

### Step 2: support projector equals P_1

Each one-axis basis state `|{a}>` is a rank-one projector `P_{ {a} }`. The
span of `{J_t, J_x, J_y, J_z}` is the four-dimensional subspace generated by
the four one-axis basis states, whose orthogonal projector is

```text
P_1 = P_{ {t} } + P_{ {x} } + P_{ {y} } + P_{ {z} } = sum_{a in E} P_{ {a} }.
```

Hence

```text
proj_{span(image(dS_link))} = P_1.
```

### Step 3: bridge premise forces P_A = P_1

Hypothesis (ii) is exactly `P_A = proj_{span(image(dS_link))}`. Combining
with Step 2,

```text
P_A = P_1.
```

This is (C1).

### Step 4: P_3 is excluded as a fundamental first variation

The Hamming-weight-three projector `P_3` is supported on the four
three-axis basis states `{ {x,y,z}, {t,y,z}, {t,x,z}, {t,x,y} }`. Each such
basis state has Hamming weight three. By hypothesis (i), each `J_a` has
Hamming weight one. Therefore each `J_a` lies orthogonal to every
three-axis basis state, and so

```text
J_a perp Im(P_3),
```

giving

```text
P_3 * proj_{span(image(dS_link))} = P_3 * P_1 = 0.
```

In particular `P_3 != proj_{span(image(dS_link))}`. Under hypothesis (ii),
`P_3` is not the active primitive response projector. This is (C2).

### Step 5: P_1 is the unique rank-four match among Hamming-weight classes

The rank-four Hamming-weight projector classes on `H_cell` are exactly
`P_1` (rank `C(4,1) = 4`) and `P_3` (rank `C(4,3) = 4`). The two other
nontrivial weight classes are `P_0` (rank `C(4,0) = 1`), `P_2`
(rank `C(4,2) = 6`), and `P_4` (rank `C(4,4) = 1`), none of which have
rank four. Step 2 selects `P_1` and Step 4 excludes `P_3`, so `P_1` is the
unique rank-four Hamming-weight class consistent with hypothesis (i) and
the bridge premise (ii).

The 17-class enumeration of locally cubic-equivariant rank-four projectors
recorded in the substrate-to-P_A retained no-go is a superset of the two
Hamming-weight rank-four classes above. The argument here restricts within
the Hamming-weight family already singled out by the source-map support
structure; the full 17-class uniqueness statement on the substrate-symmetry
surface is the parent broad row's responsibility, not this narrow theorem's.

QED.
