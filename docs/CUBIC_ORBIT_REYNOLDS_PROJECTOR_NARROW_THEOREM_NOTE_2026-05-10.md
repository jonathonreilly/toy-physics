## Statement

Let `R = R(d_max)` denote the forward-cone single-step neighbour set
on `Z^3`,

```text
R(d_max) = { (1, dy, dz) : dy, dz in Z, |dy| <= d_max, |dz| <= d_max },
```

with `d_max >= 1` an integer support radius. Let `O_h` be the full
cubic group on `Z^3`. The stabilizer of the selected `dx = +1`
spatial direction inside `O_h` acts on `(dy, dz)` as the dihedral group
`D_4`, generated by the 90-degree rotation
`(dy, dz) -> (-dz, dy)` and the reflection `(dy, dz) -> (dy, -dz)`.
This action extends linearly to a representation of `D_4` on the
real vector space

```text
V := { k : R(d_max) -> R }                                            (1)
```

of real-valued functions on `R(d_max)`. Let `V_sym` denote the linear
subspace of cubic-symmetric functions, i.e. functions that factor
through the `D_4` orbit map `o(dy, dz) := { |dy|, |dz| }` viewed as an
unordered multiset, and let `O(d_max)` denote the set of `D_4` orbit
classes appearing in `R(d_max)`. Define the **orbit-averaging map**
`P : V -> V` by

```text
(P k)(dy, dz)  :=  (1 / |orbit(dy, dz)|) * sum over orbit of k.       (2)
```

**Conclusion (T1) (idempotence).** `P` is idempotent on `V`: for every
`k in V`,

```text
P (P k)  =  P k.                                                       (3)
```

**Conclusion (T2) (image characterization).** `Image(P) = V_sym`, the
subspace of cubic-symmetric functions, and `P k = k iff k in V_sym`.

**Conclusion (T3) (`D_4`-equivariance).** `P` commutes with the `D_4`
action on `V`: for every `g in D_4` and every `k in V`,

```text
P (g . k)  =  g . (P k)  =  P k.                                      (4)
```

**Conclusion (T4) (linearity, mass-preservation, positivity).** `P`
is a linear, mass-preserving (`sum_R (P k) = sum_R k`) and
positivity-preserving (`k >= 0` pointwise implies `P k >= 0`
pointwise) operator on `V`.

**Conclusion (T5) (orbit-class basis).** `V_sym` has real dimension
`|O(d_max)|`, equal to the number of `D_4` orbit classes in
`R(d_max)`. The indicator functions of the orbit classes form a
canonical basis of `V_sym`. In particular,

```text
|O(d_max)| = (d_max + 1)(d_max + 2) / 2,                              (5)
```

so `dim V_sym = 3, 6, 10, 15, 21` for `d_max = 1, 2, 3, 4, 5`.

**Conclusion (T6) (Reynolds-operator formula and uniqueness).** `P`
equals the standard finite-group Reynolds operator

```text
P  =  (1 / |D_4|) sum_{g in D_4} g                                    (6)
```

acting on `V`. `P` is the **unique** linear projection from `V` onto
`V_sym` that commutes with the `D_4` action.

## Proof

`(T1)` Idempotence. For `(dy, dz) in R(d_max)` let `O := orbit(dy, dz)`
denote its `D_4` orbit. Because `R(d_max)` is `D_4`-invariant (the
group acts by sign flips and axis swap, which preserve the cube
`|dy|, |dz| <= d_max`), `O subset R(d_max)`. By construction `P k` is
constant on `O` with value `<k>_O := (1/|O|) sum_{p in O} k(p)`. Hence

```text
(P (P k))(dy, dz) = (1/|O|) sum_{p in O} (P k)(p)
                  = (1/|O|) sum_{p in O} <k>_O
                  = <k>_O
                  = (P k)(dy, dz).
```

So `P (P k) = P k` on `R(d_max)`, giving `P P = P`.

`(T2)` Image characterization. If `k in V_sym`, then `k` is constant
on each orbit, so `<k>_O = k(dy, dz)` for any representative, hence
`(P k)(dy, dz) = k(dy, dz)`, giving `P k = k`. Conversely, if
`P k = k`, then `k(dy, dz) = <k>_O` is constant on each orbit `O`,
so `k in V_sym`. Hence `Image(P) = { k : P k = k } = V_sym`.

`(T3)` Equivariance. Fix `g in D_4`. For any `(dy, dz) in R(d_max)`,
`g . orbit(dy, dz) = orbit(g . (dy, dz)) = orbit(dy, dz)` because `g`
permutes the orbit. Hence

```text
(P (g . k))(dy, dz) = (1/|O|) sum_{p in O} (g . k)(p)
                    = (1/|O|) sum_{p in O} k(g^{-1} . p)
                    = (1/|O|) sum_{q in g^{-1}.O} k(q)
                    = (1/|O|) sum_{q in O} k(q)         (since g^{-1}.O = O)
                    = (P k)(dy, dz).
```

So `P (g . k) = P k`. By `(T2)` and idempotence, `P k in V_sym`, so
`g . (P k) = P k`. Both right-hand sides agree, giving `(4)`.

`(T4)` Linearity. The defining formula `(2)` is a finite sum with
fixed weights `1/|O|`, hence linear in `k`. Mass preservation: each
`(dy, dz) in R(d_max)` belongs to exactly one orbit, and the sum
`sum_R (P k)` reassembles to `sum_R k` because the orbit-average is
weighted by `|O|/|O| = 1` per orbit class. Concretely,

```text
sum_{(dy,dz) in R} (P k)(dy, dz)
   = sum over orbit-classes O of |O| * <k>_O
   = sum over orbit-classes O of sum_{p in O} k(p)
   = sum_R k.
```

Positivity preservation: if `k >= 0` pointwise, then each
`<k>_O = (1/|O|) sum k(p) >= 0`, so `P k >= 0` pointwise.

`(T5)` Orbit-class basis. By `(T2)`, `V_sym` is exactly the space of
functions constant on each `D_4` orbit class. A function in `V_sym`
is therefore determined by one real value per orbit class, giving a
linear bijection `V_sym -> R^{|O(d_max)|}`. The indicator functions
`{ e_c : c in O(d_max) }` form a basis: they have disjoint supports,
are individually cubic-symmetric (constant on their support class),
and span `V_sym` because any `f in V_sym` is `f = sum_c f(c) e_c`
where `f(c)` is the common value of `f` on orbit class `c`. The
orbit-count formula `|O(d_max)| = (d_max+1)(d_max+2)/2` follows from
indexing orbit classes by unordered multisets `{ |dy|, |dz| }` with
each `|dy|, |dz| in {0, 1, ..., d_max}`: the number of multisets of
size 2 from a `(d_max+1)`-element set equals
`C(d_max+1, 2) + (d_max+1) = (d_max+1)(d_max+2)/2`.

`(T6)` Reynolds formula and uniqueness. For each `(dy, dz)`,

```text
((1/|D_4|) sum_{g in D_4} g . k)(dy, dz)
  = (1/|D_4|) sum_{g in D_4} k(g^{-1} . (dy, dz)).
```

As `g` ranges over `D_4`, `g^{-1} . (dy, dz)` ranges over `orbit(dy, dz)`,
hitting each orbit member exactly `|D_4|/|O|` times (`D_4` acts
transitively on each orbit with stabilizer of order `|D_4|/|O|`).
Hence the average equals `(1/|O|) sum_{p in O} k(p) = (P k)(dy, dz)`,
giving `(6)`.

Uniqueness. Let `Q : V -> V_sym` be any linear projection commuting
with the `D_4` action (i.e. `Q g = g Q` for all `g in D_4`,
`Q^2 = Q`, `Image(Q) = V_sym`). For any `k in V`, averaging the
identity `Q g . k = g . Q k = Q k` (the last equality because
`Q k in V_sym` is fixed by `g`) over all `g in D_4`,

```text
Q ( (1/|D_4|) sum_g g . k )  =  Q k.
```

By `(6)` the left-hand side is `Q (P k) = P k` (since `P k in V_sym
= Image(Q)` and `Q` is the identity on `V_sym` because `Q^2 = Q` and
`Image(Q) = V_sym`). Hence `P k = Q k` for all `k`, so `Q = P`. ∎
