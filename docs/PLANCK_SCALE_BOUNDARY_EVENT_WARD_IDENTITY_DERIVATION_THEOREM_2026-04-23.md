## Why This Is A Ward Identity

A Ward identity is a statement that the generator of a transformation equals
the insertion of the associated charge.

Here the transformation is not a Schur source shift. It is the primitive
boundary incidence insertion

`U_A(s) = exp(s P_A)`.

The charge is the generator at the identity:

`d/ds U_A(s)|_(s=0) = P_A`.

Thus the normalized source-free generator is

`d/ds log Tr(rho_cell U_A(s))|_(s=0) = Tr(rho_cell P_A)`.

This is the finite projector-algebra version of the usual source/charge Ward
identity. It is exact because the event algebra is finite and `P_A` is a
projector.

## Theorem 1: the primitive insertion group is unique

Let `P_A` be the forced primitive boundary incidence projector.

A one-parameter insertion group for primitive boundary incidence must satisfy:

1. `U(0) = I`;
2. `U(s + t) = U(s) U(t)`;
3. the infinitesimal generator is supported only on `P_A`;
4. each admitted primitive incidence has unit source charge.

In finite dimension, such a group has the form

`U(s) = exp(s N)`.

Support on `P_A` and unit incidence charge force

`N = P_A`.

Therefore

`U_A(s) = exp(s P_A)`

is the unique primitive incidence insertion group.

So the source group is fixed before any expectation value is taken.
