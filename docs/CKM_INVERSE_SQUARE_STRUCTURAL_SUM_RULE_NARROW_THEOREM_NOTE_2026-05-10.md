## Statement

Let `(n_pair, n_color, eta, rho, A)` be any abstract real positive symbols
satisfying the three parametric input identities

```text
(H1)   rho      =  1 / (n_pair * n_color),
(H2)   A^2      =  n_pair / n_color,
(H3)   eta^2    =  1 / n_pair^2  -  1 / n_color^2.                       (1)
```

**Conclusion (T1) (product identity).** Under `(H1)+(H2)`,

```text
rho * A^2  =  1 / n_color^2.                                              (2)
```

**Conclusion (T2) (first sum-rule).** Under `(H1)+(H2)+(H3)`,

```text
eta^2  +  rho * A^2   =   1 / n_pair^2.                                   (3)
```

**Conclusion (T3) (second sum-rule).** Under `(H1)+(H2)+(H3)`,

```text
eta^2  +  2 * rho * A^2   =   1 / n_pair^2  +  1 / n_color^2.             (4)
```

**Conclusion (T4) (positivity gate).** `eta^2 > 0` (i.e. `(H3)` has a real
square-root solution `eta`) if and only if `n_color > n_pair`.

**Conclusion (T5) (concrete instances).** The hypothesis system `(H1)+(H2)+(H3)`
admits positive-integer count solutions; e.g.

```text
(n_pair, n_color) = (2, 3):  rho = 1/6,  A^2 = 2/3,  eta^2 = 5/36,        (5a)
                             rho A^2 = 1/9,  eta^2 + rho A^2 = 1/4,
                             eta^2 + 2 rho A^2 = 13/36;

(n_pair, n_color) = (3, 4):  rho = 1/12, A^2 = 3/4,  eta^2 = 7/144,       (5b)
                             rho A^2 = 1/16, eta^2 + rho A^2 = 1/9,
                             eta^2 + 2 rho A^2 = 25/144;

(n_pair, n_color) = (1, 2):  rho = 1/2,  A^2 = 1/2,  eta^2 = 3/4,         (5c)
                             rho A^2 = 1/4,  eta^2 + rho A^2 = 1,
                             eta^2 + 2 rho A^2 = 5/4.
```

All four conclusions are universal in `(n_pair, n_color)` (subject to
`(H3)`'s positivity gate `n_color > n_pair`); the specific framework instance
`(2,3)` is exhibited for sanity, not as a load-bearing input.

## Proof

Pure substitution and elementary fraction algebra.

`(T1)`. By `(H1)` and `(H2)`,

```text
rho * A^2  =  (1 / (n_pair * n_color)) * (n_pair / n_color)
           =  1 / n_color^2.
```

`(T2)`. By `(H3)` and `(T1)`,

```text
eta^2 + rho * A^2
   =  (1 / n_pair^2  -  1 / n_color^2)  +  1 / n_color^2
   =  1 / n_pair^2.
```

`(T3)`. By `(H3)` and `(T1)`,

```text
eta^2 + 2 * rho * A^2
   =  (1 / n_pair^2  -  1 / n_color^2)  +  2 / n_color^2
   =  1 / n_pair^2  +  1 / n_color^2.
```

`(T4)`. From `(H3)`,

```text
eta^2  =  (n_color^2 - n_pair^2) / (n_pair^2 * n_color^2).
```

The denominator is positive (both `n_pair, n_color > 0`), so the sign of
`eta^2` equals the sign of `n_color^2 - n_pair^2`. With both `n_pair,
n_color > 0`, `n_color^2 - n_pair^2 > 0` iff `n_color > n_pair`. Hence
`eta^2 > 0` iff `n_color > n_pair`.

`(T5)`. By direct substitution into `(H1)`, `(H2)`, `(H3)` and the proved
`(T1)`, `(T2)`, `(T3)` at the listed `(n_pair, n_color)` tuples; the
runner exhibits each readout by exact rational arithmetic. ∎

## Derivable corollary

```text
(C1)   (eta^2 + 2 rho A^2) - (eta^2 + rho A^2)
        =  rho * A^2  =  1 / n_color^2.
```

That is, the difference of the two sum-rules `(T2)` and `(T3)` recovers
the product identity `(T1)` algebraically, providing a closed three-way
identity among `(eta^2, rho A^2, n_pair, n_color)`.
