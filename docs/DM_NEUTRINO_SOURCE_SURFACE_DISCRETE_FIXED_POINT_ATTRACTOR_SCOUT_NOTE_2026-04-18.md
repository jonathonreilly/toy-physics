## The induced chart map `F_S`

For any retained unitary `S` acting by conjugation, the image `S H S^{-1}` is
typically NOT in the chart (Theorem 2 of the Impossibility Note). We DEFINE the
induced chart map by Hilbert-Schmidt orthogonal projection onto the chart:

```
F_S(m, delta, q_+) = chart_proj( S H(m, delta, q_+) S^{-1} )
                   = A_S v + b_S
```

where `v = (m, delta, q_+)`. The equation `F_S(v) = v`, i.e.
`(I - A_S) v = b_S`, defines the CHART-FIXED LOCUS of `S`. If `det(I - A_S)` is
nonzero the fixed locus is a point; if `A_S` has eigenvalues of modulus `< 1`,
that fixed point is a CONTRACTIVE ATTRACTOR.

This is a mild axiom-native weakening of "literal fixed point": when the chart
is a gauge-fixed slice (as here), orthogonal projection is the canonical
closure of the discrete action back to the chart.
