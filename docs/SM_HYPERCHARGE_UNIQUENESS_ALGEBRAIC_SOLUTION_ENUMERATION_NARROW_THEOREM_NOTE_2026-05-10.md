## Statement

Let `(y_1, y_2, y_3, y_4) in Q^4` satisfy:

1. (**Linear anomaly:** `Tr[Y] = 0`)
   `3 (y_1 + y_2) + y_3 + y_4 = 0`. (A1)
2. (**Mixed color anomaly:** `Tr[SU(3)^2 Y] = 0` after clearing the
   Dynkin factor)
   `y_1 + y_2 = 2/3`. (A2)
3. (**Cubic anomaly:** `Tr[Y^3] = 0`)
   `3 (y_1^3 + y_2^3) + y_3^3 + y_4^3 = -16/9`. (A3)
4. (**Neutral singlet identification.**)
   `y_4 = 0`. (N)
5. (**Q-labelling convention.**)
   `Q(u_R) > 0`, i.e. `y_1 > 0` in the doubled-Y convention
   `Q = T_3 + Y/2` with `Q = Y/2` on SU(2) singlets.

Then the rational solution set of (A1)-(A3) ∧ (N) is exactly two ordered
4-tuples

```text
S_2  =  { (+4/3, -2/3, -2, 0),  (-2/3, +4/3, -2, 0) }
```

related by the `y_1 ↔ y_2` swap. Imposing (Q) selects the unique tuple

```text
S_1  =  { (+4/3, -2/3, -2, 0) }                               (★)
```

The associated one-generation electric-charge spectrum is
`{ 0, ±1/3, ±2/3, ±1 }` with denominators exactly `{1, 3}`.

## Proof

Pure rational algebra.

**Step 1. Eliminate `y_4` and `y_3` from (A1).** Imposing (N) into (A1):

```text
3 (y_1 + y_2) + y_3  =  0.
```

Using (A2): `3 · (2/3) + y_3 = 0`, hence `y_3 = -2`. (S1)

**Step 2. Reduce (A3) to a 2-variable identity in `(y_1, y_2)`.**
Substituting `y_4 = 0` and `y_3 = -2` into (A3):

```text
3 (y_1^3 + y_2^3) + (-2)^3 + 0  =  -16/9
3 (y_1^3 + y_2^3) - 8           =  -16/9
3 (y_1^3 + y_2^3)               =  72/9 - 16/9  =  56/9
y_1^3 + y_2^3                   =  56/27.       (S2)
```

**Step 3. Solve the (S2) ∧ (A2) system.** Using
`(y_1 + y_2)^3 = y_1^3 + y_2^3 + 3 y_1 y_2 (y_1 + y_2)`:

```text
(2/3)^3   =  56/27 + 3 y_1 y_2 · (2/3)
8/27      =  56/27 + 2 y_1 y_2
2 y_1 y_2 =  -48/27  =  -16/9
y_1 y_2   =  -8/9.    (S3)
```

So `y_1, y_2` are the two roots of the rational quadratic

```text
t^2 - (2/3) t - 8/9  =  0
9 t^2 - 6 t - 8       =  0
t  =  (6 ± sqrt(36 + 288)) / 18  =  (6 ± 18) / 18
t  =  4/3   or   t = -2/3.
```

The discriminant `324 = 18^2` is a perfect square, so the roots stay in
`Q` without any extension. The two roots are therefore
`{ 4/3, -2/3 }`. Hence

```text
(y_1, y_2)  in  { (4/3, -2/3),  (-2/3, 4/3) }   ⟹   S_2 = { (+4/3, -2/3, -2, 0),
                                                              (-2/3, +4/3, -2, 0) }.
```

**Step 4. Q-labelling fixes the swap.** Imposing (Q) `y_1 > 0`:

- `(y_1, y_2) = (4/3, -2/3)` has `y_1 = 4/3 > 0`. Accept.
- `(y_1, y_2) = (-2/3, 4/3)` has `y_1 = -2/3 < 0`. Reject.

So `S_1 = { (+4/3, -2/3, -2, 0) }`. (★)

**Step 5. Electric-charge spectrum.** Under `Q = T_3 + Y/2`:

- `Q(u_R) = y_1 / 2 = +2/3`,
- `Q(d_R) = y_2 / 2 = -1/3`,
- `Q(e_R) = y_3 / 2 = -1`,
- `Q(ν_R) = y_4 / 2 = 0`.

From `Q_L : (2, 3)_{+1/3}`: `Q(u_L) = 1/2 + 1/6 = +2/3`, `Q(d_L) = -1/2 +
1/6 = -1/3`. From `L_L : (2, 1)_{-1}`: `Q(ν_L) = 1/2 - 1/2 = 0`,
`Q(e_L) = -1/2 - 1/2 = -1`. The combined one-generation charge spectrum
is therefore `{ 0, ±1/3, ±2/3, ±1 }` with denominators exactly `{1, 3}`.

QED.

## Derivable corollaries

- **Rationality:** (S2) ∧ (A2) closes over `Q` (no algebraic extension is
  needed) because the discriminant of the reduced quadratic
  `9 t^2 - 6 t - 8 = 0` is `324 = 18^2`.
- **Discrete-swap-only ambiguity:** the only ambiguity that survives
  (A1)-(A3) ∧ (N) is the `y_1 ↔ y_2` swap. (Q) breaks it.
- **Counterfactual without (N):** if `y_4` is treated as free in
  (A1)-(A3) (no neutral-singlet input), the system has 3 equations in
  4 unknowns and admits a 1-parameter rational family of solutions; the
  uniqueness statement does not survive without the (N) input.

Both corollaries are pure rational-algebra consequences of (A1)-(A3) ∧
(N) ∧ (Q).
