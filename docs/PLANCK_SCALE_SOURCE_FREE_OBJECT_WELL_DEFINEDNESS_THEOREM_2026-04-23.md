## Setup

Work on the abstract labeled factorized cell object

`H_cell = H_t ⊗ H_x ⊗ H_y ⊗ H_z`,

with each factor two-dimensional.

An isomorphism of this labeled object is exactly an independent basis change on
each factor:

`U = U_t ⊗ U_x ⊗ U_y ⊗ U_z`.

Call a source-free state assignment **object-well-defined** if whenever two
presentations of the same labeled factorized object differ only by such an
isomorphism, the assigned state operators are carried into each other by
conjugation.

On the same object this means simply:

`U rho_cell U^dagger = rho_cell`

for every factor-preserving automorphism `U`.

## Theorem 1: object well-definedness forces the tracial cell state

Assume `rho_cell` is a normalized positive operator on `H_cell` and is
well-defined on the abstract labeled factorized object in the sense above.

Then

`rho_cell = I_16 / 16`.

### Proof

Object well-definedness gives invariance under the full factor-preserving
automorphism group

`U(2)_t × U(2)_x × U(2)_y × U(2)_z`.

So `rho_cell` commutes with the full local matrix algebra generated by the four
factor actions

- `M_2(C) ⊗ I ⊗ I ⊗ I`,
- `I ⊗ M_2(C) ⊗ I ⊗ I`,
- `I ⊗ I ⊗ M_2(C) ⊗ I`,
- `I ⊗ I ⊗ I ⊗ M_2(C)`.

These generate `M_16(C)`, so `rho_cell` lies in its center. Therefore

`rho_cell = lambda I_16`.

Normalization gives `16 lambda = 1`, hence

`rho_cell = I_16 / 16`.
