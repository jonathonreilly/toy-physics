# YT_WARD Step 3 Same-1PI Construction Gate Diagnostic

**Date:** 2026-05-10
**Type:** open_gate
**Claim scope:** a narrow coefficient-bookkeeping diagnostic for the proposed
same-1PI construction at `YT_WARD` Step 3. Given the retained one-hop
inputs listed below, the OGE-projected coefficient and the
`H_unit`-projected coefficient reduce to

```text
C_A(g_bare) = c_S * g_bare^2 / (2 N_c),
C_B         = F_Htt^(0)(g_bare)^2 = 1 / (N_c N_iso).
```

same equation used by the W2 pinning surface, but it does **not** by
itself prove that the OGE representation and the `H_unit` representation
are the same amputated 1PI Green's function. That same-1PI bridge remains
the open gate.

## Statement

Assume the retained `Q_L = (2,3)` bare Wilson + staggered-Dirac action,
the D12 color-singlet Fierz coefficient, the standard Clifford-scalar
coefficient `c_S`, and the retained unit-singlet overlap form of
`H_unit`. On the projected scalar singlet four-fermion channel

```text
O_S := (psibar psi)_(1,1) (psibar psi)_(1,1),
```

the coefficient bookkeeping gives:

1. **Rep A (OGE projection):**

   ```text
   C_A(g_bare) = c_S * g_bare^2 / (2 N_c).
   ```

2. **Rep B (`H_unit` projection):**

   ```text
   C_B = F_Htt^(0)(g_bare)^2 = 1 / (N_c N_iso).
   ```

3. **Gate equation:** `C_A = C_B` is equivalent to `(Gate)`.

The equality `C_A = C_B` is the missing same-1PI bridge. It cannot be
obtained merely by writing both projected coefficients and then solving
for `g_bare`; that would assume the equality whose construction is under
review.

## Proof

From the D12 color-singlet projection and the scalar Clifford projection,
the OGE tree coefficient is

```text
C_A(g_bare) = c_S * g_bare^2 / (2 N_c).
```

From the retained unit-singlet overlap form,

```text
F_Htt^(0)(g_bare) = 1 / sqrt(N_c N_iso),
```

so the `H_unit` projected coefficient is

```text
C_B = F_Htt^(0)(g_bare)^2 = 1 / (N_c N_iso).
```

The symbolic residual is

```text
C_A - C_B
  = c_S * g_bare^2 / (2 N_c) - 1 / (N_c N_iso)
  = (N_iso * c_S * g_bare^2 - 2) / (2 N_c N_iso).
```

Thus `C_A = C_B` iff `c_S * g_bare^2 = 2 / N_iso`. For
`c_S = +1` and `N_iso = 2`, this reduces to `g_bare^2 = 1`. For example,
at `N_c = 3`, `N_iso = 2`, `c_S = +1`, the residual is
`(g_bare^2 - 1) / 6`, so `g_bare = 2` is an explicit off-surface
counterexample to equality.

Therefore the algebra verifies the downstream pinning equation but does
not construct the same-1PI equivalence. A separate Wick-level argument is
still required to prove that the OGE contraction and the `H_unit`
decomposition are two complete representations of the same projected
amputated 1PI Green's function without first imposing `C_A = C_B`.
