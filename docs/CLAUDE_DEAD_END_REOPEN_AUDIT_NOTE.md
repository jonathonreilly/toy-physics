# Claude Dead-End Reopen Audit Note

**Date:** 2026-04-05  
**Scope:** audit of Claude-derived no-go / bounded-negative lanes for routes
that may have been closed too early

This note is intentionally adversarial.

It does **not** ask which claims are most exciting. It asks which claims were
actually dead ends, which were only dead in one specific implementation, and
which need a more radical architecture before they can be reopened.

## Nearby positive controls

These are the live retained lanes that keep the audit honest:

- **Exact-lattice wavefield mechanism**
  - retained phase-ramp law
  - exact zero-source reduction survives
  - weak-field `F~M` stays near linear
  - the detector-line phase ramp is coherent and source-depth dependent
- **Exact-lattice complex-action carryover**
  - exact `gamma = 0` reduction survives
  - Born stays machine clean on the frozen field
  - `TOWARD -> AWAY` crossover survives on the retained exact family
- **Wide-lattice `h^2+T` replay**
  - finite-lattice far tail stays near Newtonian
  - Born stays clean
  - `F~M = 1.000` on the retained wide replay

These are the controls to compare against the dead-end lanes below.

## Strongest Mistaken-Closure Risks

1. The `h^2+T` continuum lane is the clearest place where a route can be
   overclosed if the reduced-family audit is mistaken for the full story.
   The wide-lattice replay is still a live retained frontier.
2. QNM is the clearest place where a route can be underexplored if the missing
   controls are interpreted as fatal rather than as hardening work.
3. The exact-lattice complex-action lane is **not** a dead end; it is a valid
   narrow carryover. What remains closed is the geometry-independent / retained
   effective-theory language.

## Bottom Line

The current dead-end map is:

- **truly closed**
  - distance-law steepening as a retained `-1.5` claim
  - BMV as a discrete-spacetime correction
- **reopen with a specific narrower variant**
  - `h^2+T` `h = 0.125` bridge
  - QNM
- **reopen only if architecture changes**
  - grown propagating field v1/v2/v3
  - Poisson self-gravity
  - generated-wavefield transfer
  - readout localization

The useful discipline is to keep the nearby positives in view:

- exact-lattice wavefield
- exact-lattice complex action
- wide-lattice `h^2+T` finite-lattice replay

Those are the controls that show the difference between a real dead end and a
route that only failed under one narrow implementation.
