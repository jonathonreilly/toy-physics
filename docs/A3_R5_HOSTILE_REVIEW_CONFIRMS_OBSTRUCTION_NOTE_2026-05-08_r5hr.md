## Per-attack-line summary

### HR5.1 — GNS rigor (PASS, 12/0)

R5's claim that U_{C_3}^3 = I strictly (not projective) holds: the
3-cycle permutation matrix has det = +1 and U^3 - I = 0 to machine
precision. The OS state on hw=1 (tracial) is α_{C_3}-invariant by cubic
isotropy. The α_{C_3} *-automorphism is on the QUASILOCAL algebra A(Λ),
not on any single local A(O); R5's brief glosses this distinction but
the GNS lift is rigorous.

### HR5.2 — C_3 ∈ A(Λ) consistency (PASS, 4/0; sharpening)

The DHR-retirement note (commit `7a8c16ffd`) explicitly says C_3[111]
"is an element of A(Λ) (a lattice symmetry operator)". R5's V1 reframes
C_3 as a *-automorphism α_{C_3} of A(Λ). Are these reconcilable?

**Yes.** On hw=1, U_{C_3} = E_{21} + E_{32} + E_{13} ∈ M_3(C), where
M_3(C) is the FULL matrix algebra on C^3. So U_{C_3} IS in M_3(C) ⊂
A(Λ)|_{hw=1}. The GNS image of α_{C_3} produces the same matrix.

**Sharpening:** R5's V1 framing "C_3 is automorphism, NOT in A(Λ)"
overstates the difference. Both readings give the same Ad U_{C_3}
on M_3(C). V1 is a SUFFICIENT-NOT-NECESSARY fix of Block 02; the
in-algebra reading is also valid.

### HR5.3 — Modular structure (PASS, 5/0)

For the tracial state on M_3(C), the modular operator Δ = I (trivial
modular group). Asymmetric states have non-trivial Δ, but those states
themselves break C_3 (their density matrix doesn't commute with U_{C_3}).
The non-trivial modular labels come from external C_3-breaking, not
internal structure.

### HR5.4 — Bare retained Hamiltonian C_3 symmetry (PASS, 6/0; sharpening)

The Kawamoto-Smit kinetic on Z³ APBC restricted to hw=1 is exactly
λ_KS · I (proportional to identity) by cubic isotropy. Therefore
[H_KS, U_{C_3}] = 0 EXACTLY, and corner expectations are equal.

**Sharpening:** R5's V6 says "spectrum positivity ≠ C_3-breaking" with
toy examples. The stronger statement is: the BARE retained Hamiltonian
on hw=1 IS C_3-symmetric. The retained constraint set (RP A11, OS,
single-clock, cubic isotropy of K-S, APBC) jointly forces this.

### HR5.5 — Continuum-only effects (PASS, 6/0)

A C_3-symmetric anomalous-dim matrix on hw=1 has equal corner expectations
in the corner basis (eigenvalues differ in the C_3-Fourier basis but
each corner is a sum of all three Fourier components). Asymptotic

RG flow preserves C_3 symmetry by Coleman 1975 / Wilson 1971.

### HR5.6 — Embedding non-uniqueness (PASS, 10/0)

Two embeddings ρ_1 (standard) and ρ_2 = Ad(V) ∘ ρ_1 (random unitary V)
of M_3(C) into B(C^3) are unitarily equivalent. The cyclic permutation
transforms covariantly: U_{C_3}^{(2)} = V U_{C_3}^{(1)} V*. Eigenvalue
