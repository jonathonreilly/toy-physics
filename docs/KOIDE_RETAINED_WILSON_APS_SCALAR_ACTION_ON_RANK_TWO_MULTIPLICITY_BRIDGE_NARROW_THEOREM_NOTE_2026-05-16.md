```text
M_ζ := { v ∈ V_0 : U v = ζ v },
```

**Retained Wilson/APS generators.** Define the retained generating set

```text
G := { D,  U,  U^† }  ∪  { P_λ(D) : λ ∈ Spec(D) },
```

where `P_λ(D)` is the orthogonal projector onto the eigenspace of `D`
with eigenvalue `λ`. APS-type data (`η`-style spectral sums, sign of `D`,

polynomial algebra over `G` because they are finite spectral sums built

`U^†`, etc. The retained Wilson/APS algebra is

```text
A := ⟨ G ⟩_polynomial  ⊆  ℂ^{N×N},
```

the unital `ℂ`-subalgebra of `ℂ^{N×N}` generated by `G` under linear
combination and matrix product.

## 3. Statement

**Theorem (Scalar-on-`M_ζ`).** Let `Π_ζ : ℂ^N → M_ζ` be orthogonal
projection. For every `A ∈ A`,

```text
Π_ζ A Π_ζ  =  λ_A · Π_ζ                                                 (*)
```

for some scalar `λ_A ∈ ℂ`. Equivalently, in the orthonormal basis
`(line_0, line_1)` of `M_ζ`, every retained Wilson/APS-derived operator
restricts to a scalar `2 × 2` matrix `λ_A · I_2`.

In particular, every "retained boundary mark" built from
`{D, U, U^†, P_λ(D)}_{λ ∈ Spec(D)}` is scalar on `M_ζ`, which is the
load-bearing premise the parent runner had previously asserted as
`retained_mark = lam * sp.eye(2)`.

## 4. Proof

The proof has two parts. Part (a) reduces to checking the generators on
`M_ζ`. Part (b) checks each generator.

**(a) Reduction to generators.** We claim that for any `A ∈ A`,

```text
Π_ζ A Π_ζ  =  P(Π_ζ D Π_ζ,  Π_ζ U Π_ζ,  Π_ζ U^† Π_ζ,
              {Π_ζ P_λ(D) Π_ζ}_{λ ∈ Spec(D)})                          (1)
```

for a finite polynomial `P` (the same polynomial whose value on `G`
equals `A`). This follows because `M_ζ` is invariant under every element
of `G`:

- `D` preserves `V_0` trivially (as a self-adjoint operator with
  `V_0 = ker D`, `D V_0 = 0 ⊂ V_0`), hence preserves `M_ζ ⊂ V_0` (in
  fact, `D|_{V_0} = 0`).
- `U` preserves `V_0` because `U D U^† = D` implies `U` commutes with
  the spectral projector `P_0(D)`, hence `U V_0 = V_0`. On `V_0`, `U` is
  unitary (because `U` is unitary on all of `ℂ^N`) and has the `ζ`-
  eigenspace `M_ζ`. So `U M_ζ = M_ζ` directly.
- `U^†` preserves `V_0` for the same reason (and `U^† M_ζ = M_ζ`).
- Each `P_λ(D)` preserves `V_0`: if `λ ≠ 0`, then `P_λ(D) V_0 = 0
  ⊂ V_0`; if `λ = 0`, then `P_0(D) V_0 = V_0`. In either case
  `P_λ(D) M_ζ ⊂ V_0`. Furthermore, because `P_λ(D)` is a polynomial in
  `D` for each isolated eigenvalue `λ` (Lagrange interpolation on the
  finite spectrum), and `D` commutes with itself, `P_λ(D)` commutes
  with the `U`-isotypic decomposition of `V_0` (because `U` and `D`
  commute on `V_0` by `U D U^† = D` and `D|_{V_0} = 0` — in particular
  `U` and `P_0(D)` commute, and trivially each `P_λ(D)` with `λ ≠ 0`
  vanishes on `V_0`, hence in particular on `M_ζ`). So
  `P_λ(D) M_ζ ⊂ M_ζ` for every `λ`.

Since every generator preserves `M_ζ`, every polynomial in the
generators preserves `M_ζ`, and the restriction homomorphism

```text
ρ : A  →  End(M_ζ),   A ↦ A|_{M_ζ},
```

is a unital `ℂ`-algebra homomorphism. By construction of polynomial
algebras, `ρ(A)` is determined by `ρ(G)`, which establishes (1).

**(b) The generators are scalar on `M_ζ`.** By direct calculation:

- `D|_{M_ζ} = 0` because `M_ζ ⊂ V_0 = ker D`. So
  `D|_{M_ζ} = 0 · I_{M_ζ}`.
- `U|_{M_ζ} = ζ · I_{M_ζ}` by definition of `M_ζ` as the `ζ`-eigenspace
  of `U` on `V_0`.
- `U^†|_{M_ζ} = ζ̄ · I_{M_ζ}` because if `U v = ζ v` then `U^† v = ζ^{-1} v
  = ζ̄ v` (using `|ζ| = 1`).
- For each `λ ∈ Spec(D)`:
  - If `λ ≠ 0`, then `P_λ(D) v = 0` for every `v ∈ V_0`, hence for every
    `v ∈ M_ζ`. So `P_λ(D)|_{M_ζ} = 0 · I_{M_ζ}`.
  - If `λ = 0`, then `P_0(D) v = v` for every `v ∈ V_0`, hence for every
    `v ∈ M_ζ`. So `P_0(D)|_{M_ζ} = 1 · I_{M_ζ}`.

So every generator restricts to a scalar multiple of `I_{M_ζ}`.

**(c) Conclusion.** By (a), the restriction of any polynomial `A ∈ A` to
`M_ζ` is a polynomial in matrices that are scalar on `M_ζ`; by (b), each
generator restricted to `M_ζ` is in the (commutative) subring
`ℂ · I_{M_ζ}` of `End(M_ζ)`. Polynomials in commuting scalars are
scalars, so `A|_{M_ζ} ∈ ℂ · I_{M_ζ}`, i.e. (*) holds for some
`λ_A ∈ ℂ`. ∎

## 5. Remarks

**Why this is the right "Schur" argument.** A standard Schur lemma
states that an operator commuting with an irreducible representation of
a group is a scalar. Here `M_ζ` is the `ζ`-isotypic component of `U|_{V_0}`,
which is itself only a sum of two copies of the one-dimensional
representation `ζ`, so a general operator commuting with `U|_{M_ζ}` is
only of the form `M_2(ℂ)` (a `2 × 2` matrix), not necessarily a scalar.
The reason it is nevertheless a scalar in our setting is that the
**retained algebra `A` itself has no operators that distinguish two
copies of `ζ`**: the only generators are `D`, `U`, `U^†`, and spectral
projectors of `D`, and each of these collapses to a scalar on `M_ζ`
(because `D|_{M_ζ} = 0` and `U|_{M_ζ}, U^†|_{M_ζ}, P_λ(D)|_{M_ζ}` are
all scalars). A different (non-retained) operator, such as the rank-one
projector `|line_0⟩⟨line_0|`, would distinguish the two copies of `ζ`
— but that operator is not in `A`, and the parent no-go note already
records that introducing such a non-retained selector is exactly the
"extra data" the no-go forbids (parent residual `RESIDUAL_MARK =
derived_boundary_mark_is_scalar_on_multiplicity`).

**What this does NOT claim.**

- Does **not** identify `M_ζ` with any physical Brannen-line space.
- Does **not** select a unique rank-one line inside `M_ζ`.
- Does **not** consume any charged-lepton mass, PDG value, fitted Koide
  target, or `√m` identification.
- Does **not** assert that any non-retained extension of `A` is scalar
  on `M_ζ`; on the contrary, any non-retained rank-one boundary mark
  outside `A` would be a non-scalar operator (and the parent note's
  falsifier explicitly says this is the missing extra data).
- Does **not** depend on the value of the Wilson mass `r` beyond the
  requirement that the sibling runner certifies the rank-two
  `ζ`-character structure at that `r` (verified at `r = 1.0` and
  `r = 1.425` in the sibling).
- Does **not** claim the existing parent runner's check `B.1` is now
  vacuous; this bridge derives precisely that hard-coded scalar
  premise.
