## Question

After the new identity-rim reduction theorem, what exact bulk operator-side
object still matters for explicit class-sector `beta = 6` closure?

Is it still the whole compressed bulk operator `S_6^env`?

And if the identity rim datum `eta_6(e)` and the minimal propagated retained
triple were both fixed exactly, would that finally determine the remaining bulk
object?

## Answer

Not the whole operator, and still not uniquely.

The exact class-sector law is already

`v_6 = (S_6^env)^(L_perp-1) eta_6(e)`,

`Z_6^env(W) = <K(W), v_6>`.

So for class-sector closure, the only upstream bulk data that matter are the
ones seen from repeated action of `S_6^env` on the identity rim state
`eta_6(e)`.

Equivalently, the live bulk object is:

- the cyclic compression of `S_6^env` generated by `eta_6(e)`,
- or the spectral measure of `S_6^env` in `eta_6(e)`,
- or the Jacobi data of `S_6^env` on that cyclic subspace.

Anything orthogonal to that cyclic sector is invisible to the class-sector
boundary data.

But even after this reduction, the current bank still does **not** determine the
reduced cyclic bulk object from the finite propagated target alone. The runner
gives an explicit positive witness pair with:

- the same fixed identity rim state `eta`,
- the same exact propagated retained triple
  `(Z_6^env(W_A), Z_6^env(W_B), Z_6^env(W_C))`,
- but different cyclic moment sequences, hence different cyclic spectral /
  Jacobi data.

So the plaquette bulk front is now sharper:

- the remaining upstream object is smaller than the whole bulk operator,
- but it is still not closed by the present finite propagated data.

## Theorem 1: exact identity-rim cyclic bulk reduction

Let `eta = eta_6(e)` and let `Q_eta` be the orthogonal projector onto the cyclic
subspace

`K_eta(S_6^env) = span{eta, S_6^env eta, (S_6^env)^2 eta, ...}`.

Then for every `n >= 0`,

`(S_6^env)^n eta = (Q_eta S_6^env Q_eta)^n eta`.

Therefore:

- the propagated vector `v_6 = (S_6^env)^(L_perp-1) eta` depends only on the
  cyclic compression `Q_eta S_6^env Q_eta`,
- every boundary amplitude `Z_6^env(W) = <K(W), v_6>` depends only on that same
  cyclic compression,
- and any change to `S_6^env` on the orthogonal complement of `K_eta(S_6^env)`
  is invisible to the class-sector boundary data.

So the live upstream bulk object is not the whole compressed bulk operator. It
is exactly the cyclic bulk object seen from the identity rim state.

## Corollary 1: exact spectral / Jacobi reformulation of the bulk front

Because the cyclic compression is self-adjoint and positive on a finite
class-sector truncation, the relevant bulk object is equivalently:

- the cyclic spectral measure of `S_6^env` in `eta_6(e)`,
- the corresponding moment sequence
  `m_n = <eta_6(e), (S_6^env)^n eta_6(e)>`,
- or the Jacobi data of `S_6^env` on the `eta_6(e)`-cyclic subspace.

That is the exact bulk analogue of the earlier source-cyclic Perron / Jacobi
reduction.

## Corollary 2: exact orthogonal-complement irrelevance

If `S_A` and `S_B` are two positive self-adjoint class-sector operators with the
same cyclic compression on `K_eta` and the same identity rim state `eta`, then
for every marked holonomy `W`,

`<K(W), (S_A)^(L_perp-1) eta> = <K(W), (S_B)^(L_perp-1) eta>`.

So bulk-side differences outside the `eta`-cyclic sector are exact gauge
invisible as far as class-sector plaquette PF data are concerned.

## Theorem 2: fixed identity rim plus fixed propagated retained triple still does not determine the reduced cyclic bulk object

On the explicit higher-orbit slice

`{(0,2), (0,3), (0,4), (0,5)}`,

let `E_3` be the exact propagated three-sample evaluator to

`(Z_6^env(W_A), Z_6^env(W_B), Z_6^env(W_C))`.

The runner exhibits two distinct nonnegative propagated vectors

`v_P != v_Q`,

`E_3(v_P) = E_3(v_Q)`.

Fix one common positive identity rim state `eta`.

Define two positive self-adjoint bulk operators by

`S_P = v_P v_P^T / <v_P, eta>`,

`S_Q = v_Q v_Q^T / <v_Q, eta>`.

Then:

- `S_P eta = v_P`,
- `S_Q eta = v_Q`,
- the propagated retained triples agree exactly,
- but the cyclic moments differ:

  `m_1^P != m_1^Q`,

  `m_2^P != m_2^Q`.

So even fixed identity rim plus fixed exact propagated retained triple still
does **not** determine the reduced cyclic bulk spectral measure / Jacobi data on
the current bank.
