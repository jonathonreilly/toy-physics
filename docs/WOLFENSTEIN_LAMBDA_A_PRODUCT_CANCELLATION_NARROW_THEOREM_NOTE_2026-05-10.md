## Statement

Let `(alpha_s, n_pair, n_color)` be abstract positive real symbols and let
`(lambda, A)` be abstract positive real symbols satisfying the two
parametric input definitions

```text
lambda^2  =  alpha_s / n_pair,                                          (W1)
A^2       =  n_pair  / n_color.                                         (W2)
```

**Conclusion (T1) (product-cancellation identity).** The product
`A^2 lambda^2` reduces by pure symbolic algebra to

```text
A^2 lambda^2  =  alpha_s / n_color.                                     (W3)
```

The `n_pair` factor cancels exactly. This is a direct consequence of the
multiplicative identity `(p / c) * (q / p) = q / c` for nonzero `p`, in
the special case `q = alpha_s`, `p = n_pair`, `c = n_color`.

**Conclusion (T2) (rational-cancellation lemma).** Independently of
`alpha_s`, the pure rational identity

```text
(n_pair / n_color) * (1 / n_pair)  =  1 / n_color                       (R1)
```

holds for every abstract positive `(n_pair, n_color)`. This is the
underlying group-counting cancellation that makes `(W3)` an `alpha_s`-only
statement.

**Conclusion (T3) (squared-magnitude corollary).** Defining
`n_quark = n_pair * n_color` and identifying the Wolfenstein-leading
amplitude `|V_cb|^2 = A^2 lambda^4`,

```text
|V_cb|^2  =  A^2 lambda^4
          =  alpha_s^2 / (n_pair * n_color)
          =  alpha_s^2 / n_quark.                                       (V1)
```

Equivalently, `|V_cb| = alpha_s / sqrt(n_quark)` for the abstract
positive symbols.

**Conclusion (T4) (closure on arbitrary count tuples).** The cancellation
holds for every abstract positive `(n_pair, n_color)`. The framework
instance `(n_pair, n_color, n_quark) = (2, 3, 6)` and an alternative
instance `(5, 7, 35)` both satisfy the same exact symbolic reductions
`(W3)` and `(V1)`; the identity does not require any specific integer
choice.

## Proof

`(T1)` By direct substitution and `sympy.simplify`:

```text
A^2 lambda^2  =  (n_pair / n_color) * (alpha_s / n_pair)
              =  alpha_s * n_pair / (n_color * n_pair)
              =  alpha_s / n_color.
```

The `n_pair` factor in the numerator cancels with the `n_pair` factor in
the denominator. Cancellation is valid because `n_pair` is a positive
real symbol (`n_pair > 0`, hence `n_pair != 0`).

`(T2)` Pure rational algebra:

```text
(n_pair / n_color) * (1 / n_pair)  =  n_pair / (n_color * n_pair)
                                  =  1 / n_color.
```

`(T3)` Square `(W1)` and combine with `(W2)`:

```text
A^2 lambda^4  =  (n_pair / n_color) * (alpha_s / n_pair)^2
             =  (n_pair / n_color) * alpha_s^2 / n_pair^2
             =  alpha_s^2 / (n_color * n_pair)
             =  alpha_s^2 / n_quark.
```

`(T4)` By `(T1)`-`(T3)`, the reductions hold for every abstract positive
`(n_pair, n_color)`. Substitution of `(n_pair, n_color) = (2, 3)` gives
`A^2 lambda^2 = alpha_s / 3` and `|V_cb|^2 = alpha_s^2 / 6`. Substitution
of `(n_pair, n_color) = (5, 7)` gives `A^2 lambda^2 = alpha_s / 7` and
`|V_cb|^2 = alpha_s^2 / 35`. Each is an exact symbolic reduction. ∎
