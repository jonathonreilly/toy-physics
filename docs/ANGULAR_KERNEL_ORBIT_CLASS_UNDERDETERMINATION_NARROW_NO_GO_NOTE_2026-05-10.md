## Statement

Let `R = R(d_max)` denote the forward-cone single-step neighbour set on
`Z^3`,

```text
R(d_max) = { (1, dy, dz) : dy, dz in Z, |dy| <= d_max, |dz| <= d_max },
```

with `d_max >= 1` an integer support radius. Let `k : R -> R_{>=0}` be a
non-negative weight function on `R`.

We say `k` is **cubic-symmetric** if it factors through the action of
the cubic group `O_h` restricted to the layer. The restriction of `O_h`
to the (forward) layer fixes the time axis and acts on `(dy, dz)` as the
dihedral group `D_4` generated by the 90-degree rotation
`(dy, dz) -> (-dz, dy)` and the reflection `(dy, dz) -> (dy, -dz)`. Two
offsets `(dy, dz)` and `(dy', dz')` are in the same `D_4` orbit iff
`{|dy|, |dz|} = {|dy'|, |dz'|}` as unordered multisets. Let `O(d_max)`
denote the set of `D_4` orbit classes appearing in `R(d_max)`.

**Conclusion (T1) (orbit-class enumeration).** For every integer
`d_max >= 1`,

```text
|O(d_max)| = (d_max + 1)(d_max + 2) / 2,                             (1)
```

so `|O(d_max)| >= 2` for all `d_max >= 1` and grows linearly in
`d_max`.

**Conclusion (T2) (parameter dimension).** The space of cubic-symmetric
non-negative weight functions on `R(d_max)` modulo overall normalization
has affine real dimension exactly `|O(d_max)| - 1`. In particular at
`d_max = 3`,

```text
|O(3)| = 10,    affine free dim = 9.                                 (2)
```

**Conclusion (T3) (primitives do not pin a unique kernel).**
The four structural primitives

```text
(P1) cubic-orbit symmetry:   k depends only on the D_4 orbit of (dy, dz);
(P2) forward-cone support:   k is supported on R(d_max) (dx = +1 only);
(P3) transverse-quadratic
     isotropy:               sum_R k (dy^2) = sum_R k (dz^2),
                             sum_R k (dy * dz) = 0;
(P4) unit total mass:        sum_R k = 1.
```

do **not** uniquely determine `k`. Specifically, at `d_max = 3` there
exist seven explicit cubic-symmetric probability weights
`k_1, k_2, ..., k_7` derived from seven distinct radial profiles
(uniform; `cos`; `cos^2`; `exp(-0.4 t^2)`; `exp(-0.8 t^2)`;
`exp(-1.6 t^2)`; `linear_falloff`), each satisfying (P1)-(P4) exactly,
that produce seven pairwise-distinct values of the cubic-invariant
fourth moment

```text
I_4(k) := sum_R k(dy, dz) (dy^4 + dz^4).                             (3)
```

Equivalently, the cubic-orbit ratio

```text
Q(k) := I_4(k) / sum_R k(dy, dz) (dy^2 + dz^2)^2                     (4)
```

separates the seven witnesses, and on the cubic-symmetric weight
simplex `Q(k)` ranges at least over the interval `[1/2, 1]`, with
`Q(k) = 1` on axial-only orbits and `Q(k) = 1/2` on the diagonal
`(1, 1)`-orbit alone.

## Proof

`(T1)` `D_4` orbit classes on `R(d_max)` are indexed by unordered
multisets `{|dy|, |dz|}` with `|dy|, |dz| in {0, 1, ..., d_max}`. The
number of such unordered multisets is the number of multisets of size 2
from a `(d_max + 1)`-element set, i.e.

```text
C(d_max + 1, 2) + (d_max + 1) = (d_max + 1)(d_max + 2) / 2.
```

Each forward layer offset `(dy, dz)` lies in exactly one such orbit
class. This gives `|O(d_max)| = (d_max + 1)(d_max + 2) / 2`.

`(T2)` By (P1), a cubic-symmetric `k` is fully determined by its
restriction to a transversal `O(d_max) -> R_{>=0}`. The total-mass
condition (P4) is one linear equation in those `|O(d_max)|` free
non-negative real parameters. So the affine cubic-symmetric probability
simplex has dimension `|O(d_max)| - 1`.

`(T3)` Build seven explicit cubic-symmetric weights `k_1, ..., k_7` on
`R(d_max = 3)` from seven distinct radial profiles `f_i` by

```text
k_i(dy, dz) = f_i(theta(dy, dz)) / Z_i,
theta(dy, dz) := atan2(sqrt(dy^2 + dz^2), 1),
Z_i := sum over R of f_i(theta(.,.)),
```

with profiles `f_1 = 1` (uniform), `f_2 = cos`, `f_3 = cos^2`,
`f_4 = exp(-0.4 t^2)`, `f_5 = exp(-0.8 t^2)`, `f_6 = exp(-1.6 t^2)`,
`f_7(t) = max(0, 1 - 2 t / pi)` (linear falloff). Each `f_i` is a
function of the polar angle `theta` only, hence azimuthally symmetric in
`(dy, dz)`, hence in particular `D_4`-invariant; so each `k_i` factors
through the orbit map and satisfies (P1).

Each `k_i` is supported on `R(d_max = 3)` by construction, satisfying
(P2). By (P1) plus the reflection `(dy, dz) -> (dz, dy)`, the moments
`<dy^2>` and `<dz^2>` agree and `<dy * dz>` vanishes, satisfying (P3).
Each `k_i` is normalized to total mass `1`, satisfying (P4).

The seven `I_4` values and the seven `Q` values are pairwise distinct.
Hence the cubic-orbit primitive surface (P1)-(P4) is satisfied by at
least seven different probability weights.

The two extremal `Q`-values follow by direct computation:

- Axial-only weight `k_axial`, supported on
  `(dy, dz) in {(0, 0)} cup {(0, m), (m, 0) : 1 <= m <= d_max}` and
  normalized: every nonzero member has `dy = 0` or `dz = 0`, so
  `dy^4 + dz^4 = (dy^2 + dz^2)^2`, giving `Q(k_axial) = 1` exactly.
- Diagonal `(1, 1)`-only weight `k_diag`, uniform over the four offsets
  `(±1, ±1)`: each has `dy^4 + dz^4 = 2` and `(dy^2 + dz^2)^2 = 4`,
  giving `Q(k_diag) = 1/2` exactly.

Convex mixtures of `k_axial` and `k_diag` are themselves cubic-symmetric
probability weights with `Q` taking every value in `[1/2, 1]`. ∎
