# External Reviewer Guide

**Purpose:** fast orientation for an external reader who needs to understand
what the repo currently has without reading raw chronology.

## What this package actually claims

The current flagship package is not “everything in the repo.” It is:

1. a **retained theorem core**
2. a **broader observation-facing bounded portfolio**
3. **three live flagship gates**
4. an explicit **frozen-out registry** so important work is not silently lost

  family on `PL S^3 x R`

Three framing points matter for reading the retained core:

## Package rule

- If a result is in the repo but not in the matrix, it is not publication-captured yet.
- If it is in the matrix but not in the claims table, it is not on the retained
  flagship claim surface.
- If it is important but excluded, it should appear in the frozen-out registry.
- If you are editing the package rather than reviewing it, use
  [PUBLICATION_CONTROL_PLANE.md](./PUBLICATION_CONTROL_PLANE.md).
