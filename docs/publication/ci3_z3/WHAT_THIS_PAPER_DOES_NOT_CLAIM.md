# What This Paper Does Not Claim

**Date:** 2026-04-15  
**Purpose:** one canonical public boundary note stating which stronger claims
are intentionally not being made on the current paper surface.

This note exists so the paper can be read for what it actually proves, rather
than for a stronger slogan a reviewer might infer from breadth alone.
