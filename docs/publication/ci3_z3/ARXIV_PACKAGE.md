# arXiv Package

## Goal

Use the same retained backbone as the Nature package, but carry the full
derivation scaffolding, negative results, and bounded phenomenology with
explicit status labels.

## Include

- [arXiv draft](./ARXIV_DRAFT.md)
- [Derivation / validation map](./DERIVATION_VALIDATION_MAP.md)
- [Full claim ledger](./FULL_CLAIM_LEDGER.md)
- [Figure plan](./FIGURE_PLAN.md)
- [Figure captions](./FIGURE_CAPTIONS.md)
- [Submission checklist](./SUBMISSION_CHECKLIST.md)
- full outline / manuscript source
- retained notes and runners
- bounded-lane notes for:
  - `S^3`
  - DM relic mapping
  - renormalized `y_t`
  - CKM
- negative-result notes that block overclaiming
- claims table and reproduce guide

## Exclude or quarantine

- stale review packets
- pitch decks / acceptance-probability cards as if they were science authority
- duplicated closure notes that disagree with the current audited state

## Working rule

The arXiv package is allowed to be longer, not looser. Every bounded section
must end with its actual current status and must not silently outrun
[CLAIMS_TABLE.md](./CLAIMS_TABLE.md) or
[DERIVATION_VALIDATION_MAP.md](./DERIVATION_VALIDATION_MAP.md).
