## 0. Statement

**Theorem.** Given quark electric charges Q_u = +2/3, Q_d = −1/3 (from
the cited `Q = T_3 + Y/2` convention plus the cited SM hypercharge table)
and admitted SM quark constituent content of nucleons:
- proton p = uud
- neutron n = udd

then Q_p = +1 and Q_n = 0 exactly.

**Proof.** 
```text
Q_p  =  Q_u + Q_u + Q_d  =  2/3 + 2/3 − 1/3  =  3/3  =  +1  ✓
Q_n  =  Q_u + Q_d + Q_d  =  2/3 − 1/3 − 1/3  =  0/3  =  0   ✓
```

These reproduce the standard SM nucleon charge labels exactly.
