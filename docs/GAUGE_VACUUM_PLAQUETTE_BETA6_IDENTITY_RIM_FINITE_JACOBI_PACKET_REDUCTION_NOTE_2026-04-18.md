## Question

After the identity-rim reduction, the cyclic-bulk reduction, and the
fixed-depth finite-Krylov reduction, what is the sharpest honest
**basis-invariant scalar** bulk object for fixed-depth class-sector `beta = 6`
closure?

Is the remaining target still the finite compressed matrix

`Q_d S_6^env Q_d`?

Or does fixed-depth closure really depend only on a smaller canonical packet?

And if the exact propagated retained triple were evaluated, would that at least
determine that first canonical packet?

## Answer

At fixed propagation depth `d = L_perp - 1`, the sharp upstream bulk object is
not merely the finite Krylov matrix block. It is the finite **Jacobi packet**
of that block in the Lanczos basis generated by the identity rim state
`eta_6(e)`.

More precisely, let

`K_d = span{eta, S eta, ..., S^d eta}`,

`eta = eta_6(e)`,

and let `Q_d` be the orthogonal projector onto `K_d`.

Then in the orthonormal Lanczos basis generated from the normalized rim state,
the finite compression `Q_d S Q_d` is represented by one positive symmetric
tridiagonal matrix

`J_d = tridiag(beta_1, ..., beta_r-1; alpha_0, ..., alpha_r-1)`,

where `r = dim K_d <= d + 1`.

So fixed-depth class-sector closure depends only on the finite Jacobi packet

`(alpha_0, ..., alpha_r-1; beta_1, ..., beta_r-1)`,

not on an arbitrary matrix representation of `Q_d S Q_d`.

Equivalently, the fixed-depth cyclic moment packet

`m_n = <eta, S^n eta> = e_1^T J_d^n e_1`

is already downstream of that same finite Jacobi packet.

But the current exact propagated retained triple still does **not** determine
even the first nontrivial Jacobi packet. The runner gives two positive bulk
witnesses with:

- one common identity rim state `eta`,
- the same exact propagated retained triple
  `(Z_6^env(W_A), Z_6^env(W_B), Z_6^env(W_C))`,
- but different first Jacobi packets
  `(alpha_0, beta_1)`.

So the current propagated triple still does not close even the sharpest
canonical finite bulk packet on the present bank.

## Theorem 1: exact finite-Jacobi reduction of fixed-depth plaquette closure

Let `eta = eta_6(e) / ||eta_6(e)||` and let

`K_d = span{eta, S eta, ..., S^d eta}`.

Apply the Lanczos algorithm to the positive self-adjoint finite compression
`Q_d S Q_d` with start vector `eta`. Let

`q_0 = eta`,

and let `q_0, ..., q_r-1` be the resulting orthonormal Lanczos basis of `K_d`.

Then the compressed operator has the tridiagonal representation

`J_d = [<q_i, S q_j>]_(0<=i,j<r)`,

with

`(J_d)_(i,i) = alpha_i`,

`(J_d)_(i,i+1) = (J_d)_(i+1,i) = beta_(i+1)`,

and all other matrix entries zero.

Moreover, for every `0 <= n <= d`,

`S^n eta = Q [J_d^n e_1]`,

where `Q = [q_0 ... q_r-1]` and `e_1 = (1,0,...,0)^T`.

Therefore every fixed-depth class-sector quantity of the form

`<f, S^n eta>`

depends only on the finite Jacobi packet of `J_d`.

So the sharp fixed-depth bulk datum is not an arbitrary finite matrix block. It
is the Jacobi packet of the `eta_6(e)`-generated finite Krylov compression.

## Theorem 2: the propagated retained triple still does not determine even the first nontrivial Jacobi packet

On the explicit higher-orbit slice

`{(0,2), (0,3), (0,4), (0,5)}`,

the runner uses the same common identity rim state `eta` and the same two
positive propagated targets `v_P`, `v_Q` as the finite-Krylov nonclosure

- `E_3((S_P)^d eta) = E_3((S_Q)^d eta)`,
- but the first Jacobi packets differ:

  `alpha_0^P != alpha_0^Q`,

  `beta_1^P != beta_1^Q`.

So even the exact propagated retained triple still does **not** determine the
first nontrivial Jacobi packet of the sharp fixed-depth bulk object.
