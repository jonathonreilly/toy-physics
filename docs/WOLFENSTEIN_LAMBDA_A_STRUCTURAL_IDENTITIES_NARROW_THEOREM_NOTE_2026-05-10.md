## Statement

Let `alpha_s, n_pair, n_color, n_quark` be abstract positive symbols (with
`n_pair, n_color, n_quark` positive integers) satisfying the two parametric
input identities

```text
lambda^2          =  alpha_s / n_pair,                            (I1)
A^2               =  n_pair / n_color.                            (I2)
```

Then the three Wolfenstein structural identities

```text
(W1)  lambda^2          =  alpha_s / n_pair,                      (same as I1)
(W2)  A^2               =  n_pair / n_color,                      (same as I2)
(W3)  A^2 lambda^2      =  alpha_s / n_color,                     (n_pair cancellation)
```

are forced, and the CKM magnitude corollaries

```text
(C-Vcb-sq)  |V_cb|^2          =  A^2 lambda^4
                              =  alpha_s^2 / (n_pair n_color),    (CV1)
(C-Vub0-sq) |V_ub|_0^2        =  A^2 lambda^6 (rho^2 + eta^2)
                              =  alpha_s^3 (rho^2 + eta^2)
                                 / (n_pair^2 n_color),            (CV2)
```

are forced closed-form expressions in `(alpha_s, n_pair, n_color, rho, eta)`,
where `(rho, eta)` enter `(CV2)` as a separate abstract hypothesis from the
companion CP-phase narrow theorem.

Under the additional count constraint `n_quark = n_pair * n_color` together
with the cited CP-radius hypothesis `rho^2 + eta^2 = 1/n_quark`, `(CV2)`
reduces to

```text
|V_ub|_0^2  =  alpha_s^3 / (n_pair^2 n_color n_quark)
            =  alpha_s^3 / (n_pair^3 n_color^2).                  (CV2-q)
```

And the `|V_cb|^2` rewriting

```text
|V_cb|^2    =  alpha_s^2 / n_quark.                               (CV1-q)
```

At the framework counts `(n_pair, n_color, n_quark) = (2, 3, 6)`, these
specialise to the closed forms

```text
lambda^2          =  alpha_s / 2,
A^2               =  2 / 3,
A^2 lambda^2      =  alpha_s / 3,
|V_cb|^2          =  alpha_s^2 / 6,
|V_ub|_0^2        =  alpha_s^3 / 72                  (under rho^2+eta^2 = 1/6).
```

These specialisations are shown for sanity, not as load-bearing inputs. The
parametric algebra closes for any other count tuple that satisfies the same
input identities.

## Proof

Pure substitution.

`(W1)`, `(W2)`. Tautological with `(I1)`, `(I2)`.

`(W3)`. Multiply `(I2)` and `(I1)`:

```text
A^2 lambda^2
  =  (n_pair / n_color) (alpha_s / n_pair)
  =  alpha_s / n_color.
```

The `n_pair` factor cancels exactly.

`(CV1)`. By definition `|V_cb|^2 = A^2 lambda^4`. Substitute:

```text
A^2 lambda^4
  =  (n_pair / n_color) (alpha_s / n_pair)^2
  =  (n_pair / n_color) (alpha_s^2 / n_pair^2)
  =  alpha_s^2 / (n_pair n_color).
```

`(CV2)`. By definition `|V_ub|_0^2 = A^2 lambda^6 (rho^2 + eta^2)`. Substitute:

```text
A^2 lambda^6 (rho^2 + eta^2)
  =  (n_pair / n_color) (alpha_s / n_pair)^3 (rho^2 + eta^2)
  =  alpha_s^3 (rho^2 + eta^2) / (n_pair^2 n_color).
```

`(CV1-q)`. From `(CV1)`, applying `n_quark = n_pair * n_color`:

```text
|V_cb|^2 = alpha_s^2 / (n_pair n_color) = alpha_s^2 / n_quark.
```

`(CV2-q)`. Substitute `rho^2 + eta^2 = 1/n_quark` into `(CV2)`:

```text
|V_ub|_0^2  =  alpha_s^3 / (n_pair^2 n_color n_quark).
```

Apply `n_quark = n_pair * n_color`:

```text
|V_ub|_0^2  =  alpha_s^3 / (n_pair^3 n_color^2).
```

∎

## Derivable corollaries

```text
A^2 lambda^2 / lambda^2   =  A^2  =  n_pair / n_color   (algebraic ratio)
|V_cb|^2 / lambda^2       =  alpha_s / n_color         (uses I1)
|V_us|_0^2 / |V_cb|^2     =  n_color / alpha_s         (uses I1, |V_us|_0^2 = lambda^2)
```

All three follow algebraically from `(W1)`-`(CV1)`.
