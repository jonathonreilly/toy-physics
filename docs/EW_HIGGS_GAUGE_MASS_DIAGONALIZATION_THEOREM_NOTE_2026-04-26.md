## 1. Claim

Let the electroweak gauge group be

```text
SU(2)_L x U(1)_Y
```

with gauge fields `W_mu^a` and `B_mu`, couplings `g` and `g_Y`, and Higgs
doublet

```text
H = (H^+, H^0)^T,          Y_H = 1/2.
```

Assume a neutral Higgs vacuum

```text
<H> = H_0 = (0, v/sqrt(2))^T,        v > 0,
```

and the standard covariant derivative

```text
D_mu = partial_mu - i g (tau^a/2) W_mu^a - i g_Y Y B_mu.
```

Then the quadratic gauge-mass part of `|D_mu H|^2` is diagonalized by

```text
W_mu^+ = (W_mu^1 - i W_mu^2) / sqrt(2),
W_mu^- = (W_mu^1 + i W_mu^2) / sqrt(2),

Z_mu   =  cos(theta_W) W_mu^3 - sin(theta_W) B_mu,
A_mu   =  sin(theta_W) W_mu^3 + cos(theta_W) B_mu,
```

where

```text
sin(theta_W) = g_Y / sqrt(g^2 + g_Y^2),
cos(theta_W) = g   / sqrt(g^2 + g_Y^2).
```

The exact tree-level masses are

```text
M_W^2 = g^2 v^2 / 4,
M_Z^2 = (g^2 + g_Y^2) v^2 / 4,
M_A^2 = 0.
```

The unbroken gauge generator is

```text
Q = T_3 + Y,
```

and the electric charge is fixed by the same diagonalization:

```text
e = g sin(theta_W) = g_Y cos(theta_W)
  = g g_Y / sqrt(g^2 + g_Y^2).
```

Consequently the custodial tree relation follows algebraically:

```text
rho_tree = M_W^2 / (M_Z^2 cos^2(theta_W)) = 1.
```

For this repo's GUT-normalized convention, set

```text
g_Y = sqrt(3/5) g_1_GUT.
```

The theorem then gives the exact tree-level dictionary used by the EW and
`M_W` notes:

```text
M_W = g_2 v / 2,
M_Z = sqrt(g_2^2 + g_Y^2) v / 2.
```

## 3. Charged Gauge-Boson Sector

Use the Pauli matrices normalized by

```text
T_a = tau^a / 2,
[T_a, T_b] = i epsilon_abc T_c.
```

At the vacuum,

```text
T_1 H_0 = (v/(2 sqrt(2)), 0)^T,
T_2 H_0 = (-i v/(2 sqrt(2)), 0)^T,
T_3 H_0 = (0, -v/(2 sqrt(2)))^T,
Y H_0   = (0,  v/(2 sqrt(2)))^T.
```

The charged directions are generated by `T_1` and `T_2`. Their contribution
to the vacuum kinetic term is

```text
| -i g W_mu^1 T_1 H_0 - i g W_mu^2 T_2 H_0 |^2
  = g^2 v^2 / 8 [ (W_mu^1)^2 + (W_mu^2)^2 ].
```

Define

```text
W_mu^+ = (W_mu^1 - i W_mu^2) / sqrt(2),
W_mu^- = (W_mu^1 + i W_mu^2) / sqrt(2).
```

Then

```text
W_mu^+ W^{-\mu} = [ (W_mu^1)^2 + (W_mu^2)^2 ] / 2,
```

so the charged mass term is

```text
g^2 v^2 / 4 W_mu^+ W^{-\mu}.
```

Therefore

```text
M_W^2 = g^2 v^2 / 4.
```

This is purely the spectrum of the Higgs kinetic term at the neutral doublet
vacuum. No experimental `M_W` value is used.

## 4. Neutral Gauge-Boson Sector

The neutral contribution is generated by `T_3` and `Y`. Since

```text
T_3 H_0 = -Y H_0,
```

the neutral part of the covariant derivative is

```text
-i (g W_mu^3 T_3 + g_Y B_mu Y) H_0
  = -i (-g W_mu^3 + g_Y B_mu) Y H_0.
```

Taking the norm gives

```text
| -i (g W_mu^3 T_3 + g_Y B_mu Y) H_0 |^2
  = v^2 / 8 (g W_mu^3 - g_Y B_mu)^2.
```

Equivalently,

```text
L_mass,neutral
  = 1/2 (W_mu^3, B_mu)
      [ v^2/4 * [[ g^2,     -g g_Y ],
                 [ -g g_Y,   g_Y^2 ]] ]
      (W^{3 mu}, B^mu)^T.
```

The neutral mass-squared matrix is

```text
M_0^2 = v^2/4 * [[ g^2,     -g g_Y ],
                 [ -g g_Y,   g_Y^2 ]].
```

Its determinant is exactly zero:

```text
det(M_0^2) = (v^4/16)(g^2 g_Y^2 - g^2 g_Y^2) = 0.
```

Its trace is

```text
tr(M_0^2) = v^2/4 (g^2 + g_Y^2).
```

Because `M_0^2` is real symmetric and positive semidefinite, its eigenvalues
are therefore

```text
0,
(g^2 + g_Y^2) v^2 / 4.
```

The zero eigenvector solves

```text
g^2 x - g g_Y y = 0,
-g g_Y x + g_Y^2 y = 0,
```

so for nonzero couplings it is proportional to

```text
(x, y) = (g_Y, g).
```

The massive eigenvector is orthogonal to it:

```text
(x, y) = (g, -g_Y).
```

Normalize by

```text
G = sqrt(g^2 + g_Y^2),
sin(theta_W) = g_Y / G,
cos(theta_W) = g / G.
```

Then

```text
A_mu = sin(theta_W) W_mu^3 + cos(theta_W) B_mu,
Z_mu = cos(theta_W) W_mu^3 - sin(theta_W) B_mu,
```

with

```text
M_A^2 = 0,
M_Z^2 = G^2 v^2 / 4.
```

This proves the exact photon/Z diagonalization.

## 5. Unbroken Charge and Electric Coupling

The vacuum is invariant under generators `X = alpha T_3 + beta Y` satisfying

```text
X H_0 = 0.
```

Since `T_3 H_0 = -Y H_0`, the condition is

```text
(-alpha + beta) Y H_0 = 0,
```

hence

```text
alpha = beta.
```

The unique unbroken generator, up to normalization, is therefore

```text
Q = T_3 + Y.
```

The interaction before rotation is

```text
g T_3 W_mu^3 + g_Y Y B_mu.
```

Invert the Weinberg rotation:

```text
W_mu^3 = cos(theta_W) Z_mu + sin(theta_W) A_mu,
B_mu   = -sin(theta_W) Z_mu + cos(theta_W) A_mu.
```

The coefficient of `A_mu` is

```text
g sin(theta_W) T_3 + g_Y cos(theta_W) Y.
```

Using the definitions of `sin(theta_W)` and `cos(theta_W)`,

```text
g sin(theta_W) = g_Y cos(theta_W)
               = g g_Y / sqrt(g^2 + g_Y^2)
               = e.
```

Thus the photon couples as

```text
e A_mu (T_3 + Y) = e A_mu Q.
```

This proves both the charge generator and the electric coupling formula.

## 6. Custodial Tree Relation

From the mass formulas,

```text
M_W^2 = g^2 v^2 / 4,
M_Z^2 = (g^2 + g_Y^2) v^2 / 4,
cos^2(theta_W) = g^2 / (g^2 + g_Y^2).
```

Therefore

```text
M_Z^2 cos^2(theta_W)
  = [(g^2 + g_Y^2) v^2 / 4] * [g^2 / (g^2 + g_Y^2)]
  = g^2 v^2 / 4
  = M_W^2.
```

So

```text
rho_tree = M_W^2 / (M_Z^2 cos^2(theta_W)) = 1.
```

This is an identity of the one-doublet Higgs representation. It is not a
separate numerical assumption.

## 7. Scalar Hessian Companion

If the scalar potential is the usual one-doublet renormalizable potential

```text
V(H) = -mu^2 H^dagger H + lambda (H^dagger H)^2,
lambda > 0,
```

then the stationary condition at

```text
H = (0, (v+h)/sqrt(2))^T
```

is

```text
v^2 = mu^2 / lambda.
```

Expanding around the vacuum gives

```text
V = constant + lambda v^2 h^2 + lambda v h^3 + (lambda/4) h^4.
```

Since a real scalar mass term is `(1/2) m_h^2 h^2`, the tree-level radial
mass is

```text
m_h^2 = 2 lambda v^2.
```

The three angular scalar modes are flat directions of the potential at the
minimum and are absorbed as the longitudinal polarizations of `W^+`, `W^-`,
and `Z` after gauge fixing. This companion calculation is included only to
pin the standard Higgs-mechanism bookkeeping; it is not a numerical Higgs
mass derivation.
