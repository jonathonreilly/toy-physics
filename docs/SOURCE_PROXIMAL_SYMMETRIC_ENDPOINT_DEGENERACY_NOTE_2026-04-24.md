# Source-Proximal Rule: Symmetric-Endpoint Degeneracy

**Date:** 2026-04-24
**Scope:** detector-heuristic hygiene note, not a physics claim.
**Runner:** `scripts/frontier_source_proximal_symmetric_endpoint_degeneracy.py`
**Result:** `10/10 PASS`, wallclock ~0.05 s.

## 1. Rule under examination

On a cycle-bearing graph `G = (V, E)` with source vertex `s`, the
**source-proximal non-bridge edge rule** picks, within a given cycle,
the non-bridge edge `e = (u, v)` minimizing the midpoint distance

```
midpoint_dist(e) = (d(s, u) + d(s, v)) / 2
```

where `d(·, ·)` is graph distance. Under an isotropic kernel
`K: N -> R` with `phi(v, t) = S(t) * K(d(s, v))`, the detector reads
out the current span on `e`:

```
span_e = max_t j_e(t) - min_t j_e(t),   j_e(t) := phi(v, t) - phi(u, t).
```

The rule has been proposed as an optimal-sensitivity detector for
time-varying source amplitudes `S(t)`.

## 2. Finding

The rule has a characterized degeneracy: when the selected edge `e`
has equidistant endpoints `d(s, u) = d(s, v)`, the detector span is
**identically zero** for any `S(t)` and any isotropic `K`.

### Proof

```
phi(u, t) = S(t) * K(d(s, u))
phi(v, t) = S(t) * K(d(s, v))
d(s, u) = d(s, v)
  =>  K(d(s, u)) = K(d(s, v))
  =>  phi(u, t) = phi(v, t)   for all t
  =>  j_e(t) = 0              for all t
```

The derivation uses only that `K` is a function of graph distance
alone. It does not depend on the choice of kernel (Yukawa, Coulomb,
logarithmic, etc.) or on the choice of source amplitude `S(t)`.

## 3. Concrete counter-example

**Triangular prism.** Bottom triangle `0-1-2`, top triangle `3-4-5`,
verticals `0-3`, `1-4`, `2-5`. Source `s = 5`, BFS root `0`.

BFS spanning tree: `{(0,1), (0,2), (0,3), (3,4), (3,5)}`.
Back edges: `{(1,2), (1,4), (2,5), (4,5)}`.

The back edge `(1, 4)` generates the fundamental cycle
`1 -> 0 -> 3 -> 4 -> 1` with edges `(0,1), (0,3), (3,4), (1,4)` and
midpoint distances:

| Edge | `d(5, u)` | `d(5, v)` | midpoint |
|---|---|---|---|
| (0, 1) | 2 | 2 | 2.0 |
| (0, 3) | 2 | 1 | 1.5 |
| (3, 4) | 1 | 1 | **1.0** (minimum, equidistant) |
| (1, 4) | 2 | 1 | 1.5 |

The rule selects `(3, 4)`. Both endpoints are distance 1 from `s = 5`,
so `phi(3, t) = phi(4, t)` and `span_{(3,4)} = 0` under any isotropic
kernel. The runner confirms this for Yukawa (`mu = 1.0` and `mu = 0.3`),
a Coulomb-like `1 / (1 + d)` kernel, and a `-log(1 + d)` kernel: all
four give `span_{(3,4)} < 1e-12`.

## 4. Positive control

A different basis cycle of the same prism — the one generated by back
edge `(2, 5)` — contains edge `(2, 5)` with endpoints at distances `1`
and `0` from the source. The rule selects `(2, 5)`, whose endpoints are
non-equidistant, and the detector reports a nonzero span (`span ~ 1.26`
for Yukawa `mu = 1.0`). The degeneracy is specific to edges with
endpoint symmetry, not a graph-wide failure of the rule.

## 5. Remediation

The simplest fix: **restrict the candidate set** to non-bridge edges
with `d(s, u) != d(s, v)` before minimizing the midpoint distance. On
the problematic prism cycle, this pre-filter promotes `(0, 3)` (or an
equivalent asymmetric edge) and restores a nonzero span.

If every non-bridge edge of a cycle has equidistant endpoints under the
working kernel — a purely graph-geometric property — the detector is
declared **N/A** for that cycle. This matches the existing N/A
designation for DAG cycle-integrals, so the treatment is uniform.

## 6. Scope and non-claims

- The finding is graph-geometric. It holds for isotropic kernels
  (those depending only on `d(s, v)`). **Anisotropic kernels** — for
  example, direction-biased kernels on non-homogeneous graphs, or
  path-integral kernels depending on more than the endpoint distance —
  may not exhibit the degeneracy.
- No claim is made about the underlying physics. The detector span
  being zero is a symmetry artifact of an unmodified heuristic, not a
  statement about zero flux, zero signal, or zero holonomy on the
  cycle.
- No claim is made that the rule is unsalvageable. The remediation in
  §5 is a minimal fix; other remediations (averaging across edges,
  using a detector rule that doesn't rely on midpoint distance) are
  not examined here.

## 7. Falsifier

- A graph, isotropic kernel `K`, and non-bridge edge with equidistant
  endpoints that produce a nonzero `span_e` (would contradict §2
  derivation).
- A rule-selected edge with non-equidistant endpoints that produces
  exactly-zero span under a non-degenerate source `S(t)` (would mean
  the failure mode is not specifically characterized by endpoint
  symmetry).

The runner exposes both axes.
