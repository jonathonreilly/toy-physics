## The critique in detail

### What iter 21 ACTUALLY establishes

Iter 21's Q = p·δ identity is **genuine**:
- δ = 2/p² from iter 1 (APS η derivation, retained-forced).
- Q = 2/d from iter 2 (AM-GM derivation, retained-forced).
- p = d = 3 (Z_3 = C_3[111] subgroup from iter 1; d = 3 generations from
  Z_3 isotypes in iter 2; identification retained).
- Therefore Q/δ = p²/d = p, so Q = p·δ.

**This identity is retained-forced.** It connects two independent
retained closures.

However, iter 21 also claims SR2 "conservation law" under iter 4 deformation:
```
Δ(SR2 LHS) = Q·Δsin²θ_12 + Δsin²θ_13 = 0 at leading order
```

**This conservation depends on iter 4's specific deformation formulas.**
It's a property of iter 4 (if correct), not of the retained axioms directly.

### What iter 22 ACTUALLY establishes

Iter 22's θ_13 = 4/p³ rewrites iter 4's θ_13 = δ·Q using Q = p·δ:
- θ_13 = δ · Q = δ · p·δ = p · δ² = p · (2/p²)² = 4/p³

At p = 3: θ_13 = 4/27. Correct.

**BUT**: the equality θ_13 = δ·Q is **iter 4's conjecture**. It hasn't
been derived from retained axioms — it was observed to fit NuFit.

Iter 22 beautifully simplifies this to θ_13 = 4/p³, but:
- If iter 4's θ_13 = δ·Q is correct, iter 22 gives the p-scaling.
- If iter 4's is wrong (e.g., θ_13 = δ·Q + small correction), iter 22's
  scaling also needs correction.

### The crux: iter 4 is still a conjecture

Iter 4 claimed:
- θ_13 = δ·Q (matches NuFit central at 0.085° gap, within 1σ)
- θ_23 − π/4 = δ·Q/2 (matches NuFit at 0.04°)
- sin²θ_12 = 1/3 − δ²·Q (matches NuFit at 0.0036 gap)

These match NuFit within 1σ but they are **numerical fits**, not
derivations. The iter 21-22 analysis shows these fits are STRUCTURALLY
ELEGANT (conservation law, powers of p), which is evidence FOR them
being retained, but not a derivation.

### What "retained-derived at numerical level" honestly means

Given iter 21-22's findings, the stronger claim "I5 numerically
retained-derived" holds ONLY IF:
- iter 4's θ_13 = δ·Q is exact (not merely 1σ approximate).
- No higher-order corrections beyond leading.

Honestly, the data cannot distinguish θ_13 = δ·Q from θ_13 = δ·Q + ε
for small ε (below NuFit 1σ resolution). So iter 4's exactness is
unproven.
