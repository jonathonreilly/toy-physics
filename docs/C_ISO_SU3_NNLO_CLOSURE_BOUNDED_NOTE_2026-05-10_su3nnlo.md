## 2. What this closes vs. does not close

### Supported (bounded, this work)

- **Numerical rational identification for SU(3) Wilson NNLO `c_3 = −13/3`**
  (was numerical −4.33 in retained note, matching to 11+ digits).
- **Numerical rational identification for SU(3) Wilson NNNLO `c_4 = −37/2`**
  (new, matched to 17 digits via mpmath @ 50 dps).
- **Numerical rational identification for SU(3) Wilson N4LO `c_5 = −3071/36`**
  (new, matched to 12 digits).
- **Fit-limited target for SU(3) Wilson N5LO `c_6 = −20873/48`**
  (matched at 6×10⁻⁸; needs higher-β fits or symbolic derivation for
  final-digit confidence).
- **Convention C-iso difference (P_W − P_HK) coefficients in
  s_t to 5 orders past NLO**: explicit rationals `7/9, −5/9, −47/486,
  −16891/43740, −308999/524880` for `s_t² … s_t⁶`.
- **C-iso truncation residual decoupled from ε_witness**: at ξ = 4 with
  NNNLO truncation, the residual `4.24×10⁻⁵` is already below
  ε_witness `3×10⁻⁴`. At ξ = 8 with NNNLO truncation, residual is
  `2.25×10⁻⁶` (~130× below target).

### Not closed (frontier remaining)

- **Exact-tier `ε_witness ~ 3×10⁻⁴`** on the full Hamilton-limit `<P>_KS`:
  the stat+vol budget `√(0.0002² + 0.0006²) ≈ 6.3×10⁻⁴` is the dominant
  systematic. Tightening **stat+vol** is now the named compute frontier:

  - Statistical: PR #685 used 5-seed × 1500 sweeps; doubling seeds + 4×
    sweeps would tighten ε_stat from 0.0002 to ~0.00007 (well below
    target).
  - Volume: PR #685 used L = 3, 4, 6 with `P_∞ + a/L² + b/L⁴` extrap;
    extending to L = 8, 10 with the same extrap quality would tighten
    ε_vol from 0.0006 to ~0.0002.

  With these tightenings, ε_total would drop to `√(0.00007² + 0.0002² +
  ε_C-iso²)` ≈ 2×10⁻⁴ at ξ ≥ 8 with NNNLO C-iso correction, **at
  ε_witness target**. The C-iso analytic series is no longer the
  bottleneck.

- **Closed-form analytic c_6 = −20873/48** (N5LO): identified at fit
  precision 6×10⁻⁸. Verifying to 1e-12+ precision would require
  higher-β mpmath integration (β > 1e6 would be sufficient; current
  largest β = 1.024×10⁵). This is a marginal precision question, not a
  structural obstruction.

- **Multi-plaquette propagation**: the C-iso bound here remains a
  **single-plaquette weight discrepancy**; propagation to the full
  lattice `<P>` is captured at LO via the rel_shift, but multi-plaquette
  factors of order unity may shift the absolute systematic. This is
  inherited from the prior NLO note; not the focus of this work.

## 4. Implementation overview

The runner [`scripts/cl3_c_iso_su3_nnlo_2026_05_10_su3nnlo.py`](../scripts/cl3_c_iso_su3_nnlo_2026_05_10_su3nnlo.py)
implements:

1. **Path A (rational verification)**: SU(3) Wilson single-plaquette via
   2D trapezoidal Weyl integration with adaptive range truncation
   `min(π, 12·sqrt(3/β))`. Compared against analytic series at orders
   LO/NLO/NNLO/NNNLO/N4LO. At β = 5000:
     - LO  rel err: 5.0×10⁻⁵
     - NLO rel err: 4.3×10⁻⁸
     - NNLO rel err: 3.7×10⁻¹¹
     - NNNLO rel err: 3.4×10⁻¹³
     - N4LO rel err: 3.0×10⁻¹³  (numerical floor)

2. **Path B (HK Taylor convergence)**: Verifies the exact rational
   coefficients `h_n = (-1)^(n+1) (4/3)^n / n!` via direct truncation
   residual comparison. At ξ = 4 (s_t = 0.125):
     - NLO trunc res: 7.4×10⁻⁴
     - NNLO trunc res: 3.1×10⁻⁵
     - NNNLO trunc res: 1.0×10⁻⁶
     - N4LO trunc res: 2.9×10⁻⁸
     - N5LO trunc res: 6.9×10⁻¹⁰

3. **Path C (engineering bound)**: Convention C-iso (P_W − P_HK)
   truncation residual at each canonical ξ value, propagated to absolute
   shift on `<P>_KS = 0.4410`. Verifies that NNNLO truncation residual
   at ξ ≥ 4 is below ε_witness target.

4. **Path D (total error budget)**: Combined `ε_total² = ε_stat² +
   ε_vol² + ε_C-iso²`. Identifies that the stat+vol floor `6.3×10⁻⁴`
   exceeds the ε_witness target `3×10⁻⁴`, and at ξ ≥ 8 with NNNLO
   truncation, ε_C-iso `~2×10⁻⁶` is now negligible vs. stat+vol. The
   bottleneck has shifted from C-iso (PR #685) to stat+vol (this
   work).

The rational identification used:
- `mpmath` 2D Gauss-Legendre quad with `mp.dps = 50–100`.
- Multiple β values from 800 to 102400 to fit `(P_W − 4/β) · β² = c_2 +
  c_3/β + c_4/β² + ...` as a polynomial in `1/β`.
- Linear-least-squares fit (via mpmath `lu_solve`) gives c_k to
  precision limited by the largest β. Each rational identified to
  within fit precision via `c_k · D` integer-rounding search up to
  D ≤ 10⁵.

Verification:
- `c_3 = −13/3`: 6×10⁻¹⁷ precision (mp.dps = 100, β up to 102400).
- `c_4 = −37/2`: 6×10⁻¹⁷ precision.
- `c_5 = −3071/36`: 3×10⁻¹² precision.
- `c_6 = −20873/48`: 7×10⁻⁸ precision (fit-limited).

## 6. Boundaries

This note does NOT claim:

- **Exact-tier `ε_witness ~ 3×10⁻⁴`** on the full Hamilton-limit
  `<P>_KS`. The stat+vol systematic `6.3×10⁻⁴` is now the bottleneck;
  C-iso has been moved to a sub-dominant systematic by this work.
- **Closed-form to all orders.** The series is asymptotic; the
  rationals `c_3 = −13/3, c_4 = −37/2, c_5 = −3071/36, c_6 = −20873/48`
  are identified up to fit precision. The pattern (denominators
  containing factors of 2 and 3 only, suggesting Bernoulli-like
  structure) is not analytically derived from group-theoretic identities
  in this work; that is a documented sub-frontier (see Brower-Rebbi-Soni
  1981 and subsequent SU(3) lattice gauge theory literature for the
  structural origin).
- **Full lattice multi-plaquette propagation**. As before, the
  single-plaquette weight discrepancy is the leading systematic; the
  multi-plaquette propagation is captured at leading order and inherits
  from PR #685.
- **NNLO derivation strictly from group-theoretic Vandermonde-Gaussian
  symbolic computation.** The retained NLO closure attempted symbolic
  derivation; this work uses high-precision mpmath fitting and rational
  identification, equivalent in result but distinct in method. The
  symbolic derivation of the c_3, c_4, ... rationals from
  Vandermonde-Gaussian moment computation remains a documented
  sub-frontier — the rationals identified here serve as targets for
  that derivation.
