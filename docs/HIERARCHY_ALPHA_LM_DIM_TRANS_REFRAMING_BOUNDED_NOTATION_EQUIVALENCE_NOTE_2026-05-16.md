## 2. The trivial algebraic identity (T1)

> **Theorem (T1).** For any positive real `alpha` and any positive integer
> `N`,
>
> ```text
> alpha^N = exp(N * ln(alpha)) = exp(-N * ln(1/alpha)) = exp(-c_eff/alpha)
> ```
>
> where `c_eff := N * alpha * ln(1/alpha)`.

**Proof.** By the definition of `ln` and `exp` on positive reals,
`alpha^N = exp(N ln alpha)`. Since `alpha > 0`, `ln(1/alpha) = -ln(alpha)`,
so `N ln(alpha) = -N ln(1/alpha) = -alpha * (N * ln(1/alpha)) / alpha
= -c_eff/alpha`. Exponentiating gives `alpha^N = exp(-c_eff/alpha)`. The
identity is a tautology of the real exponential. `∎`

This is the only theorem of `(H')` as a notational object.
