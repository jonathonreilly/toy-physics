## Claim scope

Let `(X, μ)` be a finite or compact measured space and let
`Θ : X → X` be a measure-preserving involution
(`Θ² = id`, `Θ_* μ = μ`). Let `S_+ : X → R` be a real-valued
`Θ`-invariant half-action (`S_+(Θx) = S_+(x)`) and define
`ψ²(x) := exp(-S_+(x))`. Let `F : X → C` be a measurable observable
that is Hermitian under reflection:

```text
F(Θx) = overline{F(x)}.
```

Then:

- **(G1) Cauchy-Schwarz factorization identity.**

  ```text
  ∫_X dμ(x) exp(-S_+(x)) exp(-S_+(Θx)) F(Θx) F(x)
    =  ∫_X dμ(x) |ψ²(x) F(x)|²,
  ```

  so the reflected expectation reduces to the weighted norm-square

  ```text
  ⟨ Θ(F) · F ⟩  =  ‖ ψ² F ‖²_{L²(X, μ)}  ≥  0.
  ```

- **(G2) Real non-negativity.** `⟨ Θ(F) · F ⟩` is a real non-negative
  number, with strict positivity iff `ψ² F` is non-zero
  `dμ`-a.e. The reflection-positivity inequality
  `⟨ Θ(F) · F ⟩ ≥ 0` holds.

- **(G3) Sesquilinear form structure.** The map
  `(F, F') ↦ G(F, F') := ∫_X dμ(x) overline{ψ²(x)F(x)} ψ²(x)F'(x)`
  defines the standard positive semi-definite Hermitian
  sesquilinear form on the weighted `L²(X, μ)` domain. The kernel
  `Null(G) := { F : ψ² F = 0 μ-a.e. }` quotients to a positive
  inner-product space.

(G1)-(G3) are bounded measure-theoretic / functional-analytic
identities on `(X, μ, Θ, S_+)` under the explicit
`Θ`-invariance and reflection-Hermiticity hypotheses above. They do
not assert the full gauge-half Osterwalder-Seiler theorem for a
generic Wilson plaquette action.

## Proof

### (G1) Cauchy-Schwarz factorization

Start from the explicitly bounded reflected expectation

```text
⟨ Θ(F) · F ⟩
  := ∫_X dμ(x) exp(-S_+(x)) exp(-S_+(Θx)) F(Θx) F(x).
```

By the `Θ`-invariance hypothesis on `S_+`,
`exp(-S_+(Θx)) = exp(-S_+(x)) = ψ²(x)`. By the
reflection-Hermiticity hypothesis on `F`, `F(Θx) = overline{F(x)}`.
Therefore

```text
⟨ Θ(F) · F ⟩
  = ∫_X dμ(x) ψ²(x) ψ²(x) overline{F(x)} F(x)
  = ∫_X dμ(x) |ψ²(x) F(x)|²
  = ‖ ψ² F ‖²_{L²(X, μ)}.
```

This proves (G1). The claim is intentionally conditional: without
the `Θ`-invariant half-action and reflection-Hermitian observable
hypotheses, a generic expression `∫ h(x)h(Θx)dμ(x)` need not be a
non-negative norm-square.

### (G2) Real non-negativity

The expression `‖ ψ² F ‖²_{L²(X, μ)}` is a non-negative real
number by definition of the `L²` norm. Strict positivity holds
iff `ψ² F` is non-zero `μ`-a.e. (i.e. `F` does not vanish on the
support of `ψ²`). In this bounded symmetric setting,

```text
⟨ Θ(F) · F ⟩  ≥  0
```

follows. ∎

### (G3) Sesquilinear form structure

Define

```text
G(F, F') := ∫_X dμ(x) overline{ψ²(x) F(x)} ψ²(x) F'(x).
```

By linearity of the integral and the standard weighted `L²` inner
product, `G(·, ·)` is Hermitian sesquilinear:

```text
G(F, F')         =  ⟨ ψ² F | ψ² F' ⟩_{L²(X, μ)},
G(F, F)          =  ‖ ψ² F ‖²  ≥  0,
G(F', F)         =  overline{G(F, F')}.
```

The null space `Null(G) := { F : G(F, F) = 0 }` consists of
observables for which `ψ² F = 0` `μ`-a.e. The quotient by this null
space carries the induced positive-definite inner product, completing
to a Hilbert space when the usual `L²` completion is taken. ∎
