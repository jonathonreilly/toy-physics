## 2. The closure theorem

> **Theorem (Q_l = 2/3 closure via Observable-Principle locality).**
> On retained main, the physical charged-lepton scalar source on the
> 3-generation orbit is `J = jI` for `j ∈ ℝ`, by composition of
> retained Observable-Principle locality (OP) and C_3-invariance (C3)
> on the orbit. Reducing to the two-slot carrier (RED) gives
> `K = jI_2`, which has zero traceless part. By the criterion theorem
> (CRIT), this forces `Q_l = 2/3`. □

**Proof.**

*Step 1 (OP + onsite locality).* By OP Theorem 2 (property 4: "local:
it is generated by local projectors P_x"), every scalar bosonic source
on the framework's lattice is

```text
J = Σ_x j_x P_x,    j_x ∈ ℝ,    {P_x}_x = local onsite projectors.
```

*Step 2 (Z is excluded).* By ONSITE §3, the C_3-invariant operator

```text
Z = P_+ − P_perp = -I/3 + (2/3) C + (2/3) C²
```

has cross-site entries (it depends on the cyclic shifts `C, C²`).
Therefore `Z ∉ span{P_x}_x`. Equivalently, by ONSITE §3,

```text
{Σ_x j_x P_x : j_x ∈ ℝ} ∩ End_{C_3}(V) = span{I}.
```

So an OP source on the C_3-invariant orbit **cannot** carry a `Z`
component.

*Step 3 (C_3-invariance forces uniformity).* By C3, the retained 3-gen
orbit is C_3-equivariant: `C P_x C^{-1} = P_{x+1 mod 3}`. C_3-invariance
of the source `C J C^{-1} = J` forces

```text
Σ_x j_x P_{x+1} = Σ_x j_x P_x   ⇒   j_x = j_{x+1}   ⇒   j_1 = j_2 = j_3.
```

So `J = j (P_1 + P_2 + P_3) = j · I` (since `Σ_x P_x = I` on the
3-generation Hilbert space).

*Step 4 (carrier reduction).* By RED, the OP source on the orbit reduces
canonically to a source on the two-slot carrier:

```text
J = j · I  (on C^3)   ↦   K = j · I_2  (on the reduced two-slot carrier).
```

This is because `J = jI` is in the trivial isotype only (it commutes
with C_3 and is proportional to identity), so under the Plancherel
isotype split `C^3 = trivial ⊕ doublet`, `J` acts as the same scalar
`j` on both isotypes — equivalently `K = jI_2` on the reduced carrier.

*Step 5 (criterion ⇒ Q = 2/3).* By RED, `Y = (I_2 + K)^{-1}`. For
`K = jI_2`: `Y = (1+j)^{-1} I_2`. After the trace-2 normalization of
RED: `Y_norm = I_2`. Equivalently, the traceless coordinate

```text
z = (y_+ − y_perp) / (y_+ + y_perp) = 0
```

since `y_+ = y_perp` for `K = jI_2`. By CRIT,

```text
z = 0   ⇔   Q = 2/3.
```

So `Q_l = 2/3` retains. □

### 4.1 "Does OP really apply to the charged-lepton Yukawa source?"

OP is the framework's **retained** axiom-derived scalar observable
principle on the lattice
([`OBSERVABLE_PRINCIPLE_FROM_AXIOM_NOTE.md`](OBSERVABLE_PRINCIPLE_FROM_AXIOM_NOTE.md)).

(`Q_l = (Σ m_i)/(Σ √m_i)²`), hence a scalar bosonic observable in OP's
sense. OP's source `J` is the scalar mass-source for the lattice Dirac

× Higgs VEV insertion, which in the OP framing is exactly the scalar
source `J` on the orbit.

The April 22 reduced-carrier theorem (RED) explicitly identifies its

### 4.2 "Does locality really exclude Z?"

`Z = -I/3 + (2/3)C + (2/3)C²` involves the cyclic shift `C`, which is
a multi-site operator: `C P_x C^{-1} = P_{x+1}` — it shifts site `x`
to site `x+1`. Hence `Z` has matrix entries between *different* sites
in the site basis. A source of the form `J = Σ_x j_x P_x` is purely
diagonal in the site basis (each `P_x` is a single-site projector).
`Z` is not diagonal in the site basis, so `Z ∉ span{P_x}_x`.

ONSITE §3 makes this explicit: "But it is not an onsite diagonal source
function. It has offsite entries in the site basis. The exact
intersection is only the common scalar: onsite local functions ∩
End_{C_3}(V) = span{I}."

So OP locality forces sources into the onsite diagonal sector, which
intersects C_3-equivariant operators only in `span{I}`.

### 4.3 "Does the reduction step preserve K_TL = 0?"

`J = jI` on the 3-site orbit is purely in the trivial C_3 isotype (it
commutes with all C_3 operators and is proportional to identity).
Under the Plancherel reduction `C^3 = trivial ⊕ doublet`, `J`
restricts to:
- on trivial 1-dim subspace: scalar `j`
- on doublet 2-dim subspace: scalar `j` (since `J = jI` commutes with everything)

So on the two-slot reduced carrier `(+, perp)`: `K = diag(j, j) = jI_2`.

The traceless part of `K = jI_2` is identically zero: `K_TL := K − (1/2)Tr(K) I_2 = jI_2 − jI_2 = 0`.

So K_TL = 0, hence z = 0 by CRIT's `z = K_TL`-coordinate parameterization, hence Q = 2/3.

### 4.4 "Does CRIT really say z = 0 ⇔ Q = 2/3?"

Verbatim from CRIT §5:

> `W_red = log det(I+K)`, the following statements are equivalent:
> `K = 0 ⇔ Y = I_2 ⇔ z = 0 ⇔ ⟨Z⟩ = 0 ⇔ Q = 2/3`."

The chain `z = 0 ⇔ Q = 2/3` is the load-bearing equivalence.

### 4.5 "Is OP's locality property actually retained, or is it a definition?"

OP Theorem 2 derives source-locality as a CONSEQUENCE of the axiom-derived form `W = log|det(D+J)|`:
> "the local scalar curvature is: 1. bosonic ... 4. **local: it is generated by local projectors P_x**".

Property 4 is presented as a derived property of the source structure forced by OP Theorem 1's chain (Grassmann factorization + scalar additivity + CPT-even). It is not introduced as a separate axiom.

So OP's locality is a derived theorem, not a definition.
