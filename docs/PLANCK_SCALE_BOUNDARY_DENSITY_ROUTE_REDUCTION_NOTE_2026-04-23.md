## Exact reduction

### 1. The current free-fermion route is ruled out

The retained BH entropy no-go theorem is exact on the current carrier:

- `c_Widom = 1/6` in 2D on the retained square-lattice half-filled carrier;
- the current RT-ratio identification therefore does **not** land the
  black-hole coefficient `1/4`.

So the old free-fermion RT route cannot be the surviving boundary-density
theorem class.

### 2. The current bounded boundary-law probe is too weak

The bounded holographic/boundary-law probe shows:

- entropy scales more cleanly with boundary size than with region volume on
  the tested Dirac-sea construction;
- gravity preserves that boundary preference;
- but the note explicitly says it does **not** establish Bekenstein-Hawking or
  AdS/CFT.

So the present boundary-law probe is evidence for boundary sensitivity, not an
exact boundary-density theorem.

### 3. Therefore the surviving route must be gravitational and new-carrier

What survives is the following exact theorem target:

> find a new admissible same-surface carrier in which the gravitational sector
> itself, rather than the free-fermion RT ratio alone, determines the exact
> boundary density with the Einstein-Hilbert/Bekenstein-Hawking coefficient.

This is a much narrower target than "some entropy theorem."

## Safe route statement

The strongest surviving boundary-density route is:

1. preserve the physical-lattice and Einstein-Hilbert-style gravity/action
   surface;
2. abandon the current free-fermion RT coefficient carrier as a closure route;
3. look for a new gravitational boundary observable that is exact on the same
   surface and can carry the coefficient.

## What this closes

This closes the route ambiguity:

- should the next Planck-scale effort keep leaning on the existing

Answer: no.

The next boundary-density theorem, if it exists, must be a **new gravitational

It only proves the sharper strategy statement:

- the surviving boundary-density theorem cannot be a reuse of the current
  free-fermion or bounded boundary-law carriers.
