**Type:** positive_theorem

## Statement

Let `v_0 > 0` and `delta in R` be abstract real symbols, and set
`omega := exp(2 pi i / 3)`.

Define three real positive numbers
```text
x_k  :=  v_0 (1 + sqrt(2) cos(delta + 2 pi k / 3)),  k = 0, 1, 2.          (1)
```
(For `delta in R` and `v_0 > 0`, each `x_k >= v_0 (1 - sqrt(2)) > -0.42 v_0`,
so `x_k` is positive whenever `1 + sqrt(2) cos(theta_k) > 0`; the
narrow theorem treats `x_k` as a generic real symbol and does NOT
require positivity of every `x_k` for the algebraic identities to
hold.)

**Conclusion (T1) (root-of-unity sum identity).**

```text
sum_{k = 0, 1, 2} cos(delta + 2 pi k / 3)  =  0.                           (2)
```

**Conclusion (T2) (squared-cosine identity).**

```text
sum_{k = 0, 1, 2} cos^2(delta + 2 pi k / 3)  =  3/2.                       (3)
```

**Conclusion (T3) (Brannen-Rivero algebraic-Koide ratio).**

For all real `(v_0, delta)` with `v_0 > 0`,
```text
( sum_{k = 0, 1, 2} x_k^2 ) / ( sum_{k = 0, 1, 2} x_k )^2  =  2/3.         (4)
```

In particular, the ratio in `(4)` is independent of both `v_0` and
`delta`.

## Proof

`(T1)` Write `cos(theta_k) = Re(e^{i theta_k})` where `theta_k = delta + 2 pi k / 3`.
Then
```text
sum_k cos(theta_k)  =  Re( sum_k e^{i theta_k} )  =  Re( e^{i delta} sum_k omega^k ),
```
and `sum_{k = 0, 1, 2} omega^k = 1 + omega + omega^2 = 0` (standard
root-of-unity identity for the primitive cube root). Hence `sum_k cos(theta_k) = 0`.

`(T2)` Use `cos^2 theta = (1 + cos(2 theta)) / 2`:
```text
sum_k cos^2(theta_k)
   =  sum_k (1 + cos(2 theta_k)) / 2
   =  3/2  +  (1/2) sum_k cos(2 theta_k).
```
`sum_k cos(2 theta_k) = sum_k cos(2 delta + 4 pi k / 3) = Re(e^{2 i delta} sum_k omega^{2 k})`,
and `sum_{k = 0, 1, 2} omega^{2 k} = 1 + omega^2 + omega^4 = 1 + omega^2 + omega = 0`
(same identity, since the map `k -> 2 k mod 3` permutes `{0, 1, 2}`).
Hence the second sum vanishes and `sum_k cos^2(theta_k) = 3/2`.

`(T3)` Compute:
```text
sum_k x_k  =  sum_k v_0 (1 + sqrt(2) cos(theta_k))
           =  3 v_0  +  sqrt(2) v_0 * 0  by (T1)
           =  3 v_0.
```

```text
sum_k x_k^2  =  sum_k v_0^2 (1 + sqrt(2) cos(theta_k))^2
            =  v_0^2 sum_k (1 + 2 sqrt(2) cos(theta_k) + 2 cos^2(theta_k))
            =  v_0^2 (3 + 2 sqrt(2) * 0 + 2 * 3/2)  by (T1) and (T2)
            =  v_0^2 (3 + 3)
            =  6 v_0^2.
```

Hence
```text
( sum_k x_k^2 ) / ( sum_k x_k )^2  =  (6 v_0^2) / (3 v_0)^2  =  6 / 9  =  2/3. ∎
```

## Derivable corollary (Koide-form expressed as eigenvalue-squared atlas)

Define `m_k := x_k^2` (formal squaring of the abstract real symbols, no
physical-mass identification). Then the "Koide expression"
`(sum_k m_k) / (sum_k sqrt(m_k))^2` is, on positive `m_k = x_k^2`,
exactly the ratio `(sum_k x_k^2) / (sum_k x_k)^2` of `(T3)`, equal to
`2/3` independent of `(v_0, delta)`. This corollary identifies the
purely-algebraic content of the Koide ratio on the Brannen / Rivero
positive cosine ansatz; it does NOT claim that `m_k` equals any
physical charged-lepton mass (see "What this does NOT claim" below).
