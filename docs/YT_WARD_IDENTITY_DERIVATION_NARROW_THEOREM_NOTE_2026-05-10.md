## Proof

Pure algebraic substitution and ratio cancellation.

`(P1)`. Square both sides of `(I2)` to get
`F_Htt^(0)(g_bare)^2 = 1 / (N_c * N_iso)`. Equate with `(I1)`:

```text
1 / (N_c * N_iso)  =  c_S * g_bare^2 / (2 N_c).
```

Multiply both sides by `2 N_c`:

```text
2 / N_iso  =  c_S * g_bare^2.
```

Substitute `(I3)` (`c_S = +1`) and `(I4)` (`N_iso = 2`):

```text
2 / 2  =  g_bare^2,  i.e.  g_bare^2  =  1.
```

On the positive branch `g_bare = +1`.

`(P2)`. Substitute `(I4)` into `(I2)`:
`y_t_bare = F_Htt^(0)(g_bare) = 1 / sqrt(N_c * N_iso) = 1 / sqrt(6)`.

`(P2')`. Substitute `(I3)`, `(I4)`, `(P1)` into `(I1)`:
`y_t_bare^2 = (+1) * 1^2 / (2 * 3) = 1/6`. Consistent with `(P2)` on the
positive branch.

`(P3)`. From `(D2)`, `(D3)` the common factor `1/sqrt(u_0)` cancels in the
ratio:

```text
y_t(M_Pl) / g_s(M_Pl)  =  (y_t_bare / sqrt(u_0)) / (g_bare / sqrt(u_0))
                       =  y_t_bare / g_bare.
```

Using `(P2)` and `(P1)`:

```text
y_t(M_Pl) / g_s(M_Pl)  =  (1/sqrt(6)) / 1  =  1/sqrt(6)  =  1/sqrt(2 N_c).
```

[`scripts/audit_companion_yt_ward_identity_derivation_narrow.py`](./../scripts/audit_companion_yt_ward_identity_derivation_narrow.py)
verifies symbolically (sympy `simplify` / `solve` to 0 / unique positive
root) that:

1. Substituting `(I2)` squared into `(I1)` under `(I3)` reduces to the
   identity `g_bare^2 = 2 / (N_iso * c_S)` parametrically in
   `(c_S, N_iso, N_c)`.

3. `sympy.solve(g_bare^2 - 1, g_bare)` returns exactly `{-1, +1}`, and the
   positive-branch root is `+1`.

5. Substituting `(I3)`, `(I4)`, `(P1)` into `(I1)` yields `y_t_bare^2 = 1/6`,
   consistent with the `1/sqrt(6)` value on the positive branch.
6. Under `(D2)`-`(D3)` the ratio `y_t(M_Pl) / g_s(M_Pl)` reduces
   symbolically (no `u_0` left in free_symbols of the ratio) to `1/sqrt(6)`.

   the count assignment `(I4)` is load-bearing; at `c_S = -1`, no positive
   real `g_bare` satisfies the substitution under `(I2)`-`(I4)`, confirming
   `(I3)` is load-bearing.
