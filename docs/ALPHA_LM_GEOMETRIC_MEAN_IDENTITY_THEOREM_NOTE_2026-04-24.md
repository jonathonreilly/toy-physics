## Statement

Let `(α_bare, u_0) ∈ ℝ⁺²` be any pair of abstract positive scalars, and
define

```text
α_LM    := α_bare / u_0,                                                 (1)
α_s(v)  := α_bare / u_0².                                                (2)
```

**Conclusion (T1) (algebraic identity).** For all `(α_bare, u_0) ∈ ℝ⁺²`,

```text
α_LM²  =  α_bare · α_s(v).                                               (3)
```

**Conclusion (T2) (logarithmic restatement).** For all
`(α_bare, u_0) ∈ ℝ⁺²`,

```text
log α_LM  =  (log α_bare + log α_s(v)) / 2.                              (4)
```

**Conclusion (T3) (constant-ratio restatement).** For all
`(α_bare, u_0) ∈ ℝ⁺²`,

```text
α_LM / α_bare  =  α_s(v) / α_LM  =  1 / u_0.                             (5)
```

## Proof

`(T1)` Direct expansion using only `(1)` and `(2)`:

```text
α_LM²
  = (α_bare / u_0)²
  = α_bare² / u_0²
  = α_bare · (α_bare / u_0²)
  = α_bare · α_s(v).
```

`(T2)` Take `log` of both sides of `(3)`:

```text
2 log α_LM  =  log α_bare + log α_s(v),
```

valid because `α_bare, α_s(v), α_LM > 0` (positive products of
positive scalars). Divide by `2`.

`(T3)` From `(1)` and `(2)`,

```text
α_LM / α_bare  =  1 / u_0,
α_s(v) / α_LM  =  (α_bare / u_0²) / (α_bare / u_0)  =  1 / u_0.
```

The two ratios share the value `1 / u_0`, so they are equal. ∎

## Structural content

The two-power exponent progression `0, 1, 2` in `u_0^(-1)`,

```text
α_bare : α_LM : α_s(v)  =  α_bare : α_bare/u_0 : α_bare/u_0²,
```

is an arithmetic progression in the exponent of `u_0^(-1)`. By `(3)`,
this forces a geometric progression in the coupling values. The three
named couplings on this two-symbol surface contain only two independent
inputs: the bare normalization `α_bare` and the dimensionless scale
`u_0`. Any downstream lane that consumes `α_LM` and `α_s(v)` separately
should not count them as independent knobs once `α_bare` and `u_0` are
fixed.
