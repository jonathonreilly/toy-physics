# Top-Quark QFP Attractor Route No-Go

**Date:** 2026-05-10
**Claim type:** no_go
**Claim scope:** under the current framework coupling inputs used by the
top-Yukawa lane and a standard one-loop SM top-Yukawa RGE, a generic
Pendleton-Ross / quasi-fixed-point attractor scan over ultraviolet
`y_t(M_Pl)` values does not close the top-quark mass to the external

a special ultraviolet boundary near `y_t(M_Pl) ~= 0.5`. Therefore the QFP
attractor route cannot replace a derived UV boundary condition.

## Statement

The route under test is:

```text
generic UV y_t(M_Pl)  --one-loop RGE-->  IR QFP attractor  -->  m_t ~= target
```

The runner scans generic UV values `y_t(M_Pl)` over a broad range and runs
the one-loop SM RGE down to the framework electroweak scale. It finds:

```text
y_t(M_Pl) in [1, 10]  -->  y_t(v) ~= 1.18 .. 1.26
                         -->  m_t ~= 205 .. 219 GeV,
```

well above the external top-mass target. The route has strong focusing, but
the focused value is misplaced for the current framework coupling inputs.

A special point near `y_t(M_Pl)=0.5` can land near the target, but that is a
selected UV boundary, not a generic attractor closure. Without an independent
derivation of such a UV boundary, the QFP mechanism alone does not close the
top mass.

## What This Claims

- The one-loop QFP/focusing structure exists for the top-Yukawa RGE under the
  tested inputs.
- Generic large UV values focus to an IR band that is far above the external
  top-mass target.
- A near-target point in the scan is a boundary-condition selection problem,
  not an attractor derivation.

## What This Does Not Claim

- No top-mass derivation.
- No broad exhaustion of every possible heavy-quark mechanism.
- No rejection of a future derived UV top-Yukawa boundary condition.
- No use of the external top mass as a derivation input.
- No new axiom and no new retained status.

## Repair Target

To make the top-QFP route landable as positive science, the repo would need a
framework-derived ultraviolet boundary condition for `y_t(M_Pl)` that lands
in the successful part of the basin, or a modified retained coupling/RGE
packet that moves the attractor band to the target without an empirical fit.
