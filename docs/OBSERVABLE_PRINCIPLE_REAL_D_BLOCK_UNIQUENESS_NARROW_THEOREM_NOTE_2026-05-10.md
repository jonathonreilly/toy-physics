## Statement

Let `D` be a real anti-Hermitian operator on a finite-dimensional complex
Hilbert space `V = C^N` (with `N` even), so that `D^T = -D` and `D` has
real entries. Equivalently, in the language of
[`CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md`](CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md),
this is the abstract operator type satisfying premise identity (3)
`T D T = D` with `T = K` complex conjugation. Let the source `J` range
over the real-symmetric matrices `Sym_N(R) ⊂ Mat_N(R)`. Define the
candidate generator

```text
W[J]  :=  log |det(D + J)|  -  log |det D|,                            (W)
```

restricted to the open neighborhood `U_0 ⊂ Sym_N(R)` of `J = 0` on which
`det(D + J) ≠ 0`. (Such a `U_0` exists because `det(D + λ I)` is a
degree-`N` polynomial in `λ` and `det D ≠ 0` by invertibility of `D`,
checked in the runner.)

Fix three pieces of input:

**(X1) Real-D structural fact.** `D` is a real anti-Hermitian operator
on `V`, with the discrete-symmetry algebra of
[`CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md`](CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md)
holding on the block; in particular premise identity (3) `T D T = D`
holds. (Live ledger status of the cited authority: `retained_bounded`.)

**(X2) Admissibility class.** A scalar generator `F : U_0 → R` is
*admissible* on this real-D block iff it satisfies all four of:

  - (A) **block-additive on independent direct sums.** Whenever
    `D = D_A ⊕ D_B` and `J = J_A ⊕ J_B` on this block, with no hopping
    bonds across the `A | B` partition, `F[J_A ⊕ J_B] = F_A[J_A] + F_B[J_B]`.
  - (C) **CPT-even on the real-D block.** `F[J]` depends on `J` only
    through `|det(D + J)|`. (Equivalently, by (X1) and the Hermitian-lift
    invariance of
    [`CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md`](CPT_EXACT_REAL_ANTI_HERMITIAN_D_NARROW_THEOREM_NOTE_2026-05-10.md),
    `F` is invariant under the antiunitary `Θ_H = P K` action that
    transports `det(D + J) → det(D + J)^*` on real-D blocks; the
    fermionic phase is in the antiunitary-fixed eigenspace and is
    therefore not a free degree of freedom.)
  - (R) **continuous on `U_0`.**
  - (N) **normalized to** `F[0] = 0` on the real-D block.

**(X3) Cauchy–Erdős functional-equation closure.** The only continuous
function `g : R_+ → R` satisfying `g(r_1 r_2) = g(r_1) + g(r_2)` for
all `r_1, r_2 ∈ R_+` is `g(r) = c log r` for some real constant `c`.
(Standard one-variable Cauchy multiplicative-to-additive theorem; see
e.g. Erdős, *On the distribution of additive functions*, 1946.)

### Conclusion

Under (X1), (X2), (X3), there exists a real constant `c` such that, for
every admissible `F` in class (X2),

```text
F[J]  =  c · W[J]    on   U_0.                                          (U1)
```

Consequently the local source-derivative content of `F` at `J = 0`
matches that of `W` up to the overall scale `c`:

```text
∂F / ∂J_x    = c · ∂W / ∂J_x      at J = 0,                             (U2)
∂²F / ∂J_x ∂J_y  =  c · ∂²W / ∂J_x ∂J_y    at J = 0.                    (U3)
```

In particular, the **direction** of the source-response is fixed (one
real degree of freedom `c` is left, which is removed by the
overall-normalization convention; the standard choice is `c = 1`, the
canonical natural log baseline).

This is the block-local uniqueness statement. Outside scope: removing
either of the two physical-principle admissibility criteria (A) or (C)
from class (X2) — those remain selection premises on the class of
scalar generators that count as "physical scalar bosonic observables".

## Proof

Pure algebraic substitution closed by Cauchy–Erdős.

**Step 1 — `det(D + J)` is real for real `D` and real `J`.** By (X1),
`D` has real entries. The source `J` is real-symmetric, in particular
real. So `D + J` is a real matrix on `R^N`, hence
`det(D + J) ∈ R`. (The determinant of a real matrix is real because it
is a polynomial in the real entries with integer coefficients.)

**Step 2 — `det(D + J) ≠ 0` on `U_0`.** By definition of `U_0`, the
polynomial `λ ↦ det(D + λI + J)` does not vanish on `U_0`. In particular
`det(D + J) ∈ R \ {0}` is a continuous nonvanishing real-valued
function on `U_0`, so it has a well-defined locally constant sign
`σ(J) = sign(det(D + J)) ∈ {+1, -1}`. On any connected component of
`U_0`, `σ(J)` is constant.

**Step 3 — `W[J] = log |det(D + J)| - log |det D|` is well-defined and
real-analytic on `U_0`.** Because `det(D + J)` is a polynomial in the
entries of `J`, and `|det(D + J)| > 0` on `U_0`, the composite
`J ↦ log|det(D + J)|` is real-analytic on `U_0`. The normalization
constant `log|det D|` is finite because `D` is invertible (premise of
`U_0` containing `J = 0`; runner-checked).

**Step 4 — block-additivity for `W`.** If `D = D_A ⊕ D_B` and
`J = J_A ⊕ J_B` is a no-bond direct-sum decomposition, then
`det(D + J) = det(D_A + J_A) det(D_B + J_B)`, so

```text
log|det(D + J)|  =  log|det(D_A + J_A)|  +  log|det(D_B + J_B)|,
```

hence `W[J] = W_A[J_A] + W_B[J_B]` after subtracting baselines on each
block. The determinant factorization is elementary linear algebra
(block-diagonal determinants multiply); this note uses it as such and
re-verifies it in the runner without citing any external authority for
it. A sibling source note
`observable_generator_additivity_from_cluster_decomposition_theorem_note_2026-05-10`
records the same elementary fact but is not cited as a dependency here
(see "Cited dependencies" below).

**Step 5 — application of the admissibility class.** Let `F` be any
admissible generator in class (X2). By (C), `F[J]` depends on `J` only
through `|det(D + J)|`. Let

```text
r(J)  :=  |det(D + J)| / |det D|  >  0    for J ∈ U_0,                  (r)
```

so `r : U_0 → R_+` is a continuous map with `r(0) = 1`. The hypothesis
(C) says there is a function `f : R_+ → R` with `F[J] = f(r(J))`.
Continuity of `F` (criterion (R)) plus continuity of `r` (Step 3) plus
the fact that `r` is open (its image is a real-dimensional neighborhood
of `1`, verified in the runner) imply that `f` is continuous on a
neighborhood of `1`, and by patching across `U_0` connected components,
on all of `R_+`.

**Step 6 — Cauchy–Erdős multiplicative-to-additive closure.** Now apply
(A) to a no-bond direct-sum decomposition `D = D_A ⊕ D_B`,
`J = J_A ⊕ J_B`. Using `r(J) = r_A(J_A) r_B(J_B)` (Step 4 applied to
`r` rather than `W`: determinant factorizes, then absolute value
factorizes), additivity (A) gives

```text
f(r_A r_B)  =  f(r_A)  +  f(r_B)                                        (CE)
```

for all `(r_A, r_B) ∈ image(r_A) × image(r_B) ⊂ R_+ × R_+`. Because
the images of `r_A, r_B` are open neighborhoods of `1` on the
real-D block, (CE) extends by continuity (criterion (R)) to all of
`R_+ × R_+`. Cauchy's multiplicative-to-additive theorem (X3) then
forces `f(r) = c log r` for some real constant `c`.

**Step 7 — collecting.** Substituting back,

```text
F[J]  =  f(r(J))  =  c log r(J)  =  c (log|det(D + J)| - log|det D|)
       =  c · W[J]                                                       (U1)
```

on `U_0`. (Normalization (N) — `F[0] = 0` — is automatic because
`r(0) = 1` and `c log 1 = 0`.) Source derivatives (U2), (U3) follow by
differentiating (U1) on `U_0`. ∎

## Derivable corollaries

**(R1) Imaginary part of `Tr log(D + J)` has zero source derivatives.**
On the real-D block, `det(D + J) ∈ R` (Step 1). Therefore
`Im Tr log(D + J) = arg(det(D + J)) ∈ π Z` is locally constant on
each connected component of `U_0`, so its source derivatives vanish
identically:

```text
∂ Im Tr log(D + J) / ∂J_x  =  0    on each component of U_0.            (R1)
```

This is consistent with (U1): the unique CPT-even continuous additive
scalar generator on the real-D block is real-valued, and any imaginary
contribution from `Tr log(D + J)` is in the kernel of the
source-derivative map.

**(R2) `Re Tr log(D + J) = log|det(D + J)|` identically on `U_0`.**
This is a direct consequence of Step 1 — for real `det(D + J) ≠ 0`,
`log(det(D + J)) = log|det(D + J)| + i π · χ_<0(det(D + J))` with the
imaginary part purely a sign indicator. Taking real parts gives the
identity. Equivalently, the candidate generator `W = log|det(D + J)|`
is the real part of `Tr log(D + J)` on the real-D block, and the
ambiguity in the complex logarithm (winding number) drops out of all
source derivatives.

Both corollaries follow algebraically from the proof and are checked
exactly in the runner.

## What this claims

- Block-local uniqueness `(U1)`: any admissible scalar generator in
  class (X2) equals `c · W[J] = c (log|det(D + J)| - log|det D|)` on
  `U_0`, up to a single real overall-scale constant `c`.
- Source-derivative uniqueness `(U2)`, `(U3)`: the local source-response
  direction at `J = 0` is fixed; only the overall scale `c` is free,
  removed by the standard normalization convention `c = 1`.
- The two derived corollaries `(R1)`, `(R2)`.
